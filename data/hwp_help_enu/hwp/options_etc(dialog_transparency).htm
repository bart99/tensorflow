Set All Dialog Box Transparency
 

File > Options > Edit

You can set or modify the transparency of all dialog boxes and it will be synced with Set Dialog Box Transparency of each dialog box.

Transparent when a modeless dialog is not focused automatically makes the selected dialog box transparent so that it does not hide the edit window when the content of the edit window is being edited.

Steps

To set automatic transparency for all modeless dialog boxes

  1. Click File > Options.
  2. In the Options dialog box, select the Others tab.
  3. In Set Transparency, click Set Transparency of All Dialog Boxes.
  4. In the Set All Dialog Box Transparency dialog box, select Do Not Set Dialog Box Transparency in Transparency.
  5. Select the Transparent when a modeless dialog is not focused check box and click Set.
  6. In the Options dialog box, click Set.
  7. To open the modeless Find dialog box, click Edit > Find.
  8. The Find dialog box becomes transparent automatically if you select the text box of the edit window with the Find dialog box open.
  9. The Find dialog box becomes opaque again on modifying the content of the text box and pointing it to the Find dialog box.

More Information

Set All Dialog Box Transparency Dialog Box

Transparency

  • Do Not Set Dialog Box Transparency

  • Return the selected dialog box to the original transparency level.

  • 25%

  • Set the transparency of the selected dialog box as 25%.

  • 50%

  • Set the transparency of the selected dialog box as 50%.

  • 75%

  • Set the transparency of the selected dialog box as 75%.

  • User

  • Set the transparency of the selected dialog box to a user-entered value.

    Transparency value can be set from 0 through 90.

Transparent when a modeless dialog is not focused

Automatically makes the open modeless dialog box transparent so that it does not hide the edit window when the content of the edit window is being edited.

Note

What is a modeless dialog box?

A modeless dialog box is a dialog box that enables the user to move the pointer to the edit window and edit the content of the window even when dialog boxes are open. (for example: Find or Replace dialog box)
