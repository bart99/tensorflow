Set Dialog Box Transparency
 

You can use Set Dialog Box Transparency to increase the transparency of opaque dialog boxes.

When you use functions that references the body text such as Find and Replace, dialog boxes become transparent and the body text displays through the dialog boxes. Using this feature, you can see the body text even if it is overlapped by dialog boxes.

You can apply the selected transparency value of Set Dialog Box Transparency to specific dialog boxes or to all dialog boxes.

Steps

  1. Click Home > Find.
  2. In the Find dialog box, right-click the title bar.
  3. From the context menu, select Set Dialog Box Transparency.
  4. In the Set Dialog Box Transparency dialog box, select Transparency > 50%, and then click Set.
  5. Type a keyword in the Find What box of the Find dialog box, and then click Find.
  6. Words that match the keyword of Find What are shown as blocks in the edit window.

    You do not have to move the Find dialog box to see the body text from under the dialog box.

More Information

Set Dialog Box Transparency Dialog Box

Transparency

  • Do Not Set Dialog Box Transparency

  • Return the selected dialog box to the original transparency level.

  • 25%

  • Set the transparency of the selected dialog box as 25%.

  • 50%

  • Set the transparency of the selected dialog box as 50%.

  • 75%

  • Set the transparency of the selected dialog box as 75%.

  • User

  • Set the transparency of the selected dialog box to a user-entered value.

    Transparency value can be set from 0 through 90.

  • Transparent when a modeless dialog is not focused

  • Automatically makes the selected dialog box transparent so that it does not hide the edit window when the content of the edit window is being edited and a modeless dialog box such as the Find and Replace dialog boxes open.

    This option appears only when the dialog box is modeless.

All Dialog Boxes

You can choose whether to apply transparency to all dialog boxes or to return the transparency level back to the original value.

  • Apply Transparency Button

  • Apply the specified transparency value to all dialog boxes in Hwp.

  • Cancel Transparency Button

  • Cancel all transparency levels applied to every dialog box.

Note

What is a modeless dialog box?

A modeless dialog box is a dialog box that enables the user to move the pointer to the edit window and edit the contents of windows even when they are open. (for example: Find or Replace dialog box)

Set the transparency of all dialog boxes

Click Options > Others > Set the transparency of all dialog boxes to apply transparency to all dialog boxes or to change the transparency levels applied to all dialog boxes.
