Comparing Data Compatibility between Applications
 

 

If you save a document created in Hwp as a Microsoft Office Word 2007 file (*.docx) or open a document created in Microsoft Office Word 2007 (*.docx) in Hwp, you may experience compatibility issues. This is because both applications use different data processing and format designation methods.

More Information

Comparing Data Compatibility between Hwp and Word 2007

1. Save as a Word document (*.docx) in Hwp

Saving a document as a Word 2007 document (*.docx) may result in object, font, and paragraph display issues.

  • Equation
  • Proofing Symbols
  • Form Controls
  • Table > Size
  • Font > Shadow continuity, missing underline, partial strikethrough, partial outline, and partial border/background
  • Paragraph > Spread, partial line spacing, and partial border/background

 

2. Opening a Word document (*.docx) in Hwp

Opening a Word 2007 document (*.docx) in Hwp may result in object, organization chart, and SmartArt display issues.

  • Table > Merge, font, spacing, etc.
  • Chart > Legend size
  • Drawing > Partial line style, location info, etc.
  • Object > Partial arrow style, partial line weight, and partial drawing objects
  • WordArt > Partial font style and WordArt style
  • Organization Chart > Partial line position
  • SmartArt
  • OLE Object > Object video clips, object wave sounds, object bitmap images, WordPad documents, etc.

 

3. Pasting content from a Word document (*.docx) in a Hwp document

Copying a text box, picture, line, drawing, flowchart, or basic shape from a Word 2007 document and pasting it in a Hwp document may result in insertion of the object as a picture or in object formatting issues.

  • Text Box
  • Picture > Placement, etc.
  • Line > Arrow shape, etc.
  • Shape > Cube, replacement, line parentheses on both ends, multidocument, logical OR, etc.
  • Flowchart
  • WordArt
  • Organization Chart
  • SmartArt
  • Hyperlink
  • OLE Object > Object video clips, object wave sounds, object bitmap images, WordPad documents, etc.

 

4. Pasting content from a Hwp document in a Word document (*.docx)

Copying a table, picture, line, drawing object, Wordshop object, equation, or chart from a Hwp document and pasting it in a Word 2007 document may result in insertion of the object as a picture or in object formatting issues.

  • Table > Partial border, partial diagonal line, etc.
  • Picture > White space, partial line, etc.
  • Line > End style, partial shadow, etc.
  • Drawing Object (Rectangles and Circles) > Position, white space, shadow, etc.
  • Sound Object
  • Chart
  • Equation
  • Wordshop
