Open XML Document
 

File > XML Document > Open XML Document

You can open the XML file you wish to work on by clicking Open XML Document.

Steps

  1. In the document window, open an XML document (*.xml) by clicking File > Open or File > XML Document > Open XML Document.
  2. In the Open XML Document dialog box, click Open XML Document in XML Document.
  3. To open an XML document, select the file, and then click Open.
  4. A style sheet document (*.xsl) with the same path and file name as the XML Document is displayed in XML Style Sheet Document.
  5. Check the path of the XML document and the XML style sheet document, and click OK.

More Information

Open XML Document Dialog Box

XML Document

You can specify the XML document to open.

XML Style Sheet Document

Display the style sheet document (*.xsl) of the open XML document. You can apply another style sheet document, if you select Assign Another Style Sheet.

Assign Another Style Sheet

A style sheet document has the same path and file name as the document specified in the XML Document.

If you want to change the file name and path of the style sheet, select Assign Another Style Sheet.

You can specify the path and file name by typing them in XML Style Sheet Document or clicking Open XML Style Sheet.

Note

Style Sheet (*.XSL)

Style sheets (*.xsl) are used to display XML information.

Because XML instances (documents) have only logical structure, they require formatting to be recognizable.

Each element defined in DTD has structure information, but the option to display information on the screen is not available.

It is the role of XSL to figure out the method to display the content of elements. XSL contains information about the type of fonts and the colors to be used to display certain elements, and also the method to align them.

XSL is a formatting language suggested for formatting XML data and documents on the web. It provides powerful formatting, required for highly structured information.
