Save XML Document
 

File > XML Document > Save XML Document

You can save a Hwp document (*.hwp, *.hwt) as an XML file.

Steps

  1. To create as an XML document, open a document.
  2. Click File > XML Document > Save XML Document.
  3. In the Save XML Document dialog box, specify drive and file name in XML Document Saving Path.
  4. Click OK.

   If you open a Hwp document, and click File > XML Document > Save XML Document, you can convert the Hwp document to an XML document file easily.

More Information

Save XML Document Dialog Box

XML Document Saving Path

You can specify the path and file name for saving the XML file.

XML Style Sheet Document Saving Path

You can specify style sheet document (*.xsl) for the XML document to save. To specify the path and file name of a style sheet document, select Edit the style sheet file name and path.

Edit the Style Sheet File Name and Path

A style sheet document has the same path and file name as the document specified in XML Document Saving Path.

If you want to change the file name and path of the style sheet, select Edit the style sheet file name and path.

You can specify path and file name by typing them in XML Style Sheet Document Saving Path or clicking Save XML Style Sheet.

Note

Style Sheet (*.xsl)

Style sheets (*.xsl) are used to display XML information.

Because XML instances (documents) have only logical structure, they require formatting to be recognizable.

Each element defined in DTD has structure information, but the option to display information on the screen is not available.

It is the role of XSL to figure out the method to display the content of elements. XSL contains information about the type of fonts and the colors to be used to display certain elements, and also the method to align them.

XSL is a formatting language suggested for formatting XML data and documents on the web. It provides powerful formatting required for highly structured information.

Caution

Saving Document with Object as XML Document

If you save a document with inserted objects, such as table, wordshop, chart, and formula, as an XML document, the wordshop objects may not display properly.
