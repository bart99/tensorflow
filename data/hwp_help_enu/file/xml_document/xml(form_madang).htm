XML Templates
 

File > XML Document > XML Templates

You can create frequently used XML templates into template files (*.hwt) and open the files as required by using XML Templates. Template Package is a folder that contains the type of template files.

Steps

  1. Click File > XML Document > XML Templates.
  2. In the XML Templates dialog box, select Root in Template.
  3. When templates of the Root template package are listed in Template, select Resume, and click Open.
  4. When the resume template is displayed in the edit window, enter content in each click Here.
  5. Click Validate in XML toolbox to check if the content is valid.
  6. Click File > Save to save the complete document.

More Information

XML Templates Dialog Box

Template Package

Multiple template package lists are displayed in a tree structure.

Template List

If you select a template package from the Template Package list, multiple templates from the package will be displayed on a list.

Preview

If you select a template from the Template List, the template will be displayed.

Details on Icons

Icon Name Description
New Package Display the New Package dialog box to create a new package.
Modify Package Display the Modify Package dialog box to modify the name of a package.
Delete Package Display a message for confirmation of deleting a package.
Add XML Template Display the Add XML Template dialog box to add an XML template (*.hwt) to the package selected on the Template Package tree.
Modify XML Template In the Modify XML Template Information dialog box, you can modify File Name, File Path, and Template Title.
Delete XML Template A confirmation message about deleting an XML template is displayed.

Note

Partial Edit Mode

If you open an XML template document added to File > XML Document > XML Templates, the document becomes partial edit mode, and you can type in the fields only.

If repetitive field creation section is set in partial edit mode, you can repeat or delete sections.

You can run repetitive field creation section with shortcuts. The shortcut for repeating a section is CTRL+G,R. The shortcut for deleting a section is CTRL+G,D.

Open Template

If you create a document by using a document templates package, click File > New > Document Templates. You can open the selected template file from document templates as a Hwp document with the name New Document?.hwp.

When opening the original file to edit the template file in document templates, click File > Open, and select File Type > Hwp Template (*.hwt). You can open the selected template file from document templates with the original name.

Save XML Document

You can save documents created with XML templates, as a Hwp document (*.hwp) or an XML document. If you click File > XML Document > Save XML Document, you can convert a Hwp document into an XML document.
