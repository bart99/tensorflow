Create XML Templates
 

File > XML Document > Create XML Templates

You can use the Create XML Template function, which is a web data standard.

Steps

  1. Click File > XML Document > Create XML Templates.
  2. In the Create XML Template dialog box, type a title in XML Template Title.
  3. Click Open Schema Document next to Schema Document.
  4. In the Open Schema Document dialog box, select a schema file (*.xsd), and click Open.
  5. In the Create XML Template dialog box, the path of the schema file (*.xsd) is inserted in the Schema Document box.
  6. Click XML Template Path next to XML Template Saving Path.
  7. In the XML Template Saving Path dialog box, specify path and file name (*.hwt) for the XML template, and click Save.
  8. In the Create XML Template dialog box, the path of the XML template file (*.hwt) is inserted in the XML Template Saving Path box.
  9. To apply an XML template created in a Hwp document, click Path of Hwp File to Apply next to Path of Hwp File to Apply.
  10. In the Open Hwp Document dialog box, specify path and name of the Hwp document to apply, and click Open.
  11. When you complete all settings, click OK in the Create XML Template dialog box.
  12. A template file (*.hwt) will be created with the specified name in the specified path.

More Information

Create XML Template Dialog Box

XML Template Title

Type the title of a new XML template document (*.hwt) to create.

Schema Document

Specify a schema document file that defines the structure and content of XML documents. Click Open Schema Document to open an XML schema document (*.xsd).

XML Template Saving Path

Specify path for a new XML template document (*.hwt).

Path of Existing Hwp File to Apply

Specify a Hwp file on which a new XML template will be applied. When an XML template is created, it will be applied to the selected Hwp file, if you select this option.

Note

XML Schema (*.xsd) and DTD

XML Schema is a term that refers to files that define the structure and content of XML documents.

DTD (Document Type Definition) is a type of schema but it has some issues to be used. The biggest difference between DTD and XML Schema is that DTD is described in a complicated, unfamiliar language named EBNF, and XML is described in XML. Unlike DTD, XML Schema is also capable of expressing various data types and reusing elements.

The fundamental role of schema files is the same as that of DTD. Schema is regarded as an advanced form of DTD. It can express much more data types than DTD, and also solve conflicts among XML tags by using Namespace.

DTD also requires a program to interpret itself, because DTD is not XML, but schema itself is XML.
