XML Validation
 

File > XML Document > XML Validation

With this feature, you can run a validation test after opening an XML file to verify its validity.

Steps

  1. Open an XML document file to validate by clicking File > Open or File > XML Document > Open XML Document.
  2. Click Validate in the XML tab.
  3. Instructions appear in the Validation Result message box if required fields are empty in the document.
  4. Type content in correct XML grammar according to the instructions.

Note

Open XML Document

There are two ways to open an XML document file in Hwp.

  • Click File > Open. You can open an XML file by selecting XML Document (*.xml) as the File Type in the Open dialog box.
  • Click File > XML Document > Open XML Document. In the Open XML Document dialog box, click Open XML Document on the right side of the XML Document box, and select an XML file to open.
