Microsoft Word Compatible Document
 

File > New > Microsoft Word Compatible Document

You can use this feature to create a New Document as a Microsoft Word compatible document when another program is selected as default in File > Compatible Document.

Steps

To open a new Hwp document

  1. Click File > New > Microsoft Word Compatible Document.

    Or, on the Formatting toolbar, click the drop-down arrow of the New Document icon and select Microsoft Word Compatible Document.

  2. A Microsoft Word compatible document will open in a new window.
