Word Document
 

File > New > Word Document

You can start Word from Hwp, if Thinkfree Office NEO is installed over Hwp.

Steps

To open a new Word document

  1. Click File > New > Word Document.

    Or, click the New Document icon in the Formatting toolbar and select Word Document.

  2. A new Word document opens in a new window.
