Hwp Document
 

File > New > Hwp Document

You can start Show or Cell from Hwp, if Thinkfree Office NEO is installed over Hwp.

You can also use this function to create a New Document as a Hwp document, even if another program is selected as default from File > Compatible Document.

Steps

To open a new Hwp document

  1. Click File > New > Hwp Document.

    Or, click the drop-down arrow of the New Document icon in the Formatting toolbar and select Hwp Document.

  2. A Hwp document opens in a new window.
