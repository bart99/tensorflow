Cell Document
 

File > New > Cell Document

If Thinkfree Office NEO is installed over Hwp, you can start Cell from Hwp.

Steps

To open a new Cell document

  1. Click File > New > Cell Document.

    Or, on the Formatting toolbar, click the drop-down arrow of the New Document icon , and select Cell Document.

  2. Cell will begin and a Cell workbook will open in a new window.
