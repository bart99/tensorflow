Close <CTRL+F4>
 

File > Close

You can close the active document by clicking File > Close. If the active window has multiple document tabs, you can close the active document tab. If the active window has only one document tab, you can close the current window.

If the active document is not saved yet, the user gets prompted to save the file.

Note

Close Documents

If you run Close repeatedly when multiple documents are open, document tabs close first and then document windows close.

If you run Close when there is only one document window and the active document is the only document tab, the active document closes and a new document window opens just like when you click File > New.

The document closes even if you click Close Tab on the far fight of the menu bar.

If you click Close Tab when there is only one document window and the active document is the only document tab, the active document closes and a new document window opens just like when you click File > New.

Close Window <ALT+F4>

If you click Close Window on the far right of the document window title bar, the window closes.

If the active window has multiple document tabs, the tabs close altogether. If there are unsaved documents, the user gets prompted to save them.

If you click Close Window or press ALT+F4 when there is only one document window, File > Exit runs.

  • Confirm and Exit

  • If you click Close Window when Edit > Confirm and Exit is set in File > Options > Others, the following occurs.

    If you click Close Window when there is only one document window, the active document closes and a new document window opens just like when you click File > New.

    If you click Close Window when a new empty document is open, a Exit Hwp? message appears. If you click Exit here, Hwp will exit.
