Document Templates: Recent Document Templates
 

File > New > Document Templates

In the Recent Document Templates tab of the Document Templates dialog box, a list of recent document templates is displayed. You can open a document and reuse it.

If you use the Recent Document Templates list when repeatedly working on documents with the same style, you can create documents easily and quickly.

Steps

  1. Click File > New > Document Templates.
  2. The Document Templates dialog box appears with the Recent Document Templates tab selected.
  3. Select a document template from the list of recently used packages and templates and click Open.
  4. The selected template of document templates will be open in the edit window.

More Information

Recent Document Templates Tab of Document Templates Dialog Box

Preview

If you select a template (*.hwt) from the list, you can view how the document looks in Preview without opening the document.

If you right click the Preview area, the following context menus will be provided.

Menu Name Description
View Image Display the first page of the document with templates.
View Content Display only text (objects like pictures and tables excluded) that is inserted in the document with templates.
Copy All This menu will be added to the context menus if you select View Content. If you select Copy All, everything in Preview will be copied.

Details on Icons

Icon Name Description
Delete from List Select a template name and click Delete from List or press DELETE to delete the template name. The file name gets deleted from the list but the actual file does not get deleted from the disk.

Note

Managing Packages

You cannot manage document template packages when the Recent Document Templates tab is selected. You have to select the Document Templates Package tab to create or register a new package, or modify or delete packages.
