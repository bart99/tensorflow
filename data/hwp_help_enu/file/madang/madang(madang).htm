Document Templates
 

File > New > Document Templates

Document Templates enable you to create a template (*.Hwt) of frequently used document styles and open it whenever you need it. You can create a document fast by just filling in the blank area of the template. A folder that gathers templates of the same kind is called Document Templates Package.

Steps

  1. Click File > New > Document Templates.
  2. In the Document Templates dialog box, select the Document Templates Package tab.
  3. From the document templates package list, select Letter Document. All the different kinds of existing letter templates are displayed.
  4. Select Flower Print Letter 1 and click Open.
  5. Click any character above the Type Receiver guide on the letter opened on the edit screen.
  6. Type the name of the receiver without deleting characters on the screen.
  7. Point to the next line and type content of the letter.
  8. Save the file by clicking File > Save.

Note

User Info

Document Templates automatically display the content in Options > User Info. To fully use Document Templates, click File > Options and correctly type each item of the User Info tab before using document templates.

Open Template

If you create a document by using a document templates package, click File > New > Document Templates. Open the selected template of document templates as a Hwp document with the name New Document?.hwp.

When opening the original file to edit the template file in document templates, click File > Open and specify File Type - Hwp Template (*.Hwt). You can open the selected template file from document templates with the original name.
