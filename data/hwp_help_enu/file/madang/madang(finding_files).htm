Document Templates: Find Template
 

File > New > Document Templates

In the Find Template tab of the Document Templates dialog box, you can find documents you need among numerous templates (*.hwt) provided by document templates.

With the feature, you can find document files you need among various document files provided by Hwp.

Steps

  1. Click File > New > Document Templates.
  2. In the Document Templates dialog box, select the Find Template tab.
  3. Type Letter in Template to Find and press ENTER or click Find.
  4. Various letter templates are displayed in the list box.
  5. Select Flower Print Letter 1 and click Open.
  6. Flower Print Letter 1 is displayed on the screen.
  7. Type the name of the receiver without deleting characters on the screen.
  8. Point to the next line and type content of the letter.
  9. Save the file by clicking File > Save.

More Information

Find Template Tab of Document Templates Dialog Box

Template to Find

You can find templates you need among numerous document templates by typing keywords.

  • Find Button

  • If you type keywords of the template you want to search for in the Template to Find box and click Find, all templates that include the keywords will be found and displayed on a list.

Preview

If you select a template (*.hwt) from the list, you can view how the document looks in Preview without opening the document.

If you right click the Preview area, the following context menus will be provided.

Menu Name Description
View Image Display the first page of the document with templates.
View Content Display only text (objects like pictures and tables excluded) that is inserted in the document with templates.
Copy All This menu will be added to the context menus if you select View Content. If you select Copy All, everything in Preview will be copied.
