Document Templates: Document Templates Package
 

File > New > Document Templates

In the Document Templates Package tab of the Document Templates dialog box, you can register new document templates packages or modify the name and path of registered document templates packages. You can also delete unused document templates packages from the list.

More Information

Document Templates Package Tab of Document Templates Dialog Box

Preview

If you select a template (*.hwt) from the list, you can view how the document looks in Preview without opening the document.

If you right click the Preview area, the following context menus will be provided.

Menu Name Description
View Image Display the first page of the document with templates.
View Content Display only text (objects like pictures and tables excluded) that is inserted in the document with templates.
Copy All This menu will be added to the context menus if you select View Content. If you select Copy All, everything in Preview will be copied.

Details on Icons

Icon Name Shortcut Description
New Package Insert Create a new document templates package in which to store templates.
Modify Package + Change the name of registered packages or specify a new path for them.
Delete Package Delete Delete a list of document templates packages that you do not use.
Open Unregistered Template   Open a template (*.hwt) inside a folder that is not registered as a document templates package.

Note

Run from Context Menu

If you right click over the drawing objects of the Shapes tab or the Clip Arts tab in the Clip Art dialog box, context menus like New Package, Modify Package, Delete Package, Delete Object, and Rename Object appear.
