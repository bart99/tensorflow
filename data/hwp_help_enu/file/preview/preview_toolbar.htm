Preview Tab
 

File > Preview

The Preview tab provides icons for functions that are frequently used in preview mode including print, orientation, settings, view, and zoom.

More Information

Preview Tab

  • Move Page
  • Previous Page Move to the previous page.
    Next Page Move to the next page.
  • Print
  • Print Select various options in the Print dialog box to print or preview again.
    Print Options Select various print options for printing documents as needed.
    Paper Select a paper type for actual printing. This option is identical to Basic > Method > Printer Paper in the Print dialog box. Set it as Use Printer Options in most cases.
  • Watermark
  • Watermark Set picture watermark or text watermark to apply to the document only when you print.
  • Color/Grayscale
  • Color Print the preview screen and printout in color.
    Grayscale Print the preview screen and printout in grayscale.
    Light Grayscale Print the preview screen and printout in light grayscale.
  • Orientation
  • Page Setup Select type and margins of paper in the Page Setup dialog box.
    Portrait Set print orientation to portrait.
    Landscape Set page orientation to landscape.
  • Page Margins
  • Page Margins Change the page margins of the active document to the new value.
  • View
  • View Margins Display the paper margins set in the Page Setup dialog box with red dots.
    View Paper Size Display the paper size set in the Page Setup dialog box with green lines.
    Hand Tool When you increase zoom ratio and the screen becomes too big, a scroll bar appears on the right and at the bottom of the window. When this happens, click Hand Tool to move to a section invisible in the current window.
    View Page Set how to view pages among Fit it Page, Book, Many Pages.
    • Whole Page: Display one page of the document so that it fits the window on the preview screen.
    • Two Pages: Display two pages on one screen with odd pages on the right. Book means two facing pages in an open book.
    • Many Pages: Display all pages on the preview screen.
  • Zoom
  • Zoom Select a zoom ratio from the list to zoom in or zoom out the screen.
    • Many Pages: Display multiple pages on one screen.
    • Book: Display two pages on one screen with odd pages on the right.
    • Fit to Page: Display one page of a document so that it fits the window.
    • Fit to Width: Zoom in or zoom out the paper so that its width fits the preview window.
    • 100% ~ 500%: Zoom in or zoom out one page at the specified ratio. When the pointer looks like a magnifying glass, click the document to see it zoomed in/out at the specified ratio.
    Zoom Out Click the icon to zoom out the screen by 25%.
    Zoom In Click the icon to zoom in the screen by 25%.
  • Edit
  • Edit Current Page Exit Preview and return to the edit window, to the first line of the selected page which was marked with blue borders on the preview screen.
  • Close
  • Close Exit preview and return to edit mode.
