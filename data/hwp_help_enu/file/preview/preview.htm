Preview <CTRL+F2>
 

File > Preview

This function enables you to preview the document to print on the screen before sending it to the printer. Preview saves paper and also saves time because it is much faster than printing from a printer.

Steps

  1. Click File > Preview before printing the document on the screen.
  2. On the screen, you can check the printing version of the page where the pointer was located.
  3. Click Close or press ESC. You will return to the pointer on the edit screen.

Note

Content That Can Be Previewed

In Preview, you can check all content of the document to print.

This means that you can check everything including print range, print method, and zoom ratio set in File > Print in Preview.

What Can Be Done in Preview Window

The Preview window only displays how the document looks when printed. You cannot edit the document in this window.

You can only run some functions like Print and Page Setup that determine the appearance of the print-out. Besides, you can change pages by using PAGE UP, PAGE DOWN, arrow keys, scroll bar, move button, and mouse wheel.

Press CTRL+PAGE UP to move to the beginning of the document. Press CTRL+PAGE DOWN to move to the end of the document. Press CTRL+ENTER to return to the selected page (current page) in edit mode.

Press NUM + to zoom in the document by 5% from the current ratio. Press NUM - to zoom out the document by 5%.

Zoom

If you move the pointer upward in the current page in preview mode, the pointer changes to a magnifying glass.

Rest the magnifying glass on part of the document you want to see in detail and click the part. The part will be zoomed in. If you click the part again, it zooms out to the previous state.

The default zoom ratio is 100% but you can change the ratio in Zoom.

Status Bar Information

The status bar in Preview provides information about current/total page number, paper type, and screen zoom ratio.

The page number is the actual number of pages that will be printed.

When the screen zoom ratio is Zoom in, One Page, and Many Pages, the ratio is displayed as 100%, Fit to Page, and 2x2 respectively.

Many Pages

In Preview, you can check multiple pages of the document to print at the same time.

In the Preview tab, select from Fit to Page, Book, and Many Pages.

Preview Status Remembered

If you click File > Preview, preview will run with the status (View Margins, View Paper Size, Fit to Page, Book, Many Pages) remembered from the last time as long as Hwp is open.

Book

Display two pages on one screen with odd pages on the right. Book means two facing pages in an open book.
