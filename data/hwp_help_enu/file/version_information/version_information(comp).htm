Compare Versions
 

Review > Manage Document History

You can insert version in the active document when you require it or when you save the document. In the Manage Document History dialog box, select the previous version and the active document or two versions of your choice. If you click Compare Versions, you can compare the content of two different version files.

Steps

  1. Click Review > Manage Document History.
  2. In the Manage Document History dialog box, click the Save as New Version icon .
  3. In the Save As dialog box, specify location and file name, and then click Save.
  4. In the Save as New Version dialog box, type Version 1 in the Comment box, and then click OK.
  5. Check that Version 1 has been added to the Manage Document History dialog box, and click Close.
  6. Click Review > Manage Document History, after adding or modifying content in the file.
  7. In the Manage Document History dialog box, click the Save as New Version icon. In the Comment box of the Save as New Version dialog box, type Version 2, and then click OK.
  8. In the Manage Document History dialog box, select Version 1 and Version 2 while holding down CTRL or SHIFT, and then click Compare Versions.
  9. The content of the two versions will be displayed vertically in the Read-Only window for comparison.

Note

Mark with Comments

The two documents are aligned side by side and the differences between them are indicated in memos.
