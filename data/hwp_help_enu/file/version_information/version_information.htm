Manage Document History
 

Review > Manage Document History

You can insert version in the active document when you require it or when you save the document. You can compare the current document to its previous version, or select two versions to compare.

Manage Document History is useful when you send a report or when you write a large-volume document like thesis. If you use Manage Document History, you can write new content and insert it in different versions of the original document without changing the original document.

Steps

  1. Click Review > Manage Document History.
  2. In the Manage Document History dialog box, click the Save as New Version icon .
  3. In the Save As dialog box, specify location and file name, and click Save.
  4. In the Save as New Version dialog box, type content in the Comment box, and click OK.

    The User Name in File > Options > User Info gets automatically added to Author. If User Info has no User Name, nothing will be displayed in Author. So you have to type a name in the text box.

  5. After changing file content and saving the changes as different versions, click Review > Manage Document History.
  6. In the Manage Document History dialog box, select a version of your choice from the Version Info list, and click Open.
  7. The content of the version will be displayed in the Read-Only window.

More Information

Manage Document History Dialog Box

Version Info

The added Number, Date and Time, Author, and Comment of each version is displayed on a list.

Depending on the Lock Version Info settings of each version, one of the following lock icons is displayed in Number of the Version Info list.

Icon Description
(Open Lock) Lock Version Info is off.
(Closed Lock) Lock Version Info is on.
You can only view version information. The author or other users cannot modify it.

Details on Icons

Icon Name Description
Save as New Version Use this option when a version is required for the active document.
Delete Version You can delete the currently selected version or the version you select from the Version Info list.
If you use Delete Version, you can delete only one version at a time.
See Version Info Check information about the selected version on the Version Info list.
If Lock Version Info is not selected, you can modify version information.

Automatically save version information when saving document

If you select this option, every time you save a document, the Save as New Version dialog box appears and version information gets saved. You do not need to run Version Info.

Compare Versions as Split View

If you select this option, version comparison will be displayed in two windows.

Open

If you select the option to search for a version from the Version Info list, and click Open or double-click the version, information about the version will be displayed in the Read-Only window.

Compare Versions

Use this option to compare the active document and the previous version document, or just two version documents.

  • If you select one version and click Compare Versions, the active document and the selected version document will be displayed side by side in the Read-Only window. Their differences will be displayed in Comment.
  • If you select two version documents and click Compare Versions, the previous version and the next version will be displayed side by side in the Read-Only window. Their differences will be displayed in Comment.

Note

Version Info Document Size

Version Info documents are added with Hwp documents, HML documents, and version comparison results. Version comparison results include added, deleted, or modified content.

Version info documents are bigger than normal documents because they are added with HML files and version comparison results. When you save a document with version information in a removable storage device like a diskette, you have to check the available space.
