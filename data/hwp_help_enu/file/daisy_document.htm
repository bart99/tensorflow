DAISY Document
 

File > DAISY Document

DAISY (Digital Accessible Information System) is an international digital document format for visually challenged and dyslexic people. DAISY Document is a function that applies the DAISY filter to Hwp documents (*.hwp) and converts them to the DAISY format (*.xml). Files in the DAISY format (.xml) are used to create digital talking books which give visually challenged and dyslexic people access to information.

Steps

To save as DAISY

  1. Open a document to change to a DAISY document.
  2. Click File > DAISY Document > Save as DAISY.
  3. In the Save DAISY dialog box, specify the drive to save the file in and a file name in Save in.
  4. Type the name and title of the file to save in File Name and Title.
  5. Click Save. A XML file gets created in the specified location.

  If you click File > DAISY Document > Save as DAISY after opening a Hwp document, you can easily convert a regular Hwp document to a DAISY document file.

More Information

Save DAISY Dialog Box

Save in

Specify a path in which to save the DAISY document.

File Name

Type the file name of the DAISY document to save.

Title

Type the title of the DAISY document to save. The typed title will be used when explaining the document to visually challenged and dyslexic people with a voice output device.

Author

Type the title of the DAISY document to save. The content in the Author box in File > Info > Summary is typed by default. The typed author information will be used when explaining the document to visually challenged and dyslexic people with a voice output device.

Uid

UID (Unique Indentifier) is a number for identifying various DAISY documents. It is automatically created. If you want to provide a specific UID, type it in the text box.

Note

Save XML Files

If you save a file as a DAISY document, the location of the original file does not change and a new XML file is saved with a different name in the location specified in the dialog box.

Supported XML Document Format

When you are running Open/Save XML Document, the XML documents that are in HTML schema format can be open and they are saved as Hwp XML templates.

If you open or save an XML document that is not in HTML schema format, the document will be processed as binary.

DAISY Style

If you create a document with a DAISY style file (*.xsl), visually challenged and dyslexic people can easily understand the structure and location of the document.

DAISY styles can be distinguished from regular styles with the word DAISY next to them. For example, the style named Address (DAISY) in the style task pane is a DAISY style. However, normal styles have the same name as the styles supported in Hwp by default and do not have the word DAISY next to their names.

Hwp provides a total of 44 DAISY styles that include 11 text styles and 33 paragraph styles.
