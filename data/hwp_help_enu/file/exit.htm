Exit
 

File > Exit

You can close all documents and exit Hwp by clicking File > Exit. If there is any unsaved document, you will be prompted to save it.

Note

Remember User Environment

When you exit the program, the location and size of the last window is stored in the memory. If you start Hwp again, a document window is created according to the location and size of the last window.

Save Document

If there is any unsaved file when you click File > Exit, a "Do you want to save (file name)?" message appears.

  • If you click Yes, the Save As dialog box appears. Save the document and exit Hwp.
  • If you click No, Hwp will exit without saving the document.

If there are two or more unsaved files, a "You have unsaved files. Do you want to save them?" message appears.

  • If you click Yes, a "Do you want to save (file name)?" message appears for each file. Save the files or leave them unsaved and then exit Hwp.
  • If you click No, you will exit Hwp without saving the documents.
  • If you click Yes to All, the Save As dialog box appears for each document. Save the files and exit Hwp.
  • If you click Cancel, the File > Exit function will not run.

Close Window and Exit

If you click Close Window or press ALT+X when there is only one document window, File > Exit is executed.

  • Confirm and Exit

  • If you select Confirm and Exit in File > Options > Others when there is only one document window, the document closes and a new document window opens. If you click Close Window when a new empty document is open, a "Exit Hwp?" message appears. If you click Exit, Hwp will exit.

Caution

If you existed Hwp abnormally

If you exit Hwp abnormally and restart it, an automatically saved file (*.asv) in the TEMP folder of Windows will open. You can then save the file in the original format.
