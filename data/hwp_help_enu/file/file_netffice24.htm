Log in to/Log out from Netffice 24
 

File-Log in to/Log out from Netffice 24

Log in to Netffice 24 or log out from the service.

By using Netffice 24, you can save active document to the cloud storage, or open a document saved in the storage.

Steps

Sign up for Netffice 24

  1. Click File > Log in to Netffice 24 or select Netffice 24 > Log in to Netffice 24 in the Home tab.
  2. In the Connect to Netffice 24 dialog box, click Sign up for Netffice 24.
    Netffice 24 (https://www.netffice24.com/sign/up) website opens in a web browser.
  3. Review and agree to terms of use and follow next steps to subscribe the service.

Log in to Netffice 24

  1. Click File > Log in to Netffice 24 or select Netffice 24 > Log in to Netffice 24 in the Home tab.
  2. From the Connect to Netffice 24 dialog box, enter the User ID and Password and then click Set.

Open from Netffice 24

  1. Click File > Log in to Netffice 24 or select Netffice 24 > Log in to Netffice 24 in the Home tab.
  2. From the Connect to Netffice 24 dialog box, enter the User ID and Password and then click Connect.
  3. In the Open dialog box, select Netffice 24 from the Favorites list.
  4. Select the service that the file to open is saved and select the file you want from the list.
  5. Click Open.

Save to Netffice 24

  1. In the Home tab, select Netffice 24 > Save to Netffice 24.
  2. From the Connect to Netffice 24 dialog box, enter the User ID and Password and then click Connect.
  3. In the Save dialog box, select Netffice 24 from the Favorites list.
  4. Select the service to save the file to.
    Only the account that has been added to Netffice 24 is listed as an available service.
  5. Click Save.

More Information

Connect to Netffice 24 Dialog Box

User ID

Enter the Netffice 24 user ID (e-mail address).

Password

Enter the password for the Netffice 24 ID.

Sign up for Netffice 24 Button

Netffice 24 (https://www.netffice24.com/sign/up) website opens in a web browser.

Find Password Button

Netffice 24 website opens in a web browser for resetting password.

Use External Service Account

You can log in to the service with Netffice 24 account as well as with an external service account.

  • Facebook

    Log in to the service with Facebook account.

  • Google

    Log in to the service with Google account.

Note

You can check usage status, change settings such as manage account and change password from Netffice 24 website.
