Open <CTRL+O>
 

File > Open

This feature allows you to open an existing file in the document window to view, edit, or print.

Steps

  1. Click File > Open.
  2. In Find in of the Open dialog box, select a drive or folder that has a file you want.
  3. From the file list box, select a file of choice.
  4. Click Open. The selected file will open in the edit window.

More Information

Open Dialog Box

Find in

You can select a drive or folder you want and move to the folder that has the file to open. You can also type the name of the drive or folder in File Name.

File Name

Type the path or name of the file to open.

File Type

Select the type of the file to open. For information about the type of files that you can open in Hwp, see Open Different File Types.

In Active Window

Open a document in a new document tab of the active window if To New Tab in Active Window is selected under Open in the File > Options > Edit tab or In Active Window is selected in the Open dialog box.

Document tabs are useful when you work on a document separated into multiple files or edit a document while referring to other related documents.

Read-Only

Open a file as read-only.

Open as Copy

Save the selected file as a copy and then open the copy. The copied file will be indicated as (Copy) file name in the same folder as the source file.

Favorites Icon Description

If you add frequently used folders to the Favorites list on the left of the Open dialog box, you can open or select saved documents easily and quickly.

For more information about Favorites, see Favorites.

Icon Name Description
Add to Favorites In the Open dialog box, add the currently selected location to the Favorites list.
Edit Favorites Modify the name, path, or type of the location added to the Favorites list in the Open dialog box.

Delete Favorites Delete the location of Favorites added to the Favorites list of the Open dialog box.

Note

Open Icon of Toolbar

If you click the arrow of the Open icon in the General dialog box, a list of up to nine recent documents is displayed. You can open required files.

Open Documents Set as Read-Only

If you try to open a read-only document, a "This file is read-only or already in use. Read it as read-only?" message appears. If you click Yes, the document will open as a read-only, non-editable document.
