Open from Internet Folder
 

File > Open

This feature is used to open a file from the internet folder in the document window to view, edit, or print.

For you to use the internet folder, the server that provides it must support the WebDAV protocol.

Steps

To open from the internet folder

  1. Click File > Open.
  2. In the Open dialog box, select Internet Folder from the Favorites list.
  3. From the internet folder list, select the folder of choice.
  4. The Access Internet Folder dialog box appears if you are not logged in the selected folder.
  5. Type User ID and Password and click Set. The file list of the selected folder is displayed.
  6. From the file list box, select a file of choice.
  7. Click Open. The selected file will open in the document window.

More Information

Access Internet Folder Dialog Box

User ID

Type user ID to connect to the internet folder.

Password

Type user ID and password to connect to the internet folder.

Note

WebDAV

WebDAV (Web-based Distributed Authoring and Versioning) is an extended HTTP protocol that manages web server files through the web.
