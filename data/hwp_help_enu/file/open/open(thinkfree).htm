Open from Netffice 24
 

File > Open

With this feature, you can open a file from Netffice 24 in the edit window to view, edit, or print.

Steps

To open a file from Netffice 24

  1. Click File > Open.
  2. In the Open dialog box, select Netffice 24 from the Favorites list.
  3. The Connect to Netffice 24 dialog box appears if you are not signed into Netffice 24.
  4. Type User ID and Password, and click Set. The Netffice 24 folder of the user is displayed.
  5. Select a file you want in the file list box and click Open. The selected file will open in the document window.

More Information

Connect to Netffice 24 Dialog Box

User ID

Type the user ID to connect to Netffice 24.

Password

Type the password for the user ID to connect to Netffice 24.

Sign up for Netffice 24 Button

Move to the webpage where you can sign up for Netffice 24 (https://auth.netffice24.com/en/).

Find Password Button

Move to the Netffice 24 password finding webpage.

Use External Service Account

You can log in to the service with Netffice 24 account as well as with an external service account.

  • ThinkFree Online

  • Log in to the service with ThinkFree Online account.

  • Facebook

  • Log in to the service with Facebook account.

  • Google

  • Log in to the service with Google account.
