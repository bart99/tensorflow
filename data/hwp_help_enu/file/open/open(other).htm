Open Different File Types
 

File > Open

In Hwp, you can open and edit various file types over word processor documents (*.hwp;*.hwt;*.frm;*.doc;*.docx;*.odt).

File types that can be opened are Hwp Template (*.hwt), Web Documents (*.htm;*.html;*.asp;*.php), Text Documents (*.txt), Unicode Text Document (*.txt;*.2b), HWPML 1.x Document (*.hml), HWPML 2.x Document (*.hml), Rich Text Format (*.rtf), Database Documents (*.dbf), Hunminjeongeum Documents (*.gul), Word Documents (*.doc), HANA Word Documents (*.hna;*.kwp), Arirang Documents (*.hwd), Iltaerang Documents (*.jbw;*.j?w), WPS Documents (*.wps), and XML Document (*.xml).

Steps

  1. Click File > Open.
  2. In the Open dialog box, click the File Type list button and select Text Documents (*.txt).
  3. In Find in, select a folder that has the text document file of choice. Only the text documents (*.txt) in that folder will be displayed in the file list box.
  4. Select a file of choice from the list box and click Open.
  5. In the Type of Text Document dialog box, select a code in Select Character Code and click Read.
  6. The selected file opens in the document window.

Note

Specifying File Type

If you want to open a different file type by clicking Open, select a file type you want in File Type. You can also type a file type in File Name, like *.txt. If you type *.txt, a list of files with the extension .txt will be displayed.

  • All Word Processor Documents (*.hwp;*.hwt;*.frm;*.doc;*.docx;*.odt)

  • In Hwp, read documents of various word processor programs including Hwp documents. Also, read MS Word documents and ODF documents over Hwp documents and Hwp templates.

  • Thinkfree Office NEO Documents (*.hwp;*.hwt;*.hml;*.show;*.hpt;*.hsdt;*.htheme;*.cell;*.nxl;*.hcdt;*.nxt)

  • Read documents from the Thinkfree Office NEO program with the selected extension. Depending on the selected extension, Hwp (*.hwp;*.hwt;*.hml), Show (*.show;*.hpt;*.hsdt;*.htheme), or Cell (*.cell;*.nxl;*.hcdt;*.nxt) starts.

  • Hwp Template (*.hwt)

  • Read documents of the Hwp template file type. Read documents of the Hwp template file type. To use a template in document templates, click File > New > Document Templates to read it. To modify a template, click File > Open to read it.

  • Web Documents (*.htm;*.html;*.asp;*.php)

  • Read documents in the HTML format, which is the basic format of web documents. HTML stands for Hyper Text Markup Language. It is a language that creates documents with hypertext in the web browser.

      If you save a document with objects like table, wordshop, chart, and formula inserted, as a web document, and then open it, charts may not be display properly.

  • Text Documents (*.txt)

  • Text file type is a document file type that contains the most basic information made up only of regular texts, numbers, and various symbols, but not special control characters or graphic characters. When reading a text document, you have to specify character code, reading method, and font in the Type of Text Document dialog box.

  • Rich Text Documents (*.rtf)

  • RTF (Rich Text Format) is a file type that maximizes the compatibility of document information created in various applications.

  • Database Documents (*.dbf)

  • Read DBF files that are for database. When you are reading DBF files, you have to select reading method and records to read in the Read DBF File dialog box.

  • Reading Files of Other Word Processors

  • Instead of Hwp document files (*.hwp), read files that were created in other word processors including MS Word document (*.doc), Hunminjeongeum document (*.gul), HANA Word document(*.hna;*.kwp), Arirang document (*.hwd), and Iltaerang document (*.jbw;*.j?w).

Caution

Reading MS Word Files with Specific Objects

If you open an MS Word document (*.doc) with a drawing object or WordArt in Hwp, the inserted objects might display differently than in the original document.

This occurs because if you open MS Word objects in Hwp and Hwp does not have the same objects. Hwp replaces them with objects that are the closest to them.

ODF Support & .NET Framework

Thinkfree Office NEO supports an ISO-standardized Open Document Format (ODF). All data and documents are compatible regardless of program type or version.

However, Open/Save ODF is not available when Microsoft .NET Framework 2.0 or higher is not installed in the system. If you install Thinkfree Office NEO in a system without the .NET Framework program, it is installed with Thinkfree Office NEO.
