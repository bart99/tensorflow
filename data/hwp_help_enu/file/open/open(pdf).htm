Convert PDF to Office Document
 

File > Convert PDF to Office Document

This feature allows you to open, edit a PDF file and save it as the Hancom Office file format.

Steps

  1. Click File > Convert PDF to Office Document .
  2. In Find in of the Convert PDF to Office Document dialog box, select a drive or folder that has a file you want.
  3. From the file list box, select a file of choice.
  4. Click Open. The selected file will open in the edit window.

More Information

Convert PDF to Office Document Dialog Box

Find in

You can select a drive or folder you want and move to the folder that has the file to open. You can also type the name of the drive or folder in File Name.

File Name

Type the path or name of the file to open.

File Type

File type is set to *.pdf.

In Active Window

Open a document in a new document tab of the active window.

Document tabs are useful when you work on a document separated into multiple files or edit a document while referring to other related documents.

Open as Copy

Save the selected file as a copy and then open the copy. The copied file will be indicated as (Copy) file name in the same folder as the source file.

Favorites Icon Description

If you add frequently used folders to the Favorites list on the left of the Convert PDF to Office Document dialog box, you can open or select saved documents easily and quickly.

For more information about Favorites, see Favorites.

Icon Name Description
Add to Favorites In the Convert PDF to Office Document dialog box, add the currently selected location to the Favorites list.
Edit Favorites Modify the name, path, or type of the location added to the Favorites list in the Convert PDF to Office Document dialog box.

Delete Favorites Delete the location of Favorites added to the Favorites list of the Convert PDF to Office Document dialog box.

Note

Save as Hancom Office File Format after Editing a PDF File

You can save a PDF file as the Hancom Office file format after opening and editing it.
