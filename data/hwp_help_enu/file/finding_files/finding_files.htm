Document Finder
 

File > Document Finder

Document Finder is a convenient feature when you do not remember the name of an old file or in the folder in which the file is saved.

You can easily find documents that meet search criteria among documents and folders of My Computer. Document formats that can be searched are Hwp (*.hwp), Cell (*.cell), Show (*.show), Word (*.doc, *.docx), Excel (*.xls, *.xlsx), PowerPoint (*.ppt, *.pptx), and text document (*.txt).

Steps

To run from menus

  1. Click File > Document Finder.
  2. In Hancom Document Finder, type a keyword in the File or text to find text box.
  3. In Folder to Search, click Add to view the Find Folder dialog box. Then select a folder to search and click OK.
  4. When the path of the selected folder appears on the Folder to Search list, click Index.
  5. The "Index all folders added to Folder to Search?" message appears. Click Yes.
  6. When the indexing is complete, click Search.
  7. Documents and folders that meet the search criteria appear on the search result list.
  8. Double-click a file on the search result list. The file will be open.

To run Hancom Document Finder manually

  1. On the Windows taskbar, click Start > All Programs > Hancom > Thinkfree Office NEO > Hancom Document Finder.
  2. In Hancom Document Finder, type a keyword in the File or text to find text box.
  3. In Folder to Search, click Add to view the Find Folder dialog box. Then select a folder to search and click OK.
  4. When the path of the selected folder appears on the Folder to Search list, click Index.
  5. The "Index all folders added to Folder to Search?" message appears. Click Yes.
  6. When the indexing is complete, click Search.
  7. Documents and folders that meet the search criteria appear on the search result list.
  8. Double-click a file on the search result list. The file will be open.

   If the file includes the keyword typed in File or text to find, you can view the content in the Preview window by clicking the document in the Whole tab and the Document Content tab.

File or Text to Find

Type part of the name of the file to search or text in the file.

  • Search Button

Search folders and files that meet the search criteria among those in the path specified in Folder to Search, and display the result in the search result area.

Folder to Search

Select folders to search. Select multiple folders by clicking Add multiple times.

  • Add Button

  • Click Add. The Find Folder dialog box appears. If you specify a folder to search in the dialog box, the folder will be added to the Folder to Search list.

  • Delete Button

  • Delete the folder selected from the Folder to Search list.

  • Index Button

  • Index all folders added to the Folder to Search list.

Search Result Area

The search result area of the Hancom Document Finder is made up of several tabs like Whole, Document Content, File Name, and Folder Name.

The search result list displays Name, Modified on, and Path. If you right click the title bar, you can select information to display including Name, Size, Format, Modified on, Author, and Path.

Tab Name Description
All All search results appear on a list.
Document Content Search files that include the keywords and display the result.
File Name Search documents whose file names include keywords and display the result.
Folder Name Search documents whose folder names include keywords and display the result.
  • Preview

  • The Preview window is supported only for documents that include keywords.

    If you click a document from the list of the Whole tab and the Document Content tab, the document content is displayed in the Preview window.
