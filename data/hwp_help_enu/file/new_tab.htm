New Tab
 

File > New > New Tab

You can create a new document by using New Tab such that the current document tab as is and a new document tab in the same window is created.

Steps

  1. To create a new document window, click File > New > New Tab.
  2. A new document tab is displayed and the tab label at the bottom of the document is displayed as Noname 2.

Note

Creating a New Tab

When you create a new document tab, right click the document tab label and select New Tab. Or, click New Tab to insert a New Tab.

Closing a Document Tab

If you click Close on the far right of the menu, a document tab closes.

If you click Close Document Tab when there is only one document tab in the active window, the document closes and a new document tab is displayed just like when you click File > New.

Switching Document Tabs

When there are multiple document tabs in a window and you want to move to another document tab, click the tab label at the bottom of the document.

If you press CTRL+TAB on the keyboard, you can move to the next document tab. If you press CTRL+SHIFT+TAB, you can move to the previous document tab.

Moving Document Tabs

If you create multiple document tabs in a window, new document tab labels will be added to the right of the active document tab label in the order of tab creation.

To switch the location of two tabs, click the label of the tab to move and drag it to the other one.

Displaying Document Save Status

The color of the file name on the document tab label indicates whether the document on the edit screen is saved or not.

If the document has been edited, its tab name is red.

If the document has been automatically saved, its tab name is blue.

When the document is completely saved, its tab name is black.

Document Tab Number Limit

In one document window, you can create up to 30 document tabs. If you try to create more than 30 document tabs in a document window, a "You can create up to 30 document tabs. Create a new tab after opening a new window or closing one or more document tabs." message appears.
