Document Info: Statistics
 

File > Info > Statistics

The Statistics tab of the Info dialog box displays the date when the active document file was created, the date when it was last modified and saved, and the person who last saved it. The number of characters, Chinese characters, words, lines, paragraphs, pages, 200-character manuscript paper pages, tables, and pictures is also displayed.

More Information

Statistics Tab of Info Dialog Box

Date and User Name

The dates and user name displayed in the Statistics tab are recorded automatically by the program. Even if you change Date or Author in the Summary tab, they do not change in Statistics. The automatically saved name in Last Saved by is brought from File > Options > User Info. If User Info has no content, user information is taken from the system.

  • Created on

  • Click File > New to display the creation date of the document file.

  • Last Modified on

  • Display the date when File > Save was last run.

  • Last Saved by

  • Display the name of the last user who clicked File > Save.

Document Volume

To calculate Document Volume, Hwp even counts the content of tables and each cell of a table is considered as a Paragraph. For example, a table of 5 rows and 5 columns has 25 paragraphs.

  • Character (including space)

  • Display the number of characters in the active document. A space or tab is calculated as one character.
    One Korean syllable is calculated as one character and one English letter or number is calculated as one character as well.

  • Character (excluding space)

  • Display the number of characters in the active document. A space or tab is not included in the number of characters.

  • Chinese Characters

  • Display how many Chinese characters are included in the total number of characters.

  • Words

  • Display the number of clauses in the active document. A group of words (clauses) between spaces is calculated as one word.

  • Line

  • Display the number of lines in the active document.

  • Paragraph

  • Display the number of paragraphs in the active document. The number of times ENTER was pressed in edit mode is calculated as the number of paragraphs. The number of line breaks made by pressing SHIFT+ENTER is calculated as the number of lines.

  • Page

  • Display the page number of the active document.

  • Manuscript Paper

  • The page number of manuscript paper is calculated with the assumption that the active document is written on 200-character manuscript paper. Include headers/footers and footnotes/endnotes when calculating the page number of manuscript paper.

  • Table, Picture, Text Box

  • Display the number of tables, pictures, and text boxes in the active document.

Advanced Usage

Calculating Page Number of Manuscript Paper

Calculate the page number of manuscript paper based on 200-character manuscript paper. Assume that tables and pictures that cannot be moved to manuscript paper are separately added to each page that follows the end of the body text. Therefore, the number of characters in Table or Text Box and the number of Pictures in the document are excluded from the page calculation of the manuscript paper. If the active document is written on manuscript paper, its final page number is manuscript paper page number plus the number of tables, pictures, and text boxes.
