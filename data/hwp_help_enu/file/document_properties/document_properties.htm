Info
 

File > Info

This feature allows you to check or record additional information about the active document file.

The Info dialog box consists of the Normal, Summary, Font Detail, and Picture Info tabs. If you click File > Info, the Summary tab will be displayed by default.

The Normal tab displays file name, size, date, and properties.

Note

The information in Info does not display anything about the edit status of the body text. Each document file has document information. You can always view and modify information about the active document file regardless of the pointer position.
