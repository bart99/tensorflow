Document Info: Normal
 

File > Info > Normal

The Normal tab of the Info dialog box displays the save information and version information of the active document file. You can check the file type, and save location, version, and properties of the document.

More Information

Normal Tab of Info Dialog Box

Type

Display the file type of the document.

Location

Display the location of the folder where the document is saved.

Size

Display file size.

Document Version

Display the version of the file format of the document when it was last saved.

Program Version

Display the version of Hwp when the document was last saved.

MS-DOS Name

Display the file name in MS-DOS format. An MS-DOS file name is comprised of an up-to-8-character name and a 3-character extension.

Created on

Display the date when the file was created in the current location.

Modified on

Display the date when file information was last changed.

Accessed on

Display the date when the file was last opened.

Properties

Display the settings of file properties. You can adjust file properties in the Open list box or Windows Explorer. Select a file and right click it to select desired properties from Properties or Adjust.

Link Document

Link Document is used to select a linked file to print or to link reference to other files in cross-reference. When you link documents by using Link Document, Continuous Page Number and Continuous Footnote Number are available for use.
