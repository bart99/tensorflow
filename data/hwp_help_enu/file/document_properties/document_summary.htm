Document Info: Summary
 

File > Info > Summary

In the Summary tab of the Info dialog box, you can record the title, subject, author, keywords, writer, typer, editor, summary, and notes of the active document.

The content you type in the Summary tab does not affect the editing of the body text, page alignment, or printing. If you record titles or subjects appropriate for the content of the active document, they will be very useful for managing the document.

More Information

Summary Tab of Info Dialog Box

Title

Type the document title.

If you do not type a title, the first paragraph of the document with characters typed will be automatically inserted as the title. Characters from the first character to the first punctuation mark such as period (.), question mark, and exclamation mark will be automatically inserted. If there is an empty line at the beginning of the document, the empty line will be ignored.

Subject

Type a subject appropriate for the document.

Author

Type the name of the author.

  • Author Button

  • If you click this button, author information gets replaced with Company Name, Division, Job Title, and User Name which is registered in File > Options > User Info.

Date

When you create a new document file by clicking File > New, the date and time of creation get automatically inserted. You can modify them as needed.

  • Today Button

  • If you click this button, the current date and time of the system will be updated.

Keyword

Type words that are important in the document content.

Others

For using this document multiple times, you can type various pieces of information such as memos and comments about document content or editing.

Note

Document Saving Status

If you enter content in Summary, the document is considered to have changed, just like when you edit the body text. So the file name on the document tab label turns red. However, only when you change the content in the Summary tab after clicking File > Info, the change gets recorded in the document.

Save As

Calculate the page number of manuscript paper based on 200-character manuscript paper. Assume that tables and pictures that cannot be moved to manuscript paper are separately added to each page that follows the end of the body text. Therefore, the number of characters in Table or Text Box and the number of Pictures in the document are excluded from the page calculation of the manuscript paper. If the active document is written on manuscript paper, its final page number is manuscript paper page number plus the number of tables, pictures, and text boxes.
