Document Info: Picture Info
 

File > Info > Picture Info

In the Picture Info tab of the Info dialog box, you can view information about all picture files in the document and easily change the link of the picture files.

After checking information about all picture files in the active document, you can select whether to embed the picture files in the document file or simply link them to an external picture file. If you insert a picture in a document, the picture file is saved in the document file and hence it is not necessary to save the picture file separately.

More Information

Picture Info Tab of Info Dialog Box

Picture List

The picture list displays the name, type, page number, and path of all pictures in the active document.

  • Name

  • Pictures in the active document appear according to their numbers.

  • Type

  • Display whether the picture files in the active document are embedded or linked.

  • Pages

  • Display pages of the active document where picture files are inserted.

  • Path

  • If the Type of a picture file is Link on the Picture List, a path will be displayed highlighting how the picture file is linked in the local system.

Details on Icons

Icon Name Description
Insert Picture Select one linked picture file, or multiple linked pictures by pressing CTRL or SHIFT to change the pictures to embedded pictures at the same time. This function is not available if you select only an embedded picture file.
Save Inserted Picture Change the embedded picture files in the body text to linked picture files.
  • If you select a picture file in which the Type is embedded on the Picture List and click this button, the Save Inserted Picture dialog box appears. Select a folder in which to save the embedded picture and type a file name to save the picture.
  • The picture file follows the format of the active picture file.
Save All Inserted Pictures Change all picture files in the body text to linked picture files.
  • If you select all picture files on the Picture List and click this button, the Save All Inserted Pictures dialog box appears. Select a folder in which to save the embedded picture and type a file name to save the picture.
  • A serial number will be added to the file name specified by the user. (Example: image00001, image00002, image00003).
  • The picture file follows the format of the active picture file.
Embed All Change all picture files linked to the active document to embedded pictures altogether.
Change Picture Change the picture selected from the list to another picture file.
Change Path Select a different path for the picture file linked to the active document.
  • If you select a picture file whose Type is linked on the Picture List, and click this button, the Change Picture Path dialog box appears. You can select a different path for the active picture file here.
  • This function is useful when the picture file of the active document is moved to another path.
Change File Extension Select another file extension for the picture file linked to the active document. This function is useful when you need to change a picture file to one with the same file name but a different extension. However, the format of the picture file will not change.
  • For example, after inserting 200 to 300 pictures as BMP, you can change them to picture files with the same names but the JPG extension altogether.
  • If you select a linked picture file and click this button, the Change File Extension dialog box appears. Select a file extension.
Copy Picture Path Copy the path of the selected, linked picture and save it in a temporary storage. If you place the pointer on the edit screen or other programs and then press CTRL+V, you can paste the path of the copied picture at the pointer.
Save Picture List Save a list of the selected pictures. When you are saving, you can use Comma Delimited (*.csv), Tab Delimited (*.txt), and Space Delimited (*.txt) as file format. This function is useful when creating a TOC for pictures.

Note

Pictures whose information is available

If you click Insert > Illustrations > Picture in Info > Picture Info, you can view the information of only the picture files inserted in the body text. This implies that information is not available for the picture files used as the cell background or page background or those used for filling the face of an object.
