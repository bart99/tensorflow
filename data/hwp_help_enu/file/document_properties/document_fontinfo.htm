Document Info: Font Detail
 

File > Info > Font Detail

In the Font Detail tab of the Info dialog box, you can check a list of used fonts and a list of substituted fonts in the active document and replace such fonts with other fonts.

Steps

To check font details

  1. Click File > Info.
  2. In the Info dialog box, select the Font Detail tab.
  3. Display a list of used fonts and a list of substituted fonts in the active document.
  4. Click OK.

To change to other fonts

  1. Click File > Info.
  2. In the Info dialog box, select the Font Detail tab.
  3. From the list of used fonts and a list of substituted fonts, select a font to replace from the Used Fonts list and click the Change Used Fonts icon .
  4. All available fonts installed in your system appear on the Font to Apply list of the Change Used Fonts dialog box.
  5. Select a font from the Font to Apply list and click Set.
  6. Once you return to the Info dialog box, click OK.
  7. Among all fonts used in the document, the selected font will be replaced with the font to apply.

More Information

Font Detail Tab of Info Dialog Box

Language

When representation fonts are not in use, font details are not displayed on the font list of Language > Rep.. Therefore, it is good to check Used Fonts and Substituted Fonts for each language.

Used Fonts

Display a list of only the fonts that display well in your system, among all fonts used in the active document.

Icon Name Description
Change Used Fonts Replace used fonts with other fonts.
  1. Select a list of fonts to replace with other fonts from the Used Fonts list and click Change Used Fonts.
  2. In the Change Used Fonts dialog box, all installed fonts are displayed on the Font to Apply list.
  3. Select a font you want and click Set.

Substituted Fonts

Display a list of fonts that were used in the active document but are replaced with other fonts because they do not exist in your system.

Icon Name Description
Change Substituted Fonts Replace substituted fonts with other fonts.
  1. Select a list of fonts to replace with other fonts from the Substituted Fonts list and click Change to Another Font.
  2. In the Change Used Fonts dialog box, all installed fonts are displayed on the Font to Apply list.
  3. Select a font you want and click Set.

Note

Checking Unavailable Fonts

There are documents that are created on other computers and they use fonts that are not installed in your system. You can check for such types of fonts on the Substituted Fonts list of the Font Detail tab and then install the fonts or replace them with other fonts.
