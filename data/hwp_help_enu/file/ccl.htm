CCL
 

Insert > CCL

As the increased amount of UCC brings more attention to the copyright issue, an international copyright agreement, CCL (Creative Common License), is considered as the best alternative that enables content to be freely shared without copyright violation in this digital age.

CCL is provided to solve the copyright issue by inserting the CCL mark in documents.

Steps

  1. Create a document and click Insert > CCL.
  2. In the Insert Creative Common License (CCL) dialog box, set Commercial Use of the Document as Allow.
  3. Set Document Modification and Secondary Copyright as Allow.
  4. Select Display Both Text and Graphics for CCL Display and click Insert.
  5. The CCL mark is inserted in the active document according to the settings.

More Information

Insert Creative Common License (CCL) Dialog Box

Commercial Use of the Document

  • Allow

  • Allow commercial use of documents.

  • Do Not Allow

  • Never allow commercial use of documents.

Document Modification and Secondary Copyright

  • Allow

  • Allow document modification and secondary copyright.

  • Allow only if the copyright holder allows using it under the same terms.

  • Allow document modification or secondary copyright only under the terms suggested by the copyright holder.

  • Do Not Allow

  • Never allow document modification or secondary copyright.

CCL Display

  • Display Both Text and Graphics

  • Display both the graphics and text of the CCL mark.

  • Display Graphics Only

  • Display the graphics of the CCL mark.

  • Display Text Only

  • Display the text of the CCL mark.

  • Graphic Display

  • If Display Both Text and Graphics or Display Graphics Only is selected in CCL Display, you can select Large Pictures, Small Pictures, or Common Pictures for graphic display.
