New <CTRL+N>
 

File > New

This feature is used for creating a new document. You can leave the active document window as is and create a new document window for a new document.

Steps

  1. To create a new document window, click File > New.
  2. A new document window opens and a file named Noname 2 is displayed on the title bar and the document tab label.

Note

New Document Style

If you click the arrow on the Open icon in the Formatting toolbar, a list of maximum nine recent documents is displayed, you can then open the required files.

If you need a list of more recent documents, go to File > Options and increase the number in Edit > Open Recent Documents > Show Recent Documents in Menu.

Author and Date

The name of the author and date of creation of the new document gets recorded automatically in the file.

You can bring the User Name, Job Title, Division, and Company Name from Options > User Info and record them in Document Info > Summary > Author. You can record the date and time of running New in Info > Summary > Date.

Document Window Number Limit

If the system memory allows, you can open maximum 30 new document windows in Hwp. However, opening more document windows than you require complicates document management and lowers system efficiency.

Page Setup of New Document

You can set page style in File > Options > New Document, when you are creating a new document by using New Document or New Tab.

Advanced Usage

New and New Tab for a New Document

You can use New Document and New Tab to create a new document.

The difference is, New creates a new document in a new window, and New Tab creates a new document in an additional tab in the active document window.

Master Document

Hwp provides the master document (Normal.hwt) that contains the default Page Setup, Styles, Multilevel List for a new document created by New Document or New Tab. If you need to change the settings of a new document, click File > Open to open the file \HNC\Shared80\HwpTemplate\Doc\Enu\NORMAL.HWT, then modify and save Page Setup, Styles, and Multilevel List.
