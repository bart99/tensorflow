Save as PDF
 

File > Save/Send To > Save as PDF

This feature allows you to save the active document as a PDF file without printing it from a printer.

Steps

To save as PDF file

  1. Open a file to save as a PDF file.
  2. Click File > Save/Send To > Save as PDF, the Save as PDF dialog box appears.
  3. Specify the location in which to save the PDF file and its file name, and click Save.

  While the file is converted to a PDF file, the status bar displays the progress of conversion.

Note

Convert PDF

The file is saved in the PDF/A-1b format among PDF properties. The HFT font can be converted to text without the Hancom PDF Converter program.

Caution

License Related to Including PDF Document Fonts

You can only include fonts that are allowed by font providers. If you include fonts, the fonts do not get replaced when you read or print the file. Text will be displayed with the original fonts.

You can include all the fonts or only the subsets of the characters used in the file. If you use subsets, custom font names will be created and fonts and font metric will be used when you print.
