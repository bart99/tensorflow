Save As <F12>
 

File > Save As

This feature allows you to save the active file in the disk with a new name and continue editing the file under the new name.

If you click Save As, you can save a file with a different name or as a different file type.

Steps

  1. Click File > Save As for the active document.
  2. In Save in of the Save As dialog box, select a drive or folder to save the file in.
  3. In the file list box, select a folder to save the file in.
  4. Type a new name in the File Name text box.
  5. Click Save. The new name will be displayed on the document tab label and the title bar.

More Information

Save As Dialog Box

Save in

Select a drive or folder to move to a location of choice. Or, type the name of a drive or folder in File Name.

File Name

Type the path or name of the file to save.

File Type

Select the file type to save.

Document Password Button

Show the Change/Remove dialog box to set a password and security level so that no one can open the file without the password.

You can set a password only when saving a file with Hwp document (*.hwp) or Hwp Templates (*.hwt) as the File Type. You cannot set a password when saving a file in other types.

Storage Settings Button

You can set Automatic Storage Settings including Always Auto Save and Auto Save When Idle, as well as Save Options including Save a Copy as, Include Preview Image, and Compress and Save.

Besides, you can set Create Backup File, Set document password when saving new documents, Search for private information when saving documents with privacy protection when you run Save in New Document.

Favorites Icon Description

If you add frequently used folders to the Favorites list on the left of the Save As dialog box, you can select documents to open or save easily and fast.

For more information about Favorites, refer to the Favorites part.

Icon Name Description
Add to Favorites In the Save As dialog box, add the selected location to the Favorites list.
Edit Favorites Modify the location, name, path, or type added to the Favorites list of the Save As dialog box.
Delete Favorites Delete the location of Favorites added to the Favorites list of the Save As dialog box.

Note

Save as Different File Type

To open a Hwp document in other programs, you can save the file in types other than Hwp document (*.hwp).

Select a file type you need from the File Type list and save.

  • Hwp Template (*.hwt)

  • This is the template file type of Hwp. Use it when using the active document as a template of document templates.

  • HWPML 2.0 Document (*.hml)

  • HWPML (Hangul Word Processor Markup Language) is a formatting language in the same format as HTML, which displays Hwp files in markup tag structure.

    If you save a Hwp file in HWPML file type, you can open and edit the Hwp file in other applications.

  • Web Documents (*.htm)

  • The default type of web documents is HTML. HTML stands for Hyper Text Markup Language. It is a language that creates documents with hypertext in the web browser.

      If objects like table, Wordshop, chart, and equation in a document saved as a web document do not display properly, select In Line with Text in Object Properties > Position and save the document again.

  • ODF Text Document (*.odt)

  • Save the file as an ISO standard ODF (Open Document Format) text document.

  • Text Document (*.txt)

  • Text file type is a document file type that contains the most basic information made only of regular texts, numbers, and various symbols, but not special control characters or graphic characters.

  • Double-byte Document (*.2b)

  • Save Hwp content as an ASCII text file made up of double-byte (16 bits) codes. When you Open such file in Hwp, you have to select Unicode Document (*.txt; *.2b).

  • Rich Text Document (*.rtf)

  • RTF (Rich Text Format) is a file type that maximizes the compatibility of document information created in various applications.

  • MS Word Document (*.doc)

  • Save the file as a Microsoft Word file.

  • Web Presentation Document (*.htm)

  • If you save a Hwp document as a web presentation document (*.htm), you can open the document on the web for a presentation or use it as online instruction material.

  • XML Document (*.xml)

  • Save the file as a Hwp XML document.

Caution

ODF Support & .NET Framework

Thinkfree Office NEO supports an ISO standard and ODF (Open Document Format). All data and documents are compatible regardless of program type or version.

However, Open/Save ODF is not available when Microsoft .NET Framework 2.0 or higher is not installed in the system. If you install Thinkfree Office NEO in a system without the .NET Framework program, it is installed with Thinkfree Office NEO.
