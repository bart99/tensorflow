Storage Settings
 

File > Save As

When saving a file with a different name, you can select various options.

Steps

  1. Click File > Save As.
  2. In the Save As dialog box, click Storage Settings.
  3. In the Storage Settings dialog box, select Save in New Document > Search for private information when saving documents with privacy protection and click Set.
  4. Open a document with Protect Privacy on and add Phone.
  5. Save the document by clicking Save As. A guide for searching the added phone number appears.

More Information

Automatic Dialog Box

Auto Save

  • Always Auto Save

  • Run timed backup every specific minutes after the file is last saved or automatically saved. The time can be set from 1 through 60 minutes.

    This option is good for Copy. When you type a manuscript incessantly, the volume of typed content grows over time and gets automatically saved.

  • Auto Save When Idle

  • Save the file automatically if there is no typing for the specified number of seconds after the last typing. The time can be from 1 through 360 seconds.

    This option is good for Writing. The reason is that the file gets saved while the user stops typing and thinks about his or her writing.

  You can use both or either one of the two Auto Save options. When editing or correcting a document, it is ideal to set Auto Save for after 60 seconds or every 30 minutes.

Save Options

  • Save a Copy as
  • Always save a Hwp document file as one of the following documents: HWPML document, web document (HTML), text document, rich text document (RTF), MS Word document, web presentation document, and XML document.
    • File type that can be saved as a copy

    • The documents that can be saved as a copy of a Hwp document file (*.HWP) are HWPML 2.1 document (*.HML), web document text document (*.HTM), text document (*.TXT), Unicode document (*.TXT), rich text document (*.RTF), MS Word document 97(*.DOC), web presentation document (*.HTM), and XML document (*.XML).

    • Code Format

    • Select a file type for the document to save as a copy and then select a code format.

    • Path

    • If you need to save the original file and its copy in different folders, type a path and a folder name in the Save a Copy as > Path edit box. You can also select a path and a folder you want by clicking the Select Path icon.

  • Include Preview Image

  • Save the image of the Preview window of the Open dialog box in a document. If you select this option, the speed of preview increases. The size of a preview image is 24KB.

  • Compress and Save

  • Save Hwp files in reduced size. When working on an important document, it is safe to remove Compress and Save.

Save in New Document

  • Create Backup File

  • If you select Create Backup File in the Edit tab of the Options dialog box, when you save a Hwp document, the existing file (*.HWP) with the same name changes to a BAK file and the new content is saved as a HWP file. Even if you accidentally save the wrong content, you can recover the original Hwp document file by changing the extension of the BAK file to HWP.

  • Set document password when saving new documents

  • If you often work on confidential documents that require security, set for the Set Password dialog box to appear automatically without you having to click File > Set Password every time you save a new document.

  • Search for private information when saving documents with privacy protection

  • Search for private information when you save a document with privacy protection as a file with a different name.

Note

Only in New Document

Create Backup File, Set Password, and Search Privacy Information cannot be used when you save existing documents. They can be used only when you save a new document.
