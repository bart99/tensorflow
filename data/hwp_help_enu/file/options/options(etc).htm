Options: Others
 

File > Options > Others

In the Others tab of File > Options, specify requirement for a confirmation before exiting Hwp, when you close the last document window. You can also specify settings to run a spell check, select a fax driver for fax printing, and a font for hyperlinks.

More Information

Others Tab of Options Dialog Box

Search
  • Delete Search Records

  • Show the Delete Search Records dialog box to delete history of functions.

Display
  • Show measurements in units

  • You can specify a unit for the values such as length, weight. If it is specified, all values are displayed based on the specified unit in all dialog boxes. However, you can change the unit in each dialog box aside from the unit specified in advance. Available units are point (pt), millimeter (mm), centimeter (cm), inch (¢©), pica (pi), and pixel (px).

Malware Block
  • Block when Opening Document in Active Program

  • Detect malware infection on your document and prevent users from opening the infected documents.

    Deselecting this option stops detecting malware when you open a document. However, when you restart the program with Malware Block when Opening Document in the Security tab of Hancom Configuration Utility selected, this option will be automatically selected and malware block will be enabled again.

Others
  • PDF Driver

  • You can select the required PDF driver, when you save a Hwp document as a PDF document, instead of printing it from a printer.

    PDF (Portable Document Format) is a file format that saves fonts in the program and enables you to preview the document before printing. You can view or print Korean documents by using PDF, even in operating systems that do not have Korean fonts like English Windows, Japanese Windows, and Macintosh.
