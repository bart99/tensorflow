Options: General
 

File > Options

If you type user information such as your name, company name, job title, and department name in the General tab of File > Options, you can use them in Document Templates and Summary.

More Information

General Tab of Options Dialog Box

User Information

  • User Name

  • Type your name to automatically display in Insert > Field > User Info, and File >Info > Summary.

  • Company Name

  • Type the company name to automatically display in Insert > Field > User Info and File >Info > Summary.

  • Job Title

  • Type the job title to automatically display in Insert > Field > User Info and File >Info > Summary.

  • Division

  • Type the department name to automatically display in Insert > Field > User Info and File >Info > Summary.

  • Phone (Work)

  • Type the work phone number to automatically display in Insert > Field > User Info.

  • Fax Number

  • Type the fax number to automatically display in Insert > Field > User Info.

  • Phone (Home)

  • Type the home phone number to automatically display in Insert > Field > User Info.

  • Phone (Mobile)

  • Type the mobile phone number to automatically display in Insert > Field > User Info.

  • Home Page

  • Type the home page address to automatically display in Insert > Field > User Info.

  • E-mail 1

  • Type E-mail address 1 to automatically display in Insert > Field > User Info.

  • E-mail 2

  • Type E-mail address 2 to automatically display in Insert > Field > User Info.

  • Others

  • Type information besides basic user information.

Word

Set word wrapping characters that cannot be located at the beginning or end of a line.

  • Default

  • Set default characters to be used at the beginning and end of a line.

  • Customize

  • You can select specific word wrapping characters in Cannot Start Line and Cannot End Line.

    • Cannot Start Line

    • Select characters that cannot be located at the beginning of a row.

    • Cannot End Line

    • Select characters that cannot be located at the end of a row.

  • Apply Settings to New Document

  • Save the currently selected word wrapping characters in a master document (normal.hwt). The characters will be applied to all new documents.

Note

Document Templates

The templates in File > New > Document Templates automatically display the content typed in Options > User Info. To use Document Templates, type in each item of the User Info tab in File > Options before using Document Templates.

Summary

The Company Name, Division, Job Title, and User Name recorded in File > Options > User Info get automatically recorded in Author of File > Info > Summary.

Caution

Word Wrapping Character Restriction

If you add word wrapping characters to Customize in Word Wrapping Character, and open a document with the characters applied in a Hwp version older than 2014, the settings might not apply.
