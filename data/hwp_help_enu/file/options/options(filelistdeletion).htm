Delete Search Records
 

File > Options > Others

After using the multiple search functions provided by Hwp, you can select some functions and delete their recent lists and search history.

If you click Delete Search Records, search history gets deleted immediately. Some history might get deleted when you exit the program.

Steps

To delete search history of the Clip Art dialog box

  1. Click Insert > Illustrations > Clip Art.

    Or, click Clip Art in the Illustrations group of the Insert tab.

  2. In the Clip Art dialog box, type Flower in the File to Find box and click Find.
  3. When the search result is displayed in the Search Results tab, search more Clip Art by typing keywords like Pattern and Rice Cake in the File to Find box.
  4. Click the drop-down arrow of the File to Find box, and check a recent search list. Then click Cancel to close the dialog box.
  5. Click File > Options.

    Click Options in the Settings group of the Tools tab.

  6. In the Options dialog box, click the Others tab and click Delete Search Records in Search.
  7. In the Delete Search History dialog box, select the Insert > Illustrations > Clip Art > File to Find check box from the Search History List and click Delete.
  8. "Some of the history list on the Search History List will be deleted when you exit the program. Delete the selected Search History?" is displayed. Click Delete.
  9. Select Always delete search history of the selected list when exiting the program check box and click Set.
  10. To check whether the search history has been deleted or not, you can open the Clip Art dialog box again and click the drop-down arrow in the File to Find box.

More Information

Delete Search History Dialog Box

Search History List

To delete search history of a function, select the check box of the function and click Delete.

  • Select All Button

  • Select or clear all the items of the Search History List.

Options

  • Always delete search history of the selected list when exiting the program

  • To delete the search history of selected items every time you exit the program, you can set the search history from the Search History List.

Set Button

Save changes and close the dialog box.

Cancel Button

Close the dialog box without saving changes.

Delete Button

Delete the items selected from the Search History List.

If you click Delete, the message "Some of the history list on the Search History List will be deleted when you exit the program. Delete the selected Search History?" is displayed. If you click Yes, the search history will be deleted immediately. Some history might get deleted when you exit the program.
