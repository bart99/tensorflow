Options: New Document
 

File > Options

In the New Document tab of File > Options, specify the type of paper to be used when creating a new document by using New or New Tab. Select paper size, landscape or portrait, and top, bottom, left and right margins.

Note

Difference with File > Page Setup

The setting of File > Page Setup affects only the active document. The settings of Options > New Document apply to all new documents. They also apply when you exit Hwp and restart it.

Master Document

When you create a new document by using New or New Tab, Hwp creates a master document (NORMAL.HWT) which contains the default Page Setup, Styles, and Multilevel List Style. If you change the settings in Options > New Document, the page setup of the file, \Program Files\HNC\Shared80\HwpTemplate\Doc\Enu\Normal.hwt, changes.

No Affect on the Active Document

If you change settings in Options > New Documents, the active document does not get affected. New documents that you will create in the future by clicking File > New or File > New > New Tab will be affected.
