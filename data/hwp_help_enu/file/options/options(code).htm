Options: Code Format
 

File > Options

If you edit a Japanese or Chinese document in Hwp, or send a document written in Japanese or Chinese to another country, you can convert the document to the file type that supports the language code. If you set the code format in Code Format tab of File > Options, you can maximize compatibility between Hwp and the system or files, by converting the document to the standard code of each language.

More Information

Code Format Tab of Options Dialog Box

Read Text

You can select a method for converting codes when reading a text document.

To select a code format, select Check and Confirm. To read or print a document in special format, select Set.

  • Specify

  • Select a format from various code formats.

  • Check and Confirm

  • The code format of text is checked automatically, and you receive a notification for confirmation. If the format is incorrect, select another format.

  • By Paragraphs

  • Connect lines into paragraphs by connect line brakes at the end of lines.

Save Text

Select a method for converting codes when saving text files with different names.

  • Specify

  • Select a format from various code formats.

  • Check and Confirm

  • The code format of text is checked automatically, and you receive a notification for confirmation. If the format is incorrect, select another format.

  • Keep Alignment

  • Save a document keeping its alignment unchanged by using Home > Paragraph > Alignment.

Read HTML

You can select a method for converting codes when reading an HTML document.

  • Specify

  • Select a code format of your choice.

  • Check and Confirm

  • The code format of text is checked automatically, and you receive a notification for confirmation. If the format is incorrect, select another format.

  • Auto Detect

  • Detect the code format of text automatically. You can also read text fast but with low accuracy.

Save HTML

You can select a method for converting codes when saving HTML files with different names.

  • Specify

  • Select a code format of your choice.

  • Check and Confirm

  • The code format of text is checked automatically, and you receive a notification for confirmation. If the format is incorrect, select another format.

Paste From Clipboard

You can select a method for converting codes when saving HTML files with different names.

  • Specify

  • Select a code format from Korean (KS), Korean (KSSM), Japanese (SJIS), Simplified Chinese (GB), Traditional Chinese (BIG5), and DPRK (KPS).

  • Paste Options for HTML Document

  • You can paste the original format or only the text of the copied HTML document into Hwp.

Copy To Clipboard

You can select a method for converting codes when saving HTML files with different names.

  • Specify

  • Select a code format of your choice.

Send To Mail

You can select a method of converting codes by clicking File > Save/Send To > Send Mail (Body Text).

  • Specify

  • Select a code format of your choice.

  • Check and Confirm

  • The code format of text is checked automatically, and you receive a notification for confirmation. If the format is incorrect, select another format.

Send to Web Browser

By clicking File > Save/Send To > Send to Web Browser, you can select a method for converting codes.

  • Specify

  • Select a code format of your choice.

  • Check and Confirm

  • The code format of text is checked automatically, and you receive a notification for confirmation. If the format is incorrect, select another format.
