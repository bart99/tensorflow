Frequently Used Fonts
 

File > Options > Fonts

If you select Show Recent Fonts in the Fonts tab of File > Options, and select the type and number of frequently used fonts, frequently used fonts will always be on the top of the Fonts list of the Formatting toolbar.

If you click Frequently Used Fonts in the Fonts tab, the Font Settings dialog box appears, and you can select frequently used fonts.

More Information

If you click Font Settings, the Font Settings dialog box appears. In the dialog box, you can select frequently used fonts for each language. When you click the arrow from the Fonts list, the fonts selected as frequently used fonts will be on the top of the list.

  • Font Settings

  • In the Font Settings dialog box, you can select Language first, and then select up to six fonts from the Font List, and add them as Frequently Used Fonts.

Icon Name Description
Move to Frequently Used Fonts List If you select a font from the Font List and click Next, you can add it to Frequently Used Fonts.
Move to Font List If you select a font in Frequently Used Fonts and click Next, you can remove it from the frequently used fonts list.
Move One Line Up If you select a font in Frequently Used Fonts, and select this option, you can move the font to the previous line.
Move One Line Down If you select a font in Frequently Used Fonts and select this option, you can move the font to the next line.

Font Settings Dialog Box

Language

You can select a language from Rep., Korean, English, Chinese, Japanese, Foreign Languages, Symbol, and User.

The Rep. fonts are made up of multiple fonts for each language.

Font List

You can select a font for the selected language from the list.

Frequently Used Fonts

You can first select Language, and then select up to six fonts from the Font List to add them as Frequently Used Fonts.

If you set frequently used fonts, they are displayed above other fonts of the Fonts list on the Formatting toolbar.

Details on Icons

Move to Frequently Used Fonts List If you select a font from the Font List and click Next, you can add it to Frequently Used Fonts.

Move One Line Up If you select a font in Frequently Used Fonts, and select this option, you can move the font to the previous line.
Move One Line Down If you select a font in Frequently Used Fonts, and select this option, you can move the font to the next line.

Note

Font Type

You can use Hwp fonts in HFT format and Windows fonts in TTF format. The H in front of a font name on the font list refers to a Hwp font in HFT format. The T in the same position refers to a Windows font in TTF format.
