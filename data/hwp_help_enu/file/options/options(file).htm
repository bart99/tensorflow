Options: File
 

File > Options

In the File tab of File > Options, specify the settings to save or open documents or how to open recent documents in Hwp.

More Information

File Tab of Options Dialog Box

Save
  • Create Backup File

  • You can click File > Save to save a file. If there is an .HWP file with the same name in the disk, then change the extension of the .HWP file to .BAK and save the file.

    If you save a .DOCX file format, the backup file will not be created.

  • Set document password when saving new documents

  • If you work on confidential documents which require security, you can set the Set Password dialog box to display automatically. You do not have to click File > Document Security > Set Password every time you save a new document.

  • Notify when saving files to Temp folder

  • If you save a file from the temporary folder in the same folder, it might get deleted. To avoid this, a guide will be displayed, suggesting you to save it in another location.

  • Show as final version when opening with previous version of program

  • You can see the final version of your document for tracked changes when opening it with the previous version of Hwp.

  • Include Preview Image

  • You can save the image of the Preview window of the Open dialog box in a document. If you select this option, the speed of preview increases. The size of a preview image is 24KB.

  • Compress and Save

  • You can compress a file to a smaller size and save. If you are working on an important document, it is safe to remove Compress and Save.

    This option is a dedicated feature for .HWP file format. When you check this option and save a file in .DOCX file format, it will not be applied.

  • Search for private information when saving documents with privacy protection

  • If you save a document with Protect Privacy on, search for specified private information in the document and replace it with protected characters.

  • Save a Copy as

  • You can save another copy in the selected file format whenever you save a Hwp document file.

  • Save files in this format

  • You can select a format of the document to save in.

Auto Save a Temporary for Recovery

If you select Always Auto Save and set time, auto save runs at a specific interval. If you select Auto Save When Idle and set time, auto save runs when you do not use the keyboard in the specified period of time.

Specify File Location

You can create multiple macros, AutoText, or master documents required for specific documents, and save them in the same folder as the documents, for the next time you work on them.

  • Macro and AutoText File
    To select the path of macro files (*.HMS) and AutoText files (*.IDO) used in Hwp by default, click File Path.
  • Master Document
    To select the path of master documents (*.HWT) required for creating a new document by clicking New or New Tab, click File Path. When you create a new document by clicking New or New Tab, Hwp creates a master document (*.HWT) which contains the default Page Setup, Styles, and Multilevel List.
  • Default
    To revert the user-defined file location to the default Hwp location, click Default. Cancel the newly selected file location and reset the location to its original value.
Custom Data
  • You can open the data you added/saved while using Word in other systems by using Open Custom Data and Save Custom Data.
  • Custom data include macros, AutoText, AutoCorrect, and typing commands data.
  • When you save custom data, save all of them at the same time. When you open them, open all of them at the same time. You cannot save or open only the data you require.

Note

If No File Exists

If you select a path that does not contain a macro or AutoText file, the AutoText list will not be displayed, but for macros, the default macro list of Hwp will be displayed. If you select a path that does not contain a master document, the normal style of the Haansoft Batang font will be displayed, instead of the default style list.

Default File Path

The default paths of macro files (*.HMS), AutoText files (*.IDO), and master documents (*.HWT) used in Hwp by default are as follows:

  • Macro File: the \User\Hwp folder under the user data folder
  • AutoText File: the \User\Hwp folder under the user data folder
  • Master Document: the \Program Files\HNC\Shared80\HwpTemplate\Doc\Enu folder

   You can check the location of the user data folder in ? > About Thinkfree Office NEO Hwp > Folder Info.
