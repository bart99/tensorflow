Options: Object
 

File > Options

In the Object tab of File > Options, specify settings to select objects, and also specify the location to insert a frame when dragging.

More Information

Object Tab of the Options Dialog Box

Select
  • Select objects by partial selection

  • If you select an object, such as, drawing object or picture by clicking the Select Object icon in the Shapes tab and drag the pointer, the entire object gets selected even if you drag the pointer over a part of it.

    This option is useful for selecting only part of overlapped multiple objects.

Table Position
  • Wrapping Style

  • You can specify the location to insert pictures and tables, when you drag them into the body text.

    Icon Name Description
    Square Set text to flow around objects.
    Top and Bottom Fill the empty space as high as the object height.
    On Top of the Text Place objects behind the text so that they look like the background.
    Behind Text Place objects in front of text so that overlapped text is not displayed.
  • Horizontal

  • Select a basis in the document for placing inserted objects.

    You can use Horizontal to place objects in the horizontal position. You can select from Paper, Page, Column, and Paragraph.

  • Vertical

  • You can use Vertical to place objects in the vertical position. You can select from Paper, Page, and Paragraph.

Picture Object
  • Increase Picture Object Quality

  • To see the picture images clearly, when you scroll through a document with picture objects inserted. Speed may vary depending on your system specifications.

  • Calibrate WMF Picture

  • When Hwp abnormally displays WMF pictures which have been saved from the previous version of Hwp, you can calibrate it to be normally displayed in Hwp.

Object Description Guide
  • Show with Balloon Tips

  • Display object descriptions in a balloon tip when you place the pointer on objects.

    Select the Basic tab in the Object Properties dialog box and type in Others > Object Description.

    If place the pointer on the object, the content entered in Object Description will be displayed in a balloon tip.
