Options: Fonts
 

File > Options > Fonts

In the Fonts tab of File > Options, you can select the Fonts list to be displayed with the original fonts or display recently used fonts above other fonts on the list.

More Information

Fonts Tab of Options Dialog Box

View Font
  • Preview Font in Formatting Toolbar

  • If you select this option and click the Fonts list icon on the Formatting toolbar, the names of available fonts are displayed with the fonts applied. You can select a font without applying it to the body text. If you select this option, the font list might take a longer time to open.

  • Show Recent Fonts

  • If you select this option and click the Fonts list icon on the Formatting toolbar, recently used fonts will be displayed at the top of the list, and you can select them easily.

    You can specify the number of recently used fonts to be displayed at the top of the font list from 1 through 5.

  • Vivid View of Onscreen Font

  • You can make onscreen fonts attractive by rounding their edges. If you select this option, fonts will be displayed according to the value set in Hwp, regardless of the ClearType settings of the system.

Font List
  • Display in Single List

  • Display all fonts on a list that is not tab delimited.

  • Display in Tabbed List

  • The font list is categorized into All Fonts, Fonts, Title Font, Text Font, and Adorn Font tabs.

  • Font List Setup Button

  • The Font List Setup dialog box appears so that you can save frequently used fonts in a font list named as per your choice.

  • Default Button

  • Reset the font list.

Register Font
  • Representative Fonts Button

  • Click Representative Fonts to add a new representation font or modify already-added custom representation fonts.

    Representation fonts are made up of multiple fonts for each language that look good together.

Note

Font Type

You can use Hwp fonts in HFT format and Windows fonts in TTF format. The H at the beginning of a font name on the font list refers to a Hwp font in HFT format. The T in the same position refers to a Windows font in TTF format.
