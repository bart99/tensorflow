Options
 

File > Options

This feature is used to configure settings of Hwp to automatically save files, to open recent documents, and configure the option to spell check. You can also type user information and select a code format for reading and writing text files or HTML files.

The settings in File > Options are saved in the registry. When Hwp starts, the settings are applied to all documents.
