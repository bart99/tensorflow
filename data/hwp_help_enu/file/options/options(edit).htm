Options: Edit
 

File > Options

In the Edit tab of File > Options, specify the settings to save or open documents or how to open recent documents in Hwp.

More Information

Edit Tab of Options Dialog Box

Recent
  • Open Recent Documents

  • You can specify whether to open a recent document in a new tab of the active window (To New Tab in Active Window) or in a new document window (To New Window) when you click File > Open > Recent Documents. To New Window is selected by default.

  • Show Recent Documents in File Menu

  • If you open the File menu, a list of recent documents is displayed above the Exit menu under the Functions list. You can set the number of recent documents to be displayed on the list from 1 through 9. You can clear Show Recent Documents in Menu to clear the list of recent documents from the File menu.

  • Display Temp Folder Documents in Recent Documents List

  • You can specify whether to display the documents from the temp folder on the recent documents list or not.

Open
  • To New Window

  • If you open a document by clicking File > Open, the document opens in a new document window. To New Window is selected by default.

  • To New Tab in Active Window

  • If you open a document by clicking File > Open, the document opens in a new tab of the active window. To New Window is selected by default.

  • Move Cursor to First Line after Opening a Document

  • When you open a document, the pointer moves to the beginning, regardless of the last save position.

Edit
  • Apply Outline Style Automatically

  • You can automatically apply outline style like Multilevel List 1, Multilevel List 2, Multilevel List 3, ..., Multilevel List 7 according to the current outline level, when you apply outline or change outline level in an outline paragraph.

  • Use TAB for Moving in Table

  • You can use TAB to move the pointer between cells in a table. In a new empty table, you can move the pointer between cells by using arrow keys, but in a table with content, you have to use TAB to move the pointer between cells, regardless of the amount text in the table.

    • TAB: The pointer moves in front of the first letter of the next cell.
    • SHIFT+TAB: The pointer moves behind the last letter of the previous cell.
    • CTRL+TAB: Insert a Tab in the body text like on the normal edit screen.
  • Double-Click and Type

  • If you double-click on a blank location in the document, the pointer moves to the location. You can insert text, graphics, and tables in a position of your choice without adding an additional line or tab.

  • Confirm and Exit

  • If you select Confirm and Exit and click Close Window, a new document opens in the previous document window. If you click Close Window again in this document window, a confirmation message "Exit Hwp?" is displayed.

    If you do not specify this option, you will immediately exit Hwp, when you click Close Window in the last document window.

  • OLE Client Operation Support

  • If you select this option, Hwp starts properly when it is inserted in a program that supports OLE.

    This option is selected by default.

  • Check Spelling as you type

  • If you select this option, a red line will be drawn below a misspelled word in a Hwp document.

    If you place the pointer on a spell-checked word and right click, the correct spelling of the words will be suggested in the context menu.

  • Lock Insert

  • Lock typing status as Insert or Modify.

    If you set Lock Insert, the typing status will not change to Insert or Modify. The status will not change even if you click Insert/Modify on the Status Bar of the edit window or press INSERT.

  • Verify Script Execution

  • To execute a script in a Hwp document with scripts included, select Verify Script Execution and save.

  • Start Settings

  • If you select a start environment from the following list and restart Hwp, it starts according to the settings.

    • Noname: You can open a new Hwp document.
    • Open Last Used Document: You can open last used document in the edit window.
    • Recent Documents Dialog Box: Hwp starts, and the Open > Recent Documents dialog box appears.
    • Document Templates Dialog Box: Hwp starts, and the Document Templates dialog box appears.
    • XML Templates Dialog Box: When Hwp starts, and the XML Templates dialog box appears.
    • Open Dialog Box: Hwp starts and the Open dialog box appears.
Hyperlink Font

You can select font properties such as hyperlink font, font color, shape color, and underline.
