Representation Fonts
 

File > Options > Fonts

In the Fonts tab of the Options dialog box, you can add new Representative Fonts or modify already-added custom representation fonts. The Representative Fonts contain multiple fonts for each language. If you change only the Representative Fonts, the current font changes to a font containing fonts selected for each language.

Steps

To add representative fonts

  1. Click File > Options. In the Options dialog box, select the Fonts tab.
  2. Click Representative Fonts in Register Font.
  3. In the Representation Fonts dialog box, click the Add Representative Font icon .
  4. In the Add Representative Font dialog box, type the name of a representation font to add in Name, and select a font for Korean, English, Chinese, Japanese, Other Languages, Symbol, and User.
  5. Click Set to finish adding representation fonts.
  6. To return to the Representation Fonts dialog box, click Set.
  7. Exit Hwp and restart it. The added representation fonts is available.

To modify representative font

  1. Click File > Options. In the Options dialog box, select the Fonts tab.
  2. Click Representative Fonts in Register Font.
  3. In the Representation Fonts dialog box, select a representation font to modify from Representative, and click the Modify Representative Font icon .
  4. In the Add Representative Font dialog box, you can modify the name of a representation font in the Representative Font Name box or select a different font for each language.
  5. Click Set to finish editing representation fonts.
  6. To return to the Representation Fonts dialog box, click Set.

To delete representative font

  1. Click File > Options. In the Options dialog box, select the Fonts tab.
  2. Click Representative Fonts in Register Font.
  3. In the Representation Fonts dialog box, select a representation font to delete from Representative and click the Delete Representative Font icon .
  4. "Delete the selected representative font?" is displayed. Click Yes.
  5. Click Set to finish deleting representation fonts.

Note

Font Type

When you select a font for each language, you can choose from Hwp fonts in HFT format, and Windows fonts in TTF format. The H in front of a font name on the font list refers to a Hwp font in HFT format, and T in the same position refers to a Windows font in TTF format.

Registering with the Same Name

You can register a custom representation font with the same name, as any representation font provided in Hwp by default. In this case, you can only use the custom representation font.

You cannot register another representation font with the same name as the registered custom representation font.

Representation Font Information File

Information about custom representation fonts is automatically recorded in the file \User\Shared80\Fonts\Fontinfo.ini in the shared data folder.

You can check the location of the shared data folder by clicking > About Thinkfree Office NEO Hwp > Folder Info.
