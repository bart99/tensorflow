Compatible Document
 

File > Compatible Document

Hwp provides the same editing environment as MS Word so that users can write documents compatible with MS Word.

If you set Compatible Document, the layout of the active document changes to the layout of MS Word to keep the layout of the content. The user will be prompted not to use incompatible functions.

Steps

  1. Click File > New to create a new document window.
  2. Select File > Compatible Document.
  3. In the Compatible Document dialog box, select Microsoft Word Compatible Document in Select Program and click Set.

More Information

Compatible Document Dialog Box

Select Program

  • Hwp Document

  • If you open a Hwp document that is not set as a compatible document with the default value of Compatible Document, the document opens as a Hwp document.

  • Microsoft Word Compatible Document

  • MS Word Compatible Document is set. You can create a document in the same edit environment as MS Word. MS Word Compatible Document supports the latest version of Word.

    Compatible Document provides the properties of the latest version by using Hancom Update or the patch program of Thinkfree Office NEO. Therefore, it does not display the version of Word.

Apply Settings to New Document Button

Apply the selections in Select Program to all future documents as the default value.

Note

Layout Compatibility

When you are opening or saving a document from another program, you can change the layout of the active document to that of the program to keep the content layout.

If you set Microsoft Word Compatible Document, the layout, which affects the body text layout options like line spacing and spacing width, changes to the layout of MS Word.

Compatibility Notification

When you are creating a document compatible with other programs, you will be prompted not to use functions that are not compatible with those programs.

Home > Paragraph > Basic > Alignment > Spread does not exist in MS Word and thus is not compatible with MS Word.

If you set Microsoft Word Compatible Document and select Alignment > Spread, red borders will be drawn on the menu and a tool tip will notify you of incompatibility.

Open a Document of a Different Program

If you open a Word document by clicking File > Open, Microsoft Word Compatible Document will be automatically set for the document. The document compatibility is displayed after the document name on the title bar.

Save as a Document of a Different Program

If you save a document as a Word document by clicking File > Save, the Guide dialog box will open and informs that the document layout might change if the document is not a Microsoft Word Compatible Document.
