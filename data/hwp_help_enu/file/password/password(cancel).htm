Change/Remove
 

File > Document Security > Change/Remove

This feature is used to change or remove the password set for the document.

Steps

To change document passwords

  1. Click File > Open to open a document with a password.
  2. In the Document Password dialog box, type the password correctly in Current Password, and click OK.
  3. When the document opens, click File > Document Security > Change/Remove to change the password.
  4. In the Change/Remove dialog box, select Change Password and type the current password in Current Password to change it.
  5. In New Password, type a password of five or more characters.
  6. In Confirm Password, type the same password to record the correct password.
  7. Click Change. The new password will be set for the active document.
  8. Click File > Save. The new password will be recorded in the file along with document content.

To remove document passwords

  1. Click File > Open to open a document with a password.
  2. In the Document Password dialog box, type the password correctly in Current Password, and click OK.
  3. When the document opens, click File > Document Security > Change/Remove to remove the password.
  4. In the Change/Remove dialog box, select Remove Password.
  5. In the Current Password box, type the password of the active document correctly, and click Disable.
  6. The dialog box closes and the password set for the active document gets removed.
  7. Click File > Save.

More Information

Change/Remove Dialog Box

Change Password

To change the password set for the active document, select this option.

Remove Password

To remove the password set for the active document, select this option.

Current Password

Type the password of five or more characters set for the active document.

New Password

Type a new password of five or more characters for the active document.

Confirm Password

In New Password, type the password again.

Security Type

Display security level set for the active document.

  • Security Level Normal

  • Set normal security level.

  • Security Level High

  • Set high security level.
