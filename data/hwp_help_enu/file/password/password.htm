Set Password
 

File > Document Security > Set Password

With this feature, you can designate a document password so that no one can open the file without the password. The document file opens only when the document password is typed correctly.

Steps

To set document passwords

  1. Click File > Document Security > Set Password.
  2. In the Set Password dialog box, type a password of five or more characters in Document Password.
  3. Type the same password in Verify again to record the correct password.
  4. Click Set. The password is set for the active document.
  5. Click File > Save. The password will be recorded in the file.

To open a document with a password

  1. Click File > Open, and select a document with a password.
  2. In the Document Password dialog box, type the password correctly in Current Password, and click OK.
  3. The document opens if the password is typed correctly. If the password is not correct, a message about incorrect password is displayed and the document does not open.

More Information

Saving an .hwp or .hwt document prompts the Set Password dialog box, and saving a .docx document prompts the Set Password to Open/Modify dialog box.

Set Password Dialog Box

Document Password

Type a password of five or more characters for the active document.

Verify

Type the password you typed in Document Password.

Security Level

Display security level set for the active document.

  • Normal

  • Set normal security level.

  • High

  • Set high security level.

Set Password to Open/Modify Dialog Box

With .docx documents, you can set different passwords for opening and modifying the document.

Password to Open

Enter a password for opening the document.

Password to modify

Enter a password for modifying the document.

Note

Considerations for Document Password

  • Typed Password Cannot Be Checked

  • When you specify a document password, the typed password is displayed as ***** on the screen. The person who types the password will know the password.

  • Character Limit for Password

  • A password should be minimum five characters and maximum 44 characters. The limit is not counted by the number of Korean or English letters, but by the number of keys you press on the keyboard.

  • Keyboard Classification

  • You can mix English letters, numbers, and SPACEBAR on the keyboard to create the password.

  • Save File

  • You can click File > Save. You must save the document after setting the password.
    You can set the Document Password if you save a document by selecting File Type > Hwp Document (*.hwp) or File Type > Hwp Template (*.hwt). If you save a document in other file types by clicking Save As, a document password cannot be set.

  • Password Decoding Impossible

  • The user-defined password of a document cannot be modified, removed or decoded.

Caution

Document

If you insert a document with a password into a document without a password, the password will not be set for the document without a password. For Document Password, the password of the active document is the priority regardless of Document.

Resctrictions on executing PE files

Portable Executable (PE, commonly *.exe file format) files cannot be executed for security reasons if the security level is set to High.
