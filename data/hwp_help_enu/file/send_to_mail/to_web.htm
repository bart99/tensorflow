Send to Web Browser
 

File > Send To > Send to Web Browser

You can temporarily save the active Hwp document as HTML, and preview it in a web browser before uploading it to the web. If you create a home page by using Hwp, preview and modify the content before uploading it to the server.

Steps

  1. Create a Hwp document, and click File > Save/Send To > Send to Web Browser.
  2. In the Web Document Type dialog box, select a code of your choice from the Select Character Code list, and click OK.
  3. If a web browser is open, the document will be sent to the web browser immediately. If not, a web browser will start and display the document.

Note

Uploading to Web Server

You can preview the document by clicking File > Save/Send To > Send to Web Browser, before running Upload to Web Server. If the preview in the web browser is appropriate, upload the document to the web server.

Web Document Type

  • Korean (KS)

  • You cannot use characters that do not have code values assigned, because each complete character has a code value assigned. Only 2350 Korean characters, which is 21% of 11,172 total characters, have code values assigned.

  • Korean (KS)

  • You can express all of the 11,722 modern Korean characters by dividing each character into initial phoneme, medial vowel, and final phoneme, and assigning a code value to each one. Two bytes (16 bits) are used to express one Korean character. The first bit is used to distinguish the character as English or Korean. The remaining 15 bits are assigned to initial phoneme, medial vowel, and final phoneme, five bits to each.

  • Japanese (SJIS)

  • This is an advanced standard Japanese code.

  • Simplified Chinese (GB)

  • This is a standard Chinese code.

  • Traditional Chinese (BIG5)

  • This is a standard Taiwanese code.

  • Unicode (UTF-8)

  • Unicode is also called Worldwide Character Standard-Unicode, this code is designed to facilitate exchanging, handling, and expressing text, written in various languages of the world.
