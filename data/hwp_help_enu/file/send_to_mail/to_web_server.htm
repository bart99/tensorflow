Upload to Web Server
 

File > Save/Send To > Upload to Web Server

This feature allows you to upload a Hwp document to the web server.

In the Upload to Web Server dialog box, you can upload a Hwp document to the web server by specifying the path and file name. You can upload files to the web server without an FTP program.

Steps

  1. Save a Hwp document to upload, and click File > Save/Send To > Upload to Web Server.
  2. In the Upload to Web Server dialog box, type a name to use for the web page list in Page Name.
  3. In FTP Server, type the name of the FTP server of your Internet service provider.

    For example, if your service provider is Netian, type ftp.netian.com. You can find the name of the FTP server on the home page of the service provider.

  4. Type User ID and Password correctly.
  5. In the Path box, type the path of the web server where you want to save the active document. The default path of the home page is public_html.
  6. In the File Name box, type a file name for the active document.

    To display the active document on the first page of the home page, save it as Index.html.

  7. Click Upload. The active file will be uploaded to the specified path of the server when you connect to the FTP server.

More Information

Upload to Web Server Dialog Box

Web Page List

Display a list of web pages you added.

For example, if you enter information about frequently used web pages, like your own home page, and then add it to the Web Page List, you do not have to enter the information again.

  • (New Page)

  • To add a new page to the web page list, select (New Page) from the web page list.

Web Page Info

Enter the information for the web page to add to the web page list.

  • Page Name

  • Enter a name for the page to add to the web page list.

  • FTP Server

  • Enter a name for the FTP server to upload files to. For example, if your service provider is Netian, enter ftp.netian.com.

  • User ID

  • Type your account (ID) registered on the FTP server.

  • Password

  • Type your password.

  • Path

  • Enter the path of the web server where you want to save the active document. The default path of your home page is public_html.

  • File Name

  • Enter a name for the active document. To display the active document on the first page of the home page, save it as Index.html.

    • Save as HTML File

    • You can upload the active document to the web server in Internet document type (*.html). You can also view documents in web browsers like on regular web pages.

    • Save as HWP File

    • You can upload the active document to the web server as a Hwp document. The file to be uploaded to the web server is not displayed as a web document in web browsers, because it is a Hwp document.

Details on Icons

Icon Name Description
Delete from List Delete the selected web page information from the Web Page List.
Move one row up Change the order of currently added web pages. If you click this button, the selection moves to the previous line.
Move one row down Change the order of currently added web pages. If you click this button, the selection moves to the next line.
Add Add the selected web page information to the Web Page List.
Save Save the modified web page information.

Caution

Usage Restriction

Upload to Web Server is available only when your account includes Internet service.

Advanced Usage

Preview in Web Browser

You can preview a Hwp document by using File > Save/Send To > Send to Web Browser before uploading it to the web server. You can upload the document to the web server, when the preview in the web browser is appropriate.
