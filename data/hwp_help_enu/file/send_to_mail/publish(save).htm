Distribution Document
 

File > Distribution Document > Save

In Hwp, you sometimes need to create the documents as read-only, because you might accidentally delete, modify, or add to part of a document that should not be changed.

Save allows a document to remain unchanged.

Steps

To save as a distribution document

  1. Click File > Distribution Document > Save.
  2. Type a password of five or more characters in Writing Password of the Save as Distribution Document dialog box.
  3. To record the correct password, type the same password in Verify, and click Save.
  4. File > Save will run. The password will be recorded as a file with document content and the active document will become a Distribution Document.
  5. Open a distribution document by clicking File > Open. The document opens as a read-only document.

To modify the password of distribution documents

  1. To modify the password of a distribution document in edit mode, click File > Distribution Document > Change/Remove.
  2. In the Save as Distribution Document dialog box, select Change Password, and correctly type the password to modify of the active document in Current.
  3. Type a password of five or more characters in New, and Verify, and then click Change.
  4. The new password will be recorded as a file with document content.

To remove the password of distribution documents

  1. To remove the password of a distribution document, click File > Distribution Document > Change/Remove.
  2. In the Save as Distribution Document dialog box, select Remove Password, and correctly type the password to modify of the active document in Current.
  3. Click Disable. The properties of the distribution document gets removed and the document becomes a regular document.

More Information

Save as Distribution Document Dialog Box

Writing Password

You can type five or more characters to use as the password for editing a distribution document.

  • Need for Writing Password: If you require to re-open a saved distribution document and edit it again, you need a password to modify to edit the document without changing its properties.
  • Password Display: When you set a password to modify, the typed password is displayed as ***** on the screen. For security of the document, only the person who types the password knows it.
  • Character Number Limit: Password to modify should be from 5 through 44 characters. The limit is not counted by the number of Korean characters or alphabets, but by the number of keys you press.
  • Keyboard Classification: For password to modify you can mix alphabets, numbers and SPACEBAR of the keyboard to modify the password, because the keys pressed on any type of keyboard is remembered.
  • Password Decoding Impossible: You must not forget the password you have specified, because the user-defined password cannot be modified, removed, or decoded.

Verify

Type same password you typed in New.

Save Selected

When you save the active document as a distribution document, you can use Print Restriction and Copy Restriction to select the option to open the document and make it available only on the screen or to allow Print and Copy.

  • Print Restriction

  • When you select this option and open a distribution document, you cannot run Preview or Print, and you can only view it on the screen.

  • Copy Restriction

  • If you select this option, you cannot copy or cut the content of a distribution document.

Note

Save File

If you click File > Save to save a New Document as a distribution document, you can specify the file name in the Save As dialog box.

If you name a file by clicking Save or if you open a named file by clicking Open, and then click Security > Save in the file, the content created under the same name will be saved as a distribution document. A confirmatory prompt will not be displayed.

Function Restriction in Distribution Document

If you open a distribution document or save the active document as a distribution document, only few functions of Hwp will be available The functions that are available in distribution documents are as follows:

  • Functions Available in Distribution Document With Printing Restricted
    • File: New, New Tab, Document Templates, Open, Close, Exit
    • Edit: Copy, Select All, Find, Find Again, Go to
    • View, Help: All functions can be used
    • Tools: Options
  • Functions Available in Distribution Document With Copying Restricted
    • File: New, New Tab, Document Templates, Open, Close, Print, Preview, Exit
    • Edit: Find, Find Again, Go to
    • View, Help: All functions can be used
    • Tools: Options
  • Functions Available in Distribution Document With Both Printing and Copying Restricted
    • File: New, New Tab, Document Templates, Open, Close, Exit
    • Edit: Find, Find Again, Go to
    • View, Help: All functions can be used
    • Tools: Options
  • Functions Available in Distribution Document With Both Printing and Copying Allowed
    • File: New, New Tab, Document Templates, Open, Close, Print, Preview, Exit
    • Edit: Copy, Select All, Find, Find Again, Go to
    • View, Help: All functions can be used
    • Tools: Options
Documents with a Password Cannot Be Saved as Distribution Documents

Documents with a password set in File > Encrypt with Password > Set Password cannot be saved as distribution documents. You can remove the document password and save it as a distribution document.

Save as Distribution Document After Selecting Block

You can select a part of the active document to save the whole document as a distribution document by clicking File > Distribution Document > Save.

Distinguishing Document Properties

You can distinguish the active document by its properties, displayed at the end of its name. If you open a distribution document, Distribution Document will be displayed at the end of its file name. If you open a read-only document, Read-Only will be displayed.

Caution

Characteristics of Password to Modify

You cannot read distribution documents in previous Hwp versions including Hangul Wordian. To read distribution documents, you have to install the latest Hwp program.

Document

You cannot Insert distribution documents in other documents.
