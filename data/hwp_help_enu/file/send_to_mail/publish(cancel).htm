Change/Remove
 

File > Distribution Document > Change/Remove

With this feature, you can change or remove the password of distribution documents.

Steps

To modify the password of distribution documents

  1. To modify the password of a distribution document in edit mode, click File > Distribution Document> Change/Remove.
  2. In the Save as Distribution Document dialog box, select Change Password, and correctly type the current password of the active document in Current.
  3. In New and Verify, type a password of five or more characters, and then click Change.
  4. The new password will be recorded as a file with document content.

To remove the password of distribution documents

  1. To remove the password of a distribution document in edit mode, click File > Distribution Document > Change/Remove.
  2. In the Save as Distribution Document dialog box, select Remove Password, and correctly type the current password of the active document in Current.
  3. Click Disable. The properties of the distribution document is removed and the document becomes a regular document.

More Information

Save as Distribution Document Dialog Box

Change Password

Select this option to change the password of a distribution document.

Remove Password

Select this option to remove the password of a distribution document.

Current

Type five or more characters to use as the password for editing a distribution document.

New

Type five or more characters to use as the password for editing a distribution document.

Verify

Type the same password you typed in New.

Save Selected

When you save the active document as a distribution document, you can use Print Restriction and Copy Restriction to select the option to open the document and make it available only on the screen or to allow Print and Copy.

  • Print Restriction

  • If you select this option and open a distribution document, you cannot run Preview or Print, you can only view it on the screen.

  • Copy Restriction

  • If you select this option, you cannot copy or cut the content of a distribution document.

Disable Button

If you type Current and click Disable, the properties of the distribution document are removed and the document becomes a regular document.

Note

Characteristics of Password to Modify

  • Character Number Limit: Password to modify should be from 5 through 44 characters. The limit is not counted by the number of Korean characters or alphabets, but by the number of keys you press.
  • Keyboard Classification: For password to modify you can mix alphabets, numbers and SPACEBAR to modify the password, because the keys pressed on any type of keyboard is remembered.

Documents with a Password Cannot Be Saved as Distribution Documents

Documents with a password set in File > Encrypt with Password > Set Password cannot be saved as distribution documents. You can remove the document password and then save the document as a distribution document.

Tip for Saving Distribution Document

If you run Save or Save As when a distribution document is open in edit mode, the document becomes a non-editable distribution document. You have to type the password to return to edit mode.

It is not effective to click Save when you are editing a distribution document, you can click Save after editing is complete.
