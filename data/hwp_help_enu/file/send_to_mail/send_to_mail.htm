Save/Send To
 

File > Save/Send To

You can send an email message with a Hwp document as the body text or an attachment by using this function from the default mail program of Windows.

To send the active Hwp document as the body text of an email message, click Save/Send To > Send Mail (Body Text). To attach the document to an email message, click Save/Send To > Send Mail (Attachment).

Note

Body Text and File Attachment

If you send an email message without any applied style, you can use Body Text for convenience.

When you need to keep 100% of the style created in Hwp, you have to use Attachment.

Installation of Default Mail Program

This function is not available, if the default mail program is not properly installed in your system.

Cannot Return to Edit Screen

To modify a Hwp document, when a new message window of the mail program is open, close the New Message window first, and then click File > Save/Send To.

You cannot modify a document on the Hwp edit screen when a New Message window is open.
