Send Mail (Body Text)
 

File > Save/Send To > Send Mail (Body Text)

You can create an email with the content on the Hwp edit screen by using the send new mail function in the default mail program installed in your system.

Also, you can easily send the content of a Hwp document as an email message by typing To and Subject and clicking Send in the new message window of the mail program.

Steps

  1. On the edit screen, type content for an email message or open a saved document.
  2. Click File > Save/Send To > Send Mail (Body Text).
  3. In the Web Document Type dialog box, select a code in Select Character Code and click OK.
  4. The content on the Hwp edit screen is automatically inserted as the body text of a New Message in the mail program.
  5. Type To, Subject, and Content.
  6. Click Send to send the email message.

Note

Converting to HTML Document

If you send the content of a Hwp document as the body text of an email message, some styles, such as header, footer, and footnote will not be maintained.

If HTML allows, the styles created in Hwp can be used in the body text of an email message.

Character Set (Code)

Hwp supports multiple character sets (codes), such as Unicode (UTF-8), and KS for saving Hwp content in web document type.

To send an email message in the character set of your choice, select the character set in File > Options > Code Format.

You can also select a character set from the mail program, before sending a new message.

For example, you can select a character set in Format > Encoding of a New Message window in MS Outlook Express.

To select a character set in other mail programs, you can refer to instructions or Help provided by each program.
