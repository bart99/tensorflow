Send Mail (Attachment)
 

File > Save/Send To > Send Mail (Attachment)

You can create an email message and attach a Hwp document by using the default mail program installed in your system.

Also, you can send the content of a Hwp document as an email message by typing To and Subject and clicking Send in the new message window of the mail program.

Steps

  1. Open the document to send in an email message.
  2. Click File > Save/Send To > Send Mail (Attachment).
  3. The content on the Hwp edit screen is automatically inserted as an attachment of the new message in the mail program.
  4. Type To, Subject, and Content.
  5. Click Send to send the email message.

Note

File Name

The name of the Hwp document is used as the name of the file attached to the New Message.

If the document is a New Document without a name, it is attached with the file name noname.hwp.
