Edit
 

File > Distribution Document > Edit

You can edit the document and keep the distribution document properties unchanged.

Steps

To edit distribution documents

  1. Open a distribution document to edit, and click File > Distribution Document > Edit.
  2. The Edit distribution Document dialog box asks you for Password.
  3. In the Edit dialog box, type the correct Password of the document, and click Edit.
  4. Enter the correct password then the document opens. A message about incorrect password is displayed if the password is not correct, and the document does not open.

To modify the password of distribution documents

  1. Click File > Distribution Document> Change/Remove to modify the password of a distribution document in edit mode.
  2. In the Save as Distribution Document dialog box, select Change Password, and type the correct password of the active document in Current Password.
  3. In New and Verify, type a password of five or more characters, and then click Change.
  4. The new password will be recorded as a file with document content.

To remove the password of distribution documents

  1. Click File > Distribution Document > Change/Remove in a distribution document in edit mode.
  2. In the Save as Distribution Document dialog box, select Remove Password and correctly type the current password of the active document in Current.
  3. Click Disable. The properties of the distribution document is removed and the document becomes a regular document.

More Information

Edit Distribution Document Dialog Box

Password

 Correctly type the password you used, when saving the distribution document.

Note

Characteristics of Password to Modify

  • Character Number Limit: Password to modify should from 5 through 44 characters. The limit is not counted by the number of Korean characters or alphabets, but by the number of keys you press.
  • Keyboard Classification: For password to modify you can mix alphabets, numbers and SPACEBAR of the keyboard to modify the password, because the keys pressed on any type of keyboard is remembered.

Documents with a Password Cannot Be Saved as Distribution Documents

Documents with a password set in File > Encrypt with Password > Set Password cannot be saved as distribution documents. You can remove the document password and save the document as a distribution document.

Tip for Saving Distribution Document

If you run Save or Save As when a distribution document is open in edit mode, the document becomes a non-editable distribution document. You have to enter the password to return to edit mode.

It is not effective to run Save when you are editing a distribution document, you can run Save after editing is complete.
