Save as Picture
 

File > Save/Send To

In Hwp, you sometimes need to create read-only documents.

Steps

  1. Open a document to save as a picture and click File > Save/Send To > Save as Picture.
  2. In Save in of the Save as Picture dialog box, select a drive to save the file in.
  3. Open the folder in the file list window, and select the type of picture file from File Type, and then save the name of the file in File Name.
  4. Click Options.... In the Save as Picture Options dialog box, select Resolution > Medium Resolution, and Specify Color > 256 Colors, and then click OK.
  5. Click Save when you return to the Save as Picture dialog box.

More Information

Save as Picture Dialog Box

File Name

When you save a document as a picture file, save each page as one file. A serial number will be added at the end of each file name for you to distinguish the files easily. The first page file has 001 added at the end of the specified name, and the second page file has 002. For example, if you select SAMPLE.BMP as the name of the picture file to save, actual files created will have file names like SAMPLE001.BMP and SAMPLE002.BMP.

File Type

When saving as an image file, you can choose from six different file formats: BMP (*.bmp), GIF (*.gif), PNG (*.png), JPG (*.jpg), WMF (*.wmf), and EMF (*.emf). The default setting is BMP (*.bmp).

File Type Description
BMP Windows Standard Bitmap Picture Type
GIF Picture type often used for communication created in CompuServe
PNG Picture type created for web design (Portable Network Graphic)
JPG Picture type designed to reduce file size with the quality of Truecolor maintained
WMF Picture type with Windows standard outline
EMF 32-bit image format capable of containing vector and bitmap information.

Options

To set picture resolution and color, display the Save as Picture Options dialog box.

Save as Picture Options Dialog Box

Resolution

In the Save as Picture Options dialog box, click Options, and then select a resolution for the picture file to save from Lowest Resolution (72 DPI), Low Resolution (120 DPI), Medium Resolution (150 DPI), High Resolution (180 DPI), and Highest Resolution (300 DPI).

The resolution unit is DPI (Dots per Inch). The file is more accurate with a higher resolution, but with a high resolution, the file will be bigger in size and will take longer to be saved.

The default resolution is Lowest Resolution (72 DPI). The content displayed on the screen gets saved as 100% in size by using this resolution.

Specify Color

To save the picture image in black and white, 16 colors or 256 colors, select the appropriate option. The default value is 256 colors.
