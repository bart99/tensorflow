Print: Basic
 

File > Print > Basic

In the Basic tab of the Print dialog box, you can set print range, copies, and method and then run Print or Print Preview.

More Information

Basic Tab of Print Dialog Box

Select Printer
  • Printer List Box

  • In the list box of Select Printer, you can add or select a printer. You can also use Save as PDF, Save as Picture, and Send Fax.

    • Save as PDF

    • Save the document as a PDF file instead of printing it from the printer. To use Save as PDF, you have to select a general-purpose PDF driver from the PDF Driver list in the Others tab of the Options dialog box.

    • Save as Picture

    • Save the document as a picture file instead of printing it from the printer.

  • Print to File

  • Save the print result in a disk as a *.prn file without printing it on paper from the printer. Refer to Print to File for more information.

  • Settings Button

  • In the Printer Document Registration Information dialog box, you can check or reset layout, paper/quality, target and information.

  • Advanced Button

  • In the Advanced Options for Printer dialog box, select your options.

Print Range

Select a print range from Whole Document, Current Page, To Here, Current Section, and From Here.

Copies
  • Copies

  • Set copies by selecting a number in the list box. Copies can be set from 1 to 1000.

  • Collate

  • When multiple copies are printed, the page order is 1-2-3, 1-2-3, 1-2-3. There is no need to reorganize printed documents.

Method

Set method such as Automatic Print, Divide and Print, Multiple Pages, and Collect and Print.

  • Basic Print

  • You can select either Automatic Print or Fit to Paper in the list box. Automatic Print is the default value.

  • Multiple Pages

  • If you select a page number, pages will be automatically reduced in size so that the selected number of pages will be printed on each piece of paper.

  • Divide and Print

  • Print a document based on large paper in several pieces of small paper.

  • Collect and Print

  • After printing a certain number of pages, you will be prompted for confirmation on another printing. So you can choose to continue or stop printing.

  • Reverse Order

  • Print the document starting from the end.

  • Economic Printing

  • You can choose between Grayscale and Light Grayscale from the combo box. The default setting is Grayscale. You can increase or decrease the color intensity or contrast level of printouts.

  • Printer Paper

  • Select printer paper such as A3, A4, and B4.

Print Button

Finish print settings and start printing.

Preview Button

You can check the printing condition on the Preview screen.

Note

Creating Virtual Printer

By creating a copy of the printer driver on the Select Printer list, you can use a virtual printer driver. Set printing for the virtual printer driver separately from the original printer driver for easy use.

If you set virtual printers with different sources of paper, you can select any paper source you need and print from one printer. You do not need to change the paper settings every time you print.

  • Utilizing Virtual Printer

  • For example, assume that a printer has new A4 paper in Tray 1 and A4 scratch paper in Tray 2. The following is how to create a virtual printer for this printer.

    1. From the Select Printer list, select AA, the original printer driver to which Tray 1 is assigned to. Then right click it and select Create Virtual Printer from the context menu.
    2. In the Register Virtual Printer dialog box, designate a Printer Name and click Register to create virtual printer aa.
    3. Select virtual printer aa and select Set to designate Tray 2 as the paper source.
    4. When printing on new A4 paper, select printer driver AA. When printing on A4 scratch paper, select printer driver aa.

Caution

Resetting Printer List After Adding/Deleting Printer

If a printer has been externally added or deleted, the printer list of Hwp will reset. Information about existing virtual printer drivers will disappear.
