Print <CTRL+P>
 

File > Print

This feature is used to print the active document.

You can print multiple copies of the entire or part of the document. You can also change print settings such as zoom depending on the purpose of the print-out.

Besides, you can set watermark to be displayed only on printed documents.

Steps

  1. Click File > Print to print the active document.
  2. To print only the current page, select Print Range > Current Page and type 1 in Copies in the Print dialog box.
  3. Click Print.
  4. A dialog box about print progress will be appear temporarily and the page where the pointer was on the edit screen will be printed.

Note

Setting Block and Print

If you select part of the document as a block and click Print, Selected Page only will be set as default in Print Range. Also, all pages that include the block will be printed.

For example, if you select content from the end of Page 2 to the beginning of Page 3 in a three-page document and select Selected Page only, all the content on Page 2 and Page 3 will be printed.

Printing Part of Document with Discontinuous Page Number

If the page numbers of the document are not continuous, it is difficult to specify print range in Print Range > Partial in the Basic tab. In this case, use Collect and Print to print a specific number of pages from the position you specify.

Select Print Range >From Here and Method > Collect and Print in the Basic tab of the Print dialog box and select how many pages to print from the current page. When the document finishes printing to the last page you want, you will be prompted for confirmation. Click Cancel not to continue printing.
