Header/Footer for Printing
 

File > Print

In Header/Footer for Printing in the Advanced tab of the Print dialog box, you can select items for the header and footer that will be printed on top and bottom of the print-out.

Steps

  1. Click File > Print to print the document on the edit screen.
  2. In the Print dialog box, click the Advanced tab.
  3. Select Header Content in Header/Footer for Printing and click Edit.
  4. In the Edit Header/Footer for Printing dialog box, select (Empty) by clicking the arrow of the Center list in Automatic Header, and then click Set.
  5. When you return to the Print dialog box, click Preview.
  6. Header content appears only on the left and right at the top of the print-out.
  7. Click Print in the Preview toolbar. The document will be printed.

More Information

Edit Header/Footer for Printing Dialog Box

Header/Footer for Printing

In the Left, Center, and Right boxes of Automatic Header (Footer), type an item or select an item by clicking the arrow of the lists to print the item on the left, in the center, and on the right respectively.

For example, if you select Total ^P pages, now at ^p page from the Left list in Automatic Header and print Page 5 of a ten-page document, Total 10 pages, now at 5 page will be automatically printed as the header.

Automatic Header/Footer

In Automatic Header and Automatic Footer, you can select (Empty) from the Left, Center, and Right lists first and type the following special characters in the boxes.

Special Characters Description   Special Characters Description
^f File Name Only   ^x Fax Number
^d File Name Including Path   ^e Phone Home
^t Date   ^h Phone Mobile
^p (lowercase) Current Page Number   ^u UMS Number1
^P (uppercase) Total Pages   ^w Home Page
^n User Name   ^m E-mail 1
^c Company Name   ^z Zip Work
^j Job Title   ^a Address Work
^s Department Name   ^o Zip Home
^i Phone Work   ^r Address Home

Note

Entering User Info

Automatic Footer automatically displays the user name, company name, department name, phone (work), and fax number of Options > User Info. To fully use automatic footer, click File > Options and correctly type in each item of the User Info tab.

Advanced Usage

Deleting Content from Text Box

When printing file name with the path on the left, print date in the center, and current page number on the right, the file name and the print date might overlap with each other if the file path is too long. When this happens, delete all the content of the text box or empty the text box by selecting (Empty) from the list. Then the file name including the path, and the current page number will be printed on the left and right respectively.
