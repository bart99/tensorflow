Print: Advanced
 

File > Print > Advanced

In the Advanced tab of the Print dialog box, you can set print range, copies and method and then run Print or Print Preview.

More Information

Advanced Tab of Print Dialog Box

Zoom

You can specify horizontal and vertical ratio to zoom in or zoom out of the document to specific size. Select zoom ratio from 10% through 500%.

  • Horizontal

  • Specify horizontal zoom ratio.

  • Vertical

  • Specify vertical zoom ratio.

  • Lock Aspect Ratio

  • If you select this option and change the horizontal (or vertical) ratio, the vertical (or horizontal) ratio changes identically.

Header/Footer for Printing
  • Header Content

  • Print with user name on the left, company name in the center, and current page number on the right at the top of the print-out.

  • Footer Content

  • Print with filename on the left, print date in the center, and total page number and current page number on the right at the bottom of the print-out.

Print Options

Select whether to print the Drawing Object, Picture Object, Form Controls, Click Here, Highlighter, Proofing Symbols, Comments, Memo Content, Show Page Setup, and Barcode inserted in the document.

If you select Comments, Memo Content activates and gets automatically selected. You can select whether to print the page with comment content separately.

If you select Show Page Setup, you can select whether to display paper size on the print-out.

Barcode gets activated when it is possible to print barcodes. If you select Barcode, the print-out will have a 2-D barcode on the top right.

  • Binding Hole

  • When printing a document to punch a hole and keep in a binder, select a hole position for the type of binder to use.

  • Odd-Even Print

  • If you want to print a document on both sides of a paper from a printer that does not support such printing, print on Odd Pages first and turn over the print-out. Then place the print-out back in the printer and print on Even Pages.

Print Button

Finish print settings and start printing.

Preview Button

You can check the printing condition on the Preview screen.

Note

Zoom Ratio Depending on Paper Size and Paper

Specify zoom ratio for the selected paper size and paper in the printer as follows.

For example, if the paper in the printer is B5, select Paper Size > B4 in the Basic tab of File > Page Setup and select 70% for horizontal/vertical ratio in the Advanced tab of File > Print. Then the document will be reduced so that it fits B5.
