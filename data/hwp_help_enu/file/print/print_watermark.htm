Print: Watermark
 

File > Printer > Watermark / Page Layout > Watermark

In the Watermark tab of the Print dialog box, you can set picture watermark or text watermark to apply to the document only when you print.

Steps

  1. Click File > Print to insert watermark when printing the active document.
  2. In the Print dialog box, select the Watermark tab.
  3. Select Text Watermark and type Hancom Inc. in the Enter Text box.
  4. Select Haansoft YGO 250 from the Fonts list and select 80pt for Size.
  5. Select Watermark Effect and select -45 for Angle.
  6. Click Preview to check if watermark is displayed in a proper position on the Preview screen.
  7. To print the file, click the Print icon in the Preview tab.

More Information

Watermark Tab of Print Dialog Box

No Watermark

Use this option to remove watermark.

Picture Watermark

Open a picture file and insert picture watermark in the document.

  • Picture File

  • Click Select Picture to open a picture file to fill watermark with.

  • Fill Type

  • Specify how to fill the document with picture watermark.

  • Picture Effects

  • You can apply black and white or grayscale effect.

  • Watermark Effect

  • Apply watermark effect to the opened picture file to change the file to a bright and low-contrast picture.

    If you select this option, Brightness and Contrast will be automatically set as 70% and -50% respectively.

  • Brightness

  • Adjust the brightness of the selected picture file. Brightness can be set from -100 through 100.

  • Contrast

  • You can adjust the contrast of the selected picture file. Contrast can be set from -100 through 100.

  • Transparency

  • Specify transparency for the selected picture file. Transparency can be set from 0% through 100%. The default value is 70%.

Text Watermark

Type text to insert text watermark in the document.

  • Enter Text

  • Type text to insert as watermark.

  • Font

  • Select a font for text watermark.

  • Text Color

  • Select a color for text watermark. If you click the color table, you can select a color from the color palette. The default color is black.

  • Watermark Effect

  • Apply watermark effect to the entered text to shade it.

  • Size

  • Select font size for text watermark. Size can be set from 25pt through 254pt. The default size is 40.0pt.

  • Angle

  • Select an angle for text watermark. Angle can be set from 0 through 360. The default angle is 0.

  • Transparency

  • Select transparency for text watermark. Transparency can be set from 0% through 100%. The default transparency is 50%.

Position

Select a position for picture or text watermark to insert.

  • Relative to Paper

  • If you select Relative to Paper as the basis position of picture or text watermark, watermark will be inserted based on full paper size.

  • Relative to Page

  • If you select Relative to Page as the basis position of picture or text watermark, watermark will be inserted within page borders.

    For the range of page borders, the value in Position of Page Border/Background > Borders applies.

Placement

If picture or text watermark objects overlap with the body text, specify placement order so that watermark text or pictures are printed according to the Placement value.

  • Beneath the Text

  • If picture or text watermark objects overlap with the body text, send the objects behind the body text.

  • On Top of the Text

  • If picture or text watermark objects overlap with the body text, send the objects in front of the body text.

Print Button

Finish print settings and start printing.

Preview Button

You can check the printing condition on the Preview screen.
