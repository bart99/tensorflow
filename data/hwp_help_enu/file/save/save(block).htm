Save Block
 

File > Save Block

With this feature, you can save only the part selected as a block in the document. Select the part in the document that you want to save as a block, and select Save Block in the File menu to save only the selection.

Steps

  1. Select part of the document as a block by pressing F3 and arrow keys.
  2. Click File > Save Block.
  3. In the Save Block dialog box, select a save location and type a file name.
  4. Click Save Block.

Note

Difference Between Save Block and Save As

  • Save Block

  • Select part of the active file as a block and save it as a new file. The name of the active file does not change.

  • Save As

  • Save the entire active file instead a part of it.

Save in New Document

  • Press F3/F4 and Click Save in New Document

  • If you press F3 or F4 and then click Save or press ALT+S in a new document, Save Bock runs.

Difference Between Save Block and Save Style

  • Save Block

  • Save Block saves not only the style applied to the block but also all styles in the active file as a Save Block file.

  • Save Block

  • Copy Block copies only the style applied to the block.

Document Password

If you run Save Block for part of a file with a password, the same password will be set for the file in which the block will be saved.

Page Setup

If you run Save Block, the Page Setup, Master Page, and Page Border/Background on the page where the pointer is will be copied in a new file.

Save as Different File Type

When running Save Block, you can save the block in file type other than Hwp document file type (*.hwp).

Select a file type you need from the File Type list and save.
