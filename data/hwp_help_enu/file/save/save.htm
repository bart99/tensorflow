Save <CTRL+S>
 

File > Save

With this feature, you can save the active document as a Hwp file.

Steps

  1. Click File > New to create a new document window.
  2. Type content in the new document window and click File > Save.
  3. In Save in of the Save As dialog box, select a drive or folder to save the file in.
  4. In the file list box, select a folder to save the file in.
  5. Type the name of the file to save in the File Name box.
  6. Click Save.

More Information

Save As Dialog Box

Save in

Select a drive or folder you want to move the file to the folder. Or, type the name of a drive or folder in File Name.

File Name

Type the path or name of the file to save.

File Type

Select the file type to save.

Document Password Button

Show the Set Password dialog box to set a document password and security level.

Storage Settings Button

Show the Storage Settings button to set Auto Save, Save Options, and Save in New Document.

Favorites Icon Description

If you add frequently used folders to the Favorites list on the left of the Save As dialog box, you can select documents to open or save easily and fast.

For more information about Favorites, refer to the Favorites part.

Icon Name Description
Add to Favorites In the Save As dialog box, add the selected location to the Favorites list.
Edit Favorites Modify the location, name, path, or type added to the Favorites list of the Save As dialog box.
Delete Favorites Delete the location of Favorites added to the Favorites list of the Save As dialog box.

Note

Saving for the First Time and Saving Repeatedly

When you save a document started as a New Document by clicking File > Save, the Save As dialog box appears and you can type a file name.

If you Save a file that you previously named by running Save or a file that you opened by running Open, the file will be overwritten with the new content made so far under the same name. You will not be prompted for confirmation.

File Name Restriction

The characters that cannot be used in the file name are \, /, :, * ? ", <, >, and |.

Automatically Saved File Name

Using the first line of a document as the file name is good because it usually has the title of the document.

If you create a New Document and click File > Save for the first time, the content of the first paragraph automatically appears in the File Name dialog box.

If the File Name is appropriate, click Save. If not, type another name.

The automatically saved file name contains the content of the first paragraph that comes before the first instance of characters that cannot be used in the file name like punctuation marks such as period and comma. The file name can have up to 100 characters Korean or English.

Saving Multiple Documents Simultaneously

You can check a list of all open document windows and save multiple documents simultaneously by using View > Switch Windows.

Advanced Usage

Saving Pointer Position

As soon as you save a document as a Hwp document (*.hwp), the pointer position is saved.

When saving a document you are still working on, it is good to place the pointer in a position where you want to start the next time, and run Save.

Creating Backup File

If you select Create Backup File in the Options > Edit tab, every time you run Save for a file, the *.HWP file with the same name in the disk changes to a *.BAK file (a backup file created) and the active file gets saved as a *.HWP file.

Saving Copy

If you select Save a Copy as in the Options > Edit tab, every time you save a Hwp document file, an additional copy gets saved as the specified file type. Use Save a Copy as when you need a file in a different file type, besides a Hwp document file.

Caution

ODF Support & .NET Framework

Thinkfree Office NEO supports an ISO standard and ODF (Open Document Format). All data and documents are compatible regardless of program type or version.

However, Open/Save ODF is not available when Microsoft .NET Framework 2.0 or higher is not installed in the system. If you install Thinkfree Office NEO in a system without the .NET Framework program, it is installed with Thinkfree Office NEO.
