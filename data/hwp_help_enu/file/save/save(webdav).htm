Save to Internet Folder
 

File > Save As

This feature allows you to save the active document to an internet folder.

For you to use an internet folder, the server that provides it must support the WebDAV protocol.

Steps

To save to an internet folder

  1. Click File > New.
  2. Type content in the new document window and click File > Save. Or, click Save in the Formatting Toolbar.
  3. In Save in of the Save As dialog box, select Internet Folder.
  4. Select a folder you want from the internet folder list.
  5. The Access Internet Folder dialog box appears if you are not logged in the selected folder.
  6. Type User ID and Password and click Set. The file list of the selected folder appears.
  7. In the file list box, select a folder to save the file in.
  8. Type the name of the file to save in the File Name box and click Save.

More Information

Access Internet Folder Dialog Box

User ID

Type user ID to connect to the internet folder.

Password

Type user ID and password to connect to the internet folder.

Note

WebDAV

WebDAV (Web-based Distributed Authoring and Versioning) is an extended HTTP protocol that manages web server files through the web.
