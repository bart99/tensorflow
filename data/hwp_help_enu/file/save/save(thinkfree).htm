Save to Netffice 24
 

File > Save As

With this feature, you can save the active document to Netffice 24.

Steps

To save documents to Netffice 24

  1. Click File > New to open a new document window.
  2. Type in the new document window and click File > Save. Or, click Save in the Formatting Toolbar.
  3. In the Save As dialog box, select Netffice 24 from the Favorites list.
  4. If you are not logged into Netffice 24, the Connect to Netffice 24 dialog box appears.
  5. Type User ID and Password, and click Set. The ThinkFree Online folder of the user is displayed.
  6. In the File Name edit box, type the name of the file to save and click Save.

More Information

Connect to Netffice 24 Dialog Box

User ID

Type the user ID to connect to Netffice 24.

Password

Type the password for the user ID to connect to Netffice 24.

Sign up for Netffice 24 Button

Move to the webpage where you can sign up for Netffice 24 (https://auth.netffice24.com/en/).

Find Password Button

Move to the Netffice 24 password finding webpage.

Use External Service Account

You can log in to the service with Netffice 24 account as well as with an external service account.

  • ThinkFree Online

  • Log in to the service with ThinkFree Online account.

  • Facebook

  • Log in to the service with Facebook account.

  • Google

  • Log in to the service with Google account.
