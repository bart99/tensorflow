Compare Documents
 

Review > Compare Documents

You can use this feature to compare two documents and display the differences in a new edit window.

If you have edited a document without turning on the Track Changes feature, you can compare the original document with the edited document to identify the changes with ease.

Steps

To compare documents

  1. Click Review > Compare Documents.
  2. From the Compare Documents dialog box, click on the Open icon under Original Document.
  3. From the Open dialog box, select the original file and click Open.
  4. Under Revised Document, click the Open icon and select the revised document from the Open dialog box.
  5. From the Compare Documents dialog box, click OK. The two documents are opened in the Read-Only window, and arranged vertically.

From the Compare Documents tab, you can change the view settings and use the zoom controls.

More Information

Compare Documents Dialog Box

Original Document

Specify the original document.

Revised Document

Specify the revised document.

Icon

Click on the icon <-> to set the original document as the revised document, and vice versa.
