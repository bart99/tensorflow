Change Tracking Options
 

Review > Change Tracking Options

This feature allows you to specify the change indication color, balloon tip size, connector color, and connector position settings.

You can also assign different settings for inserted segments, deleted segments, and individual reviewers to achieve greater collaboration efficiency.

Steps

To set change tracking options

  1. Click Review > Change Tracking Options.
  2. From the Track Changes dialog box, specify the markup for insertions and deletions, track formatting, and balloon tip settings.
  3. Click OK.

More Information

Track Changes Dialog Box

Markup

  • Insertions: Specify a format to distinguish inserted segments from the original.
  • Deletions: Specify a format to distinguish deleted segments from the original.
  • Changed Lines: Specify the position of changed line indication.
  • Color: Specify the color for insertions, deletions, and lines.

Formatting

  • Track Formatting: Distinguish changed formatting from original document formatting.
  • Formatting: Specify a format to distinguish changed formatting from original document formatting.
  • Color: Specify formatting colors.

Balloons

  • Show lines connecting to text: Show lines connecting changes to balloon tips.
  • Preferred width: Specify the width of balloon tips.
