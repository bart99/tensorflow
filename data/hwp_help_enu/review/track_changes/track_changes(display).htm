Show Markup
 

Review > Track Changes

This feature allows you to specify the range and method of showing markups.

You can choose from Original, Final with Markup, Final, and Original with Markup and individually specify whether to show comments, inserts/deletes, formatting, and other changes by individual reviewers.

Steps

To show markup

  1. Open a document that has Track Changes switched on.
  2. Select the Review tab.
  3. From the Track group, specify the range and method of showing markups in the edit window.

Note

Show Markup Range

You can choose to show the final version only or the original and the revised versions at the same time.

  • Final - Show Markup: Display the document content with emphasis on the revised version. Display deleted segments in balloon tips and inserted segments and formatting changes directly on the body in the manner designated by the user. Choose this setting to view changes and comments made by the editor and other reviewers.
  • Final: View the final document with all changes applied.
  • Original - Show Markup: Display the document content with emphasis on the original version. Display inserted segments and formatting changes as balloon tips and deleted segments directly on the body in the manner designated by the user. Choose this setting to view changes and comments made by the author of the document.
  • Original: Display the original version with all changes removed.

Show Markup

Changes are shown in the edit window in the manner defined in Change Tracking Options.

  • Insertions and Deletions: Display inserted and deleted segments.
  • Format: Display formatting changes.
  • Reviewer: Selectively display changes made by a specific reviewer.
