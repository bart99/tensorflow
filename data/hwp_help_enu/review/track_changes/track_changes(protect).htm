Protect Changes
 

Review > Track Changes

You can protect changes in documents so that they cannot be edited by others by using the Protecting Changes feature.

Steps

To set change tracking lock password

  1. Select the Review tab.
  2. In the Track group, click on the drop-down arrow next to the Track Changes icon and select Protect Changes.
  3. From the Set Password to Lock Change Tracking dialog box, enter a password made up of 5 or more characters in the Document Password field.
  4. Type the same password in Confirm Password again to record the correct password.
  5. Click OK.

To change tracking lock password

  1. Select the Review tab.
  2. From the Review toolbar, click Track Changes and select Protect Changes.
  3. From the Change/Remove Password to Lock Change Tracking dialog box, select Change Password.
  4. Enter the Current Password and then a New Password.
  5. Enter the new password again in Verify and click Change.

To remove change tracking lock password

  1. Select the Review tab.
  2. From the Review toolbar, click Track Changes and select Protect Changes.
  3. From the Change/Remove Password to Lock Change Tracking dialog box, select Remove Password.
  4. Enter the Current Password and then click Remove.
