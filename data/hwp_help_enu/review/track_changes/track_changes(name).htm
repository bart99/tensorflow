Change User Name
 

Review > Track Changes

You can change user names that are used to sort reviewers to display changes.

Steps

To change user names

  1. Select the Review tab.
  2. Click on the drop-down arrow next to the Track Changes icon and select Change User Name.
  3. From the Change User Name dialog box, enter a user name in the User Name field.
  4. Click OK.

User name in Show Markup > Reviewer

The user name in the User Name field in the User Info tab of the File > Options is shown in the User Name field of the Change User Name dialog box.
