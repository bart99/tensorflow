Compare Versions Tab
 

Review > Compare Documents

Compare Files tab is a collection of icons for commands used to compare an original copy with a revised copy. From the Compare Files tab, you can move between comparisons, search for items, set view options, and zoom-in/out.

More Information

Compare Files Tab

  • Move
  • Previous Comparison Item The cursor moves to the previous comparison item.
    Next Comparison Item The cursor moves to the next comparison item.
  • Find/Replace
  • Find Searches the current document for a specific word (text string). You can narrow down the search and perform more complex searches by using Options.
    • Find: Display the Find dialog box.
    • Replace: Finds instances of a specific word (text string) in the current document and replaces them with another. You can perform more precise and complex find/replace tasks by using Options.
    • Find Again: Click Find or Go to again without opening the Find/Replace dialog box. Searches for the term used in the previous Find or Go to action, going Up or Down from the current cursor position, depending on the Find > Search Direction setting.
    • Go to: Delete fields which were created repeatedly. Select fields to delete first.
    Previous Finds the previous instance of the search term from the current cursor position.
    Next Finds the next instance of the search term from the current cursor position.
  • View
  • Split View Displays the comparisons in two separate windows.
    Toggle Horizontal/Vertical Switch the window arrangement between horizontal and vertical.
    Fit Size Make the width and height of windows of different sizes equal.
    Scroll Side by Side If this icon is selected, both windows scroll vertically or horizontally when you drag the vertical/horizontal scroll bar or roll the mouse wheel.
    Mark with Comments Displays the comparison result as comments on the right side of the document comparison window.
    Mark with Symbols Displays the comparison results as proofing symbols within the document comparison window.
  • Language/Settings
  • Dictionary List Hancom Dictionary (consists of MinJung K-K, K-E Essence, E-K Essence, Korean thesaurus, and English thesaurus), Chinese Dictionary, and Thesaurus/Antonyms are available for use.
    Options Configure the editing environment by specifying Auto Save, Recent Documents, and Check Spelling options. In addition, you can register the user info and set the code format for reading and writing text and .html files.
  • Zoom
  • Zoom Zoom in/out the active document without changing its actual size.
    • Fit to Page: Zoom in/out the document so that one whole page is displayed on the screen regardless of paper size.
    • Fit to Width: Zoom in/out the document so that its width fits that of the document window regardless of paper size.
    • Two Pages: Zoom in/out the document so that two papers are displayed on the screen regardless of paper size. The Two Pages displayed on the screen have nothing to do with page number. Pages are displayed by two page increments from the beginning.
    • Book: Zoom in/out the document so that two papers are displayed on the screen regardless of paper size. This option implies that even number pages are always displayed on the left of the screen and odd number pages are always displayed on the right of the screen as in a published book. Pages are shown in order.
    • 100% ~ 500%: Zoom in/out windows according to the specified ratio.
    • When the pointer looks like a magnifying glass, click the document to see it zoomed in/out at the specified ratio.
    • Custom: The screen zoom ratio can be adjusted from 10% to 500%.
    Zoom Out Click the icon to zoom out the screen 5%.
    Zoom In Click the icon to zoom in the screen 5%.
  • Close
  • Close End document comparison and close the document comparison window.
