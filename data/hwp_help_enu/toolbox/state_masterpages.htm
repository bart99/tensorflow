Master Page Tab
 

Page Layout > Master Page > Master Page Tab

Master Page tab is a collection of icons for commands used to create a master page for the document including type, properties, insert object, view, and move.

More Information

Master Page Tab

  • Master Page
  • Both Pages Create a master page that applies to both pages.
    Odd Page Create a master page that applies only to odd pages. If you select both the master page for both pages and the master page for odd pages, the master page for odd pages overrides the master page for both pages.
    Even Page Create a master page that applies only to even pages. If you select both the master page for both pages and the master page for even pages, the master page for even pages overrides the master page for both pages.
    Last Page Create a master page on the last page of the section where the pointer is. The master page applied to the last page of a section overrides the master page applied to both pages, odd pages, or even pages.
    Specified Page Create a master page that applies only to random pages in the selected section. The master page applied to random pages in a section overrides the master page applied to both pages, odd pages, or even pages.
    Previous Section Move to the master page created in the previous section in a document with more than two sections. If there is no master page before the current section, this option prompts for the Master Page Application Range so that a new master page can be created.
    Next Section Move to the master page created in the next section in a document with more than two sections.
    Previous Master Page Move to the previous master page from the active one. This button cannot be clicked if there has not been a master page previously.  
    Next Master Page Move to the next master page from the active one. This button cannot be clicked if there is no master page after the current one.
    Delete Master Page Opens delete master page commands.
    • Delete Current Master Page: Deletes the master page of the section at the current position of the pointer.
    • Delete Master Page from Current Section Deletes all master pages from the current section.
    • Delete Master Page from Entire Document Deletes all master pages from the entire document.
  • Adjust
  • On Top When Printing Set the current master page to be on top of the body text when it is printed. In other words, the master page can cover the body content when printing the document. This option is linked to the Master Page to Front icon of the Master Page toolbar. The toolbar is displayed when you are editing the master page.
    Exclude First Page Hide the master page for the first page of the current section.
    Import Master Page Import the master page of another section to the selected master page type.
    Use Previous Master Page Create a new master page using the master page from the previous section. If there is no master page from the previous section, this button functions as Delete master Page from Current Section.
    Overlap Can only be set if, for Master Page Type, Section Last Page or Section Random Page is selected. Setting this option overlaps the master page of the section on top of existing master pages (both, odd, and even pages).
  • Insert
  • Shapes Select and insert a shape you want.
    • Other Shapes: Opens the Clip Art dialog box so that you can use various shapes.
    • New Drawing Style: Opens the New Drawing Style dialog box so that you can specify the line color, face color, line style, and other settings of a new drawing object.
    Picture Opens the Insert Picture dialog box so that you can insert a picture in the master page that is being edited.
    Page Number Inserts the page number in the master page at the current cursor position.
    Create New Section Opens the Create New Section dialog box to divide a new section on the master page.
  • View
  • Gridlines With gridlines enabled, you can draw a fixed grid of dots or lines on the edit screen. Gridlines provide precision positioning control when drawing or a editing a picture.
    • Show Gridlines: Displays gridlines in the body as per settings under Gridline Settings.
    • Gridline Settings: You can change details of grid by setting gird operation, grid spacing, grid style, and grid origin.
    View Margin Select View Margin to view the margin guidelines of the current master page.
  • Zoom
  • Zoom Zoom in/out the active document without changing its actual size.
    • Fit to Page: Zoom in/out the document so that one whole page is displayed on the screen regardless of paper size.
    • Fit to Width: Zoom in/out the document so that its width fits that of the document window regardless of paper size.
    • Two Pages: Zoom in/out the document so that two papers are displayed on the screen regardless of paper size. The Two Pages displayed on the screen have nothing to do with page number. Pages are displayed by two page increments from the beginning.
    • Book: Zoom in/out the document so that two papers are displayed on the screen regardless of paper size. This option implies that even number pages are always displayed on the left of the screen and odd number pages are always displayed on the right of the screen as in a published book. Pages are shown in order.
    • 100% ~ 500%: Zoom in/out windows according to the specified ratio.
    • When the pointer looks like a magnifying glass, click the document to see it zoomed in/out at the specified ratio.
    Zoom Out Click the icon to zoom out the screen 5%.
    Zoom In Click the icon to zoom in the screen 5%.
  • Close
  • Close Ends master page editing and returns to edit body mode.
