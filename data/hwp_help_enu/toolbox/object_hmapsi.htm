Wordshop Tools Tab
 

Insert > Illustrations > Wordshop > Wordshop Tools Tab

The Wordshop Tools tab contains icons of commands used to insert and edit Wordshop text. You can edit the format, alignment, and placement of Wordshop text and specify the fill effect, line color, and line type by using the Wordshop Tools tab.

More Information

Wordshop Tools Tab

  • Wordshop
  • Create Wordshop Insert Wordshop text. Wordshop is a feature that decorates letters using a variety of effects including bending, outline, fill, shadow, and rotate. Wordshop is built on the drawing features of Hwp and allows you to edit Wordshop text even after creating it.
    • Wordshop: Open the Create Wordshop dialog box so that you can create Wordshop text.
    Edit Content Modify the content of the selected Wordshop.
    Select Object In order to change the size or color of a drawn object, change its location, group/ungroup an object, or rotate an object, you must first select the object. To do so, use the Select Object icon.
    Captions Insert a number, title, and/or brief description into the selected Wordshop text.
  • Styles
  • Wordshop Properties Display the Object Properties dialog box to modify properties of the selected wordshop such as size, location, rotation, etc.
    Styles Present a Wordshop style image collection that incorporates wide varieties of fill, line, and effect properties. Click More to open the Wordshop Style image collection, which consists of different Wordshop color, fill, line type, and line color combinations.
    Line Color Change the outline color of the selected Wordshop text into a color selected by the user. Click on the arrow next to the Line Color icon to open the color palette and choose the desired color.
    Fill Change the color of the selected Wordshop text into a color selected by the user. Click on the arrow next to the Fill icon to open the color palette and choose the desired color. Other Fill: Open the Fill tab of the Object Properties dialog box so that you can choose from other Fill options.
    Shading Increase or decrease the brightness of the selected Wordshop object.
    Line Type Change the border type of the selected Wordshop object into a line type selected by the user. Click on the arrow next to the Line Type icon and choose the line type from the list of available options.
    • Other Lines: Open the Lines tab of the Object Properties dialog box so that you can choose a different line type setting.
    Line Weight Specify the weight of the selected Wordshop object.
    Wordshop Shape Change the shape of the Wordshop text. Click on the arrow next to the Wordshop Shape icon to view more than 50 different types of Wordshop shapes. Select a shape from the available options. The selected shape is then applied as the framework of the Wordshop object, and the text is changed accordingly.
    Paragraph Align Specify the paragraph alignment of the Wordshop text. Click on the arrow next to the Wordshop Alignment icon to open the Wordshop paragraph alignment format collection and select from Left, Right, Center, Justified, and Justified Including the Last Row.
  • Shadow
  • Apply Shadow After inserting a Wordshop object, you can add a shadow effect to the Wordshop text or remove an existing one.
    Move Wordshop Shadow Move the Wordshop shadow in the selected direction. You can move the shadow up, down, left, and right. Each click moves the shadow by 2%.
  • Size
  • Width Specify the width of the Wordshop object.
    Height Specify the height of the Wordshop object.
    Fix Size To fix the size of the Wordshop object, select Fixed Value for both Width and Height settings of the chart object.
    Equal Width Set the same width for two or more selected Wordshop objects.
    Equal Height Set the same height of two or more selected Wordshop objects.
    Equal Width/Height Set the same width and height for two or more selected Wordshop objects.
  • Arrangement
  • In Line with Text The object is treated as a regular text. Therefore, its position changes when you type or delete text.
    Fusion Object and body share the same line without coming in contact so that text flows around the object in a rectangular form.
    Top and Bottom Objects occupy space because of its height hence the body text cannot fit in the space.
    On Top of the Text Arrange the selected object to be placed in front of text.
    Beneath the Text The body content fills up as if there is no object and the selected object lies beneath the body text.
    Distance from Text Select the outside margin of the object. You can directly adjust the value directly from Object Properties > Margin/Caption > Distance from Text.
  • Align
  • Group Group multiple Wordshop objects into one.
    Ungroup If two or more Wordshop objects have been grouped as one, ungroups the Wordshop objects.
    Bring to Front Bring the Wordshop object to the front when multiple Wordshop objects are overlapped. If you click the drop-down arrow , the following menu is displayed.
    • Bring to Front: Bring the selected Wordshop object to the front.
    • Bring Forward: Move the selected Wordshop object one object forward.
    Send to Back Move the Wordshop object to the back when multiple Wordshop objects are overlapped. If you click the drop-down arrow , the following menu appears.
    • Send to Back: Send the selected Wordshop object to the back.
    • Send Backward: Move the selected Wordshop object one object backward.
    Align/Distribute Specify the alignment or the space distribution between objects when multiple Wordshop objects are selected. If you click the icon, the following menus appear.
    • Align: Select from Align Left, Align Center, Align Right, Align Top, Align Middle, and Align Bottom.
    • Distribute: Select either Distribute Horizontally or Distribute Vertically.
  • Rotate/Flip
  • Rotate Object Rotate a drawn Wordshop object by a range of 1¡Æ through 360¡Æ.
    Rotate Left 90 Rotate the selected Wordshop object by 90¡Æ counterclockwise.
    Rotate Right 90 Rotate the selected Wordshop object by 90¡Æ clockwise.
    Horizontal Flip Flip the selected Wordshop object horizontally at the center.
    Vertical Flip Flip the selected Wordshop object vertically at the center.
