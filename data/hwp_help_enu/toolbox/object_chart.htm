Chart Tools Tab
 

Insert > Illustrations > Insert Chart > Chart Tools Tab

The Chart Tools tab is a collection of icons for various commands that are used to insert and edit charts. You can edit chart format and legends and specify chart style with ease by using the Chart Tools tab.

More Information

Chart Tools Tab

  • Chart
  • Column Select the desired chart format from the vertical bar chart image collection, which is divided into Basic, Expanded, and Combination groups.
    Line/Area Select the desired chart format from the line/area chart image collection, which is divided into Basic Line, Expanded Line, Basic Area, Expanded Area groups.
    Pie Select the desired chart format from the pie chart image collection, which is divided into Pie and Doughnut groups.
    Bar Select the desired chart format from the horizontal bar chart image collection, which is divided into Basic and Expanded groups.
    Scatter Select the desired chart format from the scatter chart image collection, which is divided into Basic, Expanded, and Bubble groups.
    Others Charts Select the desired format from the chart image collection, which is divided into Stock, Radar, and Surface groups.
  • Data
  • Edit Data Open the Edit Chart Data dialog box to allow editing of chart data.
    Row Draw a chart on row data basis.
    Column Draw a chart on column data basis.
  • Styles
  • Chart Styles Present a chart style image collection that incorporates wide varieties of chart colors, types, backgrounds, and effects. Click More to expand the chart style image collection and show 30+ chart styles.
    Colors The image collection unfolds so that you can select the color set for a chart.
    Effects The image collection unfolds so that you can select the effect for a chart.
    Background The image collection unfolds so that you can select the background for the entire chart object. The following options are displayed:
    • No Chart Background: Removes the background that has been applied to the chart object.
    • Set Chart Background: Shows the Background tab of the Chart Background dialog box.
    Plot Area The image collection unfolds so that you can select the background for the chart. The following options are shown: Setting a plot area background applies the selected background only to the plot area of the chart object.
    • No Plot Area: Removes the background applied to the plot area of the chart object.
    • Set Plot Area: Shows the Background tab of the Chart Properties dialog box.
  • Title
  • Chart Title Click this icon to display or hide the chart title.
    • On Top: Display the title at the top corner.
    • On Bottom: Display the title at the bottom corner.
    • On Center Left: Display the title on the center-left corner.
    • On Center Right: Display the title on the center-right corner.
    • Hide Chart Title: Hide the title.
    • Format Title: Show the Title Styles dialog box to decorate the background, characters, font, and position of the title.
    Legend Click this icon to show or hide the chart legend.
    • On Top: Display the legend at the top corner.
    • On Bottom: Display the legend at the bottom corner.
    • On Center Left: Display the legend at the center-left corner.
    • On Center Right: Display the legend at the center-right corner.
    • Hide Legend: Do not display the legend.
    • Format Legend: Display the Legend Styles dialog box to decorate the background, font, and position of the legend.
  • Axis
  • Axis Set the style and background of the chart axis.
    • Axis: If you select Axis in the Select Axis dialog box, the Axis Styles dialog box appears so that you can change the axis format.
    • Label: If you select Axis in the Select Axis Label dialog box, the Axis Label Styles dialog box appears so that you can change the characters, font, and background of the axis label.
    • Title: Select an axis from the dialog box, and the Axis Title Styles dialog box appears so that you can edit the title of the axis and the font and background settings.
  • Series
  • Series Set the series and data point of each chart.
    • Series: If you select Series in the Select Data Series dialog box, the Series dialog box appears so that you can change the color and borders of the series.
    • Series Label: If you select Series in the Select Data Series dialog box, the Series Styles dialog box appears so that you can change the characters, font, and background of the series label.
    • Data Point: If you select Series and Data Point in the Select Data Point dialog box, the Data Point Styles dialog box appears so that you can decorate the data point.
    • Data Labels: If you select Series and Data Point in the Select Data Point dialog box, the Data Point Label Styles dialog box appears so that you can change the characters, font, and percent format of the data point label.
  • Arrange
  • Wrap Text Specify chart placement position.
    • In Line with Text: The chart object is treated as a regular text. Therefore, its position changes when you type or delete text.
    • Square: Place them so that the object and the body text share the same lines without overlapping each other and the body text flows around the object.
    • Top and Bottom: The chart object occupies the line, and the body text cannot enter the space occupied by the chart object.
    • Behind Text: The body text gets filled as if there were no chart. The chart is used as the background of the body text.
    • In Front of Text: Arranges the selected chart object to be placed in front of text.
  • Size
  • Width Set chart object width.
    Height Set chart object height.
    Fix Size To fix the size of the chart object, select Fixed Value for both Width and Height settings of the chart object.
