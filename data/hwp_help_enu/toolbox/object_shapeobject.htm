Drawing Tools Tab
 

Insert > Illustrations > Shapes

Drawing Tools tab is a collection of icons for frequently used shapes style, shadow, size, placement, and alignment commands.

More Information

Drawing Tools Tab

  • Insert Shapes
  • Insert Shapes Select and insert a shape of choice.
    • Other Shapes: Opens the Clip Art dialog box so that you can use various shapes.
    • New Drawing Style: Opens the New Drawing Style dialog box so that you can specify the line color, face color, line style, and other settings of a new drawing object.
    Edit Polygon Edit the selected polygon or curve and change the shape of object.
    Add Text Inside Shape You can use a rectangle, circle, polygon, arc, or any other shape that was drawn in Hwp to create a text box.
  • Shapes Styles
  • Format Presents Copy Object Style, Paste Object Style, and Set as New Drawing Style options.
    • Copy Object Style: Pastes a style copied from a picture object or a drawing object on to another drawing object.
    • Paste Object Style: Pastes a style copied from a picture object or a drawing object on to another drawing object.
    • Set as New Drawing Style: Specifies the line type, fill style, wrapping style, text box margin, and text box alignment settings for an objects and draws the object based on the specified properties.
    Shape Fill Change the face color of the selected object or specify the face color of an object that will be drawn.
    Shape Outline Change the color of the selected straight line or the object border. You can also specify the line color for an object that will be drawn.
    Shading Increase/decrease the brightness of the selected object.
    Line Weight Specify the line weight of the selected object or an object that will be drawn.
    • Other Lines: Opens the Lines tab of the Object Properties dialog box so that you can select a different Weight setting.
    Line Type Specify the line type of the selected object or an object that will be drawn.
    • Other Lines: Open the Lines tab of the Object Properties dialog box so that you can choose a different line type setting.
    Arrows Specify the style of arrows at the two ends of the selected object or an object that will be drawn.
    • Other Arrows: Opens the Lines tab of the Object Properties dialog box so that you can select a different Arrows style.
  • Shadow
  • Shadow Style You can add a shadow to a shape object in the direction and style that you want or remove an existing shadow.
    • No Shadow: Removes the Shadow from the selected object.
    • Other Shadow Colors: Opens a color palette so that you can change the shadow color of the selected object.
    Moving Shadow Moves the shadow of the shape object in the selected direction. You can move the shadow up, down, left, and right. Each click moves the shadow by 1mm. Click + to reset the shadow.
  • Text Box
  • Vertical Writing With vertical text boxes, the text is arranged vertically.
    • English Down: Apply vertical writing. Rotate English and numbers 90 degrees clockwise.
    • English Up: Apply vertical writing. Return English and numbers to their original state.
    Align Text Specify the vertical alignment setting for the text inside the text box from top, middle, bottom, upper left, lower left, upper right, lower right, upper center, and lower center.
    Text Box Margin Set distance from text box borders to text box content.
    • Text Box Margin Settings: Selecting Text Box Margin Settings opens the Text Box tab of the Object Properties dialog box so that you can change the margin settings of the selected object.
  • Arrange
  • Wrap Text Specify the placement position of the shape.
    • In Line with Text: The shape object is treated as a regular text. Therefore, its position changes when you type or delete text.
    • Square: Place them so that the shape object and the body text share the same lines without overlapping each other and the body text flows around the shape object.
    • Top and Bottom: The shape object occupies space by the amount of its height and the body text cannot enter the space.
    • Behind Text: Arranges the shape object to be placed in front of text.
    • In Front of Text: The body text gets filled as if there were no shape object. The shape object is used as the background of the body text.
    Bring to Front Bring the object to the front when multiple objects are overlapped. If you click the drop-down arrow , the following menu is displayed.
    • Bring to Front: Bring the selected object to the front.
    • Bring Forward: Move the selected object one object forward.
    Send to Back Send the object to the back when multiple objects are overlapped. If you click the drop-down arrow , the following menu is displayed.
    • Send to Back: Send the selected object to the back.
    • Send Backward: Move the selected object one object backward.
    Align Specify the alignment or the space distribution between objects when multiple objects are selected. If you click the icon, the following menus is displayed.
    • Align: Select from Align Left, Align Center, Align Right, Align Top, Align Middle, and Align Bottom.
    • Distribute: Select either Distribute Horizontally or Distribute Vertically.
    Group Groups multiple objects into one or ungroups grouped objects. If you click the icon, the following menus appear.
    • Group: Groups multiple objects into one.
    • Ungroup: Ungroups objects when two or more objects are grouped.
    Rotate Rotates the selected picture object or flips it vertically or horizontally from its center line. If you click the icon, the following menus appear.
    • Rotate Right 90: Rotates the selected shape object by 90¡Æ clockwise./li>
    • Rotate Left 90: Rotates the selected shape object by 90¡Æ counterclockwise.
    • Flip Vertical: Flips the selected shape object vertically at the center.
    • Flip Horizontal: Flips the selected shape object horizontally from its center line.
    • Rotate Object: Rotate a shape object by a range of 1¡Æ through 360¡Æ.
  • Size
  • Height Set shape object height.
    Width Set shape object width.
    Fix Size To fix the size of the shape object, for both Width and Height settings of the shape object, select Fixed Value.
    Equal Width Sets the same width for two or more selected shape objects.
    Equal Height Sets the same height for two or more selected shape objects.
    Equal Width/Height Sets the same height and width for two or more selected shape objects.
