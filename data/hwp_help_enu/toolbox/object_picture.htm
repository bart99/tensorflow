Picture Tools Tab
 

Insert > Illustrations > Picture > Picture Tools Tab

Picture Tools tab is a collection of icons for various commands that are used to insert and edit pictures. You can edit picture formats and effects and specify picture styles with ease by using the Picture Tools tab.

More Information

Picture Tools Tab

  • Adjust
  • Brightness You can adjust the level of the brightness of the picture. Use the Brightness image collection from the Adjust group under the Picture Tools tab or the Brightness adjustment slider from the Picture tab in the Object Properties dialog box to adjust the level of brightness.
    • No Brightness: Removes the Brightness value that has been applied to the selected picture. To adjust Brightness from the Picture tab of the Object Properties dialog box, click Effects > Brightness.
    • Image Brightness Settings: Open the Picture Effects tab of the Object Properties dialog box so that you can adjust the brightness level.
    Contrast You can adjust the contrast level of pictures. Use the Contrast image collection from the Adjust group under the Picture Tools tab or the Contrast adjustment slider from the Picture tab in the Object Properties dialog box to adjust the level of contrast.
    • No Contrast: Removes the Contrast value that was applied to the selected picture. To adjust Contrast from the Picture tab of the Object Properties dialog box, click Picture Effects > Contrast.
    • Image Contrast Settings: Open the Picture Effects tab of the Object Properties dialog box so that you can adjust the contrast level.
    Adjust Tone You can apply grayscale, black and white, and watermark effects to pictures and remove existing effects.
    Margin Expand an image collection of frequently used outside margins.
    • Picture Margin Settings: Open the Picture tab of the Object Properties dialog box so that you can edit Picture Margin settings.
    Edit with Photo Make clear, vivid, and gorgeous pictures with an easy and simple adjustment. With Photo, you can also make pictures transparent partially.
    Compress Pictures Compress the size of the pictures inserted into the document to reduce the size of the document as a whole. Applies to pictures included in the document only.
    • Apply Now to All Pictures: Immediately compress all the pictures included in the document.
    • Apply Now to Selected Pictures: Immediately compress selected pictures of the document.
    • Reduce Size Settings: Present various picture compression options.
    Change/Save Replace or save the selected picture.
    • Change Picture: Replace the selected picture from the edit screen with a picture selected from the list. Here, the inserted picture maintains the dimensions of the picture being replaced.
    • Save Inserted Picture: Save the picture files that are included in the document on the local system.
    Reset Picture Remove all effects from the picture and restore it to its original state.
  • Picture Styles
  • Picture Properties Display the Object Properties dialog box to modify properties of the selected picture such as size, location, rotation, etc.
    Picture Styles Present a wide variety of picture styles. Click More to open the Picture Style image collection, which consists of different line color, line type, and line weight combinations.
    Line Color Change the line color of the selected object.
    • Others: Open the Colors dialog box so that you can choose a different color or register custom colors in the color palette.
    Line Weight Change the line weight of the selected object. Click on the line weight icon to expand the line weight image collection, which consists of a variety of line weight options.
    • Other Lines: Open the Lines tab of the Object Properties dialog box so that you can choose a different line weight setting.
    Line Dashes Change the line type of the selected object. Click on the line type icon to expand the line type image collection, which consists of a variety of line type options.
    • Other Lines: Open the Lines tab of the Object Properties dialog box so that you can choose a different line type setting.
    Picture Effects Change the line type of the selected object. Click on the line type icon to expand the line type image collection, which consists of a variety of line type options.
    • Shadow: Applies a shadow effect to the object. You can specify the direction, size, slant, distance, angle, color, and transparency of the shadow.
    • Reflection: Applies a reflection effect to the object. You can specify the size of the reflector and the distance from the reflector.
    • Neon: Applies a neon effect to the object. You can specify the color, transparency, and size of the neon.
    • Soft Edge: Applies a soft edges effect to the object. The effects softens the edges of the object, and you can specify the size of the effect.
  • Arrange
  • Wrap Text Specify picture placement position.
    • In Line with Text: The picture object is treated as a regular text. Therefore, its position changes when you type or delete text.
    • Square: Place the picture object and the body text on the same lines without overlapping one another. The body text flows around the picture object.
    • Top and Bottom: The picture object occupies space by the amount of its height and the body text cannot enter the space.
    • Behind Text: Arrange the selected picture object to be placed in front of text.
    • In Front of Text: The body text gets filled as if there were no picture object. The picture object is used as the background of the body text.
    Bring Forward Bring to Front: Bring the object to the front when multiple objects are overlapped. If you click the drop-down arrow , the following menu is displayed.
    • Bring to Front: Bring the selected object to the front.
    • Bring Forward: Move the selected object one object forward.
    Send Backward Send the object to the back when multiple objects are overlapped. If you click the drop-down arrow , the following menu appears.
    • Send to Back: Send the selected object to the back.
    • Send Backward: Move the selected object one object backward.
    Align Specify the alignment or the space distribution between objects when multiple objects are selected. If you click the icon, the following menus appear.
    • Align: Select from Align Left, Align Center, Align Right, Align Top, Align Middle, and Align Bottom.
    • Distribute: Select either Distribute Horizontally or Distribute Vertically.
    Group Groups multiple objects into one or ungroups grouped objects. If you click the icon, the following menus appear.
    • Group: Groups multiple objects into one.
    • Ungroup: Ungroups objects when two or more objects are grouped.
    Rotate Rotates the selected picture object or flips it vertically or horizontally from its center line. If you click the icon, the following menus appear.
    • Rotate Right 90: Rotates the selected picture object by 90¡Æ clockwise./li>
    • Rotate Left 90: Rotates the selected picture object by 90¡Æ counterclockwise.
    • Flip Vertical: Flips the selected picture object vertically from its center line.
    • Flip Horizontal: Flips the selected Wordshop object horizontally from its center line.
    • Rotate Object: Rotates a picture object by a range of 1¡Æ through 360¡Æ.
  • Size
  • Crop Crops unnecessary portions from the picture.
    Height Sets picture object height.
    Width Sets picture object width.
    Fix Size To fix the size of the picture object, select Fixed Value for both Width and Height settings of the picture object.
    Equal Width Makes the width of the selected objects the same as the active object.
    Equal Height Makes the height of the selected objects the same as the active object.
    Equal Width/Height Makes the width/height of the selected objects the same as the active object.
