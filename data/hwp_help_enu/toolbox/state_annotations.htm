Notes Tab
 

References > New Note > Notes Tab

Notes tab is a collection of icons for various commands that are used to insert and edit notes. You can specify footnote/endnote separator lines, save notes, and delete notes with ease by using the Notes tab.

More Information

Notes Tab

  • Footnotes
  • Footnote/Endnote Styles Opens the Note Style dialog box.
    Previous Move to the previous note.
    Next Move to the next note.
    Save Saves the Footnote or Endnote from the body so that it can be opened at a later point in time. You can check footnotes that are split into the bottom of pages and end of a document by saving them as a file. You can also gather them and use for reference.
    Delete Delete notes at the cursor.
  • Note Number
  • Numbering Styles Specify the character for numbering footnotes and endnotes and its style. You can use numbers that increase automatically or designate symbols that indicate size by repeating a specific section or character over and over again as numbers. Symbols cannot be used in endnote content.
    • Numbering Styles: Opens the Footnote Styles tab of the Note Style dialog box so that you can select a different footnote number.
    Normal Display footnote/endnote numbers in normal size.
    Smaller Display footnote/endnote numbers in smaller size.
  • Position
  • Footnote Specify whether to display footnotes above footers or below the body text.
    • Above Footer: Positions the footnotes above the footer.
    • Below Text: Positions the footnotes below the text.
    Endnote Specify whether to display endnotes at the end of the document or section.
    • End of Document: Positions the endnotes at the end of the document.
    • End of Section: Positions the endnotes at the end of the section.
    Column Footnote Set the footnote position in columns.
    • Below Each Column: Positions the footnotes below each column.
    • Below Left Column: Positions the footnotes below the column.
    • Below Right Column: Positions the footnotes below the right column.
  • Separator
  • Separator Length Set the length of the separator between the body text and footnotes/endnotes.
    Set Separator Length Set the length of the footnote/endnote separator manually.
    Shape Outline Specify the color of the separator.
    Line Styles Set the type of the separator between the body text and footnotes/endnotes.
    Weight Set the weight of the separator between the body text and footnotes/endnotes.
  • Footnote¡êEndnote
  • All Footnotes to Endnotes Change all Footnote in the current file to Endnote. You cannot select this option in a document only with Endnote. If inserting footnotes in the body, specify where in the body to insert the footnotes.
    All Endnotes to Footnotes Change all Endnote in the current file to Footnote. You cannot select this option in a document only with Footnote.
    Footnote<->Endnote Change all Footnote to Endnote and all Endnote to Footnote in the current file. You can select this option in a document only when both Footnote and Endnote are present.
  • Close
  • Close Ends footnote editing and returns to edit body mode.
