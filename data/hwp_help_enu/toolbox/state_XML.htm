XML Tab
 

File > XML Document > XML Tab

XML tab is a collection of icons for commands used to create and edit XML documents. From the XML tab, you can perform XML-related tasks, such as create an XML template, open an XML template, create fields, delete fields, set field groups, and delete field groups with ease.

More Information

XML Tab

  • XML
  • Create XML Templates Opens the Create XML Templates dialog box. Provides the Create XML Documents feature, which is used to create the web standard documents known as XML documents.
    XML Templates Opens the XML Template dialog box. XML Templates are used to create frequently used XML templates as template files (*.hwt) so that they can be opened and used whenever needed. A folder containing the same type of template files is called a Template Package.
    Open XML Document Opens the Open XML Document dialog box. You can open the XML file you wish to work on by clicking Open XML Document.
    Save XML Document Opens the Save XML Document dialog box. Saves a Hwp document (*.hwp, *.hwt) as an XML file.
    Validate Runs a validation test after opening an XML file to verify its validity.
  • Field
  • Create Field Inserts fields into an XML template document, arranges them to suit your needs, and then designates the fields as a group so that you can use them over and over again. To reproduce or delete a group, you must first select a field belonging to that group.
    Repeat Field Creates the field inserted to the XML template repeatedly as the user clicks. The field is repeated in the direction set in Field Properties > Create Field Repeatedly.
    Change Field Type Changes the field type by the value of Field Properties > Field Type.
    Delete Field Deletes fields which were created repeatedly. Select fields to delete first.
  • Field Group
  • Set Field Group Sets field group to create a field group.
    Remove Field Group Removes field grouping. In order to remove field grouping, you must first select a field belonging to the group.
    Show Field Group Sets and displays the specified field group as a block.
    Repeat Field Group Creates the specified field group repeatedly according to the user specification. The field is repeated in the direction set in Field Properties > Create Field Repeatedly.
    Delete Field Group Deletes a field group. In order to delete a field group, you must first select a field belonging to that group.
