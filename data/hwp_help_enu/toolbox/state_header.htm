Header/Footer Tab
 

Insert > Header/Footer > Header/Footer Tab

Header/Footer tab is a collection of icons for commands used to insert and edit the header/footer. You can use the features like Previous, Next, Delete, Close Header and Footer with ease on the Header/Footer tab.

More Information

Header/Footer Tab

  • Header/Footer
  • Header Changes the format of a header that has been entered.
    • Header/Footer: Opens the Header/Footer dialog box so that you can insert a header or footer in a different format.
    Footer Changes the format of a footer that has been entered.
    • Header/Footer: Opens the Header/Footer dialog box so that you can insert a header or footer in a different format.
    Delete Deletes the current header/footer and return to the body text. When you click this button, the "Delete the current header (footer)?" message appears. Click Yes to delete the header. If you delete the header by accident, click Home > Undo right away to undo the change.
  • Insert
  • Date/Time Inserts the current system date as a text.
    Quick Parts Registers frequently used text and document properties for future use.
    Picture Inserts a picture file at the cursor position in the header or footer.
    Clip Art Inserts a Clip Art at the cursor position in the header or footer.
  • Navigation
  • Previous Moves to the previous header or footer from the active one. This button is disabled if there is no previous header/footer.
    Next Moves to the next header or footer from the active one. This button is disabled if there is no previous header/footer.
  • Close Header and Footer
  • Close Header and Footer Ends header/footer editing and returns to the edit body mode.
