Table Tools Tab
 

Insert > Table > Insert Table

Table tab is a collection of icons for various commands that are used to insert and edit tables. You can select table styles and edit tables with ease by using the Table tab.

More Information

Table Tab

  • Table Style
  • Table/Cell Properties From the Table/Cell Properties dialog box, you can specify table size, position, placement, and other basic properties.
    Options Specify whether to apply table options such as Border, Font/Paragraph, Cell Background, Grayscale, Header Row, Last Line, First Column, Last Column.
    Table Styles Presents a table style image collection that incorporates wide varieties of fill, line, and effect properties. Click More to open the Table Style image collection, which incorporates varieties of cell border color, cell shade color, cell background color, and cell border style combinations.
    Cell Background Color Specify the background color for the selected cell.
    Border Change border style of the table to the border type of the selected quick table.
    Cell Shading Increase/decrease the brightness of the cell block.
  • Draw Borders
  • Line Style Select the cell border style. There are 12 cell border styles to choose from.
    Line Weight Select the cell border weight. Cell border weight can be specified from the list only. Custom weight value is not supported.
    Pen Color Select the cell border color. Displays the borders of the cells that have been set as a cell block in the color selected from the color palette.
    Draw Table Draws rows/columns of the selected cell.
    Eraser Deletes rows/columns of the selected cell.
    Transparent Line Specify whether to display table and text gridlines on the screen.
  • Rows/Columns
  • Delete Delete rows or columns of an existing table.
    Insert Above Adds a row directly above the selected cell.
    Insert Below Adds a row directly below the selected cell.
    Insert Left Adds a column to the left of the selected cell.
    Insert Right Adds a column to the right of the selected cell.
  • Merge
  • Merge Cells Merges all cells within a cell block into one.
    Split Cells From Split Rows/Columns, specify the number of rows and columns and split the current cell or cell block. Specify the number of rows and columns for the current cell or cell block and split the cell or cell block accordingly.
    Split Cells into Rows Split cells at the cursor or in a cell block into rows.
    Split Cells into Columns Split cells at the cursor or in a cell block into columns.
    Split Table If a table you created extends over to the next page, you can click Table > Split Table at the appropriate location to split the table into two or more tables. Split Table splits the table beneath the row at the cursor position.
    Merge Table If you need to merge two or more tables into one, place the cursor in the first table and click Table > Merge Table. Merge Table can only merge two tables if there is a space and a line break (ENTER) between the two tables.
  • Cell Size
  • Distribute Rows Sets the same width for all cells in the cell block.
    Distribute Columns Sets the same height for all cells in the cell block.
  • Arrangement
  • Horizontal and Vertical Alignment Specify the vertical and horizontal alignment for the selected cell. Select from Align Top Justified, Align Center Justified, Align Bottom Justified, Align Top Center, Align Center, Align Bottom Center, Align Top Right, Align Center Right, and Align Bottom Right.
    Vertical Writing Changes the text arrangement in the designated cell area to vertical. Here, you can specify whether to arrange English characters and numbers vertically as well.
    • English Down: Apply vertical writing. Rotate English and numbers 90 degrees clockwise.
    • English Up: Apply vertical writing. Return English and numbers to their original state.
    Margins Display the frequently used outside margin and cell inside margin image collection.
    • Table Margin Settings: Opens the Margin/Caption tab of the Table/Cell Properties dialog box so that you can adjust the Distance from Text setting of the table.
  • Data
  • Sort Aligns the block in the vertical direction.
    Text to Table Convert the text in a block into a table.
    Convert to Text Remove table borders and convert the content into text.
    Formula You can use basic arithmetic functions such as add, subtract, multiply, and divide, as well as sheet functions such as sum and avg and formulas made up of left, right, below, and above range designators such as sum(left) in tables.
    • Block Formula: You can obtain Block Sum, Block Average, Block Product, and other calculation results after specifying a cell block within a table. Block Formula is useful for calculating the sum or average in a document containing a complete set of numerical data, such as a report card or an account book.
    • Quick Formula: Calculates the horizontal sum, vertical sum, horizontal average, vertical average, horizontal product, and vertical product of cells adjacent to the current cell. Place the cursor on the cell into which you want the calculation result to be inserted and select the desired formula.
    • Formula: You can use basic arithmetic functions such as add, subtract, multiply, and divide, as well as sheet functions such as sum and avg and formulas made up of left, right, below, and above range designators such as sum(left) in tables.
    AutoFill Select data from the default AutoFill list to insert it at the pointer. Check the AutoFill custom list that you registered, and add a new list or modify an existing AutoFill list.
    • Table AutoFill: Looks for rules from some of the cells in a table and automatically fills all selected cells in accordance with those rules.
    • AutoFill Content: Select data from the default AutoFill list to insert it at the pointer. Check the AutoFill custom list that you registered, and add a new list or modify an existing AutoFill list.
    Thousands Separator Inserts a comma (,) after each group of three digits for all numbers within the cell block area.
    • Insert Separator: Places a comma after each group of three digits for numbers contained inside a table.
    • Remove Separator: Removes thousands separators from numbers within the table.
