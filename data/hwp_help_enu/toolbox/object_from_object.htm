Form Controls Tab
 

Insert > Form Controls > Form Controls Tab

The Form Controls tab is enabled once an inserted form object is selected. The tab is used to edit the properties of an inserted form object, to insert a new form object, or to align a form object.

More Information

Form Controls Tab

  • Edit
  • Form Edit Mode Select the Form Controls command to load Form Edit Mode with all settings set to default. While you are in Form Edit Mode, you can insert Command Buttons, Check Boxes, List Boxes, Radio Buttons, Edit Boxes, and other form objects. When you are running form controls inserted in the document, turn off Form Edit Mode and select a form control.
  • Insert
  • Command Button Insert a command button (PushButton). Command button is a form control that runs the assigned script when clicked on.
    Check Box Insert a check box. A check box is a form object that is used to have the user make one or more selections in a group of items.
    Combo Box Insert a combo box. Insert a combo box. A combo box is form object that is used when a single value from a list of values is required to be selected and is accompanied by a list of values.
    Radio Button Insert a radio button (RadioButton). A radio button is a form object that is used when a single option from a group of options is required to be selected.
    Edit Box Insert an edit box (Edit). An edit box is a form object used for input and output of text and supports entry of simple text.
  • Task Pane
  • Properties Open the Form Control Properties task pane.
    From the Form Control Properties task pane, you can specify or edit internal properties of the selected form object, which includes font, size, and background color.
    View Code Open the Script task pane. From the Script task pane, you can insert scripts for all form objects that have been inserted into the body and run scripts. You can control new functions in Hwp by typing a script directly. Type a script by using JavaScript.
  • Placement
  • In Line with Text The object is treated as a regular text. Therefore, its position changes when you type or delete text.
    Fusion Place them so that the object and the body text share the same lines without overlapping each other and the body text flows around the shape object.
    Top and Bottom Objects occupy space by the amount of the shape object height and the body text cannot enter the space.
    On Top of the Text Arrange the selected object to be placed in front of text.
    Beneath the Text The body text gets filled as if there were no object. The shape object is used as the background of the body text.
  • Align
  • Bring Forward Bring to Front: Bring the object to the front when multiple objects are overlapped. If you click the drop-down arrow , the following menu is displayed.
    • Bring to Front: Bring the selected object to the front.
    • Bring Forward: Move the selected object one object forward.
    Send Backward Send the object to the back when multiple objects are overlapped. If you click the drop-down arrow , the following menu appears.
    • Send to Back: Send the selected object to the back.
    • Send Backward: Move the selected object one object backward.
  • Protect
  • Lock Object Select an object and lock it from being resized or moved. Make sure not to select the active object or an unwanted object by accident, when multiple objects are overlapped.
    • Lock Object: Lock objects so that important objects do not get edited or deleted by accident.
    • Unlock All Objects: To unlock all objects to which Lock Object is applied, select Unlock All Objects.
