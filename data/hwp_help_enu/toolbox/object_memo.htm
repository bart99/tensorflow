Comments Tab
 

Review > Comment

The Comments tab contains icons for commands that are used to insert and edit comments. You can change the comment background color, selected color, line color, line type, line weight, comment size, and other comment style settings by using the Comments tab.

More Information

Comments Tab

  • New Comment
  • Previous Move to the previous comment.
    Next Move to the next comment.
    Delete Delete the selected comment.
  • Show/Hide
  • Show Comment Show or hide comments on the screen.
    Comment Guidelines Click on the Comment Guidelines icon to hide or show comment guidelines.
    Comments Content Open a document containing comments and click Comments Content to load the comments window at the bottom and view the comments.
