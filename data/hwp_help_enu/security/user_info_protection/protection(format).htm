Select Pattern
 

File > Document Security > Protect Private Info > Search/Replace / Review > Replace Private Info > Search/Replace

This feature shows social security numbers, telephone numbers, birth dates, and other form of numerical information that can be used to identify individuals in a different format.

More Information

Select Pattern Dialog Box

Item

Provides telephone number, social security number, alien registration number, email, account number, credit card number, IP address, and birth date display pattern options.

Current Pattern

Displays the current pattern in use. If no pattern has been specified, the default pattern for each of the items is shown.

Pattern List

Displays a list of patterns for each of the items for you to choose from.

Select a pattern and click Set to apply it.

Set Button

Applies the selected patterns. Select a pattern for each of the items and click Set to apply all the selections at once.

Cancel Button

The current pattern selection is maintained.
