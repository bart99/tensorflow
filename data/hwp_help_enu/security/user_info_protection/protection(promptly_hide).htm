Replace in Place
 

Review > Replace Private Info > Replace in Place

You can change selected block content within the active document on the screen into chosen characters.

Replaced characters can be restored only by clicking Home > Undo. The Undo option will not be able to restore the characters after you exit from Hwp.

Steps

  1. Click File > New and enter data.
  2. Set the mobile phone number of Kim Hancom as a block.
  3. Click Review > Replace Private Info and select Replace in Place.
  4. The mobile number is replaced with ***.
  5. As the characters were simply replaced with ***, you can delete the change or replace it with other characters.

Note

Recover Changed Content

  • Unable to Recover Changed Content

  • Content that has been changed using Replace Private Info can only be recovered by clicking File > Undo while editing the document. If you close the document or exit from Hwp, Undo will not be able to recover the hidden content.

    To hide certain parts of the document or unhide hidden parts, you can use the Protect Private Info feature and encrypt the document.

    Protect Private Info hides/un-hides content through encryption. To recover content with ease, you should remember the password.
