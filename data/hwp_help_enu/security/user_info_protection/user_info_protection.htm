Replace Private Info
 

Review > Replace Private Info

You can use the Replace Private Info feature of Hwp to hide sections of the document by replacing them with designated characters.

Replace Private Info is useful for implementing character change in specified sections only instead of the entire document.

Text that has been changed through Replace Private Info can only be recovered by clicking Home > Undo.
However, Undo will not be able to recover the characters if you exit from Hwp.
