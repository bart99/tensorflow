Custom Settings
 

File > Document Security > Protect Private Info > Search/Replace / Review > Replace Private Info > Search/Replace

You can create custom settings in addition to the standard items of Phone, Social Security No., E-Mail, and Date of Birth.

More Information

Custom Settings Dialog Box

Item

By having Custom items besides default ones, you can add private info items you need.

Provide Phone, E-Mail, and Credit Card No. as the default values of Item.

Format

On top of default formats provided in each item, you can add formats you need.

You can use a hyphen (-), space, or comma (,) as the separator between numbers.

Name Description Example
Phone Type a phone number format and click Add to add it to Search List.

Search List

If you set and add Item and Format, they are displayed on Search List.

  • Details on Icons
  • Icon Name Shortcut Description
    Add Insert Add the format typed in the Format edit box to Search List.
    Delete Delete Select a format from Search List.

Default Button

Delete all the added formats and revert back to the initial values.

Close Button

Close the Custom Settings dialog box.
