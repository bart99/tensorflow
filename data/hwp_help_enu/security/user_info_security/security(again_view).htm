Remove Protection
 

File > Document Security > Protect Private Info > Remove Protection

You can select this option to unhide password-protected information in the current document.

You can enter the password while working on a Protect Private Info-enabled document or after opening a Protect Private Info-enabled document, and then click Remove Protection to unhide hidden information.

If you click Remove Protection after opening a new document or without entering the password, you will first be prompted for the password before the hidden information is displayed.

Steps

  1. Open a Protect Private Info-enabled document.
  2. Click File > Document Security > Protect Private Info > Remove Protection.
  3. From the Private Info Protection dialog box, enter the correct password in Current Password and click OK.
  4. The dialog box will close, and information that was replaced with **** is displayed again.

Note

Password Entry Precautions

  • Keyboard Classification

  • For creating passwords, you can use all the keys on the keyboard regardless of the keyboard type. You can mix English letters, numbers, special characters and SPACEBAR on the keyboard.

  • Password Decoding Impossible

  • Even if you forget the password of a document, no one should be able to modify, remove, or decode the user-defined password.
