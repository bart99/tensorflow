Search/Replace
 

File > Document Security > Protect Private Info > Search/Replace

You can search the active document for format selected by the user in Private Info Options and replace the found instances with characters specified by the user. The feature uses password encryption so that private information found using Search/Replace remains protected.

Steps

  1. Click File > New and enter data.
  2. Click File > Document Security > Protect Private Info > Search/Replace.
  3. From the Protect Private Info dialog box, select Phone and E-Mail from Private Info Options and click Protect All.
  4. From the Set Protection Password dialog box, enter a password made up of 5 characters or more in Set Protection Password.
  5. In Confirm Password, type the same password again to record the correct password.
  6. Click Set. The security password is applied to the document, and all instances of Phone and E-Mail within the document are replaced with ***.
  7. Point over a section that has been changed to ***. The pointer changes to a lock which indicates that the section cannot be edited.

More Information

Search/Replace Dialog Box

Private Info Options

Select Private Info Options in the document.

Phone, E-Mail, Credit Card No., IP Address, and Date of Birth are provided by default. If you require additional patterns, register them under Custom Settings. The registered items can then be selected from Private Info Options > Customize.

If you select Others, you can hide characters by typing them in the edit box on the right.

  • Details on Icons
  • Icon Name Description
    Custom Settings Create custom settings in addition to the basic items such as Phone, Social Security No., and E-Mail.
    Select Pattern Show social security numbers, telephone numbers, birth dates, and other form of numerical information that can be used to identify individuals in a different format.

Protected Character Selection

Select characters to replace the search result of Private Info Options.

Name Description
****** Replace the search result of Private Info Options with ***. The default value is selected.
~~~~ Replace the search result of Private Info Options with ~~~.
XXXX Replace the search result of Private Info Options with XXX.
Custom Text Replace the search result of Private Info Options with custom text. You can type custom text or insert it from Symbol.

Find Button

Find the selection of Private Info Options one by one in the document and display it as a block, every time you click Find. If you click once, the name of the button changes to Find Next.

Protect Button

If you click Protect, the format of characters that match the selection in Protected Character Selection is changed.

Protect All Button

If you click Protect All, all formats that match the selection in Private Info Options are changed. When Protect All is complete, a message prompts you with information of how many replacements were made.

Close Button

Complete the process of Protect Private Info and close the Protect Private Info dialog box.

As Protect Private Info is used repeatedly in many cases, it stays on the screen until you close the dialog box by clicking Close.
