Change Password
 

File > Document Security > Protect Private Info > Change Password

You can change the current password of the document to a new password.

Steps

  1. Click File > Open and open a Private Info Protection-enabled document.
  2. Click File > Document Security > Protect Private Info > Change Password.
  3. From the Change Protection Password dialog box, enter the correct password in Current Password.
  4. In New Password, enter a new password made up of 5 characters or more.
  5. In Verify, type the same password again to record the correct password.
  6. To apply the new password to the document, click Change.
  7. To save the document with the new password setting, click File > Save.

More Information

Change Protection Password Dialog Box

Current Password

Type in the current protection password (5 characters or more) of the document.

New Password

Type in a new private info password (5 characters or more).

Verify

Type in the private info password you typed in Verify.

Caution

Document

Inserting a Private Info Protection-enabled document into a document without Private Info Protection does apply the password protection to the unprotected document. If you require Private Info Protection, you need to enable it separately.
