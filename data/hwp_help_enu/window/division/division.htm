Split Window
 

View > Split Window

This feature allows you to split the edit screen and work on a document while viewing two or four different sections of the document at the same time.

You can reference and move to different parts of a document and even copy content from different sections of the document with great ease by splitting the screen.

Steps

To split horizontally

  1. Click View > Split Window > Split Horizontally.
  2. The edit screen is split horizontally, and the cursor flashes on the bottom screen.
  3. To return the edit window to its original state, click View > Split Window > Remove Split.

To split vertically

  1. Click View > Split Window > Split Vertically.
  2. The edit screen is split vertically, and the cursor flashes on the right screen.
  3. To return the edit window to its original state, click View > Split Window > Remove Split.

To split to 4 windows

  1. Click View > Split Window > Split to 4 Windows.
  2. The edit screen is split vertically and horizontally into four screens, and the cursor flashes on the lower right screen.
  3. To return the edit window to its original state, click View > Split Window > Remove Split.

To remove split

  1. With the screen split vertically and/or horizontally, click View > Split Window > Remove Split.
  2. A selection mark is displayed in front of the Remove Split option to indicate that the edit screen is now a single screen.

Note

Resizing Split Screens

Splitting the edit screen results in either two or four split screens. Therefore, the sizes of the split screens are identical. You can enlarge or reduce the size of the selected screen by dragging on the border of a split screen.

View in Split Screen Mode

After splitting the screen, you can apply different Zoom Ratio and Hard Return, Control Codes, Transparent Line, Page Layout, and other show/hide settings to each split screen.

Maintaining Split Screen Mode

Clicking File > New or File > Open while in split screen mode removes the split and opens the document in a single window.

On creating a new document, the zoom and hard return, control code, transparent line, ruler, page layout, and other show/hide settings of the split screen at the cursor position are applied to the new document.

Removing Split

To remove split after splitting the screen into two, double-click on the vertical or horizontal split boundary. The split is then removed. Alternatively, dragging the boundary to the left/right or top/bottom of the screen also removes the split.

To remove split after splitting the screen into four, double-click on the point where the vertical boundary and the horizontal boundary intersect.

After Removing Split

After removing the split, the Zoom Ratio and Hard Return, Control Codes, Transparent Line, Page Layout, and other show/hide settings of the split screen at the cursor position are applied to the restored screen.

Shortcut

To move between split screens, use the following shortcuts:

Move Screen: CTRL+W,N

Move Order:

1 2
3 4

Advanced Usage

Drag & Drop

To move a section of the body to a different location or copy a section of the body while in split screen mode, show the source content in one screen and the target location in the other screen. Set the applicable section as a block, drag it across the split line, and drop.
