Cascade
 

View > Arrange Window > Cascade

You can arrange all open windows in a cascading configuration.

Steps

  1. Click View > Arrange Windows > Cascade.
  2. All open document windows are arranged in a cascading configuration.

Note

Window Arrangement Order

You can cascade the document windows from the back to front in the order of first opened document to last opened document, positioning the current window at the very front by clicking View > Arrange Windows > Cascade when multiple windows are open.

Applying to All Windows

Cascade functions in the same manner even when the document window is maximized. Clicking Cascade changes all windows including the ones hidden behind the maximized one, into the same size and arranges them in a cascading configuration.

However, minimized windows remain minimized.
