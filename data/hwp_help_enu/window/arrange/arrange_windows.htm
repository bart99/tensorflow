Arrange Window
 

View > Arrange Window

There is no limit to the number of document windows that can stay open simultaneously in Hwp, and each document window operates like an independent application program.

Having too many document windows, however, can become confusing and inconvenient. In such cases, you can use the Arrange Vertically, Arrange Horizontally, and Cascade options to reorganize the document windows in a structured manner.

Steps

  1. Click View > Arrange Window > Arrange Horizontally.
  2. All open windows as horizontal tiles are arranged.

Note

Document Tab Arrangement

Arranging document windows does not arrange the tabs contained in those windows. The document windows are arranged based on the active document tab of the document windows.

Minimized windows are excluded

Minimized document windows are excluded from window arrangement. Windows minimized as icons remain as icons.

Arrange Selected Windows Only

Window arrangement applies to all document windows that are open. However, you can use the SHIFT and CTRL keys in the Window List dialog box to select specific windows and arrange them vertically or horizontally.
