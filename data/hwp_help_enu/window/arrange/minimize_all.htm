Minimize All
 

View > Arrange Window > Minimize All

You can minimize all open document windows into document icons on the taskbar. The foremost opened window will go to the leftmost position.

Steps

  1. Click View > Arrange Windows > Minimize All.
  2. All document windows that are open are minimized into icons on Windows taskbar.

Note

Restoring to previous size

To restore a minimized window, click on the icon of the concerned window. The file is restored as a document window by clicking on the icon.

Excluded from Window Arrangement

Arranging document windows vertically, horizontally, or in a cascading configuration does not apply to minimized document windows. Windows minimized as icons remain as icons.
