Arrange Horizontally
 

View > Arrange Window > Arrange Horizontally

You can fit all open windows to screen and display them in a tiled horizontal arrangement.

Steps

  1. Click View > Arrange Window > Arrange Horizontally.
  2. All open windows as horizontal tiles are arranged.

Note

Window Arrangement Order

Clicking View > Arrange Window > Arrange Horizontally when only one window is open maximizes the current window to fit the screen.

You can arrange the current window at the top left and tile the remaining windows from there on in the order of last opened window to first opened window by clicking on View > Arrange Window > Arrange Horizontally when multiple windows are open.

Arrange Selected Windows Only

View > Arrange Window > Arrange Horizontally applies to all document windows that are open. However, you can use the SHIFT and CTRL keys in the Window List dialog box to select specific windows and arrange them vertically or horizontally.

Caution

Cannot Arrange Document Tabs

Arrange Horizontally arranges all document windows that are open. Document tabs contained within a document window cannot be arranged horizontally.
