Open Windows List
 

The file names and paths of up to nine windows that are currently open are displayed at the very bottom of the View menu.

Steps

  1. Select the View menu.
  2. A list of file names and paths of windows that are currently open is shown at the very bottom of the View menu.
  3. Clicking on a document from the list displays that document window on screen.

  In the open windows list, the file with the selection mark displayed in front of its name is the active window.

Note

Using Window List

A list of up to nine windows that are currently open is displayed in the View menu. To view a complete list of open windows if there are more than ten open windows, click View > Window List.

Viewing Tabbed Documents

In Hwp, multiple tabbed documents in a single document window are displayed as individual entries in the open windows list.
