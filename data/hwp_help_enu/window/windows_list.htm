Window List
 

View > Window List

You can view a list of all currently open document windows, switch to a different window, and arrange the windows vertically or horizontally.

Steps

  1. Click View > Switch Windows.
  2. From the Window List dialog box, select a window that is to be worked on and click Switch.
  3. The selected window is moved to the front.

More Information

Window List Dialog Box

Window List

The file names and paths of all open document windows are displayed on a list.

Details on Icons

Icon Name Description
Save Executes File > Save and saves the selected window list.
  • You can select and save multiple window lists at the same time by using SHIFT and CTRL.
  • The Window List dialog box remains on screen after saving the selected document.
Arrange Horizontally Arrange all the selected windows as horizontal tiles that fit the screen.
  • You cannot arrange the windows horizontally if you select two or more document tabs in the same document window.
  • The Window List dialog box remains on screen after performing Arrange Horizontally.
Arrange Vertically Arrange all the selected windows as vertical tiles that fit the screen.
  • You cannot arrange the windows vertically if you select two or more document tabs in the same document window.
  • After performing Arrange Vertically, the Window List dialog box remains on screen.

Switch Button

Switches over to the selected document and closes the Window List dialog box. You cannot switch windows if you select two or more window lists.

Note

Documents opened as Tabs

A list of all the tabs in the document window is displayed in Window List if the document window contains two or more document tabs. However, if you select two or more document tabs in the same document window, you cannot arrange them horizontally or vertically.

In the Window List dialog box, the ¦¦ symbol is displayed in front of the name and path of documents that were opened as tabs so that you can distinguish them from documents that were opened as windows.

Using Window List

The list of windows that are currently open displays up to nine windows under the View option. To view a complete list of windows when there are more than ten open windows, click View > Switch Windows.

From the Window List dialog box, you can switch over to different windows and perform Save, Arrange Horizontally, and Arrange Vertically with ease.

You can also select multiple window lists to save or arrange them vertically/horizontally by using SHIFT and CTRL.

Caution

Restriction

You cannot switch windows if you select two or more window lists. You cannot arrange the windows horizontally or vertically if you select two or more document tabs in the same document window.
