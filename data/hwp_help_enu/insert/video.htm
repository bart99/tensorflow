Video
 

Insert > Illustrations > Video

With a video file in the document, you can use the document as multimedia contents and it helps you create more vivid document.

You can insert a video with two ways; embed a video file and link to a video file. If you embed a video file, you would not be worried about loosing the embded video file and you do not need to back it up in another storage. When you have file size limit, you can link to a video file without embedding it. Also, you can play a online video in your document by using its source code.

Steps

Embed a video from local drive

  1. Click Insert > Illustrations > Video.
  2. In the Insert Video dialog box, click the Select Video File icon in Local Video > File Name.
  3. In the Insert Video dialog box, select the path and name of a video file (*.mpg, *.avi, *.asf, *.wmv, *.mp4) and click Open.
  4. The path and name of the selected file will be automatically displayed in the Local Video > File Name box of the Insert Movie dialog box.
  5. Select Embedded and click Insert.

Link to a video from local drive

  1. Click Insert > Illustrations > Video.
  2. In the Insert Video dialog box, click the Select Video File icon in Local Video > File Name.
  3. In the Insert Video dialog box, select the path and name of a video file (*.mpg, *.avi, *.asf, *.wmv, *.mp4) and click Open.
  4. The path and name of the selected file will be automatically displayed in the Local Video > File Name box of the Insert Movie dialog box.
  5. Click Insert. Make sure not to select Embedded.

Link to a video from web site

  1. Click Insert > Illustrations > Video.
  2. In the Insert Video dialog box, click Web Video.
  3. Go to a web site and locate a video that you want to link to.
  4. Copy source code or embed code in the web site. Some codes can be copied from the context menu with the right click.
  5. Paste the copied source or embed code in the Video Tags field of the Insert Movie dialog box.
  6. Click Insert.

  How to copy source code may be different from each web sites and some videos may not provide source code.

  To prevent possible problems with a video copyright, you should check before distributing your document with a video.

Note

Some videos may not be available due to OS

Some videos which cannot be played in your OS may not be inserted in your document correctly.
