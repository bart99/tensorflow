Insert Caption
 

References > Insert Caption

Number, title, and short description are added to pictures, tables, text boxes, drawing objects, and equations inserted in the body text, if needed. The title added to an object is called caption.

References > Insert Caption is a function that inserts or modifies captions. Captions are useful because once inserted, they always follow their objects even if the objects are moved.

Steps

  1. Click Insert > Illustrations > Picture to insert a picture.
  2. Select a picture to insert a caption and click References > Captions > Bottom.
  3. When Picture 1 automatically displays at the bottom of the picture, type the title of the picture and press SHIFT+ESC to return to the body text.
  4. While the picture is selected, click Home > Object Properties.
  5. In the Object Properties dialog box, select the Margin/Caption tab.
  6. Select Insert Caption > Position > Top and click Set.
  7. A caption will be displayed on the top of the picture.

More Information

Margin/Caption Tab of Object Properties Dialog Box

Caption

  • Position

  • The Caption content will be inserted outside the borders of the object.

    Select the caption position from Top, Bottom, Upper Left, Center Left, Lower Left, Upper Right, Center Right, and Lower Right.

  • Size

  • If the caption position is set to Top or Bottom, caption size remains the same as the width of the object. If the caption position set to Left or Right, you can specify the size of the caption.

  • Distance from Object

  • You can specify the distance between the object and caption content. The default value is 3mm. You can use a negative value in Distance from Object because Hwp supports negative values for the distance between the caption and the object.

  • Include Margin

  • If the outside margin is specified for the object, the caption starts from the object area excluding the specified margin. The caption starts from the left or right outside margin if you select this option.

  • Single Row

  • Everything you type in the caption area will be in one line even if the pointer reaches the right boundary as you type. if you select this option.

Note

Insert Numbering Again

Numbering[number] appropriate for the type of the current object will automatically display if you use Insert Caption. You can delete such number by pressing DELETE or BACKSPACE.

Right click the pointer to open the context menu if you want to reinsert the number that you deleted when typing a caption. The numbering[number] will be reinserted at the pointer if you select Insert Number.

Remove Caption

The caption area remains without content if you delete everything in the caption area by pressing DELETE or BACKSPACE and return to the body text. The pointer goes inside it if you click the caption area.

The caption will be completely deleted if you right click the caption area and use No Caption in the content menu.

Modify Caption

To modify a caption, select its object and click References > Captions > Bottom again, or click the caption area and modify content inside it.

Group/Ungroup

The caption of the object will disappear if you use Group to combine an object that has a caption with another object. The caption will also disappear if you add a caption to grouped objects and use Ungroup.

Numbering Type

In Hwp, drawing objects and pictures are numbered as Picture, tables are numbered as Numbering Type-Table, and equations are numbered as Numbering Type-Equation.

When you want to change the numbering type of an object, select from None, Picture, Table, and Equation in Others > Numbering Type in the Home > Object Properties > Basic tab.

Caution

Caption Settings

For an object without a caption, you cannot set caption by clicking Home > Object Properties > Margin/Caption.
