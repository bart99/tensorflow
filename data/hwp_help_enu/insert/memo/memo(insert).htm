Comment
 

Review > Comment > Insert Comment

Inserting a comment on a specific word or text block within the current document creates an editable comment box on the right side of the page.

Steps

  1. Position the cursor on the word you wish to insert a comment for.
  2. Click Review > Comment > Insert Comment.
  3. A comment box is created to the right of the selected word.
  4. Type information inside the comment box and click anywhere on the edit window to exit from the comment box.

Note

Comment Style

To change the size, border, border color, border type, border weight, and comment color settings of an inserted comment box, click Review > Comment > Comment Style.
