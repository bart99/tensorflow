Comment
 

Review > Comment

With this feature, you can attach a comment containing important information to a specific word or text block within the current document.

Note

Insert Comment

You can attach short messages to specific words and text blocks as comments by using Insert Comment.

Click Insert Comment from the current cursor position to insert a comment on the word at the cursor position.

To create a single comment for multiple words, set the words as a block using F3 and then click Review > Comment > Insert Comment. A single comment is inserted for the selected words.

Comment Style

Comment Style is an option used to specify comment box width, guideline, border color, border type, border weight, background color, and selected color settings.

To apply different comment style settings to different sections, set Apply to to New Section instead of Whole Document.

Comment Print

To exclude comments from being printed, click Print > Advanced > Print Options and uncheck the Comments option.

The document will be printed without the comments.

Inserting Comment for Table Content

It is possible to insert comments for content inside a table. Set the word you wish to attach a comment to as a block using the F3 key and then click Review > Comment > Insert Comment.

An editable comment box appears to the right of the selected word.

Inserting Object Comments

It is possible to insert comments for objects such as pictures and Wordshops.

  1. Select an object and check the In Line with Text under Object Properties > General.
  2. Set the object as a block using the F3 key and then click Review > Comment > Insert Comment.
  3. An editable comment box appears to the right of the selected word.
