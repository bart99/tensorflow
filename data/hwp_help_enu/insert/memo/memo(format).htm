Comment Style
 

Review > Comment > Comment Style

Comment Style is an option used to specify comment box width, guideline, border color, border type, border weight, background color, and selected color settings.

Steps

To change comment box size

  1. Open a document containing a comment.
  2. Click Review > Comment > Comment Style or click on the Styles group name under the Comments tab.
  3. From the Memo Style dialog box, change Width from 55mm to 70mm under Size and click Set.
  4. From the edit window, verify that the comment box size has been changed.

More Information

Memo Style Dialog Box

Size

  • Width

  • You can specify the width of the comment box. Maximum comment box width allowed is 210mm.

Borders

  • Color

  • You can specify the color of the comment box border and the guideline.

    You can click the color table to display the color palette. In the color palette, select the desired color. If the color palette does not have the desired color, click Others and specify a color or click Spuit and pick a color from a different application program.

  • Type

  • You can specify the comment box border and comment guideline border. Select No Lines or choose one of the 11 available border types.

  • Weight

  • You can specify the weight of each border type. Select from the list only. Custom weight value is not supported.

Memo Color

  • Background Color

  • You can specify the comment Background Color.

    You can click the color table to display the color palette. In the color palette, select the desired color. If the color palette does not have the desired color, click Others and specify a color or click Spuit and pick a color from a different application program.

  • Selected Color

  • Specify the Selected Color.

    You can click the color table to display the color palette. In the color palette, select the desired color. If the color palette does not have the desired color, click Others and specify a color or click Spuit and pick a color from a different application program.

Apply to

You can specify the application range of selected comment box size, border color, border type, border weight, background color, and selected color settings. Comment style settings can be applied by individual sections or to the entire document.

Normally, available choices are Whole Document and New Section.

  • Current Section

  • If the document is divided into two or more sections, Current Section, which applies the specified comment style to the current section, is available as an option.

  • Whole Document

  • It applies the current memo style to the entire document even if the document is divided into multiple sections with different Memo Style settings.

  • New Section

  • It divides the document into a new sections by breaking the page at the cursor position and applies the new comment style settings.
