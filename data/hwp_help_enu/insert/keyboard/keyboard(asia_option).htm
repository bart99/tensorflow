Language Options
 

Review > Keyboard > Language Options

If the current language of the keyboard is Japanese or Chinese, you can specify settings including how to convert characters.

Steps

To set confirmed character input format on Japanese keyboard

  1. Click Review > Keyboard > Change Keyboard.
  2. In the Keyboard Settings dialog box, select the Change Keyboard tab.
  3. Select Japanese > Hiragana for Current Keyboard and click Set.
  4. After the keyboard is set to Japanese > Hiragana, click Review > Keyboard > Language Options.
  5. In the Keyboard Settings dialog box, select the Confirm tab.
  6. Select Chinese (Yomigana) from Confirmed Character Input Format.
  7. Select Hiragana from Yomigana Character Type and click Set.
  8. When you type Japanese phonetically in English, text changes according to the settings in the Language Options dialog box.

More Information

Keyboard Settings Dialog Box

If the current language of the keyboard is Japanese or Chinese, you can specify how to convert characters. You can click Review > Keyboard > Language Options only when the Chinese or Japanese keyboard is selected.

When you type with the Chinese keyboard, only the Conversion tab is displayed. However, when you type with the Japanese keyboard, both the Conversion tab and the Confirm tab are displayed.

Conversion tab (Chinese keyboard)

Specify details including how to convert characters.

  • Conversion

  • Specify the character conversion method.

    • Manual Conversion

    • When you type a sentence and press SPACEBAR, the sentence changes to Chinese characters according to the pronunciation of the sentence entered.

    • Auto Conversion

    • Even if you do not press SPACEBAR when you type a sentence, the sentence automatically changes to Chinese characters that correspond to it.

  • Show Candidate Window

  • Display a candidate window with the same pronunciation so that you can convert the typed pinyin to its corresponding Chinese characters.

  • Show Candidate Number

  • Display numbers on the candidate window. You can select the candidate without pressing arrow keys or SPACEBAR by pressing a candidate number on the candidate list.

Conversion tab (Japanese keyboard)

    • Semi-Auto Conversion

    • When the standard text  is found when you type, the typed characters change to Chinese characters that correspond to .

  • Standard Text

  • You can specify the standard text if Semi-Auto Conversion is selected under Conversion. The default value is .

  • Show Candidate Window

  • Display a candidate window with the same pronunciation so that you can convert the typed to its corresponding Chinese characters.

  • Show Candidate Number

  • Display numbers on the candidate window. You can select the candidate without pressing arrow keys or SPACEBAR by pressing a candidate number on the candidate list.

  • Remember Conversion Result

  • Display the conversion candidates previously selected on top when the user enters a previously-converted sentence.

Confirm tab (Japanese keyboard)

You can select the location of readings, phonetic type, and spacing for Chinese characters.

  • Confirmed Character Input Format
    Input Format Description Example
    Confirm Character (No Phonetics) Display Chinese characters without their readings.
    Chinese (Yomigana) Display readings after Chinese characters.
    Yomigana (Chinese) Display readings and Chinese characters in parentheses after the readings.
    Superscript Yomigana Display readings above Chinese characters. This is the default way in Japanese to display readings of Chinese characters.
    Subscript Yomigana Display readings below Chinese characters.
  • Yomigana Character Type

  • You can select Hiragana or Katakana as a method of displaying readings of Chinese characters.

  • Auto Spacing

  • Automatically puts a space after punctuation marks like period and question mark.
