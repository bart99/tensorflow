Keyboard Settings
 

Review > Keyboard > Change Keyboard

Show All

In the Keyboard Settings dialog box, you can select the keyboard type to use and change settings for the keyboard on the screen. You can also specify the location of the Keyboard toolbar and set showing or hiding options.

Steps

To show or hide Keyboard toolbar

  1. Click Review > Keyboard > Change Keyboard.
  2. In the Keyboard Settings dialog box, select the Toolbox/Icon tab.
  3. Clear the Show Toolbox on Desktop check box and click Set.
  4. From the desktop the Keyboard toolbar disappears.
  5. Click Review > Keyboard > Change Keyboard again.
  6. In the Keyboard Settings dialog box, select the Show Toolbox on Desktop check box in the Toolbox/Icon tab and click Set.
  7. The Keyboard toolbar appears on the desktop again.

More Information

Keyboard Settings Dialog Box

Change Keyboard Tab
  • Current Keyboard

  • Displays the type of the currently used keyboard.

  • Keyboard 1

  • The keyboard that is assigned to Left SHIFT+SPACEBAR. You can assign up to two keyboards to Left SHIFT+SPACEBAR.

  • Keyboard 2

  • The keyboard that is assigned to Left SHIFT+SPACEBAR. You can assign up to two keyboards to Left SHIFT+SPACEBAR.

  • Switch Keyboard 1/2

  • If you press Left SHIFT+SPACEBAR, you can switch between assigned keyboards.

  • Keyboard 3

  • The keyboard that is assigned to Right SHIFT+SPACEBAR. You can assign up to two keyboards to Right SHIFT+SPACEBAR.

  • Keyboard 4

  • The keyboard that is assigned to Right SHIFT+SPACEBAR. You can assign up to two keyboards to Right SHIFT+SPACEBAR.

  • Switch Keyboard 3/4

  • If you press Right SHIFT+SPACEBAR, you can switch between assigned keyboards.

View Keyboard Layout Tab

Set items to display on the keyboard when you click Review > Keyboard > View Keyboard Layout.

  • Keyboard Size

  • Select from Smaller, Normal, and Larger.

    If you select Smaller, the base position, SHIFT-modified keys, and key properties are not displayed.

  • Show Base Position

  • Displays the English keyboard on other language keyboards including the Korean keyboard.

  • Show Shift-Modified Key

  • Displays the result of typing with the SHIFT key pressed.

Toolbox/Icon Tab
  • Show Icon in System Tray

  • Displays an icon on the Windows taskbar that indicates the current keyboard type.

  • Show Toolbox on Desktop

  • Displays a keyboard-related toolbar on the lower right of the screen.

  • Toolbar Items

  • Select or clear Current Language, Fullwidth/Halfwidth, Language Options, Current Keyboard, Change Keyboard, View Keyboard Layout, Options, and Help from the list to show or hide the function on the toolbar. At least one icon must be selected on the Keyboard toolbar.

    The default icons, Current Language, Current Keyboard, Change Keyboard, View Keyboard Layout, Language Options, and Help are displayed in the given order.

Others Tab
  • Enter Modern Hebrew

  • Uses the modern Hebrew keyboard. Modern Hebrew typing is the same as Hebrew typing in Microsoft products.
