View Keyboard Layout
 

Review > Keyboard > View Keyboard Layout

This feature displays the layout of the currently selected keyboard. You can enter text while viewing the layout of the currently selected keyboard by using this function.

Steps

To view keyboard layout

  1. Click Review > Keyboard > View Keyboard Layout.
  2. A keyboard appears on the screen. It displays the layout of the current keyboard.
  3. Enter text with the keyboard or mouse by viewing the key layout.
  4. Click the Close button on the right side to close the keyboard.

Note

Keyboard on Screen

If you click Review > Keyboard > View Keyboard Layout or click the View Keyboard Layout icon on the Keyboard toolbar of the desktop, a keyboard appears on the screen.

Change Keyboard

You can switch the keyboard by clicking the KO/EN key on the screen keyboard or pressing Left SHIFT+SPACEBAR. If you press ALT+F2, the Change Keyboard tab of Keyboard Settings is displayed and you can select any language or keyboard type.

Caution

Restriction

You cannot click View Keyboard Layout if you specify the keyboard as Windows Keyboard in the Change Keyboard tab of Keyboard Settings or select Use Windows Keyboard by clicking the Current Language icon on the Keyboard toolbar.
