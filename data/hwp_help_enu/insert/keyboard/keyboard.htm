Keyboard
 

Review > Keyboard

In Review > Keyboard, you can change the keyboard settings or perform keyboard-related functions.

If you use Change Keyboard, you can assign up to four frequently used keyboards to Left/Right SHIFT+SPACEBAR to change keyboard easily, however, you cannot switch between the keyboards when Windows Keyboard is set as the current keyboard. By using View Keyboard Layout, you can enter text when viewing the key layout.

Create Keyboard enables you to create a custom keyboard with the characters and symbols you want by using the Hancom KeyLayout program.

Note

Keyboard Toolbar

When you start Cell, Cell Keyboard toolbar appears on the desktop.

On the Keyboard Toolbar, you can change the language and type of the current keyboard and also display the keyboard on the screen.
