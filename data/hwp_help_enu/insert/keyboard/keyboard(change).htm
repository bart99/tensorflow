Change Keyboard <ALT+F2>
 

Review > Keyboard > Change Keyboard

You can select a keyboard to use. The currently selected keyboard is displayed on the Keyboard toolbar or on the Windows taskbar.

Steps

To change Keyboard

  1. Click Review > Keyboard > Change Keyboard.
  2. In the Keyboard Settings dialog box, select the Change Keyboard tab.
  3. Select English for language and Dvorak for keyboard in Current Keyboard, and then click Set.
  4. Now you can use the English Dvorak keyboard instead of the default keyboard.

Note

Current Keyboard

Displays the type of the currently used keyboard. If you change the Current Keyboard without assigning a keyboard, the keyboard changes back to the assigned keyboard instead of the currently used keyboard when you press the keyboard switch key.

Keyboard Assignment

You can assign different keyboards to Left SHIFT+SPACEBAR and Right SHIFT+SPACEBAR. You can assign up to two keyboards to each key combination. This means that you can assign up to four keyboards for frequent use.

The default keyboard is Keyboard 1 assigned to Left SHIFT+SPACEBAR.

ü in Pinyin Input

If you cannot input ü using SCIM, IME, or other keyboards, type the v key for ü. You can mark ü with a tone symbol adding 1, 2, 3, or 4 to v to apply four tones: ǖ, ǘ, ǚ, ǜ. nv or nv3 for าณ(nü, nǚ), lv or lv4 for 绿(lǜ), or other Chinese characters requiring ü can be applied typing v which is not used in pinyin.
