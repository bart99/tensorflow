Create Keyboard
 

Review > Keyboard > Create Keyboard

The Hancom KeyLayout program allows you to create a custom keyboard with the characters and symbols of choice and use it by registering it in Hwp.

The program speeds up your typing by creating a custom keyboard with frequently used symbols.

Steps

To change key layout
  1. Click Review > Keyboard > Create Keyboard.
  2. In the Hancom KeyLayout program, select English in the Language list.
  3. Click the key to change on the layout and type a character you want.
  4. After you change the key layout, click the Save icon .
  5. In the Save As dialog box, enter text in Save in, File Name, Keyboard Name, and Name in English, and then click Save.
    • The default saving location is the \User\Shared80\Him folder under the User Data folder. You can check the location of User Data folder by clicking Help > About Thinkfree Office NEO Hwp > Folder Info.
    • Custom keyboards are saved as UKL files (*.ukl).

More Information

Hancom KeyLayout Program

The Hancom KeyLayout program consists of icons for main functions, settings for Language, Default Keyboard, and Input Direction, and a custom keyboard.

Toolbox Icon Description

Icon Name Description
New Keyboard Create a new keyboard as required by the user.
Open Open an existing keyboard file (*.ukl).
Save Save the currently working keyboard.
Save As Save the currently working keyboard with a different name.
Register Register the currently working keyboard.
About Display the information about Hancom KeyLayout.
Exit Finish the keyboard creating task and exit Hancom KeyLayout.
Help The Hancom KeyLayout Help dialog box appears.

Select the base language for a new keyboard. You can select from English, Korean, Japanese, Simplified Chinese, Traditional Chinese, Other Language, and Special Symbol.

Input Direction

The Input Direction menu is available only for the English keyboard. It specifies whether to type text from left to right or from right to left.

Custom Keyboard

When you select a language and a default keyboard from the Language and Default Keyboard lists respectively, the key layout of the keyboard appears.

You can easily create a keyboard with frequently used characters and use the characters by using the Create Keyboard function.

Note

To move around in key layout

  • TAB: Move to the next space
  • SHIFT+TAB: Move to the previous space
  • UP ARROW: Move one row up
  • DOWN ARROW: Move one row down
  • LEFT ARROW: Move to the left
  • RIGHT ARROW: Move to the right
