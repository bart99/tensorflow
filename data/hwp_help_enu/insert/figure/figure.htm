Picture
 

Insert > Illustrations > Picture

Insert Picture is a function for inserting a prepared picture file at the pointer, instead of drawing a picture in Hwp.

When inserting a picture, you can insert the entire picture file or only part of it. You can also insert a picture as zoomed in or out, or apply or remove border and picture effect.

Steps

  1. Click Insert > Illustrations > Picture.
  2. In the Insert Picture dialog box, specify path and name of a picture file and click Insert.
  3. The added picture file will be displayed in the edit window.
  4. To modify the properties of a specific picture, select the picture and click Home > Object Properties.

    You can adjust the size and position of the picture by dragging it with the pointer.

More Information

Insert Picture Dialog Box

Find in

Select a drive or folder to move to a location you want. Or, type the name of a drive or folder in File Name.

File Name

Type the path or name of the picture file to insert.

File Type

Select the file type of the picture file to insert.

Embedded

If you select this option and in the Insert Picture dialog box click Insert, the picture file will be embedded in the active document, instead of being linked to the document. In this case, the modification or position change of the original picture file will not affect the picture embedded in the document.

In Line with Text

Apply In Line with Text to a picture to insert. If you apply In Line with Text, the picture object will be treated just like a regular text. Therefore, its position changes when you type or delete text.

Set Size with Mouse

Instead of the current pointer position, insert a picture inside the rectangle you draw by dragging the pointer.

If you click Insert without selecting this option, the picture will be inserted at the current pointer position in its 100% size.

Apply Previous Object Properties

Apply the properties of an already inserted picture to a picture to insert. The properties of the Object Properties dialog box of the already inserted picture object apply. size, rotation, angle, flip, caption, or In Line with Text do not apply.

If Set Size with Mouse is selected, you cannot apply the properties of the previous object.

Fit to Cell Size

Inserts the picture to fit cell size. This option is activated only if the In Line with Text option is selected.

If you have selected the body instead of a cell, the picture is inserted to fit the width of the edit window.

Insert File Name

Automatically inserts the name of the picture as a caption.

Note

Favorites Icon Description

If you add frequently used folders to the Favorites area on the left of the Insert Picture dialog box, you can open or select saved documents fast.

Icon Name Description
Add to Favorites In the Insert Picture dialog box, add the currently selected location to the Favorites list.
Edit Favorites Modifies the name, path, or type of the location added to the Favorites list in the Insert Picture dialog box.
Delete Favorites Deletes the location of Favorites added to the Favorites list of the Insert Picture dialog box.

Picture Toolbar

If you insert a picture file by clicking Insert > Illustrations > Picture, a picture tab will be displayed.

From the Picture toolbar, you can modify pictures, apply picture effect, cut pictures, insert caption, save inserted pictures, and reset pictures.

Modify Picture

If you select an inserted picture and click Home > Object Properties or double-click the picture, you can zoom it in or out, change its position and margins, cut part of it, surround it with borders, and apply effect to it.

Insert Picture with Shortcut

Once you click Insert in the Insert Picture dialog box, the pointer changes to a crosshair (+). If you press the following shortcut, you can insert a picture without dragging the pointer. The picture will be inserted in its 100% size.

Shortcut Description
D Pictures will be inserted with In Line with Text selected.
S Pictures will be inserted with Square/Paragraph selected.
A Pictures will be inserted with Square/Page selected.
V Pictures will be inserted with Square/Paper selected.
C Pictures will be inserted with Top and Bottom/Paragraph selected.
X Pictures will be inserted with Top and Bottom/Page selected.
Z Pictures will be inserted with Top and Bottom/Paper selected.

Insert Picture While Pressing SHIFT

If you drag the pointer while pressing SHIFT when you insert a picture, the picture will not have the identical horizontal and vertical ratio but be in the size you specified with the pointer.

Insert Caption in Picture

To add a picture caption about the title and short descriptions to a picture, select a picture file in the body text and click References > Insert Caption. You can type or modify caption content.

Picture Info

If you use File > Info > Picture Info, you can check information about all picture files embedded in the document and specify whether to completely embed the picture files in the document file or to just link external picture files to the document. You can also replace the picture files with other pictures or change the path or extension of the picture files.

Supported Picture Types

The picture file types that can be supported in Hwp are BMP, CDR, CGM, DRW, DXF, EMF, GIF, JPG, PCX, PIC, PICT, PLT, PNG, SVG, TIFF, WMF, and WPG.

File Type Description
BMP Windows standard bitmap picture type
CDR CorelDRAW picture type
CGM Outline method picture type for computer graphics
DRW Picture type of Micro Graphics for designers
DXF Picture type for AutoCAD
EMF Picture type with Windows standard outline
GIF Picture type often used for communication created in CompuServe
JPG Picture type designed to significantly reduce file size with the quality of Truecolor maintained
PICT Apple DRAW picture type
PCX Bitmap picture type of Zsoft for paint brush
PIC Picture type used to express graphs in Lotus 1-2-3
PLT Vector picture type created in earlier versions of AutoCAD
PNG Picture type for web design (Portable Network Graphic)
SVG File type based on XML to express 2-D vector graphics
TIFF Picture type for DTP created by Adobe with great compatibility
WMF Picture type with Windows standard outline
WPG WordPerfect Draw picture type
