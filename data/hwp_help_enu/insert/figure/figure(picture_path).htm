Picture Path
 

Insert > Illustrations > Picture

When you link a picture to a document without embedding it in the document, the picture will not display properly if you change the name of the folder where the picture file is and open the document. In which case, the Picture Path dialog box opens and asks for the path of the picture file.

Steps

To set the path of a picture

  1. Insert the file sample.jpg under D:\ in a document by clicking Picture in the Insert tab, save the file as sample.hwp, and exit Hwp.
  2. Cut the file sample.jpg under D:\ from the Internet Explorer and paste it under the folder D:\sample.
  3. Open the file sample.hwp and move to the page where the picture is inserted.
  4. The picture is not displayed in the position where you inserted it. Only the position is marked and the Picture Path dialog box appears.
  5. Specify path of the picture file as D:\sample in the Picture Path dialog box and click Set.
  6. The picture inserted in the document will be displayed properly.

More Information

Picture Path Dialog Box

Picture File

The name of the picture file is displayed.

Picture Path

Find and select the path of the picture file in the folder tree.

Do Not Check Picture Path Automatically

If you click Ignore All, the Picture Path dialog box will not automatically open when there is no picture file or the path of a picture file is changed.

Set Button

You have to specify the picture path of the picture file correctly to activate Set.

Ignore Button

Ignore the path of the picture file.

Ignore All Button

Ignore the path of all pictures inserted in the document.

Note

Display Dialog Box

The Picture Path dialog box is not displayed as soon as you open a document. It is displayed when you move to the page in the document where a picture is inserted.

Order of Finding Picture File

For example, if you open the document file, C:\HWP\DATA\SAMPLE.HWP, and it has the picture file, C:\PICTURE\LETTER01.JPG, the picture file will be found in the following order.

  1. C:\PICTURE\ Folder Specified in Picture File Name
  2. C:\HWP\DATA\ Folder with Document File
  3. Current Folder

If the file, LETTER01.JPG, is nowhere to be found, the Picture Path dialog box appears and asks for the path of the file, LETTER01.JPG. You have to specify folder of the picture file to activate Set and open the picture file from there.

Display Picture Position Only

If you click Ignore or Ignore All without specifying path of a picture in the Picture Path dialog box, only the borders of the picture will be displayed without any picture content.

Modify Picture

If you use File > Info > Picture Info, you can check information about all picture files embedded in a document and change the link status of the picture files.

After checking information about all picture files of the current document, you can change their path or replace them with other picture files by using Change Picture.
