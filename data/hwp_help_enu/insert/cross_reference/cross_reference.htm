Cross-reference
 

References > Cross-reference

Cross-reference marks pictures and tables on various pages to refer to them from the current body text.

The location of reference items (picture, table, equation, bookmark, multilevel list, etc) might change during document editing. If you use cross-reference, you can update the numbers of the items. Cross-reference is provided especially for users who write theses, reports, and books with Hwp.

Steps

  1. Open a document in which to create a table reference item and click References > Cross-reference.
  2. In the Cross-reference dialog box, select File > Current Document.
  3. Select Table for References Item Types and select Table Number for References Content.
  4. Select Hyperlink.
  5. Select Table 1 from the Select References Item list and click Insert.
  6. The table number 1 will be displayed at the pointer.
  7. If additional tables get added above Table 1 in the future, Table 1 will be automatically renumbered according to the number of new tables.

More Information

Cross-reference Dialog Box

File

Select a document to refer to. The default value is Current Document.

To refer to a document that is not the active document, click Select File and select the document file.

References Item Types

Select a type of object to refer to. You can select from Table, Picture, Equation, Footnote/Endnote, Outline, and Bookmark.

References Content

Select reference content to display. You can display the page number, object number, or caption content of the page where the reference item is.

Select References Item

When you select an item in File and References Item Types, all available reference items of the selected type in the selected file will be displayed in a list. Select an item to refer to in the tree-structure list box.

Hyperlink

Hyperlink cross-reference. This is not selected by default.

If you hyperlink cross-reference, you can easily move to the position of the cross-reference. Unlike regular hyperlinks, cross-reference hyperlinks have hyperlink properties only internally. So they follow the text properties at the pointer.

Open Linked Document

  • Close Current Document

  • When opening a document linked to cross-reference, close the active document and open the linked document.

  • To New Tab

  • When opening a document linked to cross-reference, open a new tab on the right of the active document and open the linked document.

  • To New Window

  • When opening a document linked to cross-reference, open a new window and open the linked document in the window.

Note

Link References to Other Files

To insert information about a reference item of another file, type the name of the file in File and insert the information in the same way you link information about a reference item in the active file.

Hyperlink References Content

If you hyperlink cross-reference, you can easily move to the position of the cross-reference. Unlike regular hyperlinks, cross-reference hyperlinks have hyperlink properties only internally. So they follow the text properties at the pointer. Also, the name of the sub-document will be automatically registered as a screen tip.

Cross-reference Update

When the document with reference items changes and the page number or content of the reference items changes accordingly, cross-reference needs to get updated. Cross-reference content gets automatically updated when you print, preview, or open the active document or move through a hyperlink.
