Inserting Foreign Characters

Insert > Symbol

You can insert alphabets of various European languages that are not available on the keyboard by pressing CTRL and two regular English keys.

Steps

To insert with key combinations

  1. Click Review > Keyboard > Change Keyboard.
  2. In the Keyboard Settings dialog box, select the Change Keyboard tab.
  3. Select English and Qwerty from the Current Keyboard list and then click Set.
  4. Type a right after pressing CTRL+GRAVE ACCENT in the edit screen.
  5. à is inserted in the body text.

To insert from Symbol

  1. Click Insert > Symbol.
  2. In the Symbol dialog box, select the Unicode tab.
  3. From the Character Domain list, select Latin.
  4. Select à in the Characters list box and click Insert.
  5. à is inserted in the body text.
