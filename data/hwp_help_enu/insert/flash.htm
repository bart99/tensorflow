Flash
 

Insert > Illustrations > Flash

With this feature, you can insert a flash (*.swf) object in the document. You can play flash animation in real time by inserting a flash object in a Hwp document.

Steps

  1. Click Insert > Illustrations > Flash.
  2. In the Insert Flash dialog box, click the Select Flash File icon in Position > File Location.
  3. In the Select Flash File dialog box, select the path and name of a flash file (*.swf) and click Open.
  4. The path and name of the selected file will be automatically displayed in the Position > File Location box of the Insert Flash dialog box.
  5. Set playing quality, background color, view settings, and Windows mode of the flash object and click Insert.

More Information

Insert Flash Dialog Box

Position
  • File Location

  • In File Location, type the name of a flash file to be inserted or the name combined with the path. If you type only the file name or the relative path of the file location in File Location, you have to type the absolute path in Default Location.

  • Default Location

  • Type the absolute path of the folder in Default Location if the flash files you want to open are in the same folder. Then you do not have to type it again when opening other flash files.

About Display
  • Playing Quality

  • Set playing quality so that the screen looks clear without any shaking.

    Selection Description
    Low Do not increase playing quality. If you select this option, the play speed increases.
    Auto Decrease Avoid increasing playing quality if you already increased playing quality in the beginning and speed decreased.
    Auto Increase When a flash first plays, do not increase playing quality to increase speed. When speed reaches a certain point, it automatically increases playing quality.
    High Increase playing quality. If you select this option, play speed might decrease.
  • Background Color

  • Select a background color to use in the document window when a flash document opens. However, this excludes the area occupied by a Hwp document.

  • View Settings

  • Set screen size when a flash object plays.

    Selection Description
    Show All Display the entire flash by changing its horizontal and vertical ratio according to the size of the browser window. This option is selected by default.
    No Margin The size of a flash is decided by the horizontal or vertical size of the browser window, whichever one is bigger. The horizontal and vertical ratio of a flash does not change. If you increase the horizontal size of the browser window, the top and bottom part of the flash will be cut.
    Original Size Set a flash file to insert in the document to be viewed in its original size without any size increase or decrease.
    Fit to Object Size Display the entire flash so that it fills the browser window. Either the horizontal or vertical size of the flash will be increased or decreased because the horizontal and vertical ratio of a flash changes according to the size of the browser.
Others Settings
  • Autoplay

  • Immediately play all animation scenes when you insert a flash object in the body text.

  • Repeat Play

  • Repeat all scenes after a flash finishes playing.

  • Show Menu

  • If you select this option and right click a playing flash object, context menus appear with menus like Zoom/View All, Select Quality, Play/Repeat, Rewind/Forward/Back, Print, and Properties.
