Date/Time Format
 

Insert > Date/Time > Date/Time Format

Select a format for inserting the current date/time set in the system.

If you select Nation and Default Format in Date/Time Format and then use Date/Time Code or Date/Time String, the date/time will be inserted in the selected format at the pointer in the active document.

Note

Nation

You can select from Korea, U.S., Japan, China, and Taiwan. You can change the nation when needed.
