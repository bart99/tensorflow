Date/Time
 

Insert > Date/Time

Insert the current date/time set in the system in the selected format at the pointer. Or, insert the file name and path of the document in the form of code in the document.

When inserting the date/time in the body text, you can use either String or Code. By using String, you can insert today's date. Like regular strings, you can delete and modify the date/time inserted through String by pressing DELETE or BACKSPACE. However, the date/time inserted in a document as a Code will be updated to current date/time you open the document.

When inserting the file name, insert either the file name only or the path as well. If you save a document file with the File Name inserted in a different folder or path, the document file will be displayed with the new folder or path. This function is useful for managing documents because it shows you on a printed document what kind of file is in which folder.

Note

Creation Date of Field

The date you insert by using Insert > Date/Time is the current date set in the system. To insert a date in various forms, use Creation Date of Insert > Field.
