Date/Time String
 

Insert > Date/Time > Date/Time String

Insert the current date/time set in the system at the pointer. Like regular strings, you can delete and modify the date/time inserted through String by pressing DELETE or BACKSPACE.

Steps

  1. Click Insert > Date/Time > Date/Time String.
  2. The current date/time set in the system will be inserted at the pointer.

Note

Difference from Code

To insert today's date, use String. Like regular strings, you can delete and modify the inserted date/time by pressing DELETE or BACKSPACE.

If you want the date/time inserted in a document to be updated to the current date/time of the system when you open the document, insert date/time by using Code. Code is useful when you insert date/time in a form document.
