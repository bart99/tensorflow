Insert File Name
 

Insert > Date/Time > Insert File Name

Insert the file name and path of a document in the form of code in the document. When you save the document with the File Name stored in a different path, the new path will be displayed in the File Name.

Steps

  1. Place the pointer in a position to insert the file name and click Insert > Date/Time > Insert File Name.
  2. In the Insert File Name dialog box, select Include Path in Format.
  3. Click Insert. The file name and path of the active document will be inserted at the pointer.

Note

File Name Only

Insert only the file name of the document in the form of code. If you use Save As for the document file, the file will be displayed with the new name.

Include Path

Insert the file name that includes the path in the form of code. If you save the document file in a different folder or path, the folder or path will be displayed.

Advanced Usage

Insert File Name in Document

Click Insert > Date/Time > Insert File Name on the Header/Footer screen to display the file name and path in a document.
