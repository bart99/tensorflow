Sound
 

Insert > Illustrations > Sound

With this feature, you can insert a sound object in the document. In Hwp, you can create various documents by inserting sound material.

Steps

  1. Click Insert > Illustrations > Sound.
  2. In the Insert Sound dialog box, click the Record icon and start recording with the microphone connected to the system.
  3. To stop recording, click the Stop icon .
  4. In the Add Comment box, type a short description about the sound to insert and click Insert.
  5. Double-click the icon inserted at the pointer to check the inserted sound.

More Information

Insert Sound Dialog Box

Recording

Display recording progress.

Details on Icons
Icon Name Description
Record Starts recording to create a sound file. When you create an object by recording sound, a microphone and a sound card have to be installed in your system.
Stop Stops the on-going recording.
Play Plays recorded sound.
Copy Copies the current recording to the Clipboard. You can paste the copied sound in an application like audio editor.
Open Open a saved sound file (*.WAV) instead of recording sound.
Save You can save the current recording as a separate file (*.WAV) and use it when needed.
Quality and Maximum Time

Specify recording quality and recording time in seconds. The higher the value, the better the quality. The maximum recording time is 3600 seconds.

Show as Icon

It is impossible to insert sound in visible form in a document because sound files are invisible objects. So you have to display the objects with the icon of the application that can handle them, and connect the sound files to the objects.

Click the Open Icon icon to select an icon to use.

Add Comment

Add a short comment to a sound object to insert in the body text. If you add a comment, the selected icon will be inserted as Caption.

Note

Run Object

You can check sound objects displayed as icons in the body text by double-clicking them.
