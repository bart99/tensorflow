Linking to Hwp Document
 

Insert > Hyperlink

You can insert links to specific bookmarks, outline paragraphs, or objects (image, table, equation, etc.) within a Hwp document for quick and easy information referencing.

Steps

  1. Type Hwp in the body.
  2. Set Hwp as a block and then click Insert > Hyperlink.
  3. The Hyperlink dialog box is shown with Hwp automatically inserted in the Text to Display field. If you want a different name, type it.
  4. Verify the Current Document setting under Link to and then select Document Home from the tree menu.
  5. Click Insert.
  6. In the body, Hwp turns into blue and underlined hyperlink text.
  7. Rest the mouse pointer on Hwp and left click once. The cursor then moves to the beginning of the document.

Note

Link to Different Document

You can insert links to specific locations within a document that is not the active document. Type the exact file name and path of the destination document in the Link to field, or click on the Select File icon and then select the destination document.

Relative Path Support for Different Document Links

You can use either an absolute path or a relative path to hyperlink the active document to a Hwp document that is located on a different path.

  • Hyperlink to sample2.hwp in the same folder (banana1)

  • Absolute Path: C:\apple\banana1\sample2.hwp

    Relative Path: .\sample2.hwp

  • Hyperlink to sample3.hwp in a subfolder (melon)

  • Absolute Path: C:\apple\banana1\melon\sample3.hwp

    Relative Path: .\melon\sample3.hwp

  • Hyperlink to sample4.hwp in a different folder (banana2) under the same parent folder (apple)

  • Absolute Path: C:\apple\banana2\sample4.hwp

    Relative Path: ..\banana2\sample4.hwp

Open Linked Document

Specify the linked document action for the Hwp document hyperlink under Open Linked Document.

  • Close Current Document

  • Closes the current document and opens the linked document in the same window.

  • To New Tab

  • Opens the linked document in a new tab in the current window.

  • To New Window

  • Opens the linked document in a new document window without changing the current window.

Hyperlink Types

Setting a hyperlink to a web or Excel document as External Application Document under Type starts the corresponding program and opens the linked document when the hyperlink is followed.

However, setting a hyperlink to a web or Excel document as Hwp Document under Type opens the linked document in Hwp when the hyperlink is followed.

Advanced Usage

Bookmark and Hyperlink

Bookmark is useful when dealing with a large document and is capable of providing links to bookmarks in other documents when used in conjunction with the Insert > Hyperlink function.
