Hyperlink <CTRL+K>
 

Insert > Hyperlink

Hyperlinks provide references or links to areas within the active document, to other documents, to Web pages, to e-mail addresses, etc. from specific locations in current document.

Steps

To insert hyperlink

  1. Type Thinkfree Office NEO Hwp in the body.
  2. Set Thinkfree Office NEO Hwp as a block and then click Insert > Hyperlink.
  3. The Hyperlink dialog box is shown with Thinkfree Office NEO Hwp automatically inserted in the Text to Display field.

    If you want a different name, type it.

  4. Verify the Current Document setting under Link to and then select Document Home from the tree menu.
  5. Click Insert.
  6. In the body, Thinkfree Office NEO Hwp turns into blue and underlined hyperlink text.
  7. Rest the mouse pointer on Thinkfree Office NEO Hwp and left click once.
  8. The cursor then moves to the beginning of the document.

More Information

Hyperlink Dialog Box

Type

In Type, specify the format of the target document.

Specify the target document as a Hwp Document, Web Address, E-mail Address, or External Application Document.

Link to

  • If you select Type > Hwp Document, a list of bookmarks, multi-level lists, tables, images, and equations inserted in the document is shown under Link to.
    • Absolute Path<->Relative Path Button

    • If you select Type > Hwp Document and then specify a document that is not Current Document under Link to or select Type > External Application Document and then specify the document path, the Absolute Path<->Relative Path button under Link to is activated.

      Click on this button to toggle between absolute path and relative path. In other words, click on the button to switch an absolute path to a relative path and vice versa.

  • If you select Type > Web Address, a list of Internet Explorer Favorites is shown under Link to.

Open Linked Document

  • If you have selected Type > Hwp Document, you will need to specify the linked document action.
    • Close Current Document

    • Closes the current document and opens the linked document in the same window.

    • To New Tab

    • Opens the linked document in a new tab in the current window.

    • To New Window

    • Opens the linked document in a new document window without changing the current window.

  • If you have selected Type > Web Address, you will need to specify the web browser action.

  • However, following actions may not perform correctly in Internet Explorer 7.0 or above.

    • To Same Window

    • Opens the linked web address in the current web browser window.

    • To New Window

    • Opens the linked web address in a new window without changing the current web browser window.

Note

Hyperlink Display

Hyperlinks inserted in the body are shown as blue and underlined text. These properties are shown both on screen and on printouts.

To change hyperlink display options, go to File > Options > Others and specify the font, color, shading color, and underline options.

Go to Hyperlink

To follow a hyperlink, left click on the mouse once on the hyperlink or press ENTER.

Hyperlink to Objects

To hyperlink objects such as an image, table, drawing object, text box, or equation, select one and click Insert > Hyperlink.

Select Object

To select an object hyperlink, select it while holding down the ALT key.

Link Information

To see a hyperlink's target, rest the mouse pointer on the hyperlink. The information is then shown on the status bar.

Advanced Usage

Bookmark and Hyperlink

Bookmark is useful when dealing with a large document and is capable of providing links to bookmarks in other documents when used in conjunction with the Insert > Hyperlink function.

Automatic Hyperlink

Automatically converts web and e-mail addresses typed in the body into hyperlinks.

If you do not wish to automatically format web and e-mail addresses as hyperlinks, go to Review > AutoCorrect and disable AutoRun Typing Command.
