Linking to External Application Document
 

Insert > Hyperlink

You can create links to related information in web documents (*.html), Excel documents, MS Word documents, Hunminjeongeum documents, and other non-Hwp documents using hyperlinks.

Steps

  1. Type Link External Document in the body.
  2. Set Link External Document as a block and then click Insert > Hyperlink.
  3. The Hyperlink dialog box is shown with Link External Document automatically inserted in the Text to Display field. If you want a different name, type it.
  4. Click Type and select External Application Document.
  5. Type the exact file name and path of the target external document in the Link to field, or click on the Select File icon and then select the target external document.
  6. Click Insert.
  7. In the body, Link External Document turns into blue and underlined hyperlink text.
  8. Rest the mouse pointer on Link External Document and left click once. The corresponding program starts and opens the linked document.

Note

Relative Path Support for External Application Document Links

You can use either an absolute path or a relative path to hyperlink the active document to an external document that is located on a different path.

For example, if attempting to hyperlink from sample1.hwp, whose path is as shown below, to an external document, the path in the Link to field can be specified in the following manner:

  • Hyperlink to sample2.html in the same folder (banana1)

  • Absolute Path: C:\apple\banana1\sample2.html

    Relative Path: .\sample2.html

  • Hyperlink to sample3.html in a subfolder (melon)

  • Absolute Path: C:\apple\banana1\melon\sample3.html

    Relative Path: .\melon\sample3.html

  • Hyperlink to sample4.html in a different folder (banana2) under the same parent folder (apple)

  • Absolute Path: C:\apple\banana2\sample4.html

    Relative Path: ..\banana2\sample4.html

Go to Hyperlink

A hyperlink cannot be followed if an application program needed to open the hyperlink is not installed on the system.

Hyperlink Types

Setting a hyperlink to a web or Excel document as External Application Document under Type starts the corresponding program and opens the linked document when the hyperlink is followed.

However, setting a hyperlink to a web or Excel document as Hwp Document under Type opens the linked document in Hwp when the hyperlink is followed.
