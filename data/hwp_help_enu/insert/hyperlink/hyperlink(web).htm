Linking to Web Address
 

Insert > Hyperlink

You can insert hyperlinks to specific web pages to provide convenient access to those pages.

Steps

  1. Type Hancom Home Page in the body.
  2. Set Hancom Home Page as a block and then click Insert > Hyperlink.
  3. The Hyperlink dialog box is shown with Hancom Home Page automatically inserted in the Text to Display field.

    If you want a different name, type it.

  4. Click on the drop-down arrow next to Type and select Web Address. Web addresses from your Favorites list is shown.
  5. Type www.hancom.com/en in the Link to field and click Insert.
  6. In the body, Hancom Home Page turns into blue and underlined hyperlink text.
  7. Rest the mouse pointer on Hancom Home Page and left click once.
  8. A web browser installed on your system starts and connects to the Hancom Inc. website.

Note

Open Linked Document

Specify the web browser action for the web address hyperlink.

However, following actions may not perform correctly in Internet Explorer 7.0 or above.

  • To Same Window

  • Opens the linked web address in the current web browser window.

  • To New Window

  • Opens the linked web address in a new window without changing the current web browser window.

Automatic Link to Web Address

It automatically converts web addresses typed in the body into hyperlinks.

If you do not wish to automatically format web addresses as hyperlinks, go to Review > AutoCorrect and disable AutoRun Typing Command.
