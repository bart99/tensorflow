Linking to E-mail Address
 

Insert > Hyperlink

You can insert a hyperlink to an e-mail address as a convenient way to send mail to that address.

Steps

  1. Type Send Mail in the body.
  2. Set Send Mail as a block and then click Insert > Hyperlink.
  3. The Hyperlink dialog box is shown with Send Mail automatically inserted in the Text to Display field. If you want a different name, type it.
  4. Click Type and select E-mail Address.
  5. Type the exact e-mail address of the recipient in the Link to field.
  6. Click Insert.
  7. In the body, Send Mail turns into blue and underlined hyperlink text.
  8. Rest the mouse pointer on Send Mail and left click once.
  9. An e-mail client installed on the system, such as Outlook Express or Hanmail, starts. The recipient's address is automatically entered in the recipient field.

Note

Automatic Link to E-mail Address

Automatically converts e-mail addresses typed in the body into hyperlinks.

If you do not wish to automatically format e-mail addresses as hyperlinks, disable References > AutoCorrect > AutoRun Typing Command.
