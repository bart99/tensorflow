Add to Custom Symbol
 

Insert > Symbol

You can reduce the time required to search for characters, if you register frequently used characters in the character domain of custom symbols.

You can select a character in the Symbol dialog box and register it in an existing character domain of Custom Symbol. You can also create a new character domain and register the character in it.

Steps

To add to custom symbol

  1. Click Insert > Symbol.
  2. In the Symbol dialog box, select a tab you want.

    Select the Unicode tab for this example.

  3. Select a character domain from the Character Domain list and then select a character from the Characters list. Click the Add to Custom Symbol icon .

    Select Character Domain > Greek and Characters > ет, and click the Add to Custom Symbol icon.

  4. In the Add to Custom Symbol dialog box, select a character domain from the Character Domain List to Add and click Add.

    Follow the steps below to create a new character domain in Custom Symbol and to register a character.

    1. In the Add to Custom Symbol dialog box, click the Register Character Domain icon .
    2. In the Register Character Domain dialog box, type My Symbols in Domain Name and click Add.
    3. Confirm that My Symbols is registered on Character Domain List to Add of the Add to Custom Symbol dialog box and click Add.
  5. When the registration is complete, click the Custom tab to confirm that the characters are registered in the character domain that was earlier specified.

    When you select Character Domain > Greek of the Custom tab, you can view that ет is registered on the Characters list.

To add all to custom symbol

  1. Click Insert > Symbol.
  2. In the Symbol dialog box, select a tab you want.
  3. Select all the required characters from the Characters list and add them to Input Text. Click Add.
  4. In the Add All to Custom Symbol dialog box, view all the characters added to Input Text in the Symbol dialog box which are now displayed in the Text to Add list.
  5. Select a character domain from the Character Domain List to Add and click Add.

    If you select a character from Recent Text in the Symbol dialog box and click the Add All to Custom Symbol icon , you can add all characters displayed in Recent Text to Custom Symbol.

More Information

Add to Custom Symbol and Add All to Custom Symbol Dialog Boxes

Text to Add

All the characters selected in the Symbol dialog box are displayed, which are selected to add to Custom Symbol.

Character Domain List to Add

The Character Domain list of the Custom tab is displayed. Select the character domain to add the selected character.

Details on Icons

Icon Name Description
Register Character Domain Adds a new character domain to Character Domain List to Add.
When you type a name in Domain Name of the Register Character Domain dialog box and then click Add, a new character domain is added to the Character Domain list of the Custom tab.
Modify Character Domain Modifies the name of the character domain selected on Character Domain List to Add.
Once you type a name in Domain Name of the Modify Character Domain dialog box and click Modify, the name of the character domain changes and the modified name appears on the Character Domain list of the Custom tab.
Delete Character Domain Deletes the character domain selected on Character Domain List to Add.
If you delete a character domain, it gets deleted from the Character Domain list of the Custom tab.
Delete Deletes the characters currently selected in Text to Add.

Note

Insert Multiple Characters at Once

To add a character in Input Text, select the character from the Characters list of the Symbol dialog box and click Select. You can also press SPACEBAR on the character or double-click it to add the character in Input Text. If you select multiple characters, add them to Input Text, and then click Insert, the characters are inserted at the pointer in the edit window all at once.

You can also add adjacent characters to Input Text by selecting the characters and hold down SPACEBAR in the Symbol dialog box. If you select  with the pointer on the Characters list, hold down SPACEBAR, and release SPACEBAR at , the 10 characters from  through  are added to Input Text.

Open Symbol from Another Dialog Box, CTRL+F10

To insert symbols when you edit the body and also when you type in the text box of the dialog box of certain functions.

For example, to insert a symbol in Home > Find, place the pointer in the text box and press CTRL+F10 or click the right mouse button. In the context menu, select Symbol to open the Symbol dialog box.
