Screenshot
 

Insert > Illustrations > Screenshot

Screenshots are useful for capturing contents that might change or expire. You can easily add a screenshot to your document without leaving the program that you are working on.

You can use this feature to take a picture of all or part of the windows open on your computer.

Steps

Add whole window

  1. Click Insert > Illustrations > Screenshot, or click Screenshot in the Insert tab.
  2. Select a thumbnail image in the Capturable list and click Insert.

Add part of the window

  1. Click Insert > Illustrations > Screenshot, or click Screenshot in the Insert tab.
  2. Click Capture. The mouse pointer changes to a cross.
  3. Press and hold the left mouse button to select the area of your screen that you want to capture.

More Information

Screenshot Dialog Box

Capturable

Display the list of thumbnail for active windows. If you place the mouse pointer on one of the thumbnails, a balloon tip appears and indicates the program name or document title. You can select the screenshot you want to insert.

Embedded

Embed the selected screenshot from Capturable.

Set Size with Mouse

Specify the size of screenshot with the mouse before inserting it.

Treat as Text

Insert the selected screenshot in line with text.

Insert

Insert the selected screenshot from Capturable.

Close

Close the dialog box without inserting a screenshot.

Capture

Select the area of your screen that you want to capture by pressing and holding the left mouse button. The mouse pointer changes to a cross.

Note

Only one screenshot at a time can be added. To add multiple screenshots, repeat steps 1 and 2 above.
