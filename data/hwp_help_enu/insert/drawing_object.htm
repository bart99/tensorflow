Drawing Object
 

Insert > Illustrations > Drawing Object

This feature is used to insert a drawing object like line, rectangle, circle, and arc on the edit screen of Hwp. You can set properties of a drawing object such as its size, position, shadow, and line, and then insert the drawing object by using Drawing Object.

More Information

Insert Drawing Object Dialog Box

Object

Select a drawing object to draw in the body text from Lines, Rectangles, Circle, and Arc.

Size

Specify width and height of the object.

  • Width

  • For Width, type a value or use the increase or decrease icon.

  • Height

  • For Height, type a value or use the increase or decrease icon.

Position

You can specify the position and position basis of the inserted object, its distance from the position basis, and wrapping style.

  • Horizontal/Vertical Position

  • In the first list box, specify the horizontal and vertical position. In the second list box, specify the guideline origin of the object to be inserted and then type horizontal and vertical position values.

  • Wrapping Style

  • You can specify the position of the object and the body text.

    • Fusion : Place them to allow the object and the body text share the same lines without overlapping each other and to let the body text flow around the object.
    • Top and Bottom: Objects occupy space because of its height hence the body text cannot fit in the space.
    • On Top of Text: The body text gets filled as if there were no object. The object is used as the background of the body text.
    • Beneath the Text: The body text gets filled as if there were no object. The object is placed on top of the body text so that it covers the body text.
Others
  • In Line with Text

  • The object is treated as a regular text. Therefore, its position changes when you type or delete text.

  • Create by Dragging

  • Insert a picture inside the rectangle you draw by dragging the pointer instead of the pointer position.

  • Make Text Box

  • Set rectangles, circles, and arcs to have text box properties when created.

    Others > Make Text Box is activated when you select Rectangles, Circle, or Arc in Object.

Properties Button

In the Object Properties dialog box, you can set detailed properties of a drawing object.

  • The Object Properties dialog box with Lines and Shadow tabs will open if you select Lines in Object and click Properties.
  • The Object Properties dialog box with Lines, Shadow, and Fill tabs will open if you select Rectangles, Circle, or Arc in Object and click Properties.
