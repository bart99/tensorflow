Horizontal Line
 

Home > Horizontal Line

Horizontal Line is a separator of specific weight and color that you draw as a page decoration in a document. Unlike Straight Line that you draw by dragging the pointer, you draw a horizontal line from the left margin of a paragraph to the right margin at once.

Steps

  1. Click Home > Horizontal Line, a horizontal line will be drawn from the left margin to the right margin of the paragraph where the pointer is.
  2. Select the inserted horizontal line and click the arrow next to Fill in the Styles group of the Shapes tab.
  3. When the color palette opens, select blue.
  4. The color of the horizontal line changes to blue.

Note

Changing Properties

Horizontal Line is a rectangular object of weight: 1 mm, width: 100% depending on the paragraph, anchor: paragraph, face color: black, and line type: no line.

You cannot change the default value. When a horizontal line is drawn, you can change its size, anchor, margins, line properties, and fill properties by clicking Home > Object Properties.

New Drawing Style

If you change New Drawing Style of a drawing object, the default value of the Horizontal Line does not change. However, you can set the properties of the inserted Horizontal Line as a new drawing style by using Set as New Drawing Style.

Advanced Usage

Header/Footer

If you insert a horizontal line in the header or footer, you can easily create a decoration line that separates the body text from the header or footer on every page.
