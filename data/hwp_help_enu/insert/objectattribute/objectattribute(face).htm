Object Properties: Fill
 

Home > Object Properties

Show All

You can apply a wide range of fill effects to drawing objects including color, pattern, gradation, and picture.

More Information

Fill Tab in Object Properties Dialog Box

No Fill

You can select No Fill if you are not applying Color, Gradation, or Picture.

Color
  • Face Color

  • You can click the color table to display the color palette, and then select a background color from the color palette.

  • Pattern Color

  • If Pattern is selected, click the color table in Pattern Color to display the color palette. In the color palette, select a pattern color to fill the background.

  • Pattern

  • From the Pattern list, select a pattern to fill the background. For more information, you can refer to the Fill with Pattern section.

Gradation

Fill the background with gradation for gradual color spread.

  • Beginning Color
    You can click the color table to display the color palette. In the color palette, select a color that spreads from the center of the object to the outside.

  • Ending Color

  • You can click the color table to display the color palette. In the color palette, select a color that comes into the center from the outside of the object.

  • Type

  • You can select a style of your choice from the various gradation templates. You can create a type or style by selecting Custom.

  • H Center

  • You can display the position of the horizontal center of the object in percent.

  • V Center

  • You can display the position of the vertical center of the object in percent.

  • Angle

  • You can specify the degrees to rotate gradation.

  • Blur Level

  • You can specify the blur level for the beginning color and the ending color. Level 0 means no blur effect. Blur level can be set from 0 through 255.

  • Blur Center

  • Specify spread level for the beginning color and the ending color. Level 0 means no face color. Level 2 displays only the beginning color and the ending color. This can be set from 0 through 100.

Picture

You can open a picture file, and then fill the background by using picture.

  • Picture File

  • You can open a picture file by clicking Open to fill the background.

  • Embedded

  • You can insert the picture file in the document without connecting it to the document. If you select this option, the modification or changed location of the original picture file will not affect the picture inserted in the document.

  • Fill Type

  • You can select Tile, Fit to Size, or Center from the list for fill type to fill the background with a picture.

  • Effects

  • You can apply grayscale or black and white to the opened picture.

  • Watermark Effect

  • You can apply 70% brightness and -50% contrast to the opened picture file to change it to a bright and low-contrast picture.

  • Brightness

  • You can adjust the brightness of the selected picture file. Brightness can be set from -100 through 100.

  • Contrast

  • You can adjust the contrast of the selected picture file. Contrast can be set from -100 through 100.

Set Transparency

You can specify the transparency, after filling the face of a drawing object with a color, a pattern, gradation, or a picture.

  • Transparency

  • You can type a value in the edit box or adjust the value by using the increase and decrease icons.

    You can set the transparency value from 0 through 100.

Note

Set Simultaneously

You cannot fill a drawing object face with color and gradation effect at the same time. However, you can set two effects like Fill with Both Color and Picture and Fill with Both Gradation and Picture simultaneously.

To apply two effects, first fill with a color or gradation effect and then fill with a picture.

New Drawing Style

You can specify the Color and Gradation settings for New Drawing Style. You can alsoassign frequently used fill settings to New Drawing Style.

No Fill Property

You can select No Fill from the dialog box to disable the fill effect.
