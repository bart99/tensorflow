Object Properties <P>
 

Home > Object Properties

You can specify the properties common to all objects of the document, such as size, position, and margins, on Basic and Margin/Caption tabs in Object Properties dialog box. You can also specify properties to individual objects from Lines, Fill, Text Box, Shadow, Reflection, Neon, Soft Edges, Picture, Equation, Wordshop, and OLE Properties.

Steps

  1. Click Insert > Illustrations > Picture, specify the location and name of the picture file in Insert Picture dialog box, and click Insert.
  2. When the pointer becomes a cross (+), click the point to insert the picture, and drag the pointer diagonally in the desired direction to set the size.
  3. Once the picture has been inserted into the edit window, select the picture and either press the P key or click Home > Object Properties.
  4. From the Object Properties dialog box, select the Picture tab.
  5. Check Lock Aspect Ratio in Zoom Ratio and then click the Make 1/2 icon.
  6. Verify the Horizontal and Vertical values are same, and click Set.
  7. The picture inserted in the edit window scales down to 50% of its original size.

Object Properties

You can use common settings for position, margin, caption, and other general properties, for all picture, drawing, text box, Wordshop, equation, table, and chart objects used in Hwp.

The properties that vary depending on the type of the object selected are shown under corresponding tabs in Object Properties dialog box.
