Object Properties: Wordshop
 

Home > Object Properties

Show All

You can insert a Wordshop object and press the P or click Home > Object Properties. The Object Properties dialog box appears with the Basic, Margin/Caption, Lines, Fill, and WordArt tabs.

From the WordArt tab, you can edit the Wordshop object and change the font, line spacing, text spacing, and paragraph alignment. You can also apply or remove a shadow and specify the color and direction of the shadow.

More Information

WordArt Tab of Object Properties Dialog Box

Wordshop Body
  • Contents

  • You can type in text to convert into a Wordshop object.

  • Font

  • Select a font.

  • Property

  • Most fonts have four font properties: Regular, Italic, Bold, and Bold Italic.

    After selecting an English font whose properties can be specified (e.g., Arial or Verdana) from the Font list, you can specify the properties of the selected font from the Property list.

      To display English fonts on the Font list of Wordshop, you have to select Register Latin Fonts as Representative Fonts of Register Font in Options > Fonts. Exit from Hwp and restart it to add the English fonts to the list.

  • Line Spacing

  • You can adjust the spacing between lines.

  • Text Spacing

  • You can adjust the spacing between characters.

  • WordArt Shape

  • In the font shaped palette, you can select from the 55 available shapes including Wave, Butterfly, Flag, Semicircle, Circle, and Crescent Moon.

Paragraph Align
  • Details on Icons
  • Icon Name Description
    Align Left You can align the text to the left.
    Align Right You can align the text to the right.
    Align Center You can align the text to the center.
    Align Justified You can align the text on both sides so that there is an equal margin, except for the last line.
    Align Justified (including last sentence) You can align the text on both sides so that there is an equal margin including the last line.
Shadow
  • None

  • No shadow.

  • Separate

  • Applies a shadow effect beneath the text.

    • Color

    • To display the color palette click the color table. In the color palette, select a shadow color of your choice.

    • X Position

    • You can adjust the horizontal position of the shadow by typing a value into the edit box or by using the increase and decrease icons.

      The value can be set from -48 through 48 in relation to the size of the text. A negative value places the shadow to the left of the text, and a positive value places the shadow to the right of the text.

      You can also adjust the X and Y positions of the shadow using the 9 arrow icons.

    • Y Position

    • You can adjust the vertical position of the shadow by typing a value into the edit box or using the increase and decrease icons.

      The value can be set from -48 through 48 in relation to the size of the text. A negative value places the shadow to the top of the text, and a positive value places the shadow to the bottom of the text.

      You can also adjust the vertical and horizontal positions of the shadow using the 9 arrow icons.

    • Details on Icons
    • Icon Name Description
      Upper Left You can move the shadow of the selected object 1mm to the top and 1mm to the left.
      Left You can move the shadow of the selected object 1mm to the left.
      Lower Left You can move the shadow of the selected object 1mm towards the bottom and 1mm to the left.
      Top You can move the shadow of the selected object 1mm to the top.
      Default You can reset the shadow of the selected object to its original state.
      Bottom You can move the shadow of the selected object 1mm towards the bottom.
      Upper Right You can move the shadow of the selected object 1mm to the and 1mm to the right.
      Right You can move the shadow of the selected object 1mm to the right.
      Lower Right You can move the shadow of the selected object 1mm towards the bottom and 1mm to the right.

Note

X, Y Position Adjustment for Wordshop Shadow

The positions can be specified from -48 through 48 in relation to the size of the text.

A negative value for the X position places a shadow to the left of the text, and a positive value places a shadow to the right of the text. A negative value for the Y position places a shadow to the top of the text, and a positive value places a shadow to the bottom of the text.
