Object Properties: Margin/Caption
 

Home > Object Properties

Show All

You can insert an object and press P or click Home > Object Properties. From the Margin/Caption tab in the Object Properties dialog box, you can specify the margin between the object and the body, the caption position, size, and object spacing settings.

Steps

To insert caption and adjust caption position

  1. Click Insert > Illustrations > Picture to insert a picture.
  2. Select the picture to insert a caption, and right click to open the context menu. From the menu, select Insert Caption or click References > Insert Caption.
  3. When Picture 1 automatically displays at the bottom of the picture, type the title of the picture and press SHIFT+ESC to return to the body text.
  4. Select the picture and press P to open the Object Properties dialog box. Then select the Margin/Caption tab.
  5. Select Caption > Insert Caption > Center Right and then click Set.
  6. The position of the Picture 1 caption changes from bottom of the picture to the right of the picture.

More Information

Margin/Caption Tab of Object Properties Dialog Box

Distance from Text
  • Left

  • You can adjust the distance between the left edge of the object and the body.

  • Right

  • You can adjust the distance between the right edge of the object and the body.

  • Top

  • You can adjust the distance between the top edge of the object and the body.

  • Bottom

  • You can adjust the distance between the bottom edge of the object and the body.

  • All

  • You can click All to specify the left, right, top, and bottom margins.

Caption
  • Insert Caption

  • The Caption content will be inserted outside the borders of the object.

    You can choose from Top, Bottom, Upper Left, Center Left, Lower Left, Upper Right, Center Right, Lower Right, and None as the caption position.

  • Size

  • If the caption position is set to Top or Bottom, caption size remains the same as the width of the object. If the caption position set to Left or Right, you can specify the size of the caption.

  • Distance from Object

  • You can specify the distance between the object and caption content. The default value is 3mm. Hwp supports negative distance from object values. You can also specify a negative value for Distance from Object.

  • Include Margin

  • If the outside margin is specified for the object, the caption starts from the object area excluding the specified margin. If you specify this option, the caption starts outside the left or right margin.

  • Single Row

  • Everything you type in the caption area will be in one line even if the pointer reaches the right boundary as you type. if you select this option.

Note

Outside Margin and Caption

If the outside margin is specified for the object, the caption starts from the object area excluding the specified margin. If Caption > Include Margin is enabled, the caption starts outside the left or right margin.

Relationship with Object

Outside margin is the space between the object and the in-body text. This margin does not affect the distance between the object and another object that has not been set as In Line with Text.

Wrapping Style

Outside margin can be applied only to objects that have been set as In Line with Text or whose wrapping style has been set as Square or Top and Bottom. Outside margin cannot be applied to objects whose wrapping style has been set as On Top of the Text or Beneath the Text.

Size

If you specify margin settings for an object, it does not affect its size.

Unit

The default unit is millimeter (mm). Other available options are point (0.35146 mm), centimeter (cm), pica (pi), pixel (px), geup (geup), and inch (¢©).

To change the unit, type the unit along with the numerical value or right click on the unit to open the context menu.
