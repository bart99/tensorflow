Object Properties: Text Box
 

Home > Object Properties > Text Box

Show All

You can insert a text box and click Home > Object Properties. The Object Properties is shown with the Basic, Margin/Caption, Lines, Fill, Text Box, and Shadow tabs.

You can specify the margin, vertical alignment, and vertical writing settings from the Text Box tab.

More Information

Text Box Tab of Object Properties Dialog Box

Text Box Margin
  • Left

  • You can adjust the left margin of the text box.

  • Right

  • You can adjust the right margin of the text box.

  • Top

  • You can adjust the top margin of the text box.

  • Bottom

  • You can adjust the bottom margin of the text box.

  • All

  • You can click All to specify the left, right, top, and bottom margins of the text box.

Properties
  • Vertical Alignment

  • You can specify whether to align the text inside the text box to the top, bottom, or middle.

  • Icon Name Description
    Top You can align the text to the top of the text box.
    Center You can align the text to the middle of the text box.
    Bottom You can align the text to the bottom of the text box.
  • Vertical Writing

  • You can enable vertical writing of text inside the text box.

  • Icon Name Description
    English Down Set English and numbers to be laid down when text is entered vertically.
    English Up Set English and numbers to be put up when text is entered vertically.
