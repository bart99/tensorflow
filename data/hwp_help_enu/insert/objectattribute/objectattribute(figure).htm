Object Properties: Picture
 

Home > Object Properties

Show All

You can insert a picture by clicking Insert > Illustrations > Picture and press P on the keyboard or click Home > Object Properties. The Object Properties dialog box appears with the Basic, Margin/Caption, Lines, and Picture tabs.

From the Picture tab, you can insert or link to a picture, enlarge or reduce the ratio of the selected picture, crop the picture, set the margins, and apply effects.

More Information

Picture Tab of Object Properties Dialog Box

File Name
  • File Name

  • You can click Open and select a picture file to insert.

  • Embedded

  • You can insert the picture file in the document without connecting it to the document. If you select this option, the modification or changed location of the original picture file will not affect the picture inserted in the document.

Zoom Ratio
  • Horizontal

  • You can specify the horizontal size ratio of the picture by entering a value or using the increase and decrease icons.

  • Vertical

  • You can specify the vertical size ratio of the picture by entering a value or using the increase and decrease icons.

  • Lock Aspect Ratio

  • If Lock Aspect Ratio is selected and Horizontal or Vertical settings are changed by using the increase and decrease icons, the other settings also change.

  • Details on Icons
    Icon Name Description
    Original size Ignores the current picture size and restores the picture to 100% of its original size.
    Make 1/2 Reduces the picture size to 1/2.
    Make 2/3 Reduces the picture size to 2/3.
    Make 3/2 Increases the picture size to 3/2.
    Double Doubles the picture size.
  • Reset Picture Button

  • You can reset a picture that has been resized, applied with an effect, or cropped to 100% of its original size and state.

Crop Picture
  • Left

  • You can specify the area of picture to crop from the left.

  • Right

  • You can specify the area of picture to crop from the right.

  • Top

  • You can specify the area of picture to crop from the top.

  • Bottom

  • You can specify the area of picture to crop from the bottom.

  • All

  • You can click All to specify the area of picture to crop from the left, right, top, and bottom.

Picture Margin
  • Left

  • You can specify the left margin.

  • Right

  • You can specify the right margin.

  • Top

  • You can specify the top margin.

  • Bottom

  • You can specify bottom margin.

  • All

  • You can click All to specify the left, right, top, and bottom margins.

Effects

You can convert a picture file, inserted into the document as a grayscale image, a black and white image, or a watermark.

  • Effects

  • You can convert the opened picture into a grayscale image, a black and white image, or a watermark.

    Icon Name Description
    No Effects Returns the picture file to which effects like Grayscale, Black and White, and Watermark are applied to the original image.
    Grayscale Converts the selected picture file into a grayscale image.
    Black/White Converts the selected picture file into a black and white image.
    From Original Picture Automatically selects defined Brightness, Contrast, Watermark Effect, and Invert Picture settings.
  • Brightness

  • You can adjust the brightness of the selected picture. Brightness can be set from -100 through 100.

  • Contrast

  • You can adjust the contrast of the selected picture. Contrast can be set from -100 through 100.

  • Watermark Effect

  • You can apply 70% brightness and -50% contrast to the opened picture file to change it to a bright and low-contrast picture.

  • Invert Picture

  • You caninvert the colors of the picture.

Note

Objects with Shadow Support

If you select a shape or text box and click Object Properties, the Shadow tab is displayed. You can use it to apply a shadow to the selected object. However, picture and Wordshop objects do not support the shadow effect and does not contain the Shadow tab in the Object Properties dialog box.
