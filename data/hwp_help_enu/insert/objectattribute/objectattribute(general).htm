Object Properties: Basic
 

Home > Object Properties

Show All

You can insert an object and click Home > Object Properties. From the Basic tab in the Object Properties dialog box, you can specify the size, position, placement, and other general object properties.

More Information

Basic Tab of Object Properties Dialog Box

Size

To change the height and width of the object. Select Fix Size to lock the size of the object.

  • Width

  • You can choose from Fixed Value, Based on Paper, Based on Page, Based on Column, and Based on Paragraph as the basis for the horizontal Width of the object.

    • Fixed Value : An absolute value that fixes the typed value as object size.
    • Paper, Page, Column, Paragraph: A relative value that displays object size in percent, based on all criteria.
  • Height

  • You can choose from Fixed Value, Based on Paper, and Based on Page as the basis for the vertical Height of the object.

    • Fixed Value : An absolute value that fixes the typed value as object size.
    • Based on Paper, Based on Page: A relative value that displays object size in percent, based on all criteria.
  • Fix Size

  • To fix the size of the object, select Fixed Value for both Width and Height.

Position

You can specify the position and position basis of the inserted object, its distance from the position basis, and wrapping style.

  • In Line with Text

  • The object is treated as a regular text. Therefore, its position changes when you type or delete text.

  • Wrapping Style

  • You can specify the position of the object and the body text.

    Icon Name Description
    Square Place them to allow the object and the body text share the same lines without overlapping each other and to let the body text flow around the object.
    Top and Bottom Objects occupy space because of its height hence the body text cannot fit in the space.
    On Top of the Text The body text gets filled as if there were no object. The object is used as the background of the body text.
    Beneath the Text The body text gets filled as if there were no object. The object is placed on top of the body text so that it covers the body text.
  • Horizontal/Vertical Position

  • In the first list box, specify the horizontal and vertical position. In the second list box, specify the guideline origin of the object to be inserted and then type horizontal and vertical position values.

  • Keep Inside Page Area

  • The object will be carried over to the next page by using Vertical set to Paragraph, if its vertical position goes outside the current page area.

  • Allow Overlap

  • If the Wrapping Style settings of objects (pictures or tables, but not drawing objects) are set to Square or Top and Bottom, objects are arranged away from one another without overlapping. Enabling Allow Overlap, however, makes it possible to overlap objects that have been set as Square or Top and Bottom.

  • Hold on control code and object at the same page

  • If the Wrapping Style setting of an object is set to Top and Bottom, enabling Hold on control code and object at the same page relocates the object and its control codes even if the object goes beyond page or gets pulled while the document is being edited.

Rotate Object

To rotate the object by a specific angle, click Rotation Angle. From the Rotation Angle dialog box, enter a rotation angle from 0 through 360.

Tilt

To tilt a drawn object in the vertical or horizontal direction, click Change Tilt and then set the vertical horizontal tilt values from -89 through 89.

Others
  • Numbering Type

  • You can choose from Picture, Table, and Equation as the numbering type for the selected object. Select None if you do not want to number the object.

  • Lock Object

  • Lock the object to prevent it from being moved or deleted accidentally. Objects that have been applied with the Lock Object setting cannot be numbered or selected using the mouse. To unlock all objects, you can go to the Protect group in the Drawing Tools tab and click Lock Object > Unlock All Objects.

  • Object Description

  • You can insert the information about a selected object. The default description is the name of the object. However, the description can be edited as per your requirement. Object description supports text input only and can be referenced as a balloon tip directly from the document.

Caution

Rotate/Tilt Objects in Line with Text

Rotate Object and Tilt are not available for objects that are set as In Line with Text.
