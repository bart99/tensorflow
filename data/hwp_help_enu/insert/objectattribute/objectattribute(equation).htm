Object Properties: Equation
 

Home > Object Properties

The Equation Properties dialog box appears with General, Margin/Caption, and Equation tabs, if you insert an equation and press P, or click Home > OLE Properties.

You can specify the style of the equation from the Equation tab including Font, Equation Region, and Apply to settings.

More Information

Equation Tab of the Equation Properties Dialog Box

Font
  • Text Size

  • You can enlarge or reduce the text size of the equation.

  • Font Color

  • You can click the color table to display the color palette, and choose the font color for the equation.

Equation Region
  • Area by Text

  • You can insert equations in the body as a part of the text.

  • Area by Rows

  • You can insert equations on individual rows, apart from text elements of the document.

Apply to
  • Apply to > Selected Equation Object

  • You can apply equation object properties to the selected equation object only.

  • Apply to > Whole Document

  • You can apply equation object properties to all equation objects in the document.

Edit Button

You can edit the equation from the Equation Editor dialog box.
