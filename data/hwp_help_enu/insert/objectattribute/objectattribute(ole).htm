Object Properties: OLE Properties
 

Home > Object Properties

You can insert an OLE object by clicking Insert > Object and press P or click Home > Object Properties. The Object Properties dialog box appears with the Basic, Margin/Caption, Lines, and OLE Properties tabs.

From the OLE Properties tab, you can check and edit the properties of the inserted OLE object.

Steps

  1. Click Insert > Object.
  2. In the Insert Object dialog box, select New.
  3. From the Object Type list box, select Thinkfree Office NEO Cell Document and click OK.
  4. Enter data into the new Cell worksheet and exit from Cell. The Cell worksheet is inserted into the cursor position in the Hwp edit window.
  5. Select the inserted object and either press P or click Home > Object Properties.
  6. From the OLE Properties tab in the Object Properties dialog box, check or edit the properties of the OLE object and click Set.
