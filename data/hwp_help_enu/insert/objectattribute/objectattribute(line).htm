Object Properties: Lines
 

Home > Object Properties

Show All

After inserting an object, press P or click Home > Object Properties. From the Lines tab in the Object Properties dialog box, you can specify the Line Type, Arrow Style, Rectangle Corner Curvature, and Arc Border settings for the selected line.

More Information

Lines Tab of Object Properties Dialog Box

Lines

You can specify the color, weight, type, and end style settings for the selected line.

Arrows

You can specify the style and size of arrows to use on the two ends of the selected object.

Rectangle Corner Curvature

To change the rectangular corners to rounded or semicircular corners or specify the exact curvature.

Icon Name Description
Right Angle To specify the rectangular corner curvature as 0% to set corners as right angles.
Rounded To specify the rectangular corner curvature as 20% to set rounded corners.
Semicircle To specify the rectangular corner curvature as 50% to set semicircular corners.
  Curvature To specify the degree of rectangular corner curvature. Specify a value between 0% and 50%. Enter 0% for Right Angle, 20% for Rounded, or 50% for Semicircle. Larger the curvature, more rounded the corner is.
Arc Border

You can change the arc border to a fan or a bow.

Icon Name Description
Arc No border on the radius of the arc.
Fan Apply a border on the radius of the arc to create the shape of a fan.
Bow Connect the two ends of the arc with a line to create the shape of a bow.
Set Transparency

You can specify the transparency of the line color.

  • Transparency

  • You can type a value in the edit box or adjust the value by using the increase and decrease icons.

    You can set the transparency value from 0 through 100.

Others
  • Line Weight Does Not Affect Size

  • The line is formed inside the object and therefore does not affect the size of the object.

    If this option is not selected, the line is formed on both sides of the base line, resulting in the enlargement of the object.

Note

New Drawing Style

You can register an object line type in New Drawing Style. If you frequently use a specific line type, register it as a new drawing style.
