Clip Art: Search Results
 

Insert > Illustrations > Clip Art

In the Find tab of the Clip Art dialog box, you can find an object of your choice from numerous objects of Clip Art packages.

Steps

  1. Click Insert > Illustrations > Clip Art.
  2. In the Clip Art dialog box, type Giraffe in the File to Find box and click Find.
  3. The name and type of the found Clip Art objects will be displayed on the list.
  4. Select a giraffe and click Insert to insert it in the edit window.

More Information

Search Results Tab of Clip Art Dialog Box

File to Find

Type the name of the file to find. If you do not remember the exact name of the file to find, you can use wildcards (*, ?) in the File to Find box.

Find Button

Once you type the name of the file to find in the File to Find box and click Find, the search result will be displayed on the list.

Each madang.ini file of Clip Art will be open and the name of the picture file will be searched for. Picture files that meet the search criteria will be found.

Found List

Register found shapes and Clip Art in the Search Results tab in the order of finding. If more than 15 items are found, a vertical scroll bar will appear. You can use the scroll bar to check all the items.

Even if you exit Hwp, the objects displayed in the Search Results tab will be maintained until other objects are found.

Preview

When you select an image object in the Object List box, the zoomed image of the object will be displayed in Preview.

Insert Directly

  • If you select this option

  • If you select this option, you can insert the drawing object selected from the list into the body text without closing the dialog box.

    Select a drawing object with the dialog box open and specify size of the object with the mouse in the body text. If you repeat this process, you can insert a series of drawing objects.

  • If you do not select this option

  • If you do not select this option, select a drawing object from the list and close the dialog box by clicking Insert. Specify object size with the pointer and the drawing object will be inserted in the body text.
