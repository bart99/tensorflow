Clip Art: Shared Clip Art
 

Insert > Illustrations > Clip Art

In the Shared Clip Art tab of the Clip Art dialog box, you can register new Clip Art packages or modify the name and path of registered Clip Art packages. You can also delete Clip Art packages you do not use from the list.

Steps

  1. Click Insert > Illustrations > Clip Art.
  2. In the Clip Art dialog box, select the Shared Clip Art tab.
  3. If you select a package from the Package to Select list, objects in the package will be displayed as icons in the Object List.
  4. When you select an icon from the Object List and point to the position in the body text where you want to draw that object, the pointer changes to a crosshair (+).
  5. Once you click a starting point, drag it to desired size, the object will be inserted.

  6. After inserting the object, click Cancel to close the dialog box.

More Information

Clip Art Tab of Clip Art Dialog Box

Package to Select

You can select a package from the list of Clip Art packages that include picture files created externally.

Object List

If you select a Clip Art package from the Package to Select list, objects in the package will be displayed as icons.

Preview

When you select an image object in the Object List box, the zoomed image of the object will be displayed in Preview.

Insert Directly

  • If you select this option

  • If you select this option, you can insert the drawing object selected from the Object List into the body text without closing the dialog box.

    To insert a drawing object, select one and specify size in an appropriate position in the body text with the pointer when the dialog box is still open. After inserting the object, you can insert other drawing objects by continuously selecting them in the dialog box.

  • If you do not select this option

  • If you do not select this option, select a drawing object from the Object List and close the dialog box by clicking Insert. Specify object size with the pointer and the drawing object will be inserted in the body text.

Details on Icons

Icon Name Shortcut Description
New Package Insert Creates a Clip Art package where you want to store new shapes or Clip Art.
Modify Package + Changes the name of registered Clip Art packages or specifies a new path for them.
Delete Package Delete Deletes Clip Art packages that you do not use.
Delete Object   Deletes objects selected in the selected Clip Art package. Once deleted, objects cannot be recovered.
Rename Object   Modifies the name of the object selected in the selected Clip Art package.

Note

Run from Context Menu

If you right click over the drawing objects of the Shapes tab or the Shared Clip Arts tab in the Clip Art dialog box, context menus like New Package, Modify Package, Delete Package, Delete Object, and Rename Object appear.
