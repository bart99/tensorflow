Clip Art
 

Insert > Illustrations > Clip Art

Clip Art means creating and adding frequently used objects in advance to draw pictures easily and fast.

Each drawing file (*.drt) in Clip Art is called shape. A collection of shapes of the same type is called Clip Art package.

Hwp provides various shapes and Clip Art pictures in Clip Art. So you can save time and energy when drawing a picture.

Steps

  1. Click Insert > Illustrations > Clip Art.
  2. In the Clip Art dialog box, select the Shapes tab.
  3. Select a package from the Package to Select list.
  4. Select one of the various pictures from the Object List and click Insert.
  5. Move the pointer in the form of crosshair (+) to a position where you want to insert the picture and specify its size by dragging the pointer diagonally.

  6. If you let go of the pointer after dragging, the selected picture will be inserted in the position in the specified size.

More Information

Clip Art Dialog Box

File to Find

Type the name of the file to find. If you do not remember the exact name of the file to find, you can use wildcards (*, ?) in the File to Find box.

Find Button

If you type the name of the file to find in the File to Find box and click Find, the Search Results tab gets selected and the search result appears on a list.

Shapes Tab

Provide objects drawn in Hwp with the drawing function in the Shapes tab.

Shared Clip Art Tab

Approximately 2,500 kinds of Clip Art are provided in various packages.

Search Results Tab

In the File to Find box, the result of keyword search appears. The search results will be displayed as Shapes or Clip Art.

Note

Insert as Default Size

The pointer changes to a crosshair if you select a shape to insert into the body text and then click Insert. If you click an area once without dragging the pointer, the shape will be inserted as the default size.

Difference from Insert Picture

If you insert a picture file with Clip Art, only the picture files in registered packages are available.

If you click Insert > Illustrations > Picture, you can use all kinds of picture files that Hwp can open, regardless of which folder they are in.

Clip Art Type

Clip Art consists of Shapes and Clip Art. The former is a collection of pictures drawn with the picture drawing function of Hwp. The latter is a collection of picture files drawn externally.

Package Type

Hwp provides approximately 1,000 shapes in approximately 35 packages including Tradition, Science, Transportation, Anniversary, Shape, Background, and Life.

Insert as Default Size

The pointer changes to a crosshair if you select a shape to insert into the body text and then click Insert. When you click an area once without dragging the pointer, the shape will be inserted as the default size.

Insert while Pressing SHIFT

Once you drag the pointer while pressing SHIFT when you insert Clip Art, the Clip Art will not have the identical horizontal and vertical ratio but be in the size you specified with the pointer.

Difference from Insert Picture

If you use Clip Art of Clip Art when inserting an external picture file into a document, only the picture files in the registered packages are available.

If you click Insert > Illustrations > Picture, you can use all kinds of picture files that Hwp can open, regardless of which folder they are in.
