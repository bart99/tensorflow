Add to Clip Art
 

Insert > Illustrations > Clip Art

With this feature, you can add to Clip Art the pictures drawn in Hwp or external picture files inserted in documents, and open them whenever you need them.

Steps

To add a drawing object to Clip Art

  1. Select a drawing object to add to Clip Art and right click it to open context menus.
  2. From the context menus, select Add to Clip Art.
  3. In the Add Clip Art dialog box, select a package from Package List to Add to add the drawing picture to the package.
  4. Type a name for the drawing object in the Name box and click Add.
  5. Click Insert > Illustrations > Clip Art. In the Clip Art dialog box, select the Clip Art tab.
  6. Check the added object on the Package to Select list.

Note

Add to Shapes

If you select an object that you drew, and run Add to Clip Art, the Add Shapes dialog box appears. Package List to Add only displays a list of packages that are already registered in the Shapes tab of Clip Art.

Add to Clip Art

If you select an external picture file inserted in the document and use Add to Clip Art, the Add Clip Art dialog box appears. Package List to Add only displays a list of packages that are added to the Shared Clip Art tab of Clip Art.

Add Grouped Objects

If two or more drawing objects, an external picture file and a drawing object, or external picture files are grouped as one object, you can only add them to the Shapes package.

Difference from Insert Picture

If you insert a picture file with Clip Art, only the picture files in registered packages are available.

If you click Insert > Illustrations > Picture, you can use all kinds of picture files that Hwp can open, regardless of which folder they are in.

Caution

One at a Time

You can add shapes to Clip Art only one object at a time. If you select two objects at the same time and press the right mouse button, there will not be Add to Clip Art in the context menus.
