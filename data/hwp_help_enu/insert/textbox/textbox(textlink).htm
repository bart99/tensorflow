Create Link
 

Insert > Illustrations > Text Box

Create Link joins multiple text boxes together so that text entered into one text box flows over to the next text box if the first box becomes full.

Create Link is highly useful for editing newspapers, multiple column articles, and other documents that feature a complex layout.

Steps

  1. Click Insert > Illustrations > Text Box.
  2. The mouse pointer turns into a cross (+). Click the area of choice to insert a text box or drag.
  3. Repeat steps 1 and 2 to insert additional text boxes to link together.
  4. Select the first text box.
  5. From the Text Box group under the Drawing Tools tab, locate the Create Link toolbar and click on the Create Link icon.
  6. The mouse pointer turns into a link pointer. Click on the text box of choice to link to.
  7. To link another text box, select the second text box and repeat steps 5 and 6.
  8. In the first text box, enter text or paste in copied text.
  9. Once the first text box becomes full, the text flows over to the second text box.

Note

Linkable Drawing Objects

Create Link not only joins empty text boxes, but is also capable of joining drawing objects with access to the Make Text Box option in their context menu of Drawing Tools tab.

Remove Text Box Link

If you click on the Create Link icon and do not select an empty text box, a warning message will appear.

Advanced Usage

Linking to Text Box that Contains Text

With Create Link, you can link a text box to an empty text box or a text box that already contains text.

If you click on the Create Link icon and select a text box that already contains text, the "The target text box already contains information. Overwrite?" message appears. Click Yes to replace the existing text with text from the source text box.

Linking Fixed Size Text Boxes

To use the Create Link feature more effectively in situations where the text entered exceeds the area of the text box, click Home > Object Properties > Basic and select Size > Fix Size to lock in the size of the text box.
