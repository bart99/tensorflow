Vertical Text Box
 

Insert > Text Box

You can use Vertical text box to arrange text vertically.

Steps

  1. Insert a text box by clicking Insert > Text Box > Vertical Text Box.
  2. Type text in the text box.
  3. To open the Object Properties dialog box, click Home > Object Properties.
  4. From the Object Properties dialog box, click on the Text Box tab.
  5. Check the Vertical Writing option, click on the English Down icon, and click Set.
  6. The text inside the text box will be arranged vertically.

Note

English Down and English Up

If you select the Vertical Writing option, you can specify whether to turn Latin characters and numbers on their sides or to stack them.

  • English Down

  • If you select the Vertical Writing option, this setting rotates Latin characters and numbers by 90 degrees.

  • English Up

  • If you select the Vertical Writing option, this setting stacks Latin characters and numbers on top of one another.

Alternative Symbols

If you select the Vertical Writing option, symbols such as the period, comma, and quotation mark are replaced with their vertical alternatives. In other words, the period mark is replaced by the Asian full stop (¡£), the comma mark by the vertical comma, the single quotation marks by the hook brackets (¡¸¡¹), and the double quotation marks by the double hook brackets (¡º¡»).

Alignment

If you select the Vertical Writing option, text typed into the text box is entered from right to left.

The basis for Vertical Alignment is the width (horizontal) of the text box. Top is horizontal right, Bottom is horizontal left, and Center is horizontal middle.

Caution

No Automatic Text Box Width Increase

If you select the Vertical Writing option, the width of the text box does not increase automatically even with continuous entry of text. Text entered beyond the width of the text box is not visible. In order to make the whole text visible, you must increase the width of the text box manually.
