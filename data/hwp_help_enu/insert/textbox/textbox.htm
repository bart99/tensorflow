Text Box
 

Insert > Text Box

The Text Box is used to insert a long title into a multiple column document, and also to insert a boxed summary in the middle of the body.

You can easily adjust a text box by changing its position and size, fill effect for the inside, and also modify the shape and border color.

Steps

  1. Click Insert > Text Box > Horizontal Text Box .
  2. Point to the edit window. The pointer changes to a cross (+).
  3. Point to a location to insert the text box. Drag and release when the desired size is achieved.
  4. A rectangular text box is formed with the cursor placed inside of it.
  5. In the text box, type Hwp is the new standard in Korean word processors.

    If the text typed in the box exceeds the size of the box, the height of the text box increases automatically.

  6. From the Format dialog box, set the content inside the text box as a block and adjust the font and font size settings.

Note

Come out of Text Box

Once you have finished typing text in the text box, point to any edit window area outside the text box and left click once to exit from the text box. Alternatively, click SHIFT+ESC to exit from the text box.

Modify Text Box

To open the Object Properties dialog box, select a text box and click Home > Object Properties or double-click on a text box border. From the Object Properties dialog box, you can adjust the size and position of the text box, change the border style, and apply a fill effect.

Insert with Shortcut

Click Insert > Text Box > Horizontal Text Box. The mouse pointer changes to a cross. In this state, use the following shortcuts to insert a text box without using the mouse. Here, the standard text box size is 30mm¡¿30mm.

Shortcut Description
F or ENTER A text box is inserted at the object position specified in New Drawing Style.
D A text box set as In Line with Text is inserted.
S A text box set as Square and Paragraph is inserted.
A A text box set as Square and Page is inserted.
V A text box set as Square and Paper is inserted.
C A text box set as Top and Bottom and Paragraph is inserted.
X A text box set as Top and Bottom and Page is inserted.
Z A text box set as Top and Bottom and Paper is inserted.

Series of Text Boxes

To insert two or more text boxes in succession, under the Drawing Tools tab, click on the Horizontal Text Box icon twice. The Horizontal Text Box icon remains pressed and the mouse pointer stays as a cross (+) after inserting a text box, enabling you to insert additional text boxes in succession.

To exit from successive text box insert mode, press the ESC key or click on the Horizontal Text Box icon once more.
