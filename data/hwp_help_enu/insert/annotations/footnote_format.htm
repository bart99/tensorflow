Note Style: Footnote Styles
 

References > Footnotes > Footnote/Endnote Styles

In the Footnote Styles tab of the Note Style dialog box, set footnote numbering style, type of the separator between the body text and footnotes, margins, numbering method, and footnote position.

Steps

  1. To change footnote styles in a document with footnotes, click References > Footnotes > Footnote/Endnote Styles.
  2. In the Note Style dialog box, select the Footnote Styles tab.
  3. Select *, **, *** in Numbering Styles and click Set.
  4. The footnote numbers and footnote content numbers in the body text change as follows. 1) changes to *, 2) to **, and 3) to ***.

More Information

Footnote Styles Tab of Note Style Dialog Box

Number Format

Select a character style for footnote numbers. If you select a fore/back character, it will be automatically inserted with numbers.

  • Numbering Styles

  • You can set automatically increasing numbers or repeatedly used characters as footnote numbers.

  • Symbol

  • Display the symbol to use as the footnote number If you selected Symbol from the Numbering Styles list. The default symbol is ¡Ú. You can select a different symbol by clicking Symbol.

  • Fore Character/Back Character

  • To use before or after numbers, type a character like (', '). A Fore/back character will not be used if you leave the boxes empty.

  • Insert Separator

  • Draw a separator between the body text and footnotes. You can set Length, Type, Weight, and Color of the separator.

Margins

Set distance between the body text, separator, and footnotes.

  • Above Separator

  • Set distance between the body text and the footnote separator if footnotes and the body text meet. The default value is 2mm.

  • Below Separator

  • Set distance between the footnote separator and footnotes. The default value is 2mm.

  • Between Footnotes

  • Set distance between footnotes. The default value is 3mm.

Numbering

Set range for linking and numbering footnotes.

  • Continued from Previous Section

  • You can number footnotes with numbers that continue from the previous section, regardless of page break or section break in the document.

  • Restart Each Page

  • Start the numbering of footnotes on every page with 1.

  • Restart from Current Section

  • You can number footnotes with new numbers from the current section, regardless of the footnote numbers of the previous section.

Footnote Content Number Properties

Select Normal or Smaller for the numbering style in footnotes.

Footnote Vertical Position

Specify where to place the footnotes when the body text does not fill a page.

  • Above Footer

  • Type footnote content right above the footer area set in page setup. The footnote expands upward to the body text when its content increases and thus the body text area decreases.

  • Below Text

  • Type footnote content right below the body text. The footer content gets pushed downward when the body text increases.

Position for Multiple Column

Specify under which column to type footnote content in a document with multiple columns.

  • Below Each Column

  • Type footnote content under the column that has the footnote number.

  • Below Left Column

  • Type footnote content under one column, regardless of the column editing in the body text.

  • Below Right column

  • Type footnote content under the column on the far right on the page that has the footnote number.

Apply to

Set range for applying the footnote style you have set so far to the active document. You can use one footnote style for the entire document or different footnote styles for different sections.

In most cases, you can select either Whole Document or New Section.

  • To create a new section at the pointer and apply a new footnote style to the new section, select New Section.
  • You can select Current Section in Apply to if the document is divided into two or more sections.

Note

Default Footnote Number

The default footnote number looks like 1). The numbering style is 1, 2, 3 and the back character is ). There is no fore character.

Footnote numbers in the body text are inserted as superscript and footnote numbers in footnotes are inserted in normal size.

Restart Symbol Number Style on Each Page

Footnote numbers that use symbols (*, ¢Ó, ¢Ô, ¡×, *, **, ***, #, ##, ###, +, ++, +++, Symbol) are not appropriate for expressing high numbers. So if you select a symbol in Numbering Styles, Page Number > Restart Each Page will be automatically set and footnotes will be numbered starting with 1 on every page.

Footnote Style Is Section Properties

You can use different footnote number styles for different sections. To use different footnote styles in one document, click Page Layout > Section Break first.

Footnote Position in Multiple Columns

You can place footnotes under each column, all columns as one column, or the column on the far right in a document with multiple columns.

You have to select One for all column for footnote position when using Distribute or Parallel.
