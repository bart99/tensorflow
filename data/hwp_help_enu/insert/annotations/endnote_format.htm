Note Style: Endnote Styles
 

References > Footnotes > Footnote/Endnote Styles

In the Endnote Styles tab of the Note Style dialog box, set endnote numbering style, type of the separator between the body text and endnotes, margins, numbering method, and endnote position.

Steps

  1. To change endnote styles in a document with endnotes, click References > Footnotes > Footnote/Endnote Styles.
  2. In the Note Style dialog box, select the Endnote Styles tab.
  3. Select 가, 나, 다 in Numbering Styles and click Set.
  4. The endnote numbers and endnote content numbers in the body text change as follows. 1) changes to 가, 2) to 나, and 3) to 다.

More Information

Endnote Styles Tab of Note Style Dialog Box

Number Format

You can select a character style for endnote numbers. On selecting a fore/back character, it will be automatically inserted with numbers.

  • Numbering Styles

  • You can set automatically increasing numbers or repeatedly used characters as endnote numbers.

  • Fore Character/Back Character

  • To use before or after numbers, type a character like (', '). A Fore/back character will not be used if you leave the boxes empty.

  • Insert Separator

  • Draw a separator between the body text and the endnotes. You can set Length, Type, Weight, and Color of the separator.

Margins

Set distance between the body text, separator, and endnotes.

  • Above Separator

  • Set distance between the body text and the endnote separator if endnotes and the body text meet. The default value is 2mm.

  • Below Separator

  • Set distance between the endnote separator and endnotes. The default value is 2mm.

  • Between Endnotes

  • Set distance between endnotes. The default value is 3mm.

Numbering

Set range for linking and numbering endnotes.

  • Continued from Previous Section

  • Number endnotes with numbers that continue from the previous section, regardless of page break or section break in the document.

  • Restart from Current Section

  • Number endnotes with new numbers from the current section, regardless of the endnote numbers of the previous section.

Endnote Content Number Properties

Select Normal or Smaller for the numbering style in endnotes.

Endnote Position

All endnotes are displayed at the End of Document or End of Section.

  • End of Document

  • Display all endnotes at the end of the active document.

  • End of Section

  • In a document comprised of multiple chapters, divide each chapter into sections and display endnotes at the end of each chapter.

Apply to

Set range for applying the endnote style you have set so far to the active document. You can use one endnote style for the entire document or different endnote styles for different sections.

In most cases, you can select either Whole Document or New Section.

  • You can create a new section at the pointer and apply a new endnote style to the new section by selecting New Section.
  • You can select Current Section in Apply to if the document is divided into two or more sections.

Note

Default Endnote Number

The default endnote number looks like 1). The numbering style is 1, 2, 3 and the back character is ). There is no fore character.

Endnote numbers in the body text are inserted as superscript and endnote numbers in endnotes are inserted in normal size.

Endnote Style Is Section Properties

You can use different endnote number styles for different sections. To use different endnote styles in one document, click Page Layout > Section Break first.

Symbol Number Style

You can use symbols (*, †, ‡, §, *, **, ***, #, ##, ###, +, ++, +++, Symbol) in Numbering Styles for footnotes. However, you cannot use such symbols for endnotes. The reason is that symbols are not appropriate for expressing high numbers.
