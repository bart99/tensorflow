Footnote/Endnote Styles
 

References > Footnotes > Footnote/Endnote Styles

You can set footnote/endnote numbering styles, type of the separator between the body text and footnotes/endnotes, margins, numbering method, and footnote position in the Note Style dialog box by clicking References > Footnotes > Footnote/Endnote Styles.

Steps

  1. To change footnote styles in a document with footnotes, click References > Footnotes > Footnote/Endnote Styles.
  2. In the Note Style dialog box, select the Footnote Styles tab.
  3. Select *, **, *** in Numbering Styles and click Set.
  4. The footnote numbers and footnote content numbers in the body text change as follows. 1) changes to *, 2) to **, and 3) to ***.

More Information

Note Style Dialog Box

Footnote Styles Tab

In the Footnote Styles tab, set footnote numbering style, type of the separator between the body text and footnotes, margins, numbering method, and footnote position.

Endnote Styles Tab

In the Endnote Styles tab, set endnote numbering, type of the separator between the body text and endnotes, margins, numbering method, and endnote position.

Note

Difference Between Footnote and Endnote

The difference between Footnote and Endnote is their position.

Footnotes are located at the bottom of the page where footnote numbers are inserted. Footnotes are useful because they are located on the same page as the text they refer to.

Endnotes are located at the end of a chapter or document regardless of the pages where their numbers are inserted. The strength of endnotes is that you can see all of them in one place.
