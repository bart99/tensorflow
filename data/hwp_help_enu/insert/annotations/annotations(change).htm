Convert Footnotes/Endnotes
 

References > Footnotes > Footnote°ÍEndnote

You can change all footnotes to endnotes and all endnotes to footnotes in the active file. Or, change footnotes to endnotes and endnotes to footnotes at the same time.

You can rectify the mistake by using this function if you accidentally insert endnotes instead of footnotes.

Steps

  1. Click References > Footnotes > Footnote°ÍEndnote in a file with footnotes and endnotes.
  2. In the Convert Footnotes/Endnotes dialog box, select Conversion Method > Swap Footnotes with Endnotes and click Convert.
  3. The footnotes and endnotes in the file will change to endnotes and footnotes respectively.

More Information

Convert Footnotes/Endnotes Dialog Box

Conversion Method

  • All Footnotes to Endnotes

  • Change all Footnote in the current file to Endnote. You cannot select this option in a document only with Endnote.

    If inserting footnotes in the body, specify where in the body to insert the footnotes.

  • All Endnotes to Footnotes

  • Change all Endnote in the current file to Footnote. You cannot select this option in a document only with Footnote.

  • Swap Footnotes with Endnotes

  • Change all Footnote to Endnote and all Endnote to Footnote in the current file.

    You can select this option in a document only when both Footnote and Endnote are present.

Note

Apply to

Insert Footnote and Insert Endnote apply per section but Convert Footnotes/Endnotes applies to the entire active file.
