Save Notes
 

References > Footnotes > Save Notes

This feature allows you to save the Footnote or Endnote from the body so that it can be opened at a later point in time. You can check footnotes that are split into the bottom of pages and end of a document by saving them as a file. You can also gather them and use for reference.

Steps

  1. Click References > Footnotes > Save Notes in a document with footnotes and endnotes.
  2. In the Save Notes dialog box, type a save location and file name in the File Name box or click Path to specify path.
  3. To save both footnotes and endnotes in Notes to Save, select Insert Footnote and Insert Endnote.
  4. To create (*.hwp) with the footnotes saved with the specified name in the specified location, click Save

More Information

Save Notes Dialog Box

File Name

Specify the file name and path for the file in which to save footnotes and endnotes. To select a path in the Save Notes, click Path dialog box.

Notes to Save

  • Footnote

  • Save all footnotes in the active file.

  • Endnote

  • Save all endnotes in the active file. The footnote file saved here gathers footnotes and endnotes in the order of insertion without distinguishing them.
