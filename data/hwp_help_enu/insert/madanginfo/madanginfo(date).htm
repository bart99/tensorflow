Field: Creation Date
 

Insert > Quick Parts > Field

It inserts today's date in the selected format at the cursor. By inserting Creation Date in a Document Template, you can have the system's current date entered in the date information area each time the template is used.

Steps

  1. Click Insert > Quick Parts > Field.
  2. In the Field dialog box, select the Creation Date tab.
  3. In the Creation Date tab, select the country and the date format.
  4. Click Insert.

More Information

Creation Date Tab of the Field Dialog Box

Nation

Select which country's date format to use.

Date Format List

Select the date display format to use. Select a country from Nation, and a list of date formats used in that country appears under Date Format List.

Note

Current Date

By inserting Creation Date in a Document Template, you can have the system's current date entered in the date information area each time the template is used.

Modify Date Code

To change the format of a date inserted in the body, place the cursor inside the date code and right click and select Modify Date Code.
