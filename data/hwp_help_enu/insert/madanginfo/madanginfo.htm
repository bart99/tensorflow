Field
 

Insert > Quick Parts > Field

Even beginners of Hwp can compose appealing documents quickly and with ease by using template (*.hwt) in Document Template. When creating a document template (*.hwt), you may insert User Info, Creation Date, Summary, and File Name/Path fields by clicking Insert > Field.

Steps

To create and add report template to document templates package

  1. Click File > New to create a new document window.
  2. Type Report at the cursor, press ENTER, and type Reported by on the new line.
  3. Press ENTER and type Date on the new line.
  4. Add Report Title and Report Content in the same manner.
  5. Position the cursor at the end of the Reported by line, press the SPACEBAR once, and click Insert > Quick Parts > Field.
  6. From the Field dialog box, select the User Info tab, select User Name, and click Insert.
  7. Position the cursor at the end of the Date line, press the SPACEBAR once, and click Insert > Field.
  8. From the Field dialog box, select the Creation Date tab, select the desired date format, and click Insert.

To save document templates

  1. Click File > Save.
  2. From the File Type drop-down list, select Hwp Template (*.hwt).
  3. Specify the Document Templates Package for the new document template in the Save Location field.
  4. Type Report.hwt in the File Name text box and click Save. The newly added report has is now saved in the designated folder.
  5. To open the newly added report as a document template, click File > New > Document Templates and select Everyday Document under Document Templates Package.

Note

Hwp Template File

Files saved as a Hwp Document (*hwp) cannot be used as a document template. In order to use a file as a document template, it must be saved as a Hwp Template (*.hwt) file.
