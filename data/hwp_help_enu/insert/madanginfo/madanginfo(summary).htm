Field: Summary
 

Insert > Quick Parts > Field

You can insert a document summary that includes the title, subject, author, date, and keywords.

The Summary under Field, which is used when creating a Hwp template (.hwt), shows the information entered in the Summary tab under File > Info. Only the field names are shown for Summary fields that have not been filled in by the user.

Steps

  1. Click Insert > Quick Parts > Field.
  2. In the Field dialog box, select the Summary tab.
  3. From Document Summary under the Summary tab, click on an item you wish to insert into the template and click Insert.
  4. To change the document summary information, click Edit under the Summary tab.

Note

Automatic Summary Information Fill

When a user clicks File > New > Document Templates and opens an existing document template that has been inserted with Summary, the summary information entered by the original creator of the document template is not shown.

Instead, the author information entered under Options > User Info on the template user's system is retrieved and filled in automatically. Title, however, is left as entered by the original creator of the document template.
