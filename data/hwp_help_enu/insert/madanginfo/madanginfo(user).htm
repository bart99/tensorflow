Field: User Info
 

Insert >Quick Parts > Field

You can insert your information into the document including name, company name, department name, telephone number, address, and e-mail address.

The User Info under Field, which is used when creating a Hwp template (.hwt), shows the information entered in the User Info tab under File > Options. Only the field names are shown for User Info fields that have not been filled in by the user.

Steps

  1. Click Insert > Quick Parts > Field.
  2. In the Field dialog box, select the User Info tab.
  3. From User Info List under the User Info tab, click on an item you wish to insert into the template and click Insert.
  4. To change the user info, click Modify User Info under the User Info tab.

Note

Automatic User Info Fill

When a user clicks File > New > Document Templates and opens an existing document template that has been inserted with User Info, the user info entered by the original creator of the document template is not shown.

Instead, information entered under Options > User Info on the template user's system is retrieved and filled in automatically. Therefore, User Info fields under Options must be filled out accurately to ensure proper use of document templates.

Modify User Info

To change the user information, click on the Edit icon under the User Info tab.
