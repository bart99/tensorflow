Equation
 

Insert > Equation

Equation Editor of Hwp supports both easy-to-use visual editing and script as equation entering method.

If you use Equation Editor, you can write any kind of math expressions ranging from simple arithmetic expressions to complicated equations by using equation templates and commands.

Steps

To enter an equation in the equation edit window

  1. If you click Insert > Equation, the Equation Editor dialog box appears.
  2. Click the Fractions icon in the Equation toolbox.
  3. When a fraction is displayed in the equation edit window, type 1 as the numerator.
  4. Press TAB to move to the box where you can type a number for the denominator.
  5. Type 2 as the denominator.
  6. Click the Insert icon in the Equation toolbox or press SHIFT+ESC to finish editing the equation.
  7. The equation will be displayed in the body text.

To type an equation in the script window

  1. If you click Insert > Equation, the Equation Editor dialog box appears.
  2. Type 1 over 2 in the script window at the bottom of the equation edit window.
  3. The fraction will be displayed as 1/2 in the equation edit window.
  4. Click Insert in the Equation toolbox or press SHIFT+ESC to finish editing the equation.
  5. The equation will be displayed in the body text.

To set equation object properties

  1. Select an equation inserted in the body text and press P or click Home > Object Properties.
  2. In the Equation Properties dialog box, select the Equation tab.
  3. Select 12 for Text Size and Blue for Font Color in Font.
  4. Select Selected Equation Object from the Apply to list in Apply to and click Set.
  5. The text size and font color of the equation in the body text will change to 12pt and blue.

To edit an equation

  1. Double-click an equation in the body text or click Edit in the Equation Properties dialog box.
  2. In the Equation Editor dialog box, edit the equation by using the Equation toolbox and the equation edit window or script window.
  3. Click Insert in the Equation toolbox or press SHIFT+ESC to finish editing the equation.
  4. The equation will be displayed in the body text according to the modification you made.

More Information

Equation Editor Dialog Box

Equation Toolbox

Various function symbols, equation templates, equation symbols, and equation commands are provided. If you use various equation icons in the Equation toolbox, you can insert equations without typing complicated math symbols or commands.

Equation Edit Window

You can write an equation by selecting an equation template in the Equation toolbox and typing required values in the equation edit window.

Script Window

Create an equation by typing an equation command. If you type an equation command in the script window, you can check the result immediately in the equation edit window.
