Equation Fonts and Basic Functions
 

Insert > Equation

If you type English in Equator Editor, it will be typed in Italic by default. Font format is restricted in equations.

Note

English Font Command

If you type English in the equation edit window in Equator Editor, it will be typed in Italic by default.

To use Roman instead of Italic, type rm, the command for converting to Roman in the Enter Command box of the Equation dialog box or select RM from the list and type an equation in the equation edit window.

To type in bold, type bold in the Enter Command box or select bold from the list and type an equation in the equation edit window.

  • rm: Roman
  • it: Convert Roman back to Italic
  • bold: Bold
  • rmbold: Roman+Bold

Remove Font Properties

If you press TAB while typing in bold in the equation edit window, the current font properties will be removed.

Basic Function

Display basic functions of equations in Roman instead of converting them to Italic.

  • The following basic functions are always displayed in Roman (case sensitive).

  • sin cos coth log tan cot lg sec cosec

    max min csc arcsin lim LIM arccos arctan

    exp Exp arc sinh det gcd cosh tanh mod

  • The following are commands displayed in Roman.

  • if for and hom ker deg arg dim Pr or

Caution

Enter Function in Enter Command Box

If you type basic functions in the Enter Command box of the Equation toolbox instead of the equation edit window, they will not be recognized as basic functions.
