Description of Equation Commands
 

Insert > Equation

The following are a list of commands that can be used in Equation Editor and how to use them.

More Information

Basic Command

Command Description Example Result
TIMES Display the multiplication sign. 2 times 5=10
OVER Displays a fraction. Fractions are aligned in the center by default. To align them on the right or left, use SPACEBAR. 1 over 2
ATOP This is the same as the OVER command but skips the horizontal line that indicates a fraction. x atop y
SQRT Displays a root. sqrt 2
^ Displays superscript. Type ^ in the equation edit area or type SUP in the command window. E=mc^2
_ Displays subscript. Type _ in the equation edit area or type SUB in the command window. H_2 O
INT, OINT Displays the integral sign. int _1 ^2 {3x^2}dx
SUM Displays Sigma. sum_{x=0} ^{inf}
BIGG Symbol Zoom out the next symbol. {a+b} over {a-b} bigg / {x+y} over {x-y}
lim, Lim Display the limit sign. When typing lim and Lim, make sure to match case. y= lim _{x -> 0} {{1} over {x}}
UNION, INTER, PROD Display the set symbol. If you add SMALL in front of a command, the set symbol will be entered without subscript or superscript. U=(A SMALLUNION B) SMALLINTER C
NOT If you add not in front of a character, a diagonal line will be drawn on the character. not =
REL This function expresses the correlation between two items. You can insert a relational expression above or below the arrow that connects two items. A REL -> {+2} {-5} B
BUILDREL This is similar to REL. You can skip the content below the arrow. A BUILDREL <-> {+2} B
CASES Displays a brace ({) that indicates multiple lines. Braces will be displayed automatically zoomed out depending on the situation. cases {2x+y=4 # 3x-4y=-1}
PILE, LPILE, RPILE You can select from center (PILE) alignment, left (LPILE) alignment, and right (RPILE) alignment based on the character above.    
EQALIGN Adjusts the vertical position of an equation based on &, the column alignment mark.    
CHOOSE or BINOM Enter combining symbols. Total Items CHOOSE Selected Item or BINOM Total Items Selected Item  
MATRIX Enters a matrix.
You can enter a matrix per line or per column by typing col after the matrix.
When you enter a matrix per column by typing col, you can also specify alignment position by typing lcol (left alignment) or rcol (right alignment) instead.		 matrix{a_{1} &b_{1}&c_{1} # a_{2}&b_{2}&c_{2} # a_{3}&b_{3}&c_{3}}
PMATRIX If you use pmatrix instead of matrix, a matrix will be displayed with a parenthesis (()) before and after. pmatrix{a_{1} &b_{1}&c_{1} # a_{2}&b_{2}&c_{2} # a_{3}&b_{3}&c_{3}}
BMATRIX If you use bmatrix instead of matrix, a matrix will be displayed with a bracket ([]) before and after. bmatrix{a_{1} &b_{1}&c_{1} # a_{2}&b_{2}&c_{2} # a_{3}&b_{3}&c_{3}}
DMATRIX If you use dmatrix instead of matrix, a matrix will be displayed with a pipe (|) before and after. dmatrix{a_{1} &b_{1}&c_{1} # a_{2}&b_{2}&c_{2} # a_{3}&b_{3}&c_{3}}

Character Decoration Command

You can decorate a character or word in various ways by typing a simple command in front of the character or word. Type a command before a character.

ACUTE, GRAVE, DOT, DDOT, BAR, VEC, DYAD, and UNDER are some examples.

Type Result Type Result
acute A bar A
grave A vec A
dot A dyad A
ddot A under A
hat A arch A
hat AA arch AA
hat AAA arch AAA
check A tilde A
check AA tilde AA
check AAA tilde AAA

Symbol Summary

You can edit equations by inserting simple commands in characters or words.

Some examples are ~, `, { }, " ", #, and &.

Type Result
~ Empty Space (Spacebar Effect)
' 1/4 of an empty space
{   } Merging of multiple items
"   " Merging of a word with 9 or more characters
# Change Lines (ENTER Effect)
& Align Vertical Columns (TAB Effect)

Caution

Match Case

Most commands are not case-sensitive. However, the commands whose first letter is written in uppercase must be typed in uppercase.
