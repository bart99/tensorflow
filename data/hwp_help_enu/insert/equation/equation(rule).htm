Considerations for Editing Equations
 

Insert > Equation

Equation Editor has its own rules unlike regular document editors. To use Equation Editor efficiently, learn the following basic rules.

More Information

Command Box

To create an equation by typing an equation command without using an equation template, press \ on the keyboard or click the arrow of the Enter Command in the Equation toolbox.

When the pointer blinks in the equation edit window, create an equation by typing the command for it and finish entering the command by pressing ENTER or ESC.

Script Window

Create an equation by typing the command for it.

Note

Enter Korean/Chinese

When you type Korean or Chinese in the Equation Editor dialog box, select the script window and type in the languages by changing keyboard type.

Move Item

Press TAB to move to the next item in the equation window. Press SHIFT+TAB to move to the previous item.

Remove Properties

Press TAB also to convert superscript or subscript to regular characters or bold texts to regular texts.

Change Lines

To change lines in the equation edit window, press ENTER like in the normal edit screen.

When you type an equation in the script window or the dialog box with the equation entering method of earlier versions, you have to type # at the end of a line to change lines.
