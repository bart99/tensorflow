Equation Toolbox
 

Insert > Equation

If you click Insert > Equation, the Equation Editor dialog box appears.

If you use various equation icons in the Equation toolbox of the Equation Editor dialog box, you can create equations by selecting a required equation template and typing values in empty boxes. You do not have to type equation commands.

Note

Edit Icon

The Equation toolbox appears at the top of the Equation Editor dialog box. You cannot edit it as an additional icon like icons of other toolboxes.

Command Box

Once you press \ on the keyboard in the equation edit window, the pointer will move inside the Enter Command box of the Equation toolbox. You can also create an equation by typing or selecting an equation command here.

If you type a command in the Enter Command box and press ENTER, the equation of the command will be immediately displayed in the equation edit window.

Open MathML File

As one of the applications of XML, MathML (Mathematical Markup Language) is a standard language for the meaning and expression of mathematical expressions. For more information, refer to the W3C website (http://www.w3.org/TR/MathML/).

If you select a MathML file (*.mml) in the Open MathML File dialog box and open it, the equations of the MathML file will be displayed in the equation edit window and script window of the Equation Editor dialog box.
