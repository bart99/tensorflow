Wordshop
 

Insert > Illustrations > Wordshop

You can decorate characters by curving them or applying outline, face fill, shadow, or rotation effects with the Wordshop function.

Wordshop is a drawing function in Hwp. You can edit the results of Wordshop in documents.

Steps

To create Wordshop

  1. Click Insert > Illustrations > Wordshop.
  2. In the Create Wordshop dialog box, type Hancom in the Contents box.
  3. Select Gulim in Fonts and click Set.
  4. A Wordshop object will be inserted in the edit screen.

To modify Wordshop

  1. Select the Wordshop object inserted in the edit screen and click Edit Content in the Wordshop tab.
  2. In the Modify Wordshop dialog box, modify the content of the Contents box.
  3. Click Wordshop Shape. In the character palette, select Wave 1 and click Apply.
  4. The change will apply to the edit screen while the Modify Wordshop dialog box stays open.
  5. Once all the changes are made, click Set to close the dialog box.

  You can also modify settings in the Wordshop tab of the Object Properties dialog box that displays on double-clicking a Wordshop object inserted in the edit screen.

More Information

Create Wordshop and Modify Wordshop Dialog Box

Contents

You can type in text to convert into a Wordshop object.

Font

You can use all fonts provided by Hwp and Windows in Wordshop. You cannot select different fonts for different languages.

Property

Most fonts have four font properties: Regular, Italic, Bold, and Bold Italic.

You can assign font properties in the Adjust list by selecting an English font which allows you to assign properties (example: Arial, Verdana, etc).

  To display English fonts on the Font list of Wordshop, you have to select Register Latin Fonts as Representative Fonts of Register Font in Options > Fonts. Exit from Hwp and restart it to add the English fonts to the list.

Line Spacing

Set line spacing when you type two or more lines. The default value is 120. Value can be set from 50 through 500.

Text Spacing

Adjust the distance between characters. The adjusted distance applies to each character. The default value is 100. Value can be set from 50 through 500.

Wordshop Shape

You can select from 55 character shapes by clicking Wordshop Shape. Select from various shapes including wave, butterfly, circle, semicircle, and crescent moon.

Set Button

Apply the settings and close the dialog box.

Apply Button

Apply the changed settings to the document while the dialog box stays open.
