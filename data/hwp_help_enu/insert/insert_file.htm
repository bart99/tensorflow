Contents from Document
 

Insert > Contents from Document

This feature is used to insert contents from the the selected document file at the pointer in the active document.

You can combine two or more document files using this function.

Steps

  1. Click Insert > Contents from Document while editing the original document Document 1.
  2. In the Insert Document dialog box, select Document 2, a document to insert in the active document.
  3. Select Preserve Font, Preserve Paragraph, and Keep Page Format and click Open.
  4. All the content of Document 2 will be inserted at the pointer with font, paragraph, and page format maintained.

More Information

Insert Document Dialog Box

Preserve Font

Insert a document with its font format maintained.

Insert the target document with its font format maintained even if the original document has a style with the same name.

Preserve Paragraph

Insert a document with its paragraph format maintained.

Insert the target document with its paragraph format maintained even if the original document has a style with the same name.

Keep Style

Insert a document with copy(number) added to the end of its style name.

Insert the target document with copy(number) added to its style name if the original document has a style with the same name.

Keep Page Format

Create sections before and after the current pointer position and insert a file between the sections with its page format maintained.

Note

Difference Between Open and Document

You can open a document in a new document window (document tab) if you use Open. If you use Contents from Document, you will insert a document at the pointer in the active document and combine the content of the two documents.

File Name

When you use Contents from Document, the file name of the active document does not change.

Style of Inserted File

Once you use Contents from Document, all content of the inserted document including its body text will be included in the active document. The Styles registered in the inserted document will be added to the Styles list of the active document. When the active document and the inserted document have a style with the same name, the style of the active document will be kept while the style of the inserted document will be ignored.

Select Block and Insert Document

If you use Contents from Document when part of the active document is selected as a block, the block will be overwritten by the content of the inserted document.

If you type new content when a block is selected the block gets overwritten by the new content.

Document Password

When you insert a document with a Document Password by using Contents from Document, you will be asked for the password but the password will not be set for the active document.
