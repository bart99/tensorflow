AutoText
 

Insert > Quick Parts > AutoText

You can register abbreviations for frequently used text and enter the entire text by typing the registered abbreviations when needed with Auto Text. AutoText is useful for long or complicated words.

You can register all content you can create in Hwp including not only simple words but tables and pictures.

More Information

AutoText Submenus

Frequently used info is provided in the submenus of AutoText by default. The following information will be entered if you select each menu.

Menu Feature
Last Saved on Insert the last saving date in the form of text at the pointer.
Last Saved on is brought from Normal > Created on in File > Info.
Last Saved by Insert the name of the person who saved the file last in the form of text.
Last Saved by is brought from Normal > User Name in File > Options.
Creation Date Insert the last saving date in the form of code at the pointer.
Author Insert author in the form of text at the pointer.
Author is brought from User Info > User Name in File > Options.
Author, Current Page, Creation Date Insert the author, current page, and creation date in the form of text at the pointer.
Document Title Insert the document title in the form of text at the pointer.
If a document title is typed in Summary > Title of File > Info, it gets inserted in the body text.
Total Pages Insert the total page number of the document at the pointer.
Use this function in a position where you want to insert the total page number after finishing the document.
File Name Insert the file name of the document in the form of code at the pointer.
File Name and Path Insert the file name and path in the form of code at the pointer.
If you save the document file in a different folder or path, the folder or path will be displayed.
Current Page Number Insert the current page number in the form of code at the pointer.
Current Page/Total Pages Insert the current page/total pages in the form of code at the pointer.

Note

AutoText Type

There are two types of AutoText in Hwp. They are AutoText and Formatted AutoText.

  • AutoText

  • You can only add text to this AutoText.

    You can add or edit AutoText and save or open AutoText as an AutoText file (*.ido) by using Add AutoText or AutoText Content.

  • Formatted AutoText

  • You can add everything that is used in Hwp including text, tables, and pictures to this AutoText.

    You can add AutoText only by clicking Add AutoText. You cannot add Formatted AutoText to the AutoText dialog box that displays when you click AutoText Content.
