Add AutoText
 

Insert > Quick Parts > AutoText > Add AutoText

With this feature, you can register frequently used sentences and words or symbols that are hard to insert as AutoText for your convenience.

Steps

To add Formatted AutoText

  1. Type I can do it in the edit window and select it as a block.
  2. Click Insert > Quick Parts > AutoText > Add AutoText.
  3. In the Add AutoText dialog box, the block will be registered in Original Word and the first letter in the block in Abbreviation.

    In this example, icd will be displayed in Abbreviation and I can do it in Original Word.

  4. Select Preserve Text Properties and click Add.
  5. Click Insert > Quick Parts > AutoText > AutoText Content. In the AutoText dialog box, select the Formatted AutoText tab.
  6. Check that icd and I can do it are added to Abbreviation and Content Description respectively on the AutoText list.

To add AutoText

  1. Type Hancom in the edit window and select it as a block.
  2. Click Insert > Quick Parts > AutoText > Add AutoText.
  3. In the Add AutoText dialog box, the block will be registered in Original Word and the first letter in the block in Abbreviation.

    In this example, Han will be displayed in Abbreviation and Hancom in Original Word.

  4. Change the content of Abbreviation to HC, select Discard Text Properties, and click Add.
  5. Click Insert > Quick Parts > AutoText > AutoText Content. In the AutoText dialog box, select the AutoText tab.
  6. Check that HC and Hancom are added to Abbreviation and Original Word respectively on the AutoText list.

To add AutoText in the AutoText Content dialog box

  1. Click Insert > Quick Parts > AutoText > AutoText Content.
  2. In the AutoText dialog box, click the Add AutoText icon in the AutoText tab.
  3. In the Add AutoText dialog box, type abbreviation and original word and click Set.

    In this example, type Han in Abbreviation and Hancom in Original Word.

  4. Check that Han and Hancom are added to Abbreviation and Original Word on the AutoText list of the AutoText tab.

More Information

Add AutoText Dialog Box

Abbreviation

An abbreviation is the name of AutoText. The first letter of an original word is added as its abbreviation by default.

Comment

If the AutoText to add is Formatted AutoText, type descriptions about the AutoText.

If the AutoText to add is just AutoText, type the original word.

Preserve Text Properties

You can add the AutoText to the AutoText list of the Formatted AutoText tab of the AutoText dialog box with the properties of the current characters maintained.

You can add everything that is used in Hwp including text, tables, and pictures to Formatted AutoText.

Discard Text Properties

You can add the AutoText to the AutoText list of the AutoText tab of the AutoText dialog box without keeping the properties of the current font properties.

Note

AutoText Type

There are two types of AutoText in Hwp. They are AutoText and Formatted AutoText.

  • AutoText

  • You can only add text to this AutoText.

    You can add or edit AutoText and save or open AutoText as an AutoText file (*.ido) by using Add AutoText or AutoText Content.

  • Formatted AutoText

  • You can add everything that is used in Hwp including text, tables, and pictures to this AutoText.

    You can add AutoText only by clicking Add AutoText. You cannot add Formatted AutoText to the AutoText dialog box that displays when you click AutoText Content.

Limit on Abbreviation

You can type up to 10 characters for an abbreviation.

  • AutoText

  • If there is already a one-character abbreviation with the same character you are trying to use, two characters will be automatically entered as an abbreviation. The AutoText content that comes first on the AutoText list will apply if there are two or more pieces of AutoText with the same abbreviation.

  • Formatted AutoText

  • The old AutoText will be overwritten by the new AutoText if the abbreviation that is being used is already there.

AutoText Range

You can create up to 1,024 pieces of AutoText and Formatted AutoText respectively. When the number of AutoText definitions exceeds 1,024, the oldest AutoText will be deleted when a new one is added.

Including Control Code

You cannot add the block as AutoText if it includes control codes.

Adding Table and Picture as Formatted AutoText

To add a table or picture as AutoText, select a table or picture and click Insert > Quick Parts > AutoText > Add AutoText. In the Add Formatted AutoText dialog box, add an abbreviation and description to the picture or table and add them as Formatted AutoText.

You can add a short description about the AutoText in the Original Word box while adding Formatted AutoText. The description typed here will not be inserted in the body text. The original word to be inserted in the body text is the block you selected.
