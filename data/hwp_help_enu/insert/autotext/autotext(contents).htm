AutoText Content <ALT+F3>
 

Insert > Quick Parts > AutoText > AutoText Content

You can modify, delete, insert, or save currently added AutoText. You can also bring it forward or send it backward.

Steps

To insert AutoText in the edit window

  1. Place the pointer in a position where AutoText has to be inserted and click Insert > Quick Parts > AutoText > AutoText Content.
  2. In the AutoText dialog box, select AutoText you want in the AutoText or Formatted AutoText tab.
  3. Click Insert. AutoText will be inserted at the pointer.

More Information

AutoText Tab of AutoText Dialog Box

Select AutoText to Enter

Lists of abbreviations and original words will be displayed.

The original word of the AutoText will be inserted at the pointer in the edit window if you select AutoText from the list and click Insert.

Details on Icons

Icon Name Description
Add AutoText Add the abbreviation and original word of new AutoText.
Edit AutoText Edit the abbreviation or original word of the AutoText you selected from the list.
Delete AutoText Delete the AutoText you selected from the list.
Move One Line Up Move the selected AutoText one line up on the list.
Move One Line Down Move the selected AutoText one line down on the list.
Open AutoText Open a saved AutoText file (*.ido).
Save AutoText Save AutoText content defined in the active file as a separate AutoText file (*.ido).

Formatted AutoText Tab of AutoText Dialog Box

Select AutoText to Enter

Descriptions about abbreviations and AutoText original words will be displayed on a list.

The original word of the AutoText will be inserted at the pointer in the edit window if you select AutoText from the list and click Insert.

Details on Icons

Icon Name Description
Edit AutoText Edit the abbreviation or descriptions of the Formatted AutoText you selected from the list. The original words of Formatted AutoText cannot be modified.
Delete AutoText Delete the Formatted AutoText you selected from the list.
Move One Line Up Move the selected Formatted AutoText one line up on the list.
Move One Line Down Move the selected Formatted AutoText one line down on the list.

Note

AutoText Type

There are two types of AutoText in Hwp. They are AutoText and Formatted AutoText.

  • AutoText

  • You can only add text to this AutoText.

    You can add or edit AutoText and save or open AutoText as an AutoText file (*.ido) by using Add AutoText or AutoText Content.

  • Formatted AutoText

  • You can add everything that is used in Hwp including text, tables, and pictures to this AutoText.

    You can add AutoText only by clicking Add AutoText. You cannot add Formatted AutoText to the AutoText dialog box that displays when you click AutoText Content.

No Identical Abbreviation

You cannot register the same abbreviation.

Save AutoText

AutoText is saved as an HWP.IDO file when you exit Hwp.

AutoText files are saved in the folder, \User\Hwp\60, under the User Data folder by default.

You can check the location of the User Data folder by clicking > About Thinkfree Office NEO Hwp > Folder Info.

Once defined, AutoText content can be used next time you use Hwp.
