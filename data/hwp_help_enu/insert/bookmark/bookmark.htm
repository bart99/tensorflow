Bookmark <CTRL+SHIFT+F5>
 

Insert > Bookmark

You can use Bookmark to mark various positions in the body text and return the pointer to the positions regardless of the current pointer position.

Steps

  1. Type Hwp in the body text and point to a position that is to be marked with the bookmark.

    Move the pointer in front of H.

  2. Click Insert > Bookmark.
  3. Hwp will be automatically entered in the Bookmark Name box of the Bookmark dialog box.

    If you want a different name, type it.

  4. Hwp will be added to the Bookmark List by clicking Insert and the dialog box will close.

More Information

Bookmark Dialog Box

Bookmark Name

Type the name of a bookmark to add.

Bookmark List

A list of bookmarks added to the active document will be displayed.

Sort Bookmark By

  • Name

  • Sort the bookmark list by name.

  • Position

  • Sort the bookmark list by bookmark position.

Insert Button

Insert the bookmark whose name you typed in the edit screen. Bookmarks will not be displayed in the edit window or print-out.

Cancel Button

Close the dialog box without saving changes.

Move Button

Move to the position where the bookmark you selected from the bookmark list is.

Details on Icons

Icon Name Description
Rename Bookmark Change the name of the bookmark you selected from the Bookmark List.
Delete Delete the bookmark you selected from the Bookmark List.

Note

Block Bookmark

The entire block will be remembered as a bookmark by using a bookmark when a block is selected. The bookmark will be selected as a block when you move it to the block bookmark. In the Bookmark dialog box, if you insert a block bookmark, it will be displayed as Block in Bookmark Type.

Possible Bookmark Area

Bookmarks can be inserted not only in the body text but also in objects like footnotes/endnotes, header/footer, master page, tables and text boxes.

Bookmark Marker

Bookmarks are not displayed on the edit screen. They also do not affect printing or paragraph aligning. Bookmark markers are inserted as control codes in the positions where you clicked Insert > Bookmark. You can check the control codes of the inserted bookmarks by clicking View > Show > Control Codes.

Bookmark with Identical Name

You cannot add bookmarks with the same name. The message "A bookmark with the same name is already registered. Assign another name." appears.

Copy and Paste Bookmark

On copying body text that includes a bookmark and paste it in another document, if a bookmark with the same name exists in the document, only the body text will be pasted without the control codes of the bookmark.

Information about Bookmark

Information about bookmarks you added to a document is saved in the document. So the position, number, and name of the bookmarks will be stored when you open the document again.

Advanced Usage

Bookmark and Hyperlink

Bookmark is useful when dealing with a large document and is capable of providing links to bookmarks in other documents when used in conjunction with the Insert > Hyperlink function.
