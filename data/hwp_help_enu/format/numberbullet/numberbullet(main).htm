Numbering
 

Home > Numbering

When you are listing multiple items, you can add numbers, bullets and image bullets to the head of the paragraph. Numbering provides seven levels of numbers. When there is a change in the order of numbered sentences, the numbering changes automatically.

You can select or customize numbering, bullet, and image bullet styles in the Numbering/Bullets dialog box that is displayed when you click Home > Numbering > Numbering Styles.

More Information

Numbering/Bullets Dialog Box

Numbering Tab

If you select an item in Numbering Starts by and Numbering in the Numbering tab, you can number the head of the current paragraph.

If there is no style you want from the Numbering list, you can set Custom Numbering Styles.

Bullets Tab

If you select a bullet style in the Bullets tab, you can add bullets to the head of the current paragraph.

If there is no style you want from the Bullet Styles list, you can set Custom Bullet Styles.

Image Bullets Tab

If you select an image bullet style in the Image Bullets tab, you can add image bullets to the head of the current paragraph.

If there is no style you want from the Image Bullet Styles list, you can set Custom Image Bullet Styles.

Note

By Paragraphs

Numbering/Bullets applies per paragraph. If you place the pointer anywhere in a paragraph and apply Numbering/Bullets, numbers or bullets will be inserted at the head of the current paragraph. The above and below paragraphs will not be affected.

If you want to insert numbers or bullets in multiple paragraphs, you have to select the paragraphs as a block and apply Numbering/Bullets.

Difference from Multilevel List

Multilevel List and Numbering provide seven levels of numbers. When there is a change in the order of numbered sentences, the numbering changes automatically.

You can change Multilevel List styles per section. One section has only one multilevel list style and its multilevel list should continue the numbers of the previous multilevel list.

Unlike Multilevel lists, different paragraphs can have different Numbering styles and there is no restriction on starting numbers.
