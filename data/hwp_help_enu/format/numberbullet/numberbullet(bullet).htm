Numbering/Bullets: Bullets
 

Home > Numbering

When you are listing multiple items, you can add bullets to the head of the paragraph.

In the Bullets tab of the Numbering/Bullets dialog box, you can select a bullet style. If you click Custom, you can customize a bullet style.

Steps

To change bullet styles

  1. To type content in the edit window and insert bullets, do of the following.
    • Click Bullets in the Home tab.
    • Click Home > Numbering > Toggle Bullets.
    • Press CTRL+SHIFT+L.
  2. Place the pointer in the second paragraph and click Home > Numbering > Numbering Styles.
  3. From the Bullets list, select ¢¹ as a bullet style to use.
  4. Click Set. The bullet style of the second paragraph will change to the selected style.
  5. Select all paragraphs below the second paragraph as a block and repeat the above process. The bullet style will change to ¢¹.

More Information

Bullets Tab in Numbering/Bullets Dialog Box

Bullet Styles

18 bullet styles and custom styles are displayed. To remove properties of bulleted paragraphs, select None from the Bullet Styles list.

Custom Button

If you select a style from the Bullet Styles list and click this button, the Custom Bullet Styles dialog box will appear. You can reorganize bullet styles to your preference.

Delete Icon

If you select a newly added bullet style from the Bullet Styles list and click Delete, you can delete the style.

The Delete icon is activated only if you add new bullet styles by using Custom > Custom Bullet Styles > Add Current Format to List.

Note

Font of Bullet

The font of bullets follows the font of the last character in the paragraph. So even if you select part of a paragraph with bullets as a block and change fonts, the font of the bullets do not change.

Only when you select the entire paragraph or only the end of the paragraph as a block and change fonts, the font of the bullets changes to the font of the body text.

Change Bullet Font

If you want to customize a font for bullets, select a style from the Bullet Styles list of the Bullets tab and click Custom.

In the Custom Bullet Styles dialog box, select Set Font and click Font to set text size, font, and text properties to apply to numbers. Then clear the Follow Paragraph Font Size check box.

Change Bullet Style

You can use different bullet styles for different paragraphs.

If you want to use a bullet style different from the bullet style of the previous paragraph, select a bullet style you want from the Bullet Styles list of the Bullets tab and click Set. The bullet style will be inserted from the current paragraph.

Numbering/Bullets Toolbar

If the pointer is in a bulleted paragraph, the Format tab is displayed automatically. Use this toolbar to change bullet styles or remove bullet properties easily.

If you close the toolbar when the Format tab is displayed, the toolbar will not appear even if the pointer is in a bulleted paragraph.
