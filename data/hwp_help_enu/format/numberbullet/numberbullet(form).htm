Custom Numbering Styles
 

Home > Numbering

You can reorganize numbering styles for each level to your preference.

Steps

To set custom numbering styles

  1. Click Home > Numbering > Numbering Styles.
  2. In the Numbering/Bullets dialog box, click Custom in the Numbering tab.
  3. In the Custom Numbering Styles dialog box, select Level 1 in the Level list box that is comprised of seven levels.
  4. Select I, II, III from the 15 numbering styles that appear when you click the arrow of Numbering Styles.
  5. To add a specific numbering style to the Numbering list, select Add New Number Style and type a name for the new numbering style.
  6. Click Set.
  7. In the Numbering tab of the Numbering/Bullets dialog box, the customized numbering style will be added to the Numbering list.
  8. Click Set to apply the customized numbering style to the body text.

   When you change settings in the Custom Numbering Styles dialog box, you can check the changed numbering style in the preview area.

To change fonts of numbering styles

You can change fonts for numbers by using Font of Custom Numbering Styles.

  1. Click Home > Numbering > Numbering Styles.
  2. In the Numbering/Bullets dialog box, click Custom in the Numbering tab.
  3. In the Custom Numbering Styles dialog box, select Font > Set Font and click Font.
  4. In the Font dialog box, set text size, font, and text properties to apply to numbers and click Set.
  5. When you return to the Custom Numbering Styles dialog box, clear Font > Follow Paragraph Font Size and click Set.
  6. In the Numbering/Bullets dialog box, click Set to change the current font of numbering to a font of choice.

   When you change settings in the Custom Numbering Styles dialog box, you can check the changed numbering style in the preview area.

More Information

Custom Numbering Styles Dialog Box

Level

Select a level for number format in the Level list box that is comprised of seven levels. You can specify Number Format, Insert Number, and Numbering Styles for each level.

  • Number Format

  • Decorate the level of numbers selected in the Level list box by inserting characters before and after the numbers.

  • Insert Number

  • To insert a high level number code in Number Format, click the arrow of the list and select a level of choice. On the Insert Number list, only the current level and the higher level appear.

  • Numbering Styles

  • Click the arrow of the list and select from 15 styles for numbers of the current level.

Font

You can select a font for each level of numbering.

  • Set Font

  • The font of numbering follows the font of the last character in the numbered paragraph.

    If you want to customize a font for numbering, select Set Font and click Font. In the Font dialog box, set text size, font, and text properties to apply to numbers.

  • Follow Paragraph Font Size

  • If you select this option, the font selected in the Font dialog box will be used as the font of numbering. However, text size will be the same as the last character of the numbered paragraph.

    This option is activated only when Set Font is selected.

Number Position

Adjust the width of the area occupied by numbering and specify how to align.

  • Adjust Width to the Number of Digits

  • If you select this option, when you number text with numbers of two or three digits, the width of the numbers will be aligned according to their digits.

    If you do not select this option, two-digit numbers will occupy the same area as one-digit numbers.

  • Adjust Width

  • Specify width of numbering. The width of numbering is automatically calculated according to text and text size in Number Format.

  • Alignment

  • Select from Left, Center, and Right for the arrangement of numbers within numbering width.

Wrapping Style

Set distance between numbering and the body text to maintain a certain distance regardless of text size.

  • Distance from Text

  • Set distance between numbers and the body text in percent (%) when automatic outdent is used.

  • Automatic Outdent

  • Automatically outdent as much as the width of numbers to align the body text vertically.

Add New Number Style

Add customized numbering styles to the numbering list.

  • Name

  • Type the name of a new numbering style to add to the list.

Note

Remember Custom Style

Newly added numbering, bullet, and image bullet styles are saved in Hwp. So you can use customized styles when you create a new document window or restart Hwp.
