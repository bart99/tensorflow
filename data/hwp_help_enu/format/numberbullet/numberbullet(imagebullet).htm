Numbering/Bullets: Image Bullets
 

Home > Numbering

When you are listing multiple items, you can add image bullets to the head of the paragraph.

In the Image Bullets tab of the Numbering/Bullets dialog box, you can select an image bullet style. If you click Custom, you can customize an image bullet style.

Steps

To change image bullet styles

  1. To type content in the edit window and insert bullets, click the Home > Numbering > Numbering Styles.
  2. In the Numbering/Bullets dialog box, select the Image Bullets tab.
  3. From the Image Bullet Styles list, select Blue Square as a bullet style to use.
  4. Place the pointer in any bulleted paragraph and click Home > Numbering > Numbering Styles.
  5. In the Numbering/Bullets dialog box, select the Image Bullets tab.
  6. From the Image Bullet Styles list, select Pink Square as a bullet style to use.

More Information

Image Bullets Tab in Numbering/Bullets Dialog Box

Image Bullet Styles

Various image bullet styles and custom styles are displayed. To remove the properties of image bulleted paragraphs, select None from the Image Bullet Styles list.

Custom Button

If you select a style from the Image Bullet Styles list and click this button, the Custom Image Bullet Styles dialog box will appear. You can reorganize image bullet styles to your preference.

Delete Icon

If you select a newly added image bullet style from the Image Bullet Styles list and click the Delete icon , you can delete the style.

The Delete icon is activated only if you added new image bullet styles by using Custom > Custom Image Bullet Styles > Add Current Picture Format to List.

Note

Change Image Bullet Styles

You can use different image bullet styles for different paragraphs.

If you want to use an image bullet style different from the image bullet style of the previous paragraph, select an image bullet style you want from the Image Bullet Styles list of the Image Bullets tab and click Set. The image bullet style will be inserted from the current paragraph.
