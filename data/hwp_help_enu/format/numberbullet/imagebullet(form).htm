Custom Image Bullet Styles
 

Home > Numbering

You can reorganize image bullet styles for each level to your preference.

Steps

  1. Click Home > Numbering > Numbering Styles.
  2. In the Numbering/Bullets dialog box, select a style from Image Bullet Styles of the Image Bullets tab and click Custom.
  3. In the Custom Image Bullet Styles dialog box, click Select Picture in Image Bullets.
  4. In the Insert Picture dialog box, specify path and name of a picture file and click Open.
  5. When the path and name of the picture file is inserted in the Image Bullets box of the Custom Image Bullet Styles dialog box, click Set.
  6. The customized bullet style will be added to the Bullet Styles list in Numbering/Bullets > Bullets.
  7. To apply the customized bullet style to the body text, click Set.

   When you change settings in the Custom Image Bullet Styles dialog box, you can check the changed image bullet style in the preview area.

More Information

Custom Image Bullet Styles Dialog Box

Image Bullets

Specify path and name of a picture file to use as an image bullet style.

Image Bullet Position

Adjust the width of the area occupied by image bullets and specify how to align the bullets.

  • Adjust Width

  • Specify width of image bullets. The width of image bullets is automatically calculated according to the text size of the image bullets.

  • Alignment

  • For the arrangement of image bullets within image bullet width, select from Left, Center, and Right.

Wrapping Style

Set distance between image bullets and the body text to maintain a certain distance regardless of text size.

  • Distance from Text

  • Set distance between image bullets and the body text in percent (%) when automatic outdent is used.

  • Automatic Outdent

  • Automatically outdent as much as the width of image bullets to align the body text vertically.

Add Current Picture Format to List

Add customized image bullet styles to the Image Bullet Styles list.

  • Name

  • Type the name of a new image bullet to add to the list.

Note

Change Numbering and Bullet Style

You can define numbering and bullet styles to be used for each level by using Custom Style. For styles for each level, define number format, numbering style, numbering font style, adjust numbering width, number alignment, and distance from text.

Remember Custom Style

Hwp remembers newly added numbering/bullet styles. You can use them again when creating a new document window or restart Hwp.
