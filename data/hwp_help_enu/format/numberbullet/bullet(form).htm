Custom Bullet Styles
 

Home > Numbering

You can reorganize numbering styles to your preference.

Steps

  1. Click Home > Numbering > Numbering Styles.
  2. In the Numbering/Bullets dialog box, select a style from Bullet Styles of the Bullets tab and click Custom.
  3. In the Custom Bullet Styles dialog box, click Symbol in Bullet Character.
  4. In the Symbol dialog box, select the symbol ❏ and click Insert.
  5. When the symbol ❏ is inserted in the Bullet Character box of the Custom Bullet Styles dialog box, click Set.
  6. The customized bullet style will be added to the Bullet Styles list in Numbering/Bullets > Bullets.
  7. To apply the customized bullet style to the body text, click Set.

   When you change settings in the Custom Bullet Styles dialog box, you can check the changed bullet style in the preview area.

More Information

Custom Bullet Styles Dialog Box

Bullet Character

Insert a new bullet style or click Symbol to select a bullet style in the Symbol dialog box.

Font

You can select fonts for bullets.

  • Set Font

  • The font of a bullet follows the font of the last character in the bulleted paragraph.

    If you want to use a different font for bullets, select Set Font and click Font. In the Font dialog box, set text size, font, and text properties to apply to bullets.

  • Follow Paragraph Font Size

  • If you select this option, the font selected in the Font dialog box will be used as the font of bullets. However, text size will be the same as the last character of the bulleted paragraph.

    This option is activated only when Set Font is selected.

Bullet Position

Adjust the width of the area occupied by bullets and specify how to align the bullets.

  • Adjust Width

  • Specify width of bullets. The width of bullets is automatically calculated according to the text size of the bullets.

  • Alignment

  • For the arrangement of bullets within bullet width, select from Left, Center, and Right.

Wrapping Style

Set distance between bullets and the body text to maintain a certain distance regardless of text size.

  • Distance from Text

  • Set distance between bullets and the body text in percent (%) when automatic outdent is used.

  • Automatic Outdent

  • Automatically outdent as much as the width of bullets to align the body text vertically.

Add Current Format to List

Add customized bullet styles to the Bullet Styles list.

  • Name

  • Type the name of a new bullet to add to the list.

Note

Remember Custom Style

Newly added numbering, bullet, and image bullet styles are saved in Hwp. So you can use customized styles when you create a new document window or restart Hwp.
