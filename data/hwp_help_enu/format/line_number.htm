Line Numbers
 

Page Layout > Line Numbers

This feature allows you to display line numbers in all or part of the document. Or you can display line numbers at intervals such as every tenth line (10, 20, 30, and so on), or set numbering style.

Steps

To show/hide line numbers

  1. Click Page Layout > Line Numbers > Show Line Numbers.
  2. A selection mark appears at the head of Show Line Numbers and line numbers are shown.
  3. Click Page Layout > Line Numbers > Show Line Numbers again.
  4. A selection mark disappears at the head of Show Line Numbers and line numbers are not shown any more.

To skip line numbers for the current paragragh

  1. Click Page Layout > Line Numbers > Show Line Numbers.
  2. A selection mark appears at the head of Show Line Numbers and line numbers are shown.
  3. Place the cursor for which you want to skip line numbers.
  4. Click Page Layout > Line Numbers > Skip Current Paragraph. Line numbers are renumbered except for the current paragraph or line.

More Information

Line Number Settings Dialog Box

Start at

Specify the inital number that you want to start line numbering.

Distant from

Set distance between numbering and the body text in milimeter (mm) .

Count by

Display line numbers at intervals such as every fifth line (5, 10, 15, and so on).

Every Page

Restart line numbers on each page. The first line number starts from 1 or the initial number that you have specified from Start at.

Every Section

Restart line numbers on each section. The first line number starts from 1 or the initial number that you have specified from Start at.

Continue More Previous List

Continue a line number that is closest to the previous numbering.
