Restart Numbering
 

Page Layout > Restart Numbering

Hwp is capable of numbering pages, pictures, tables, equations, and footnotes/endnotes in order.

Steps

  1. In the active document, point to the page where you want to start new numbering.
  2. Click Page Layout > Restart Numbering.
  3. In the Restart Numbering dialog box, select Numbering Type > Page Number.
  4. In the Start at box, type 1 and click OK.
  5. The current page number becomes 1 even though the pointer is in the middle of the document.

More Information

Restart Numbering Dialog Box

Numbering Type

Select a numbering type from Page Number, Picture Number, Table Number, Equation Number, Footnote Number, and Endnote Number.

Start at

Select a number to start with for the selected numbering type. The number can be set from 1 through 65,535. If you select a number to start with, the following numbers will be renumbered automatically.

Note

Purpose

To start the body text as Page 1 regardless of TOC and header in the beginning of the document, use Starting Page Number.

Difference from Section > Start at

You can select a starting page number for the active section and a starting number for panes of pictures, tables, and equations by using Starting Page Number and Starting Object Number in Page Layout > Section Settings. When you are using the functions, you can start numbering with only one number in one section. You can restart with a new number in a different section.

On the other hand, if you use Page Layout > Restart Numbering, you can restart numbering pages, and panes of picture, tables, and equations in one section unlimited times.

Caution

Restriction

There is no restriction on the number of page numberings in one file or page. When there are multiple page numberings, however, the last numbering displayed on each page will apply when it is printed.
