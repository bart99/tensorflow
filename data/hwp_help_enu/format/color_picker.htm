Color Palette
 

With the color palette, you can select colors for color-related items like text color, line color, and face color.

Steps

To select colors by using Color Palette

  1. In the edit window, type Hancom and select as a block.
  2. On the Formatting toolbar, click the arrow next to the Font Color icon.
  3. From the Color Palette, select a color.
  4. The text color of Hancom changes to the selected color. On the Formatting toolbar, the Font Color icon changes to the selected color.

To select colors by using the Colors dialog box

  1. In the edit window, type Hancom and select as a block.
  2. On the Formatting toolbar, click the arrow next to the Font Color icon .
  3. From the Color Palette, click Others.
  4. In the Colors dialog box, type 70 in Red, 130 in Green, 230 in Blue, and then click Add.
  5. Confirm that the color added to Custom Color is registered, and click Set.
  6. The text color of Hancom changes to the selected color. On the Formatting toolbar, the Font Color icon changes to the selected color.

More Information

Color Palette

Separator

Click and hold the separator in the color palette, to change the pointer to a four-headed arrow . Then you can separate the color palette from the document window by dragging the mouse towards the edit window. When you click the separator of the color palette once, the color palette gets separated from the document window.

None

Set Fill Color as None.

Others

If you do not see the color you want in the color palette, click Others and select a color or register custom colors in the Colors dialog box.

Spuit

You can extract colors from other applications by clicking the Spuit icon.

Color Theme Icon

When you click the Color Theme icon, the color theme table is displayed.

You can select a theme among various themes in the color theme table including Basic, Office, Afterimage, Light, Orange, Yesterday, Spring, Dream, Forest, Sea, Nongravitation, and Deep Sea.

When you select Single Color Gradation, the Colors dialog box is displayed.

Basic Palette

Basic palette colors for each color theme are displayed.

Custom Color/Recent Color

Custom Color displays the last 10 registered custom colors in the Colors dialog box. Recent Color displays the last 10 colors used.

Transparency Slider

Move the slider of the color palette left and right to specify transparency for the selected object from 0% through 100%.

Other Fill

Apply other fill effects to objects.

Note

Toolbar Separation

You can separate the color palette from each toolbar.

Click and hold the separator in the color palette, to change the pointer to a four-headed arrow . Then you can separate the color palette from the document window by dragging the mouse towards the edit window. When you click the separator of the color palette once, the color palette gets separated from the document window.
