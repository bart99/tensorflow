Page Setup: Line Grid
 

File > Page Setup / Page Layout > Page Setup

You can set a line grid in the Line Grid tab of the Page Setup dialog box.

The number of lines in a particular page of a document should be limited at times. The text and pictures get aligned along the line grid if you limit the maximum line number in one page.

Steps

  1. Click File > Page Setup or Page Layout > Page Setup.
  2. In the Page Setup dialog box, select the Line Grid tab.
  3. Select Line Grid Settings and type 20 in Number of Lines > Lines Per Page.
  4. When Line Spacing changes to 32.9pt, click Set.

More Information

Line Grid Tab of Page Setup Dialog Box

Line Grid
  • No Grid

  • Do not limit the maximum number of characters and lines in a page. The default text spacing and line spacing are decided by the size of each character and by the height of the tallest character in the line, respectively.

  • Line Grid Settings

  • Specify the maximum number of lines in one page (minimum line spacing). You can change the line spacing on selecting this option and also create a document where lines of different columns are well aligned.

  • Line and Text Grid Settings

  • Limit the maximum number of lines in a page or set standard text spacing. On selecting this option, the space between characters increases or decreases by a certain amount and thus the document looks loose or tight.

Number of Lines
  • Lines Per Page

  • Set maximum number of lines in a page. The line spacing value changes automatically on changing this value.

    The maximum line number in a page is 41 by default when you use regular A4 paper and 10pt font.

  • Line Spacing

  • Set the amount of space to be left between the lines. The number of lines in a page changes on changing this value.

    The line spacing in the body text is 16pt by default when regular A4 paper and 10pt font is used.

Number of Characters
  • Grids Per Line

  • Specify the maximum number of characters in a line. The text spacing value changes automatically on changing this value.

    The maximum character number in a line is 42 by default when regular A4 paper and 10pt font is used.

  • Text Spacing

  • Specify the amount of space to be left between the characters. The number of characters in a line changes automatically on changing this value.

    The line spacing in the body text is 10pt by default when regular A4 paper and 10pt font is used.

Apply to

Suggest settings for the section number, pointer position, and block setting of the active document. You can select either Whole Document or New Section. You can select Current Section if the active document is divided into two or more sections.

  • New Section

  • Create a new section by breaking pages at the current pointer position and apply the following properties.

  • Whole Document

  • Apply the same section properties to the whole document even if the document is divided into two or more sections.

  • Current Section

  • Apply properties to the current section.

Note

Normal Style and Line Grid

The already specified line grid does not get affected on setting Number of Lines > Lines Per Page and changing the text size of the normal style. The default value of Lines Per Page changes according to the text size of the current normal style on setting Lines Per Page after changing the normal style.

Using Default Line Number

The number of lines and line spacing is decided by the height of the tallest character in the line unless you limit the number of lines in a page by selecting No Grid in the Line Grid tab.

Snap to Grid When Document Grid is Defined

Line Grid only applies to the paragraphs where Snap to Grid When Document Grid is Defined of Paragraph > Basic is selected.

Specifying the Line Number per Paper Size

Lines per Page and Line Spacing automatically change to the optimal value according to paper size.

Caution

Text Grid

You cannot use Text Grid that limits the number of grids in a line if you select Text Grid Settings.
