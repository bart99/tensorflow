Page Setup: Basic
 

File > Page Setup / Page Layout > Page Setup

In the Basic tab of the Page Setup dialog box, you can set paper size, orientation, gutter type, and margins.

More Information

Basic Tab of Page Setup Dialog Box

Paper Size

Select a paper size for the document.

  • Paper Size

  • You can register the current paper size and margins as a new paper type and use it later.

    You can adjust the size of the body text in one page by setting paper size between 20¡¿20mm and A0 size, 1188¡¿1188mm.

Orientation

Select either portrait or landscape for orientation.

Gutter Type

Select whether to edit the document in one page at a time, in both, an even page and an odd page, or vertically continuous pages.

Paper Margin

Set top and bottom/left and right margins.

Apply to

Suggest settings for the section number, pointer position, and block setting of the active document. You can select either Whole Document or New Section. You can select Current Section if the active document is divided into two or more sections.

  • New Section

  • Create a new section by breaking pages at the current pointer position and apply the following properties.

  • Whole Document

  • Apply the same section properties to the whole document even if the document is divided into two or more sections.

  • Current Section

  • Apply properties to the current section.

Apply Settings to New Document

Apply Paper Size, Orientation, Gutter Type, and Paper Margin set in the Page Setup > Basic tab to Options > New Document. The values set in Options > New Document apply to all new documents. They also apply when you exist Hwp and restart it.
