Page Setup
 

File > Page Setup / Page Layout > Page Setup

You can select the paper size, portrait or landscape, and top-bottom/left-right margins by using Page Setup.

Steps

  1. Click File > Page Setup or Page Layout > Page Setup.
  2. In the Page Setup dialog box, select the Basic tab and click the arrow of the Paper Size list.
  3. Select B5 (46 Size) for paper type. Set the margins of the selected paper size.
  4. In Paper Margins > Top, type 30mm.
  5. Click Set. The length of the top margin changes to 30mm and the size of the body text decreases accordingly.

Note

Apply to

Page Setup > Apply to suggests settings for the section number, pointer position, and block setting of the active document.

In most cases, you can select either Whole Document or New Section. On selecting New Section, the page will break at the current pointer position and a new section will be created and the selected page setup will apply from the new section.

On selecting Whole Document, the selected page setup will apply to the entire document even if the active document is divided into two or more sections.

You can select the Current Section in Apply to if the document is divided into two or more sections.

Default Paper Size

The default paper size is brought from the master document of Hwp for a new document you create by clicking New or New Tab.

To change the default page size, modify Page Setup in File > Options > New Document.

Caution

Restriction

You can only set one paper size for a section. To use different paper sizes in one document, you have to create a new Section.
