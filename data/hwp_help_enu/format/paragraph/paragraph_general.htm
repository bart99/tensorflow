Paragraph: Basic
 

Home > Paragraph > Basic

From the Basic tab of the Home > Paragraph dialog box, you can set paragraph alignment, paragraph margins, first line style, line spacing in paragraphs, and line break.

More Information

Basic Tab of Paragraph Dialog Box

Alignment

Select an alignment method for when a word in a paragraph breaks at the right margin. You can set paragraph alignment by using the icons.

Margins

Specify the left margin and right margin of the active paragraph.

You can specify an additional left/right margin for the current paragraph by using Margins in Paragraph > Basic based on the left/right margin specified in Page Setup. For more information, refer to Change Paragraph Margins.

First Line

Specify whether to leave the first line of the paragraph as is or to indent/negative indent it.

  • Normal

  • This is how a paragraph looks without indentation or negative indentation.

  • Indent

  • Set the first line of a paragraph to be pushed to the right from the left margin of the paragraph. You can adjust indentation while checking it in the Preview window below.

  • Outdent

  • Set the left margin of an entire paragraph excluding the first line to be pushed as much to the right as the negative indent value specifies. You can adjust indentation while checking it in the Preview window below.

Spacing
  • Line Spacing

  • Change the line spacing of the active paragraph.

  • Before/After Paragraph

  • The distance between paragraphs grows if you set spacing above and below the paragraph.

  • Snap to Grid When Document Grid is Defined

  • Apply the line grid set in Page Layout > Page Setup only to the paragraph for which this option is selected.

Line Break

If part of a word gets too close to the right margin and the entire word moves to the next line, spacing between words in the previous line becomes too big. The Line Break option prevents this situation by adjusting line breaking criteria and spacing between words in the previous line.

  • Korean

  • Select Clause or Text as the line breaking criteria for when Korean text is at the end of lines.

  • English

  • Select Text or Word as the line breaking criteria for when English text is at the end of lines. Select whether to insert a Hyphen or not.

  • Minimal Spacing

  • Specify minimal spacing for when reducing spacing between words so that a word that does not end at the end of a line stays in the line without getting pushed to the next line.
