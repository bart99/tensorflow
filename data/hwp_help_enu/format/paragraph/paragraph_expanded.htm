Paragraph: Advanced
 

Home > Paragraph > Advanced

From the Advanced tab of the Home > Paragraph dialog box, you can set various options such as Paragraph Type, Keep Orphaned Line, Keep Lines Together, and Vertical Alignment.

More Information

Advanced Tab of Paragraph Dialog Box

Paragraph Type
  • None

  • Set the head of the current paragraph where the pointer is, as a normal paragraph.

  • Outline Paragraph

  • Set the head of the current paragraph where the pointer is, as an outline paragraph. Paragraph level can be set from 1 through 7.

  • Numbered Paragraph

  • Set the head of the current paragraph where the pointer is, as a numbered paragraph. Paragraph level can be set from 1 through 7.

  • Bulleted Paragraph

  • Set the head of the current paragraph where the pointer is, as a bulleted paragraph.

Others
  • Keep Orphaned Line

  • Prevent only one line of a paragraph with two or more lines from being printed on the front page or the back page.

  • Keep with Next

  • If the current paragraph ends in the last line of a page or column, the first line of the next paragraph will be pushed up to the current page or the last line of the current paragraph will be pushed down to the next page so that there is no page or column change between two paragraphs.

  • Keep Lines Together

  • Prevent a paragraph from being split into the front page and the back page, when printed.

  • Page Break Before

  • Always break pages in front of the current paragraph.

  • Use Optimized Height for Font

  • Use specified text size, instead of the default size of the font, as the base for line spacing.

  • Single Row

  • If you select this option, everything you type will be in one line without moving on to the next line.

  • Automatically Adjust Space between Asian and Latin Text

  • If you select this option, space between Asian and Latin text will be automatically adjusted to optimal space.

  • Automatically Adjust Space between Asian Text and Numbers

  • When you select this option, space between Asian text and numbers will be automatically adjusted to optimal space.

  • Vertical Alignment

  • Specify base for vertically aligning characters in one line.
