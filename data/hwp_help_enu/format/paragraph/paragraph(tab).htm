Tab Settings
 

Home > Paragraph > Tab Settings

Tab is used to move the body text a certain distance. Tab is useful when listing multiple items vertically aligned.

Steps

To set tab in a dialog box

  1. Click Home > Paragraph.
  2. In the Paragraph dialog box, select the Tab Settings tab.
  3. Select Tab Type > Left and type 50pt in the Tab Position box.
  4. Click Add.
  5. A left tab stop of 50pt spacing will be added to the Tab List box.
  6. Click Set.
  7. A left tab stop appears at 50pt on the horizontal ruler.

To set tab on the horizontal ruler

  1. Click gridlines of the Horizontal Ruler. A tab for aligning text left will be added.
  2. Rest the pointer on the added left tab stop and drag it to the right to change the position of the tab stop.
  3. To add another type of tab stop, click the Change Tab Type icon at the left end of the horizontal ruler.
  4. Tab type changes in this order, Left Tab Stop ¡æ Right Tab Stop ¡æ Center Tab Stop ¡æ Decimal Tab. When the tab type is displayed, point to the gridline of the horizontal ruler and click it.
  5. The selected tab will be added at the current position.

More Information

Tab Type

Tab Settings Display Description
Left Tab Stop Align the left starting point of the words in the position where you set the left tab stop.
Right Tab Stop Align the right end of the words in the position where you set the right tab stop.
Center Tab Stop Align the center of the words in the position where you set the center tab stop.
Decimal Tab Align the decimal part of the words in the position where you set the decimal tab.

Note

Default Tab

In Hwp, a left tab stop is set at 40pt (eight English letters) by default. This default tab is displayed under the gridline of the horizontal ruler. If you insert a new tab, all default tabs on the left of the added tab are no longer displayed.

Default Tab Size is part of section properties. So it cannot be set differently for different paragraphs. You can change Default Tab Size in Page Layout > Section Settings but you cannot delete the default tab.

Unit

Tab settings are remembered per paragraph. If you change tab settings, only the tab settings of the current paragraph change. To change the tab settings of multiple paragraphs, select the paragraphs as a block and then change tab settings.

Pattern to Fill

Select a line type with which to fill the empty space created by a tab from the end of a word to the beginning of the word at the current tab stop. If you select a type and press TAB in the edit screen, a line of the selected type will be drawn to the tab position.

Horizontal Ruler

On the horizontal ruler, you can check the position of the default tab and the type and position of the tabs you insert.

If you click part of the ruler that does not have a tab, a new tab will be added. If you click a tab and drag it to a position on the ruler, the tab will move to the position. If you drag a tab outside the ruler, the tab will be deleted.

Change Tab Type

The icon for changing tab type is at the left end of the horizontal ruler. When you click this icon, tab type changes in this order, Left Tab Stop ¡æ Right Tab Stop ¡æ Center Tab Stop ¡æ Decimal Tab.

When the tab type you want is displayed, point to the gridline of the horizontal ruler and click it. The selected tab will be added at the position.

The icon for changing tab type is only displayed in Show Both Horizontal/Vertical Rulers mode.

Context Menu

If you right click a position on the horizontal ruler, a context menu appears with Left Tab Stop, Right Tab Stop, Center Tab Stop, Decimal Tab, Delete All Tab Stops, Tab Settings, and Paragraph included. You can add or delete tabs easily by using this menu.

Shortcut on Ruler

If you drag a tab mark on the horizontal ruler while pressing SHIFT, you can change the position of all tabs past the current tab. You can copy the current tab type if you drag a tab mark while pressing CTRL.

To cancel moving or copying a tab, press ESC while pressing the left mouse button.
