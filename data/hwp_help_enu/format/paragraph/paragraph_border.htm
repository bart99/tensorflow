Paragraph: Border/Background
 

Home > Paragraph > Border/Background

From the Border/Background tab of the Home > Paragraph dialog box, you can set the type, weight, color, and position of borders and the face color, pattern color, and pattern of the background.

More Information

Border/Background Tab > Paragraph Dialog Box

Borders

Paragraph applies per paragraph.

  • Type

  • Select a line type from the list of 12 line types.

  • Weight

  • Select a line weight of borders.

  • Color

  • Click the color table to display the color palette. Select a line color for borders in the color palette.

  • Single Border for Several Paragraphs

  • Connect the current paragraph and the following paragraphs with one set of paragraph borders.

  • Preview Line Style

  • After you select a border type, it applies to the border whose border icon is selected in the preview area.

  • Preview Area

  • If you select a type, weight, and color of borders and click Left, Right Top, Bottom, or All in the preview area, the selected line type will apply.

Background
  • Face Color

  • Click the color table to display the color palette. In the color palette, select a background color of paragraphs.

  • Pattern Color

  • Click the color table to display the color palette. In the color palette, select a pattern color to fill the paragraph background with.

  • Pattern

  • Click the color table to display the color palette. In the color palette, select a pattern to fill the paragraph background with.

Spacing
  • Left

  • Adjust spacing between the left part of paragraph borders and the body text.

  • Right

  • Adjust spacing between the right part of paragraph borders and the body text.

  • Up

  • Adjust spacing between the top part of paragraph borders and the body text.

  • Bottom

  • Adjust spacing between the bottom part of paragraph borders and the body text.

  • All

  • Click All arrows to adjust spacing between the body text and the left, right, top, and bottom parts of paragraph borders at one time.

  • Ignore Paragraph Margin

  • Ignore the left and right margins of paragraphs and apply paragraph borders only around text.
