Paragraph: Tab Settings
 

Home > Paragraph > Tab Settings

From the Tab Settings tab of the Home > Paragraph dialog box, you can set tab or automatic tab to align multiple items vertically as you type.

More Information

Tab Settings Tab of Paragraph Dialog Box

Tab Type

Select Left, Right, Center, or Decimal as the base of tab alignment.

Pattern to Fill

Select a line type with which to fill the empty space created by a tab from the end of a word to the beginning of the word at the current tab stop. If you select a type and press TAB in the edit screen, a line of the selected type will be drawn to the tab position.

Tab Position

  • Specify tab position by typing or clicking the increase or decrease icon.

  • Details on Icons
  • Icon Name Description
    Delete If you select a tab to delete from the Tab List and click Delete, the tab will be deleted from the Tab List.
    Delete All You can delete all tab settings added to the Tab List.
Tab List
  • Tab List

  • Display a list of tabs that are added by specifying position and type. For more information about how to set tab, refer to Tab Settings.

  • Details on Icons
  • Icon Name Description
    Add If you specify Tab Position and Type and click Add, the specified tab will be added to the Tab List.
Deleted Tab List

Display a list of tabs deleted from the current paragraph or block.

Automatic Tab Stops

If you use Automatic Tab Stops, you can align a numbered or bulleted paragraph vertically and can also push text in a paragraph to the left or right.

  • Automatic Tab for Outdent

  • Insert a left tab stop at the left end of a paragraph.

  • Automatic Right Tab Stop

  • After deleting all default tabs, insert a right tab stop at the right end of a paragraph.

Default Tab
  • Default Tab Size in the Section

  • In Hwp, a left tab stop is set at 40pt (eight English letters) by default. This default tab is displayed under the gridline of the horizontal ruler. If you insert a new tab, all default tabs on the left of the added tab are not displayed.

  • Change Button

  • Once you click Change, the Section Settings dialog box appears just like when you click Page Layout > Section Settings. In the dialog box, you can change default tab size but are unable to delete default tabs. Also, default tab size is part of section properties. So you cannot set it differently for different paragraphs.
