Header/Footer
 

Insert > Header/Footer

Header and Footer is the repetitive contents at the top and bottom of every page. The title of the book, the title of the current chapter, and page number are usually inserted in the footer and header.

Steps

  1. Click Insert > Header/Footer.
  2. In the Header/Footer dialog box, select Type > Header and Position > Both Pages and click Create.
  3. The header screen is displayed.
  4. Type Hancom as the title and select a font.
  5. In the Header/Footer tab, click Close to return to the edit screen of the body text.

Note

Header/Footer Start Position

Header and Footer are inserted on the page of cursor position. For example, if you place the pointer on Page 2 and insert Header and Footer, Header and Footer will not be displayed on Page 1. Header and Footer will be displayed starting from Page 2.

Display

When the pointer is in the header or footer area, the body text turns vague. The current position is displayed in the header area on top as <<Header (Both Pages)>>, <<Header (Even Page)>>, or <<Header (Odd Page)>>. This makes it easy to check the header you are editing.

Auto Activation of Print Layout

If you create Header/Footer without selecting View > Print Layout, View > Print Layout is switched on and the mode will change to View Print Layout. Also, in View Print Layout mode, View > Print Layout becomes unavailable.

Even if you finish editing the header/footer in this mode, the View Print Layout mode continues and you will not return to Print Layout Off mode.

Exit

To finish typing footer or header, click Close in the Header/Footer tab. You can also press SHIFT+ESC on the keyboard.

Go to Header/Footer

If multiple footers or headers are inserted in the active document, click Home > Go to in the body text to find header/footer control codes. You can modify the content of the found header/footer or delete unnecessary control codes.

When the pointer is inside a header or footer area, you can move to another header or footer area by using Previous or Next in the Header/Footer tab.

Restriction on Header/Footer Number

There is no restriction on the number of headers/footers in one document or page.

However, you cannot create the same footer/header in one paragraph.

When there are multiple headers/footers, the last header/footer on a page is printed on the page.

Range

In the header and footer, you can insert all characters, pictures, tables, and drawing objects that you can insert in the body text. You can also select a paragraph style or font style completely separate from the body text.

Function Restriction

On the header or footer screen, Page Layout > Page Border, Page Layout > Page Background, Page Layout > Master Page, Page Layout > Section Settings, and References > Footnotes are not available.

Even if you click Insert > Header/Footer in column edit mode, the header and footer areas will be created as a single column.

Caution

Hide Print Layout

In Hide Print Layout mode, the header and footer inserted in the document is not displayed on the screen. You have to select Show Print Layout to edit page margins and header/footer that will be printed while looking at all content and styles on the screen.

Advanced Usage

Tab Stop in Header/Footer Area

In the header and footer, the paragraph is usually aligned for example the title is left and the page number is right of the paragraph.

In the Hwp header or footer paragraph, a Left Tab Stop is on the far left of the paragraph and a Right Tab Stop is on the far right of the paragraph by default.

If you type content for the header or footer and press TAB, text will be aligned to the right. Just by pressing TAB, you can change paragraph alignment.

Create Automatic Header/Footer on Print-out

If you click File > Print and use Header/Footer for Printing in the Advanced tab of the Print dialog box, you can create Header for Printing and Footer for Printing that will be printed on top and bottom of printouts. However, you cannot edit the header/footer for printing on the edit screen because they are not displayed on the edit screen.
