Section Settings
 

Page Layout > Section Settings

When creating multiple chapters in a document, it is common to use new page numbers, different background pictures and different borders for different chapters. If you create sections, you can use different page setup, master pages, footnote/endnote styles, page borders/backgrounds, outline styles, and comments in one document. The reason being that the content created after creating sections is considered completely separate from the content before it.

Steps

  1. Place the pointer in the spot from which you want to start a new section, and click Page Layout > Section Settings.
  2. In the Section Settings dialog box, click the arrow of the Apply to list, select New Section, and click Set.
  3. A new section will be created and the pointer will move to the beginning of the new page.
  4. To set page setup in the current section differently from the previous section, click File > Page Setup.

More Information

Section Settings Dialog Box

Starting Page Number

Select a starting page number for the current section.

If you select Type > Custom and type a starting number, numbers will restart with the starting number.

Starting Object Number

Select a starting number for Picture, Table, and Equation.

If you select Custom from the first list of each object and insert a starting number, numbers will restart with the starting number.

Others
  • Hide Header/Footer in First Page

  • Select whether to hide the footer/header only in the first page of each section.

    Use Page Layout > Hide to prevent the header, footer, page number, page borders, document background, and master page from being printed only on the page where the pointer is, regardless of Section Settings.

  • Hide Master Page in First Page

  • Select whether to hide the master page only in the first page of each section.

  • Hide Border/Background in First Page

  • Select whether to hide the page border/background only in the first page of each section.

  • Hide Empty Lines

  • If there are empty lines at the beginning of a page, up to two empty lines will be regarded as non-existent and the body text will be pushed up two lines.

    Only up to two empty lines at the beginning of a page will be hidden. The rest of the empty lines will be displayed at the beginning of the page.

  • Between Columns

  • If you created different types of columns in one document by using New Column Settings, specify space between two sets of columns.

  • Default Tab Size

  • Set default tab size of Hwp. In Hwp, a left tab stop is set at 40pt (eight English letters) by default. This default tab is displayed under the gridline of the horizontal ruler. If you insert a new tab, all default tabs on the left of the added tab are no longer displayed.

  • Apply to

  • Suggests settings for the section number, pointer position, and block setting of the active document. In most cases, you can select either Whole Document or New Section.

    • Current Section: If the document is divided into two or more sections, you can select the Current Section in Apply to.
    • Whole Document: Apply section properties to the entire document (even if the active document is divided into two or more sections).
    • New Section: Create a new section by breaking pages at the current pointer position.

    If you select part of a document as a block and click Page Layout > Section Settings, you can select Selected Text, Selected Section, and Whole Document in Apply to.

    • Selected Section: If there are two or more sections in the block, you can select Selected Section in Apply to.
    • Selected Text: Separate the text selected as a block from content before and after it and make it into a new section.
    • Current Section: If the active document is divided into two or more sections, you can select the Current Section that includes the block in Apply to.
    • Whole Document: Apply section properties to the entire document (even if the active document is divided into two or more sections).

Note

Forced Page Break

If you create a section, a new page will be created no matter what. This means that a page cannot have multiple sections.

Forced Section Break

If you click Page Layout > Section Break in the spot where you want to create a new section, the page will break at the current pointer position and a new section will be created.

A new section not only has the section properties of the previous section but also follows its page setup, master page, footnote/endnote styles, page border/background properties, outline style, and column style. To apply a new style to a new section, reset the style and apply it only to the Current Section.

Delete Section

Click Page Setup > Section Settings or Page Layout > Section Break, and press DELETE or BACKSPACE in front of or behind the spot where you created a new section to delete the section break mark. Sections combined into one follow the style of the previous section.

Check Section

From the section information on the Status Bar, you can check how many sections the active document is divided into and in which section the pointer is.

Go to Section

If there are multiple sections in one document, click Home > Go to > Section to go to the section you want.

Edit Header/Footer and Columns

You can insert different styles of header/footer and edit various types of columns in one section without creating another section.

Hide Current Page Only

To hide the header/footer, master page, and page border/background not only in the first page but also in specific pages of a section, use Page Layout > Hide.

Start from Odd Page

Chapters always start on odd pages in a book. If you select Starting Page Number > Odd when the current page is an even page, the page is left as an empty page and the starting page number always becomes an odd number.

Caution

Restriction

You cannot create sections in Header/Footer, Footnote/Endnote, Caption, Table and Text Box.
