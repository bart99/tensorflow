Delete Current Page
 

Page Layout-Delete Page-Delete Current Page

Delete the current page right away.

Steps

  1. Place the pointer in the page which you want to delete.
  2. Click Page Layout > Delete Page > Delete Current Page.
  3. The current page will be deleted.

Note

Restorable

If you deleted pages by accident, press <Ctrl+Z> to undo the action.
