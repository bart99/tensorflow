Font: Basic
 

Home > Font > Basic

In the Basic tab of the Home > Font dialog box, you can set text size, font, scale, spacing, text properties, font color, and shading color.

More Information

Basic Tab of Font Dialog Box

Standard Size

Enlarge or reduce text size. Text size can be set from 1pt through the size that fills the entire page in 0.1pt increments.

Language-Specific Settings
  • Language

  • Select a language among Rep., Korean, English, Chinese Ideograph, Japanese, Other Languages, Symbol, and User. The Rep. option combines complementary fonts from different languages.

    By using Register Font > Representative Fonts in File > Options > Fonts, you can add a new representation font or modify existing custom representation fonts.

  • Font

  • For the languages selected in Language, you can select fonts for them at the same time under one representative name or select different fonts for different languages.

  • Relative Size

  • Specify standard size first and then select different text sizes for different languages. If you specify text size for each language when using Korean, English, and Chinese at the same time, text will look natural in terms of size and format.

  • Scale

  • Adjust scale (the horizontal/vertical ratio of a font) to decrease or increase the width of a font without changing font size.

  • Text Position

  • Adjust text position by moving text up or down the standard line of text.

  • Spacing

  • Adjust spacing (distance between characters). Adjust spacing between characters to create an easy-to-read document.

Properties

Set various text properties by clicking Bold, Italic, Underline, Outline, Emboss, and Engrave.

  • Text Color

  • Click the color table to display the color palette. In the color palette, select a font color of choice.

  • Shading Color

  • Click the color table to display the color palette. In the color palette, select a shading color of choice.
