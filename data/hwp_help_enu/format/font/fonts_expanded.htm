Font: Advanced
 

Home > Font > Advanced

In the Advanced tab of Home > Font, you can set font shadow, underline, outline, and emphasis mark.

More Information

Advanced Tab of Font Dialog Box

Shadow

Select None to remove shadow, or select Separate or Adjacent to set shadow shape.

  • Color

  • When you are applying shadow to fonts, you can click the color table in Color to select a shadow color from the color palette.

  • Horizontal

  • Adjust the horizontal slope of the shadow when applying shadow to fonts.

  • Vertical

  • Adjust the vertical slope of the shadow when applying shadow to fonts.

Underline

You can apply various underlines to fonts by specifying underline Location, Style, and Color.

Strikethrough

You can apply various strikethroughs to fonts by specifying strikethrough Location, Style, and Color.

Others

You can select an Outline or set Emphasis Mark that emphasizes text by inserting a dot in the center above characters.

  • Outline

  • You can apply various outlines such as solid, dot, dash, and thick to fonts.

  • Emphasis Mark

  • Emphasize text by inserting an emphasis mark in the center above the active character. You can also insert four Chinese tone marks including flat, rising, dipping, and falling marks.

  • Use Optimized Space for Font

  • Use the original space width of each font as space width. If you do not select this option, space width is set as 1/2 of font size.

  • Kerning

  • Adjust spacing between two continuous English letters.
