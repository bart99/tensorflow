Font: Border/Background
 

Home > Font > Border/Background

From the Border/Background tab of the Font dialog box, you can select a character border type and set width, color, and position of borders as well as face color, pattern color, and pattern format of the character background.

More Information

Border/Background Tab of Font Dialog Box

Borders
  • Type

  • Select a line from a list of 12 lines for character borders.

  • Weight

  • Select a line width of character borders.

  • Color

  • Click the color table to display the color palette. Select a line color for character borders in the color palette.

  • Preview Area

  • Specify the border type, weight, and color and specify which sides to apply the style to using Left, Right, Top, Bottom, or All icons from the preview area. This applies the style on the selected sides.

Background
  • Face Color

  • Click the color table to display the color palette. In the color palette, select a background color of characters.

  • Pattern Color

  • Click the color table to display the color palette. In the color palette, select a pattern color to fill the character background with.

  • Pattern

  • From the Pattern list, select a pattern to fill the character background with.

Note

Color Palette

Click the color table in Face Color or Pattern Color to display the color palette. If there is no color you want in the color palette, click Others.

In the Colors dialog box, you can specify the exact color that you wish to use.
