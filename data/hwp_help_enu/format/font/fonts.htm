Font
 

Home > Font

This feature is used to change all the font styles for the content to enter or a block.

You can select one font style, which includes font, text size, scale, and spacing, for all languages. Or, you can select different font formats for Korean, English, Chinese, Japanese, other languages, and symbols.

Also, it allows you to decorate fonts by applying various font properties such as font color, Italic, bold, underline, shadow, emboss, engrave, outline, and, super/subscript.

Steps

  1. Click Home > Font.
  2. In the Font dialog box, select Arial in Font of the Basic tab.
  3. In Properties, select Bold and click Set.
  4. In the edit window, type Hancom for it to be displayed in the selected font.
  5. Press F3 to select Hancom as a block and click Home > Font.
  6. In the Basic tab of the Font dialog box, click Bold again to clear it, and click Shadow.
  7. For Text Color, select Red and click Set.
  8. Hancom will be displayed in red with shadow.

Note

Change Font

You can specify Font before typing content. You can also type content and select it as a block first, and then change fonts.

Copy Font

Use Home > Format Painter to copy the font at the pointer position and overwrite a font in another position with it.

Text Style

Use Home > Styles to add frequently used fonts as font styles. You can change fonts simply by applying a style when needed.

Formatting Toolbar

Language Settings, Fonts, Text Size, Bold, Underline, Italic, and Font Color that are frequently used in Font are added to the Formatting toolbar so that you can use them without opening the dialog box.

Caution

Enter Right After Setting Font

If you change fonts before typing content, you can start typing without pointing to another position.

If you point to another position after changing fonts, the font you select is cancelled as the font of the new position overrides the selection.
