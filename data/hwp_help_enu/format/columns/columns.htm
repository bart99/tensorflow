Multiple Column Settings
 

Page Layout > Multiple Column Settings

When you wish to create a newspaper, magazine or Browse through content, this function divides pages into multiple columns for the convenience of readers.

Steps

  1. Click Page Layout > Multiple Column Settings.
  2. In the Columns dialog box, select Two in Frequently Used Style.
  3. Select Column Type > General Columns and click Set.
  4. Text on the edit screen will be divided into two columns.
  5. To cancel column editing, select One in Frequently Used Style.

More Information

Columns Dialog Box

Column Type

Column types are categorized according to how content gets filled in the column. You can select a column type only when there are Two or more columns.

  • General Columns

  • If you select this most basic column editing method, columns get filled with content one by one. When one column is full, content continues to the next column.

  • Distribute

  • If you select Distribute, content volume gets automatically adjusted so that the height of each column is identical on the last page.

  • Parallel

  • If you select Parallel, even when the current column is not full, you can type content in the next column. This option is often used in documents like a glossary in dictionary format where a set of a title and its description keeps repeating.

Frequently Used Style

Select a frequently used style easily.

Number of Columns

Specify how many columns to create in the document.

Insert Separator

Separate columns by drawing a line in the empty space between them. You can select a type, width, and color for the separator.

Note

Columns Inside Table or Text Box

If you click Page Layout > Multiple Column Settings inside a table or text box, you can divide its content into multiple columns.

When the pointer is inside a text box or table, you can select Whole Document, To New Column in Apply to. If you select To New Column, you can create a new column from the pointer position in an object just like in the body text.

If there are two or more columns in the object, you can select Current Column in Apply to.

Move Between Columns

To move between columns, click the column to which you want to move.

To move from a column to the column on its left, press CTRL+ALT+LEFT ARROW. To move from a column to the column on its right, press CTRL+ALT+RIGHT ARROW.

Range of Column Settings

There is no restriction on the maximum value in Number of Columns. The range of numbers depends on the Page Setup settings of the current document.

However, the range of Width and Spacing depends on page setup and the number of columns. The minimum column width is 10mm.

Difference Between ¡æ and ¡é at the End of Column

If you press RIGHT ARROW in the last line of a column in a document with multiple columns, the pointer will move to the first line of the column on its right. If you press DOWN ARROW, the pointer will move to the first line of the next page, which is the next line on the screen.

In other words, when there are multiple columns, RIGHT ARROW and DOWN ARROW move the pointer to the line on the right and the line below respectively as you see on the screen.

Footnote Position in Multiple Columns

When you are inserting footnotes in a document with multiple columns, you can insert them below each column or all columns combined. You can also gather all footnotes of a page below the column on the far right.

In Position for Multiple Columns of the Footnote Styles tab, click References > Footnotes > Footnote/Endnote Styles to specify footnote position.

When you are using Distribute or Parallel, select One for all column as the footnote position.

When Objects Are in Body Text

In a document with objects (table, picture, drawing object, etc.), you can place the objects between columns by adjusting their position.

Click the object to move and adjust Position in Object Properties.

Column Separator

In the previous Hwp versions, if you insert a Column Separator, it will be drawn regardless of content in the body text.

In Hwp, as an added feature, if you insert a Column Separator, it will be drawn only for the typed content.

Text Entering and Multiple Column Settings

Columns are usually made in the last stage of a document when most of the content is typed and needs general formatting.

When there is no content on the screen, it is hard to create columns because they will not be displayed properly on the edit screen.
