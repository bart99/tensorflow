Object Properties: Neon
 

Home > Object Properties > Neon

In the Neon tab of the Object Properties dialog box, you can set color, transparency, and size of neon and apply or remove neon.

More Information

Neon Tab of Object Properties Dialog Box

Neon

  • No Neon

  • Remove neon applied to drawing objects.

  • Neon

  • 18 options of neon according to neon color and size.

  • Color

  • Click the color table to display the color palette. Select a neon color in the color palette.

  • Transparency

  • Adjust transparency of neon.

  • Size

  • Adjust the size of neon. Size can be set from 0pt through 150pt.
