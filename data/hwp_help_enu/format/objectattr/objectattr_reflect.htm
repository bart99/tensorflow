Object Properties: Reflection
 

Home > Object Properties > Reflection

In the Reflection tab of the Object Properties dialog box, you can set size and distance of reflectors and apply or remove reflection.

More Information

Reflection Tab of Object Properties Dialog Box

Reflection

  • No Reflection

  • Remove reflection applied to drawing objects.

  • Reflection

  • 15 options for reflection according to the distance between a drawing object and the reflector or the size of the reflector.

  • Size

  • Specify size of the reflector based on the selected original drawing object.

  • Distance

  • Specify distance between the original object and the reflector with the original drawing object as the basis.
