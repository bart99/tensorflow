Object Properties: Shadow
 

Home > Object Properties > Shadow

In the Shadow tab of the Object Properties dialog box, you can set direction, color, transparency, size, and distance of shadow and apply or remove shadow.

More Information

Shadow Tab of Object Properties Dialog Box

Shadow

  • No Shadow

  • Do not apply shadow. If shadow is applied to drawing objects, remove the shadow.

  • Shadow

  • Provide nine kinds of outer shadow and nine options for inner shadow with the drawing object as the basis as well as five kinds of perspective shadow.

  • Color

  • Click the color table to display the color palette. Select a shadow color in the color palette.

  • Transparency

  • Adjust transparency of shadow. The adjustment range is from 0% through 100%.

  • Blur

  • Adjust concentration of shadow. The adjustment range is from 0pt through 100pt.

  • Size

  • Adjust size of shadow. The adjustment range is from -200% through 200%.

    If inner shadow is selected, the Size control will not be active.

  • Slant

  • Adjust slanting of shadow. The adjustment range is from -90 degrees through 90 degrees.

    This is activated only when perspective shadow is selected.

  • Distance

  • Adjust distance between a drawing object and shadow. The adjustment range is from 0pt through 100pt.

  • Angle

  • Adjust rotation angle of shadow based on the shadow position from the drawing object. The adjustment range is from 0 degree through 360 degrees.
