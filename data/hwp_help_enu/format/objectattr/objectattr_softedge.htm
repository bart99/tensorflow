Object Properties: Soft Edges
 

Home > Object Properties > Soft Edges

In the Soft Edges tab of the Object Properties dialog box, you can set size of soft edges and apply or remove soft edges.

More Information

Soft Edges Tab of Object Properties Dialog Box

Soft Edges

  • No Soft Edges

  • Remove the soft edges applied to drawing objects.

  • Soft Edges

  • Six size options for soft borders applied to drawing objects.

  • Size

  • Specify size of soft edges by typing or clicking the increase or decrease icon. Size can be set from 0pt through 50pt.
