Hide Current Page Only
 

Page Layout > Hide Current Page Only

Documents such as book, report, and thesis do not usually have header, footer, page number, page border/background, and master page on their covers. For the cover, you can hide only the current page by using Page Layout > Hide Current Page Only.

Steps

  1. Place the pointer on the page with content to hide and click Page Layout > Hide Current Page Only.
  2. In the Hide Current Page Only dialog box, select Header, Footer, and Page Number in Item to Hide.
  3. Click Set. Footer, header, and page number will not be on the page with Protect on when the page is printed or displayed on the edit screen.

Note

Item to Hide

The items that can be hidden on the current page are Header/Footer, Page Background, Page Border, and Page Number that are displayed on each page once inserted in the document.

Cancel Hide

To cancel Hide, click Page Layout > Hide Current Page Only again for the hidden paragraph.

To go to the paragraph with Hide applied, click Home > Go to and select Control Codes > Hide. If you click Go, the paragraph where Hide is applied will be found.

Hide First Page Only

Click Page Layout > Section Settings to hide header/footer, page borders/background, and master page only on the first page of each section.

Hide Many Pages

To apply Hide to many pages, click Page Layout > Hide Current Page Only and select items to hide on each page.

Hide Only Page Number Inserted Through Page Numbering

If you select Item to Hide > Page Number, only the page numbers inserted by using Insert > Page Number will be hidden.

The page number that you inserted in the footer/header or the master page by using Page Number will not be hidden.

If you want to hide the page number inserted in the header/footer or master page, you have to hide the header/footer or master page.

Caution

Location for Inserting Hide Control Code

The Protect control code is inserted at the beginning of the paragraph of cursor position. If a paragraph covers Page 1 and Page 2 and you use Protect in the first line of Page 2, the Protect control code will be created at the beginning of the last paragraph of Page 1. The content on Page 1, where the Protect control code will be inserted, will be hidden, instead of the content on Page 2 to which Protect is applied.

When you use Protect on the current page, use it in the first paragraph instead of the beginning of the page.
