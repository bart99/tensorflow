Page Background
 

Page Layout > Page Background

From the Background tab of the Page Border/Background dialog box, you can fill the background of a document with a color, a pattern, a picture or gradation.

Steps

  1. Click Page Layout > Page Background.
  2. Select Color in the Background tab and click Face Color. From the color palette, click a background color.
  3. Click Set. The selected background color applies to all pages of the document.

More Information

Background Tab of Page Border/Background Dialog Box

No Fill

You can select No Fill if you are not applying Color, Gradation, or Picture.

Color
  • Face Color

  • Click the color table to display the color palette. Select a background color in the color palette.

  • Pattern Color

  • If Pattern is selected, click the color table in Pattern Color to display the color palette. In the color palette, select a pattern color to fill the background with.

  • Pattern

  • From the Pattern list, select a pattern to fill the background with.

Gradation

Fill the background with gradation for gradual color spread.

  • Beginning Color

  • Click the color table to display the color palette. From the color palette, select a color that spreads from the center of the object to the outside.

  • Ending Color

  • Click the color table to display the color palette. From the color palette, select a color that comes into the center from the outside of the object.

  • Type

  • Select a style from the various gradation templates. You can create a type or style you want by selecting Custom.

  • H Center

  • Display the position of the horizontal center of the object in percent.

  • V Center

  • Display the position of the vertical center of the object in percent.

  • Angle

  • Specify how many degrees to rotate gradation.

  • Blur Level

  • Specify blur level for the beginning color and the ending color. Level 0 means no blur effect. Blur level can be set from 0 through 255.

  • Blur Center

  • Specify spread level for the beginning color and the ending color. Level 0 means no face color. Level 2 only displays the beginning color and the ending color. This can be set from 0 through 100.

Picture

Open a picture file and fill the background with the picture. Refer to Fill with Picture for more information.

  • Picture File

  • By clicking Open, open a picture file to fill the background.

  • Embedded

  • Include a picture file in a document as an inserted picture without connecting it to the document. The modification or position change of the original picture file will not affect the picture included in the document if you select this option.

  • Fill Type

  • From the list for fill type, select Tile, Fit to Size, or Center when filling the background with a picture.

  • Effects

  • Apply grayscale or black and white to the opened picture.

  • Watermark Effect

  • Apply 70% brightness and -50% contrast to the opened picture file to change it to a bright and low-contrast picture.

  • Brightness

  • Adjust the brightness of the selected picture file. Brightness can be set from -100 through 100.

  • Contrast

  • You can adjust the contrast of the selected picture file. Contrast can be set from -100 through 100.

Border/Background Type

You can apply different borders and backgrounds to odd and even pages of a book.

Select Even Pages in Border/Background Type of the Background tab to create a page background for even pages located on the left part of the book. Click Page Layout > Page Borders again and select Border/Background Type > Odd Page in the Background tab to create a background for odd pages located on the right part of the book. Then different page backgrounds will be inserted on even pages and odd pages.

  • Both Pages

  • Apply the same border/background to odd pages and even pages.

  • Odd Page

  • Create a page border/background for even pages located on the left part of the book.

  • Even Page

  • Create a page border/background for odd pages located on the right part of the book.

Page to Apply

Selects pages to apply the selected background to.

  • All

  • Display the background on all pages.

  • Exclude First Page

  • Do not display the page background only on the first page of the current section.

    Do not insert header/footer, page number, page border/background, or master page in the first page of a section because the first page of a section is used as the cover of a chapter or paragraph.

  • First Page Only

  • Display the selected page background only on the first page of the current section.

Apply to

Suggests settings for the section number, pointer position, and block setting of the active document.

You can select either Whole Document or New Section. If the active document is divided into two or more sections, you can select Current Section.

  • Whole Document

  • Apply the selected page background to the whole document even if the document is divided into two or more sections.

  • New Section

  • After creating a new section by dividing pages at the current pointer position, insert the selected page background from the new section.

  • Current Section

  • Insert the selected page background in the section where the pointer is.

Area to Fill

Select paper, page, or borders as an area to fill with the page background.

  • Paper

  • Fill the entire paper with a background.

  • Page

  • Fill the inside of the body text with a background.

  • Borders

  • Fill the inside of borders with a background.

Note

Hide Background on Current Page Only

To hide the background of the current page, click Page Layout > Hide Current Page Only and select Page Background. Then the page background of the current page will not be displayed on the edit screen or print-out.

Hide Background on First Page Only

If you select Page to Apply > Exclude First Page in the Background tab or Hide Border/Background in First Page in the Section Settings dialog box, the page background will not be displayed only on the first page of the current section.

Do not insert header/footer, page number, page border/background, or master page in the first page of a section because the first page of a section is used as the cover of a chapter or paragraph.
