Page Borders
 

Page Layout > Page Borders

From the Borders tab of the Page Border/Background dialog box, you can insert various page border lines in different pages and show/hide only the borders of certain sides.

Steps

  1. Click Page Layout > Page Borders.
  2. Click the arrow of the Borders > Type list from the Borders tab to select the second item, Solid, and select 0.5mm in Weight.
  3. Click All in the border preview area to apply the selected borders to all sides.
  4. Click Set. Borders will be drawn in all pages of the active document.

More Information

Borders Tab of Page Border/Background Dialog Box

Borders

You can select from 11 types of lines for page border and select a weight and color for each border type. You can only select a line weight from the weight list. You cannot type a value for weight.

  • Preview Line Style

  • After you select a border type, it applies to the border whose border icon is selected in the preview area.

    If you do not select this option after selecting a border type, you have to click the icon of a side to apply the border type to the side.

Position

The position of page borders depends on the base of page borders.

  • Whole Paper-Based

  • You can set page borders within the range of 25mm at maximum to the left, right, top and bottom inward from the paper edges (the blue area in the picture below).

 
  • Page Body-Based

  • You can set page borders within the range of 25mm at maximum to the left, right, top and bottom outward from the edges of the body text (the blue area in the picture below).

 
Border/Background Type

You can apply different borders and backgrounds to odd and even pages of a book.

Select Even Page in Border/Background Type of the Borders tab to create page borders for even pages located on the left part of the book. Click Page Layout > Page Borders again and select Border/Background Type > Odd Page in the Borders tab to create page borders for odd pages located on the right part of the book. Then different page borders will be inserted in odd pages and even pages.

Page to Apply

Select pages to apply the selected borders to.

  • All

  • Display borders on all pages.

  • Exclude First Page

  • Do not display page borders only on the first page of the current section.

    Do not insert header/footer, page number, page border/background, or master page in the first page of a section because the first page of a section is used as the cover of a chapter or paragraph.

  • First Page Only

  • Display the selected borders only on the first page of the current section.

Apply to

Suggests settings for the section number, pointer position, and block setting of the active document.

You can select either Whole Document or New Section. If the active document is divided into two or more sections, you can select Current Section.

  • Whole Document

  • Apply the selected page borders to the whole document even if the document is divided into two or more sections.

  • New Section

  • After creating a new section by dividing pages at the current pointer position, insert the selected page borders from the new section.

  • Current Section

  • Insert the selected page borders in the section where the pointer is.

Note

Change Borders

To change the border of the side, select a line type, line color, and line weight. Then click the border icon of the side in the preview area to change only the border of the side back and forth between current border and previous border.

Adjust the Border of Each Side

To adjust the border of each side in the preview area. Click the left/right/top/bottom icon, the left/right/top/bottom border changes back and forth between current border and previous border. Click the All icon to change the borders of all sides back and forth between current border and previous border.

Borders and Header/Footer

If the body text or objects are in the position where page borders are to be inserted, page borders will be drawn behind the body text and the header and footer will be located outside page borders. However, if you select Position > Page Body-Based from the Borders tab, you can include the header and footer inside the page borders.

Hide Current Page Only

To hide the borders of the current page, click Page Layout > Hide Current Page Only and select Page Border. Then the borders of the current page will not be displayed on the edit screen or print-out.

Hide First Page Only

If you select Page to Apply > Exclude First Page in the Page Border/Background dialog box or Hide Border/Background in First Page in the Section Settings dialog box, the page borders will not be displayed only on the first page of the current section.

Do not insert header/footer, page number, page border/background, or master page in the first page of a section because the first page of a section is used as the cover of a chapter or paragraph.

Print Layout

You can check page borders on the edit screen only when View > Print Layout is on.

Caution

Restriction

On the edit screen, you cannot adjust the position of page borders inserted in a document with the pointer.
