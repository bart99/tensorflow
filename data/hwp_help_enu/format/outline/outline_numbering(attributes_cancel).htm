Toggle Multilevel List
 

Home > Multilevel List > Toggle Multilevel List

You can run the command that toggles multilevel list by clicking Home > Toggle Multilevel List.

Steps

  1. Click File > New and type Multilevel List Test in a new document.
  2. If you click Home > Multilevel List > Toggle Multilevel List, a multilevel list will apply.

    When a multilevel list is applied, the multilevel list number 1 is displayed at the beginning of the current paragraph and the Styles of the formatting toolbar changes to Outline 1.

  3. If you change lines by pressing ENTER, the multilevel list number 2 will be inserted in the next paragraph. Type Small Multilevel List after 2.
  4. If you click Home > Multilevel List > Toggle Multilevel List in the Small Multilevel List paragraph, the multilevel list will be removed.

    As the multilevel list gets removed, the Styles of the formatting toolbar changes to the previous style Normal.

Note

Multilevel List

If you place the pointer in a paragraph with a multilevel list and apply Toggle Multilevel List, the paragraph changes to a normal paragraph and the multilevel list of the rest of the paragraphs will automatically change accordingly.

If you place the pointer in a normal paragraph and apply Toggle Multilevel List, the paragraph changes to a multilevel list paragraph and an appropriate multilevel list number will be inserted automatically.
