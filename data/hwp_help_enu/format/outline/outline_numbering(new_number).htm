Numbering Starts by
 

Home > Multilevel List

If the active document is divided into two or more sections, you can select whether to start the multilevel list in the current section with new numbers or continue the numbers in the multilevel list in the previous section.

Steps

  1. In a document divided into two or more sections, place the pointer in the section where you want to change the starting number of the list.
  2. Click Home > Multilevel List > Multilevel List.
  3. In the Multilevel List dialog box, select Numbering Starts by > Restart Numbering.
  4. Type 1 as the starting number in the Level 1 Start at box.
  5. Select Current Section as Apply to.
  6. Click Set. The multilevel list in the current section will be renumbered starting with the number you typed.

More Information

Continue Previous Section

Continue the multilevel list of the previous section.

Restart Numbering

Restart the multilevel list from the current section. Type the starting number for the new multilevel list to apply.

Apply to

You can specify the application range based on the number of sections in the current document, the cursor position, block setting status, etc. In general, available options are Whole Document and New Section.

  • New Section: Divide the document into new sections by breaking the page at the pointer and apply a new multilevel list style.
  • Whole Document: Apply the same multilevel list style to the whole document even if the document is divided into two or more sections.
  • Current Section: If the document is divided into two or more sections, you can select the Current Section in Apply to.

Note

Change Multilevel List

You can change multilevel list styles per section. One section has only one multilevel list style and numbers on a multilevel list have to be continuous. Therefore, you cannot change the continuous numbers in a multilevel list or start the multilevel list with a new number in the same section.

If you want to use a new multilevel list style or start a multilevel list with a new number, you have to create a new Section.
