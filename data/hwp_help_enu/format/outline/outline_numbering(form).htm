Custom Multilevel List Style
 

Home > Multilevel List

Show All

You can reorganize multilevel list styles for each level to your preference.

Steps

  1. Click Home > Multilevel List > Multilevel List.
  2. In the Multilevel List dialog box, click Custom.
  3. In the Custom Multilevel Number Style dialog box, select Level 1 in the level list box that is comprised of seven levels.
  4. Select from the 17 multilevel list styles that appear when you click Numbering Styles.
  5. To add a specific multilevel list style to the Outline Number Style list, select Add New Number Style and type a name for the new multilevel list style.
  6. When you finish customizing a multilevel list style, click Set.
  7. The customized multilevel list will be added to the Outline Number Style list of the Multilevel List dialog box.
  8. Click Set to apply the customized multilevel list style to the body text.

More Information

Custom Multilevel Number Style Dialog Box

Level

Select a level for number format in the Level list box that is comprised of seven levels. You can specify Number Format. Insert Number. and Numbering Styles for each level.

  • Number Format

  • Decorate the level of numbers selected in the Level list box by inserting characters before and after the numbers.

  • Insert Number

  • To insert a high level number code in Number Format, click the arrow of the list and select a level of choice. On the Insert Number list, only the current level and the higher level appear.

  • Numbering Styles

  • Click the arrow of the list and select from 15 styles for numbers of the current level.

Font

You can select a font for each level of a multilevel list.

  • Set Font

  • The font of a multilevel list follows the font of the last character in the multilevel list.

    If you want to customize a font for a multilevel list, select Set Font and click Font. In the Font dialog box, set text size, font, and text properties to apply to numbers.

  • Follow Paragraph Font Size

  • If you select this option, the font selected in the Font dialog box will be used as the font of numbering, However, text size will be the same as the last character of the numbered paragraph.

    This option is activated only when Set Font is selected.

Number Position

Adjust the width of the area occupied by a multilevel list and specify how to align.

  • Adjust Width to the Number of Digits

  • If you select this option, when you number text with numbers of two or three digits, the width of the numbers will be aligned according to their digits.

    If you do not select this option, two-digit numbers will occupy the same area as one-digit numbers.

  • Adjust Width

  • Specify width of a multilevel list. The width of numbering is automatically calculated according to text and text size in Number Format.

  • Alignment

  • Select from Left, Center, and Right for the arrangement of numbers within numbering width.

Wrapping Style

Set distance between a multilevel list and the body text to maintain a certain distance regardless of text size.

  • Distance from Text

  • Set distance between numbers and the body text in percent (%) when automatic outdent is used.

  • Automatic Outdent

  • Automatically outdent as much as the width of numbers to align the body text vertically.

Add New Number Style

Add customized multilevel list styles to the multilevel list style list.

  • Name

  • Type the name of a new multilevel list style to add to the list.

Note

Change Multilevel List Style

There are 13 default multilevel list styles and custom styles.

By using Custom Multilevel Number Style, you can customize multilevel list styles for each level. For multilevel list styles for each level, define number format, numbering style, numbering font style, adjust numbering width, number alignment, and distance from text.

Remember Custom Style

Newly added multilevel list styles are saved in Hwp. So you can use customized styles when you create a new document window or restart Hwp.
