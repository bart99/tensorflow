Spuit
 

You can select a color from other applications or from the monitor screen by using the Spuit feature. This is a convenient feature as it replaces the use of the color palette or the Colors dialog box.

Steps

To use spuit in Color Palette

  1. In the edit window, type Hancom and select as a block.
  2. On the Formatting toolbar, click the arrow next to the Font Color icon.
  3. From the Color Palette, click the Spuit icon.
  4. The pointer changes to a spuit and a zoom preview window is displayed on the upper left corner of the screen.
  5. You can check the location and color using the zoom preview window and click the spot having the color you want.

  6. The text color of Hancom changes to the selected color. On the Formatting toolbar, the Font Color icon changes to the selected color.
  7. On the Formatting toolbar, click the arrow next to the Font Color icon to view a list of recently used colors in the color palette.

To use spuit in the Colors dialog box

  1. In the edit window, type Hancom and select as a block.
  2. On the Formatting toolbar, click the arrow next to the Font Color icon.
  3. From the Color Palette, click Others.
  4. From the Colors dialog box, click the Spuit icon.
  5. The pointer changes to a spuit and a zoom preview window is displayed on the upper left corner of the screen.
  6. You can check the location and color using the zoom preview window and click the spot having the color you want.
  7. The selected color is displayed as the Current Color of the Colors dialog box. To register the color in the Custom Color list, click Add.

  8. When you click Set, the text color of Hancom changes to the selected color. On the Formatting toolbar, the Font Color icon changes to the selected color.
