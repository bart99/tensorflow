Text Direction Options
 

Page Layout > Text Direction

You can use Vertical Writing not only in specific sections such as table and text box but also in normal edit mode. Alternatively, you can specify the target area (block, specific sections, or the entire document) and apply the Vertical Writing setting only to that area.

More Information

Vertical Writing Dialog Box

Orientation

Select either portrait or landscape for the area to apply Vertical Writing to.

Type

Type Description
Horizontal Writing Select this option to switch from vertical writing to horizontal writing.
English Down Apply vertical writing. Rotate English and numbers 90 degrees clockwise.
English Up Apply vertical writing. Return English and numbers to their original state.

Options

  • Apply Vertical Writing to Header/Footer

  • Apply vertical writing to not only header and footer but also page numbers.

  • Apply to

  • Select a range to apply vertical writing to.
