Export Styles
 

Home > Styles > Export Styles

With this feature, you can save the style defined in the active document as a separate style file (*.STY).

Steps

  1. Click Home > Styles.
  2. In the Styles dialog box, click the Export Styles icon .
  3. In the Export Styles dialog box, select New Style File from the Select File list.
  4. In the New Style File dialog box, type the style title in the Title box and click the Export Styles icon in File Name.
  5. In the Export Styles dialog box, specify path and name of the style file to save as below and click Save.

    Example) C:\Documents and Settings\Hancom\Application Data\HNC\User\Hwp\New Style File.sty

  6. When the name and path of the style file to save are entered in the File Name box of the New Style File dialog box, click Create.
  7. The Normal style displays on the Target list of the Export Styles dialog box.
  8. The style selected as Target gets copied if you select a style from the Current list and click the Copy icon .
  9. Click Close. A message "Save [C:\Documents and Settings\Hancom\Application Data\HNC\User\Hwp\New Style File.sty]?" displays.
  10. Click Yes to save the style.

More Information

Export Styles Dialog Box

Current

Display the name of the active document and a list of styles applied to the active document.

Target

Display the name of the target file you opened by using Select File and a list of styles applied to the target file.

Select File

Select a document to export the style applied to the active document by selecting it from the list or clicking the Select File icon .

From the Select File list, you can select Normal or New Style File.

  • Normal

  • Apply the style of the master document that is read by default when Hwp starts.

  • New Style FIle

  • To create a style file (*.sty) display the New Style File dialog box.

    The Normal style will be displayed on the Target list of the Export Styles dialog box if you create a style file by specifying its name and path in the New Style File dialog box.

  • Select File Icon

  • To select a Hwp document, style file (*.sty), or HWPML file (*.hml) that has a style applied display the Open dialog box.

Details on Icons

Icon Name Description
Copy Copy the style selected from the Current list in a target document.
Copy All Copy all styles of the Current list in a target document.
Delete Delete the selected style. You cannot delete the normal style.
Delete All Delete all styles of the Target list. You cannot delete the normal style.

New Style File Dialog Box

Title

Type the title of a style file (*.sty) to save. Use a title that indicates the content of the style.

You can check the title if you select a saved style file in the Open dialog box and click the Summary icon .

File Name

Select a style file by typing its path and name or by clicking the Select File icon .

  • Select File Icon

  • To select a Hwp document, style file (*.sty), or HWPML file (*.hml) that has a style applied, display the Open dialog box.

Note

Normal

Normal style always exists in Hwp. You cannot delete, rename, or move it. The normal style gets created by default when you create a new style file.
