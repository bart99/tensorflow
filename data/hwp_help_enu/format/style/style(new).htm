Add Style
 

Home > Styles

This feature allows you to create frequently used font and paragraph format into a style.

You can change font and paragraph format in a document all at once by applying the style once you create a style. You do not have to change the format of each font and paragraph. This helps reduce document editing time.

Steps

To add a style

  1. Adjust font and paragraph format to the desired format before creating a style.
  2. Place the pointer in a paragraph where you want to add a new style, and click Home > Styles.
  3. In the Styles dialog box, click the Add Style icon .
  4. In the Add Style dialog box, type Big Title in the Style Name box and select Paragraph in Style Type.
  5. Click Paragraph. In the Paragraph dialog box, type 10pt in Margins > Left and click Set.
  6. Click Font. In the Font dialog box, type 14pt in Standard Size, select Bold and Underline in Properties and click Set.
  7. Once you finish setting a style, click Add and close the Add Style dialog box.
  8. Big Title will be displayed on the Style List of the Styles dialog box.

More Information

Add Style Dialog Box

Style Name

Type the name of a new style.

Name in English

Type the English name of a new style.

Style Type

Specify style type.

  • Paragraph

  • Add the font format as well as paragraph format as a style.

    If you add a paragraph style, a paragraph style icon displays in front of the name of the style on the Style List in the Styles dialog box.

  • Character

  • Add only font format as a style.

    If you add a font style, a font style icon displays in front of the name of the style on the Style List in the Styles dialog box.

Style for Following Paragraph

You can decide which paragraph style to apply to the next paragraph by applying the current style and then making a forced line break by pressing ENTER.

All styles in the active document display on a list when you edit Paragraph. You cannot select Style for Following Paragraph when you edit Font.

Paragraph Button

To specify a paragraph style open the Paragraph dialog box.

Font Button

To specify a font style open the Font dialog box.

Numbering/Bullets Button

Specify numbering style, bullet style, and image bullet style.

The paragraph type of the paragraph style will be automatically set as number or bullets if you set Numbering/Bullets for a style. The numbering style of the current paragraph will be stored in the style as well if you create a style from a paragraph with numbering or bullets.

Note

Maximum Number of Styles

The maximum number of styles that can be used in a file is 160. Prefer using 10 or so styles in one document.

Name in English

While creating a new style, type an English name that indicates the content of the style. English style names are displayed on the style list when you open a Hwp document (*.HWP) from Hwp of multi-language resources.

You do not have to type an English name when creating a new style.

Change both the style name and the English name when creating a new style by editing the default style.

Hwp uses the English name of a style to distinguish the style from other styles. As Hwp has two different styles with different style names but the same English name as the same style, it uses the existing style with the same name when you use Paste or Contents from Document.
