Applying Styles
 

Home > Styles

This feature allows you to select a style from the style list to apply it to the current pointer position or the current paragraph.

You can change the style of an entire document by applying a different style.

Steps

To apply styles by using the Styles menu

  1. Point to the paragraph whose style has to be changed.
  2. The Styles dialog box appears when you click Home > Styles.
  3. Select the name of a style from the Style List and click Set.
  4. The font format and paragraph format of the paragraph with the pointer changes.

To apply styles by using the Formatting toolbar

  1. Point to the paragraph whose style has to be changed.
  2. On the Formatting toolbar, select the name of a style from the Styles.
  3. The font format and paragraph format of the paragraph with the pointer changes.

To apply styles by using the Styles task pane

  1. Point to the paragraph whose style has to be changed.
  2. To display the Styles task pane on the left of the edit window, click View > Task Pane > Styles.
  3. Click the name of a style on the Style List.
  4. The font format and paragraph format of the paragraph with the pointer changes.

Note

Style Shortcut

You can apply styles easily by using shortcuts. In the Styles dialog box, continuous 10 numbers are assigned to the 10 styles with the Normal style at top as No. 1. The style applies immediately on placing the pointer in a position where you want to apply a style and press its number while pressing CTRL.

Apply Style Automatically

Hwp automatically applies a style appropriate for each on inserting a multilevel list, page number (by using Page Numbering), or footnotes/endnotes in the body text.

Hwp does so by finding a style on the style list of the active document, which matches the English name of a style that Hwp remembers.

On inserting a multilevel list, Hwp applies a style named English Name-Outline 1 from the style list of the active document. On inserting a page number, Hwp applies a style named English Name-Page Number. On inserting footnotes, Hwp applies a style named English Name-Footnote.

Apply Style Repeatedly

The changed format remains while the style applies to the rest on changing some font or paragraph format after applying a style.

On applying the style to the same paragraph again, a message "Overwrite the body text with the style?" displays. The format you changed will be overwritten by the style by clicking Yes.

Style Application

Font styles apply to the current pointer position and paragraph styles apply per paragraph. You can use multiple font styles but only one paragraph style in a particular paragraph.

The new style applies to the word or paragraph where the pointer is when you change styles, To apply the same style to multiple words or paragraphs, you have to select them as a block and then apply the style.

Check Current Style

On the Formatting toolbar in Styles, you can check which style is applied to the current pointer position. The name of the font style will be displayed if a font style is applied to the pointer position. The name of the applied paragraph style will be displayed if no font style is applied.

In the Styles dialog box, the style used in the active document has a check mark in the name part of the Style List.

Copy Style

You can copy the style at the pointer and overwrite content in another position with the style by using Home > Format Painter.

Overwrite Style

Replace the font or paragraph style with a new style if you have not changed any font or paragraph format after applying a style.

Advanced Usage

Select Block and Apply Style

The style of the block changes to the new style whether you have changed any font or paragraph format or not on making a block and apply a new style.

Style When Copying Block

The content as well as the style applied to the block will be copied on selecting a part of a document as a block and using Copy or Cut.

The style of the document will be used if there is a style with the same name as the copied style in the document where the copied style has to be pasted. The styles that do not exist in the active document will be added to the bottom of the style list in the active document.

Style When Using Paste

The style of the paragraph changes depending on where the copied content is pasted if you copy content that has a style applied and paste it in another paragraph.

The copied content will lose its style and use the style of the current paragraph when pasted if you use Paste in the middle or at the end of the paragraph.

The style of the paragraph will be overwritten by the style of the copied content if you use Paste at the beginning of the current paragraph.

Style When Using Document

The styles saved in the inserted document follow by inserting a document saved in the disk into the active document by clicking Insert > Contents from Document. The styles of the active document will be used if the active document has styles with the same names. The styles that do not exist in the active document will be added to the bottom of the style list in the active document.

To figure out why the message box does not open when applying a style repeatedly

A message "Overwrite the body text with the style?" is displayed on changing styles and then applying the same style to the same paragraph again by using the Styles dialog box.

Everything including format that is changed will be overwritten by the style without a message box by pressing CTRL and the number of the style without opening the Styles dialog box.
