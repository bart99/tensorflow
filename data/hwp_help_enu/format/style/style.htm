Styles
 

Home > Styles

A style is frequently used font or paragraph format that you save and use later. You can change the font and paragraph format of a paragraph all at once of a newly created style by using the style.

Styles are necessary for maintaining consistent paragraph format in a long piece of writing and to select font or paragraph format easily.

Steps

To apply styles by using the Styles menu

  1. Point to the paragraph whose style has to be changed.
  2. If you click Home > Styles, the Styles dialog box displays.
  3. Select the name of a style from the Style List and click Set.
  4. The font format and paragraph format of the paragraph with the pointer changes.

To apply styles by using the Formatting toolbar

  1. Point to the paragraph whose style has to be changed.
  2. On the Formatting toolbar from the Styles list, select the name of a style.
  3. The font format and paragraph format of the paragraph with the pointer changes.

To apply styles by using the Styles task pane

  1. Point to the paragraph whose style you want to change.
  2. To display the Styles task pane on the left of the edit window, click View > Task Pane > Styles.
  3. Double-click the name of a style you want on the Style List.
  4. The font format and paragraph format of the paragraph with the pointer changes.

More Information

Styles Dialog Box

Style List

A list of the currently registered styles appears.

Style List displays the font and paragraph format applied to a style in preview.

  • Paragraph Style

  • A paragraph style is made up of font format, paragraph format, tab settings, and paragraph border/background that are required in a paragraph. A paragraph style icon is displayed in front of the name of a paragraph style.

    The mark with a paragraph style icon and a check mark overlapped is displayed in front of the style used in the active document.

  • Text Style

  • A text style is a set of the selected font, text size, and text properties. A Font icon is displayed in front of the name of a font style.

    The mark with a font style icon and a check mark overlapped is displayed in front of the style used in the active document.

Details on Icons
Icon Name Description
Add Style You can create frequently used font and paragraph format into a style.
Edit Style Edit the name, paragraph format, font, next paragraph style, and other settings of an existing style.
Delete Style Delete styles you do not use or need to reorganize the style list.
Change to Current Format The font or paragraph style at the pointer will be overwritten with the new style by selecting a style from the Style List and click the icon.
Move One Space Up If you select a style from the Style List and click the icon, you can move the style one line up on the list.
This order applies to the Styles list of the Formatting toolbar as well.
Move One Space Down If you select a style from the Style List and click the icon, you can move the style one line down on the list.
This order applies to the Styles list of the Formatting toolbar as well.
Style Templates Show the Style Templates dialog box.
Import Styles Import a Hwp document with a style applied or a saved style file (*.sty).
Export Styles Save the style defined in the active document as a separate style file (*.sty).
Paragraph

In the preview area, you can check the paragraph format of the style that you selected from the Style List.

Paragraph Info

Display information about the Left Margin, Right Margin, Line Spacing, First Line, Alignment of the style that you selected from the Style List.

Font Info

Display information about the Font, Size, Scale, and Spacing of the style that you selected from the Style List.

Numbering/Bullets Info

Display information about the Numbering/Bullets of the style that you selected from the Style List.

Cursor Position Style

Display the name of the Style applied to the current pointer position.

  • Remove Text Style Button

  • Remove the font style applied to the current pointer position.

Note

Limit on Style Number

The maximum number of styles that can be used in a file is 160. Prefer using 10 or so styles in one document.
