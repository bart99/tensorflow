Normal Style
 

Home > Styles

A Normal style is set by default for paragraphs in a new document that is created by clicking New. Normal style always exists in Hwp. You cannot delete, rename, or move it. You can change its content, if needed.

Steps

  1. Click Home > Styles.
  2. In the Styles dialog box, select Normal from the style list and click the Edit Style icon .
  3. In the Edit Style dialog box, edit paragraph format, font format, and style to apply to the next paragraph, and click Set.

Note

Information about Normal Style

Normal style has a Batang font of 10pt, 100% scale, 0% spacing, and black normal characters and paragraph format of 160% line spacing, Justified, 0 left and right margin, and default tab.

Caution

Normal Style Cannot Be Renamed

You can only edit Paragraph and Font of the normal style. You cannot change its Style Name or Name in English.
