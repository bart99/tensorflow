Master Page
 

Page Layout > Master Page

In the Master Page, you can specify a page format that will apply to all pages of the document. You can apply the master page to odd pages and even pages separately. When you are creating a book document, you can edit the master page according to how it looks on an open book.

If you apply the master page differently to different sections, you can easily create different backgrounds for different chapters and paragraphs.

More Information

Master Page Dialog Box

Type

You can create different kinds of master page for different sections.

You can apply the master page for both pages or the master page for even pages and odd pages separately to one section. For example, if you create both the master page for both pages and the master page for odd pages in one section, the master page for both pages will be displayed on even pages and the master page for odd pages will be displayed on odd pages.

  • Both Pages

  • Create a master page that applies to both pages.

  • Odd Pages

  • Create a master page that applies only to odd pages.

    If you select both the master page for both pages and the master page for odd pages, the master page for odd pages overrides the master page for both pages.

  • Even Pages

  • Create a master page that applies only to even pages.

    If you select both the master page for both pages and the master page for even pages, the master page for even pages overrides the master page for both pages.

  • Last Page

  • Create a master page on the last page of the section where the pointer is.

    The master page applied to the last page of a section overrides the master page applied to both pages, odd pages, or even pages.

  • Specified Page

  • Create a master page that applies only to random pages in the selected section.

    The master page applied to random pages in a section overrides the master page applied to both pages, odd pages, or even pages.

Properties

  • Overlap with Existing Master Page

  • This option is activated only when you select Type as Last Page or Specified Page. If you select this option and apply the master page to Last Page or Specified Page, the section master page will be displayed on top of the normal master page (both pages, odd pages, or even pages).

  • Send Master Page on Top When Printing

  • Set the current master page to be on top of the body text when it is printed. In other words, the master page can cover the body content when printing the document.

Apply to

Specify range to apply the master page.

  • From Current Section

  • Apply the same master page to the section where the pointer is and all sections below it.

  • From First Section

  • Apply the same master page to all sections regardless of the current pointer position.

Note

Difference from Page Background and Page Border

You can use the Page Background and Page Border, only to specify border style, background color, and background picture that apply to all pages of a document.

However, you can use the Page Layout > Master Page to edit pages in an one-page area with almost all functions available on the normal edit screen including text entering, table inserting, and picture drawing.
