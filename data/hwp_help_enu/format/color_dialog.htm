Selecting Other Colors
 

You can select or introduce a new color in by using colors in the Colors dialog box.

Steps

To select colors by using the Colors dialog box

  1. In the edit window, type Hancom and select as a block.
  2. On the Formatting toolbar, click the arrow next to the Font Color icon.
  3. From the Color Palette, click Others.
  4. In the Colors dialog box, type 140 in Red, 173 in Green, 214 in Blue, and then click Add.
  5. Confirm that the color added to Custom Color is registered, and click Set.
  6. The text color of Hancom changes to the selected color. On the Formatting toolbar, the Font Color icon changes to the selected color.

More Information

Colors Dialog Box

Default Color

Default Color consists of a color palette in gradation and a color palette where you can select hue, saturation, and brightness.

RGB

RGB displays the RGB value of the selected color.

Type RGB values in the Red, Green, and Blue boxes or specify RGB values by using the increase or decrease icon to create a color you want.

HSL

HSL display the HSL value of the selected color.

Type HSL values in the Hue, Saturation, and Brightness boxes or specify HSL values by using the increase or decrease icon to create a color you want.

HTML

HTML displays the HTML hexadecimal code of the selected color.

You can select a color by entering values in the HTML box.

Custom Color and Add Button

If you enter RGB, HSL, or HTML code values, or select a color in the color palette and then click Add, you can add up to 12 colors to Custom Color.

Current Color/Previous Color

The current color and the previous color appear on the right of Custom Color. You can compare the colors and select one.

Spuit Icon

You can extract colors from other applications by clicking the Spuit icon.
