Section Break <ALT+SHIFT+ENTER>
 

Page Layout > Section Break

You can divide a document into multiple sections and apply different page setup to each section or create a new multilevel list. Click Page Layout > Section Break in a position where you want to create sections. A new page will start as a new section.

Steps

  1. Place the pointer in the spot from which you want to start a new section, and click Page Layout > Section Break.
  2. A new section will be created and the pointer will move to the beginning of the newly created page.

Note

Section Break

As sections are divided by page, if you forcefully break a document into sections, a new section will be created past the empty area for filling a page.

When Print Layout is not set, such forced section separators are displayed as a red line.

The control codes of the content created after the document is divided into sections are regarded as completely separate. Therefore, you can set section properties different from the previous section.

Delete Forced Section Break

If you press DELETE or BACKSPACE before or after the spot where you used Page Layout > Section Break, the section break will be deleted. Sections combined into one follow the style of the previous section.

Follow Properties

A new section not only has the section properties of the previous section but also follows its page setup, master page, footnote/endnote styles, presentation, page border/background properties, and column style.

Caution

Restriction

You cannot use Section Break in Header/Footer, Footnote/Endnote, Caption, Table and Text Box.
