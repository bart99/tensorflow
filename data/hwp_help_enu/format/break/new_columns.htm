New Column Settings
 

Page Layout > New Column Settings

At times, we use multiple column styles in one page or one document. If you use New Column Settings when using multiple column styles, you can define a new column style independent from the previous column. So you can keep changing column number in one page.

Steps

  1. Place the pointer in the spot where you want to start a new section style, and click Page Layout > New Column Settings.
  2. Content after the pointer is regarded as separate from the content of the previous column.
  3. To select a new column style, click Page Layout > New Column Settings.

Note

Follow Properties

A new column follows not only the column style of the previous column but also its page setup, master page, footnote/endnote styles, presentation, page border/background styles, and section properties.

Delete New Column Settings

If you press DELETE or BACKSPACE before or after the spot where you used Page Layout > New Column Settings, the new column settings will be deleted and the column style of the previous column will apply.

Between Columns

If you create more than one column style in one document by using New Column Settings, you can specify distance between columns in Between Columns of Page Layout > Section Settings.

Advanced Usage

Edit Different Columns in One Document or Page

If you use New Column Settings, you can edit various kinds of columns in one page. You can also edit columns of different column number or type in one page. For example, you can edit Column 2 and Column 3 together in one page.
