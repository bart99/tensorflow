Column Break <CTRL+SHIFT+ENTER>
 

Page Layout > Column Break

If you point to the next column when the current column is not fully filled, you can separate the content after the pointer into a new column by using Column Break.

Steps

  1. Place the pointer in the spot from which you want to start a new column, and click Page Layout > Column Break.
  2. A new column will be created and the pointer will be at the beginning of the selected column.

Note

Delete Forced Column Break

When Print Layout is not set, a forced column separator is displayed as a black dash.

If you press DELETE or BACKSPACE before or after the spot where you have used Page Layout > Column Break, the column break will be deleted. The control code of Page Break will be deleted but the previous and the next paragraphs will not be combined.

Column Break in One Column

Column Break in one-column document is the same as the forced page break.

Parallel Column Break

In Parallel, even if you keep using Column Break, pages will not change until the content of a column fills the current page, and text will be entered back and forth between the two columns.

If the content of a column on one page goes over the page limit, it does not continue in the next column but in the same column on the next page. The content will be continuously entered in the same column unless you use Column Break.

Column Break in Distribute

If you use Column Break in Distribute, the current content will be distributed into columns of the same height and the pointer will move to Column 1 so that you can type new content.

You can use Column Break in distribute in the same way as finishing a piece of writing and writing a new piece about a different subject in a document like newspaper.
