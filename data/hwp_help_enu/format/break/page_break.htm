Page Break <CTRL+ENTER>
 

Page Layout > Page Break

In Hwp, when the content is larger than the size of one page, a new page gets created. You can also create a new page by clicking Page Layout > Page Break.

Steps

  1. Place the pointer in the spot from which you want to start a new page, and click Page Layout > Page Break.
  2. A new page will be created and the pointer will move to the beginning of the new page.

Note

Forced Page Break

If you want to finish the current page and type content in a new page before the current page gets filled, click Page Layout > Page Break to create a page forcefully. If you forcefully break the document into pages, a new page will be created leaving the previous page as it empty.

When Print Layout is not set, a forced page separator is displayed as a red dash.

Delete Forced Page Break

If you press DELETE or BACKSPACE before or after the spot where you used Page Layout > Page Break, the page break will be deleted. The control code of Page Break will be deleted but the previous and the next paragraphs will not be combined.

CTRL+ENTER Inside Table

Inside a table, CTRL+ENTER is the shortcut for inserting one row.

Caution

Restriction

You cannot use Page Break in Header/Footer, Footnote/Endnote, Master Page, Caption, Table and Text Box.
