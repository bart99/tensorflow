Format Painter <CTRL+SHIFT+C>
 

Home > Format Painter

Format Painter copies the font, paragraph format, and style at the pointer to another position. This function is useful when you repeat specific styles.

Steps

  1. Place the pointer behind the character whose font you want to copy, and click Format Painter in the Home tab or click Home > Format Painter.
  2. In the Format Painter dialog box, select Font in Body Format and click OK.
  3. Select as a block the part whose format is to change.
  4. Click Copy Format in the Home tab or click Home > Format Painter.
  5. The font of the block changes to the copied font.

More Information

Format Painter Dialog Box

Body Format

  • Font

  • Copy the font at the pointer.

  • Paragraph

  • Copy the paragraph format at the pointer.

  • Font and Paragraph

  • Copy both the font and the paragraph format at the pointer.

  • Text Style

  • Copy the text style at the pointer. If a text style is not applied to the character at the pointer, you cannot select Text Style.

  • Paragraph Style

  • Copy the paragraph style at the pointer.

Cell Format

  • Cell Properties

  • Copy the font of the cell where the pointer is, and other cell properties including margins inside the cell, left alignment, title cell properties, single row, and vertical writing.

  • Cell Border

  • Copy the border type, weight, color, diagonal type, and application direction of the cell where the pointer is.

  • Cell Background

  • Copy the background fill properties of the cell where the pointer is.

Format in Table

  • Both Body and Cell Format

  • Apply selections of both Body Format and Cell Format.

  • Cell Format Only

  • Copy only the items selected in Cell Format excluding Body Format.

Note

No Block and Block

Format Painter copies the format at the pointer to a temporary storage when no block is selected. When a block is selected, Format Painter overwrites it with the copied format.

Remember Copied Format

If you specify what to copy in Format Painter, the format copied most recently will be remembered until you exit Hwp. You can apply the copied format to not only the active document but also other documents.

Format Painter in Table

In a table, you can copy the font, paragraph format, cell properties, line type, and cell background of the current cell and overwrite other cells with them.

You can select Cell Properties, Cell Border, and Cell Background only when you click Home > Format Painter in a table.

Use Styles

Use Styles instead of Format Painter when the same font or paragraph format is used repeatedly in a document. Styles are useful because you can change font and paragraph format at once when you edit a document again.

Format Painter and Copy Object Style

  • Format Painter

  • Format Painter is used to copy the body format (font, paragraph format, style) at the pointer and apply it to other parts of the body text, or to copy both the body format and cell format (cell properties, cell borders, cell background) of the cell selected as a block in a table and apply the format to other cells.

    The shortcut for copying/pasting format is ALT+C.

  • Copy Object Style

  • Copy Object Style is used to copy the style of a drawing object or picture object and apply it to other drawing objects or picture objects.

    The shortcut for Copy Object Style is ALT+SHIFT+C. The shortcut for Paste Object Style is ALT+SHIFT+V.

Caution

Pointer Position

When copying a font or font style, be careful with pointer position. Place the pointer right behind the character to copy and click Home > Format Painter.
