Page Numbering
 

Insert > Page Numbering

Page Numbering is a function that numbers pages in documents without complicated process.

Steps

To number pages

  1. Place the pointer on the page that you want to print with a page number, and click Insert > Page Numbering.
  2. In the Page Number dialog box, select Bottom Center for Number Position.
  3. For Numbering Styles, select an Arabic numeral such as 1, 2, and 3 and click OK.
  4. An Arabic numeral page number will be displayed in the lower center of all pages in the current section regardless of the current pointer position.

More Information

Page Number Dialog Box

Number Position

You can select from the following 10 number positions when inserting page numbers.

1: Top Left of Each Page 7: Outside Top of Odd and Even Pages
2: Top Center of Each Page   8: Inside Top of Odd and Even Pages
3: Top Right of Each Page   9: Outside Bottom of Odd and Even Pages
4: Bottom Left of Each Page   10: Inside Bottom of Odd and Even Pages
5: Bottom Center of Each Page    
6: Bottom Right of Each Page    

Numbering

From the list, select a style for page numbers to be printed.

  • Wrap with Hyphen

  • Insert hyphens before and after a page number.

    Example: Arabic Numeral: -3-

         Uppercase Roman Alphabet: -III-

         Lowercase Roman Alphabet: -iii-

Note

Apply per Section

Page Number numbers all pages of the current section regardless of the current pointer position.

Delete Page Number Position

To delete the specified page number position, click Home > Go to and delete the control code named Page Number Position.

Hide All Page Numbers After Current Section

If you insert page numbers by using Page Number and want to hide all page numbers located past a certain section, select Number Position > No Page Number. If you use No Page Number, no page past the current section will display page number.

Font of Page Number

The page numbers that you inserted by using Page Number use the English font in the style English Name: Page Number on the style list of the current document.

When you are creating a Page Number style to change page number fonts in a document without any page number style, you must name the style Page Number (case sensitive) in English.

When Header and Page Number Overlap

If the page numbers and footer (or header) you insert overlap each other when printed, follow the below instructions.

  • Adjust the Page Number position and the Header/Footer position so that the selected Page Number position and the Header/Footer position do not overlap when printed.
  • Instead of using Page Number, select Page Number in the Insert group of the Header/Footer tab so that Header/Footer and Page Number will not overlap each other.

Hide

When you are writing a thesis or report, you sometimes need to insert a new page in the middle. In this case, do not insert page number in the page.

Click Page Layout > Hide and select Page Number to not display the page number only on the page where the pointer is at. You can then hide the page number of the only the current page.

Difference Between Page 2 and Page ii

The international numbering practice is that page numbers start with the Arabic numeral, 1 from the first chapter of the body text and that preliminaries in a book, where the title, TOC, and preface are printed, start with the Roman alphabet, i from the title page in the very beginning.

Difference from Page Number

If you want to adjust the header/footer or the page number position in detail or insert custom page numbers, insert page numbers in the header/footer by using Page Number, instead of Insert > Page Number.

Insert Page Number in Desired Position

If you use Page Number, you can select only from 10 styles of number position for page numbers to insert.

If you want to insert page numbers of a desired style at a desired position, click Page Layout > Master Page and create a text box. Then click Page Number in the Enter group of the Master Page tab in the text box to insert page numbers. Next, select a font and move the text box to a position you want.

Caution

Restriction

There is no restriction on the number of page numberings in one file or page. When there are multiple page numberings, however, the last numbering displayed on each page will apply when it is printed.
