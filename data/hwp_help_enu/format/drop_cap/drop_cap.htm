Drop Cap
 

Insert > Drop Cap

You can apply a drop cap to the first letter of paragraphs as a decoration.

Steps

  1. Rest the mouse pointer on the paragraph whose first letter to decorate, and click Insert > Drop Cap.
  2. In the Drop Cap dialog box, select Three Lines in Format.
  3. From the Font list, select a font not used in the body text as the first letter of paragraphs should stand out.
  4. Click Set.

More Information

Drop Cap Dialog Box

Styles

Select a drop cap style from None, Two Lines, Three Lines, and Margin.

Font/Borders

  • Font

  • Select a font for the drop cap.

  • Line Type

  • Select a weight for borders of the text box that surrounds drop cap.

    The default value is None in the Format. If you select other Lines in the Format, you can specify Line Color and Line Weight.

  • Line Weight

  • Specify line weight.

  • Line Color

  • Click the color table to select a line color.

  • Face Color

  • Specify face color for the text that surrounds drop cap.

  • Distance from Text

  • Specify distance between drop cap and the body text.

Caution

Drop Cap Cannot Be Applied to Multiple Paragraphs at Once

You can only apply Drop Cap to one paragraph at a time. You cannot use Format > Drop Cap when multiple paragraphs or the current paragraph is selected as a block.
