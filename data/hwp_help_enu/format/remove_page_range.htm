Delete Page
 

Page Layout-Delete Page-Delete Page

Delete multiple pages by designating the range of pages in the Delete Page dialog box.

Steps

  1. Click Page Layout > Delete Page > Delete Page in a document with three or more pages.
  2. In the Delete Page dialog box, type "2-3" in Partial field under Ranges to delete.
  3. Click OK to delete from 2 to 3 page.

More Information

Delete Page Dialog Box

Ranges to Delete

  • Current Page

Delete the current page.

  • Selected Page only

Delete the selected page only.

  • Partial

Delete partial pages by typing the range of pages. If you type "3-5" and click OK, 3-5 pages are deleted from your document right away.

Note

Restorable

If you deleted pages by accident, press <Ctrl+Z> to undo the action.
