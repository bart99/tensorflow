Style Templates
 

Home > Style Template

Style Templates is a function that provides a collection of frequently used styles for each type of document and helps create various types of documents easily and fast.

Style Templates contains a collection of styles for each purpose. You can quickly modify a document you created with the default style just by selecting a style from Style Templates after checking the style in the Preview window.

Steps

  1. Create a document with the default style.
  2. Click Home > Style Template.
  3. On the Style Templates List of the Style Templates dialog box, double-click Report Style.
  4. To check the style in the Preview window, click Report 1 on the sublist Report Style and click Apply.
  5. The font and paragraph format of the document created with the default style will change to the format selected in Style Templates at once.

More Information

Style Templates Dialog Box

Style Templates List

Display a collection of styles per purpose in tree structure.

A collection of styles per purpose includes Thesis Style, Book Style, Report Style, Paper Style, Letter Style, and Present Style. Each style contains sample documents in various formats.

Preview

You can apply the style after checking how the style applies to the background text, body text, multilevel list, page number, header, and footnotes in the Preview window if you select a sample document of a style from the Style Templates List.

Default Button

In the Style Templates dialog box, if you want to change the style of a document back to the default style from the style you selected in Style Templates, click Default. The style of the document will be overwritten by the default Hwp style.

Note

Using Default Style

Style Templates only apply to documents that use the default style of the master document of Hwp. You cannot use the function of Style Templates properly for documents that do not use the default style.

Undo

To return a document back to its previous format after changing its format by using Style Templates, click Home > Undo.
