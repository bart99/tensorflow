Output Direction
 

Review > Mail Merge > Start Mail Merge

You can specify the option to send the mail merge result to the printer, the screen, a file, or an email.

More Information

Start Mail Merge Dialog Box

Output Direction

  • Printer

  • Sends the mail merge result to the printer. The number of copies printed is equal to the number of records in the data file. To change the printing method before printing the mail merge result, click Print Settings and then modify the settings.

    • Print Settings Button

    • Opens the Print dialog box so that you can modify the print settings.

    • Continuous Page Number

    • Prints with continuous page numbers.

    • Apply to Field Position

    • If you select this option and insert a mail merge mark in the table, the marked location is interpreted and mail merge data is displayed in the location.

  • Screen

  • Sends the mail merge result to the preview window. Displays the mail merge result in the preview window, you can also display the same result by clicking File > Preview.

    • Print Settings Button

    • Opens the Print dialog box so that you can modify the print settings.

    • Continuous Page Number

    • Prints with continuous page numbers.

    • Apply to Field Position

    • If you select this option and insert a mail merge mark in the table, the marked location is interpreted and mail merge data is displayed in the location.

  • File

  • Saves the mail merge result as a Hwp file (*hwp), so that it can be opened and used as per requirement. You can type a name for the file and specify the option to insert the page numbers in sequence. When you save the mail merge result as a file, the total number of pages is equal to the total number of records*template file pages.

    • File Name

    • To save the file as. Type a name, and then click on the Save icon to open the Save Hwp File dialog box. You can then specify the path and name of the file.

    • Continuous Page Number

    • Creates separate pages, but assigns the same page number to all pages.

      If you merge mail without specifying this option, a New Number is assigned automatically to each page, therefore each page will have 1 as its page number.

    • Apply to Field Position

    • If you select this option and insert a mail merge mark in the table, the marked location is interpreted and mail merge data is displayed in the location.

  • E-mail

  • Sends the mail merge result as an email message or as an email attachment. If you select Data Type > Address Book, the mail merge result is sent to all email addresses of the designated address records.

    If you select Hwp Document or DBF File for Data Type, you must specify the field number of the field containing the email addresses.

    • Mail Title

    • Type a title for the email message.

    • Body Text

    • Sends the mail merge result as an email message.

    • Attachment

    • Sends the mail merge result as an email attachment.

    • Address Field

    • If you select Hwp File, Cell/Excel File, or DBF File for Data Type, you must specify the field number of the field containing the email addresses.
