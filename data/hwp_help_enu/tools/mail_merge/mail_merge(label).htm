Printing the Results of the Mail Merge
 

Review > Mail Merge

You can easily create labels for letters and envelopes by printing the output of mail merge on label paper instead of printing on plain paper.

Steps

  1. To create a label template for mail merge, click Review > Labels > Create Label Template.
  2. In the Create Label Template dialog box, select the Label Template Package tab.
  3. You can select the manufacturer of the label paper from the label manufacturer list. A list of label papers from the selected manufacturer is then displayed in the label paper list.

    For example, select Formtec, and then Address (16) - 3107.

  4. Check the label paper in Preview and click Open.
  5. A label is created with red dots that trace the shape of the label paper.

  6. Position the cursor in the first label and click Review > Mail Merge > Mark.
  7. From the Insert Mark dialog box, select the Create Field tab.
  8. In the Enter field number of name field, type Address as the substitution field name, and then click Insert.
  9. After you enter Address on the first line of the first label, press ENTER and mark Name in the same way.
  10. Specify the font style, paragraph style, and cell margin settings, and then save the label template as labeltemplate.hwp.
  11. Open labeltemplate.hwp by clicking File > Open, and then click Review > Mail Merge > Start Mail Merge.
  12. From the Start Mail Merge dialog box, select Cell/Excel File under Data Type, and then click the Select File(2) icon. In the Open Cell/Excel File dialog box, select a file, and then click Open.
  13. Select the desired sheet and records from the Select Sheet and Select Records dialog boxes, respectively, and then click Select.
  14. From the Start Mail Merge dialog box, set Output Direction to Screen, and then click OK.
  15. In the preview window, a preview of the entries is displayed. The number of entries displayed is equal to the number of records contained in the data file inserted into the labels.
  16. To print the mail merge result, click the Print icon.

Note

Create Label Template

To print the mail merge result on label paper, you must click Review > Labels > Create Label Template from the template file, and insert mail merge marks into the labels.

Caution

Restriction

If you create a label template from a template file, insert mail merge marks, and then attempt to merge mail, data contained in other labels are ignored and labels equal to the number of records contained in the data file are printed, beginning from the first label inserted with the mail merge mark.

If you wish to print the mail merge result on label paper, you must insert the mail merge mark in the first label and leave all other labels empty.
