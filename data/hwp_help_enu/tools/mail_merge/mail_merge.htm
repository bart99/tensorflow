Mail Merge
 

Review > Mail Merge

Mail Merge merges a data file containing the names, addresses, and other information about numerous contacts with a template file to create tens or hundreds of identical email messages, addressed to multiple recipients of different positions and addresses. For example, if you need to send out regular newsletters to subscribers, you can process large amounts of email messages by using the Mail Merge feature. To merge mail, you need to have a data file and a template file.

Steps

To merge mail using Hwp data file

  1. Click File > New, fill the new document with data, and then save it as data.hwp.

  2. To create a template with the new document, click File > New. Place the cursor at the required location to insert the field data from the data file, and then click Review > Mail Merge > Mark.
  3. From the Insert Mark dialog box, select the Create Field tab. In the field, type 1 as the field number, and then click Insert.
  4. After you insert 1 at the cursor position, mark other fields in the same manner, and save the file as mail_form1.hwp.

  5. Click File > Open, open mail_form1.hwp, and then click Review > Mail Merge > Start Mail Merge.
  6. From the Start Mail Merge dialog box, select Data Type > Hwp File, and then click the Select File(1) icon.
  7. From the Open Hwp File dialog box, select data.hwp, and then click Open.
  8. From the Start Mail Merge dialog box, select Output Direction > Screen, and then click OK.
  9. View the preview of the Mail Merge result in the preview window. To print the result, click the Print icon.
  10. The data file and template file are merged, and the result is printed.

To merge mail with Cell data file

  1. Fill a new Cell document with data and save it as data.nxl.

  2. To create a template with the new document, click File > New. Place the cursor at the required location to insert the field data from the data file, and then click Review > Mail Merge > Mark.
  3. From the Mark dialog box, select the Create Field tab. In the field, type Name, and then click Insert.
  4. After you insert Name at the cursor position, mark other fields in the same manner, and save the file as mail_form2.hwp.

  5. Click File > Open, open mail_form2.hwp, and then click Review > Mail Merge > Start Mail Merge.
  6. From the Start Mail Merge dialog box, select Data Type > Cell/Excel File, and then click the Select File(2) icon.
  7. From the Open Cell/Excel File dialog box, select data.nxl, and click Open.
  8. From the Start Mail Merge dialog box, select Output Direction > Screen, and then click OK.
  9. From the Select Sheet dialog box, select Sheet1 under Sheet List, and then click Select.
  10. From the Select Records dialog box, select only the necessary records, and then click Select.
  11. If the field name for the data file differs from the field name for the template file, the Link Field Name dialog box appears.

    In the Link Field Name dialog box, specify the same field name for the data file and the template file, and then click OK.

  12. You can view the preview of the Mail Merge result in the preview window. To print the result, click the Print icon.
  13. The data file and template file are merged, and the result is printed.

Note

Data File

A data file contains names, addresses, and other information about individuals. This data set is used to substitute fields in the template file.

In Hwp, you can use Windows Address Book, Outlook Address Book, Hwp files (*.hwp), DBF files (*.dbf), Cell files (*.cell), Nexcel files (*.nxl), and Excel files (*.xls) as data files to merge mail.

Template

A template file contains information that is included in all mail merge results.

When you assign field numbers from the data file to the template file by using Mark, you can assign the same field number to multiple fields. It is not required to assign the fields in numerical order.
