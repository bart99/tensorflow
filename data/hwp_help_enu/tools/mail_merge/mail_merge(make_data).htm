Creating Data File
 

Review > Mail Merge

To merge mail, you need to have a data file and a template file.

You can create a data file from Hwp or use records that have already been registered to the address book of Windows, a compiled dBASE III Plus (*.dbf) file, a Cell file (*.cell), a Nexcel file (*.nxl), or an Excel file (*.xls).

Steps

To create a data file from Hwp

  1. Create an empty document window by clicking File > New.
  2. In the first line, type the number of fields, and then press ENTER to enter the field data for each type of record.

    For example, if a record consists of 3 fields (name, phone, and address), type 3 in the first line and then enter new records in groups of three lines. The field number for Name becomes 1, Phone 2, and Address 3.

  3. Click File > Save to save the data file.

To create a data file from Cell

  1. Launch Hancom Cell, and enter the data.

  2. Click File > Save to save the data file.

Note

Data File

A data file contains names, addresses, and other information about individuals. This data set is used to substitute fields in the template file.

In Hwp, you can use Windows Address Book, Outlook Address Book, Hwp files (*.hwp), DBF files (*.dbf), Cell files (*.cell), Nexcel files (*.nxl), and Excel files (*.xls) as data files to merge mail.

  • Address Book

  • If you select Windows Address Book or Outlook Address Book as the data file, the data from the selected address book is retrieved automatically.

  • Hwp File

  • If you are using a Hwp (*.hwp) file as the data file, the first line of the data file must contain a number indicating the total number of fields available in that data file.

    For example, if you are creating a data file containing the fields Name, Phone, Address, and Birthday, you must enter 4 in the first line. Enter data from the second line. Press ENTER after each field entry.

    A paragraph is recognized as a field, and records are identified by the number of paragraphs containing the number of fields identified in the first line of the data file. In other words, the number of paragraphs from the second line are the number of records.

  • DBF, Cell, and Excel Files

  • If you are using a DBF (*.dbf), Cell (*.cell), Nexcel (*.nxl), or Excel (*.xls) file as the data file to merge with a template file, you must enter field numbers of the field names used in the data file when you perform Mark on the template file.

Caution

Restriction

If you are using a Hwp file as the data file, the number of fields in each record must remain consistent, even if you need to include empty lines. You must to ensure that a field does not exceed one line in the template file and the data file. Mail Merge treats each data file field as a single character and aligns the paragraphs. Therefore, it does not break fields that run over to the second line after being merged.
