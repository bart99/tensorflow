Mark
 

Review > Mail Merge > Mark

Marks areas of a template file where names, positions, addresses, and other information from the data file will be substituted. You can use Windows Address Book, Outlook Address Book, Hwp files, .dbf files, Cell files, and Excel files as data files.

Steps

  1. To create an empty document window, click File > New.
  2. When you compose the template file, place the cursor at the required location to insert field data from the data file, and then click Review > Mail Merge > Mark.

  3. From the Insert Mark dialog box, select the Create Field tab. Type Name in the field, and then click Insert.
  4. Name is inserted as the field name at the cursor position.

  5. Set the marked Name as a block, and then click Formatting Toolbar > Font Color. Click the purple color from the color palette.
  6. The font style of the marked section changes as specified.

  7. Mark Phone and Address in the similar way, and then save the template file.

More Information

Insert Mark Dialog Box

Depending on the type of data file, you must select either the Address Book tab or the Create Field tab and mark the fields.

Address Book Tab

  • Select field name.

  • If you are using Windows Address Book or Outlook Address Book as the data file, select the field name of your choice from the field name list.

Create Field Tab

  • Type the field number or name.

  • If you are using a Hwp (*.hwp), DBF (*.dbf), Cell (*.cell), Nexcel (*.nxl), or Excel (*.xls) file as the data file, type the field number or name.

    If you are using a Hwp (*.hwp) file as the data file, use the number field only.

    If you are using a DBF (*.dbf), Cell (*.cell), Nexcel (*.nxl), or Excel (*.xls) file as the data file, use number and name fields.

Note

Mail Merge Font Style

The designated font style at the cursor position applies to marked template file areas. To apply a different font style to a mail merged section of a template, select the section, and then click Font to specify the font properties.
