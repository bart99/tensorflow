Start Mail Merge <ALT+M>
 

Review > Mail Merge > Start Mail Merge

It combines a data file with a template file to produce tens or even hundreds of identical e-mails addressed to multiple recipients with different positions and addresses.

Steps

  1. Open a template file and place the cursor in the template file.
  2. Click Review > Mail Merge > Start Mail Merge.
  3. From the Start Mail Merge dialog box, select Cell/Excel File under Data Type, and then click the Select File(2) icon.
  4. From the Open Cell/Excel File dialog box, select a Cell data file (*.cell), and then click Open.
  5. Set Output Direction to Screen and click OK.
  6. From the Select Sheet dialog box, select Sheet1 under Sheet List, and then click Select.
  7. From the Select Records dialog box, select only the necessary records, and then click Select.
  8. If the field name for the data file differs from the field name for the template file, the Link Field Name dialog box appears.
  9. In the Link Field Name dialog box, specify the same field name for the data file and the template file, and then click OK.
  10. In the preview window, check the displayed mail merge preview, and then click the Print icon.

More Information

Start Mail Merge Dialog Box

Data Type

To merge data from a file to their respective fields in a template file, you need to have a data file.

The supported data file types under Data Type are Hwp files (*hwp) containing record-specific data, records that have already been registered to the address book of Windows, compiled dBASE III Plus (*.dbf) files, and data-containing Cell (*.cell), Nexcel (*.nxl), and Excel (*.xls) files.

Output Direction

You can send the mail merge result to the printer, the screen, a file, or an email and other related settings by specifying it in Output Direction.

Caution

Pointer Position

To merge mail, you must first position the cursor inside the template file, and then click Review > Mail Merge > Start Mail Merge.

Number of Fields

If you attempt to open a data file created in Hwp, and merge it with a template file, the first line of the data file must specify the number of fields for each record. The Start Mail Merge cannot be performed without the specification.
