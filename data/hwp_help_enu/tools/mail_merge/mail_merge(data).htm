Data Type
 

Review > Mail Merge > Start Mail Merge

The types of data files that can be used to merge mail are, data-containing Hwp files (*hwp), records that have already been registered to the address book of Windows, compiled dBASE III Plus (*.dbf) files, and field data-containing Cell (*.cell), Nexcel (*.nxl), and Excel (*.xls) files.

More Information

Start Mail Merge Dialog Box

Data Type

You can use Hwp files (*hwp), records that have already been registered to the address book of Windows, compiled dBASE III Plus (*.dbf) files, and spread sheet files such as Cell (*.cell), Nexcel (*.nxl), and Excel (*.xls) files as data files.

  • Address Book

  • You can select Address Book to open the Select Records dialog box, and insert the records contained in the address book to the mail.

  • Outlook Address Book

  • You can select Outlook Address Book to open the Select Records dialog box, and insert the records contained in the address book to the mail.

  • Hwp File

  • Displays the file path and file name of all windows, excluding the current document window. You can select your required document as the data file, or click the Select File(1) icon to open the Open Hwp File dialog box, and then select a Hwp file (*.hwp) from a different location.

  • Cell/Nexcel/Excel File

  • If the file you require was created in Cell (*.cell), Nexcel (*.nxl), or Excel (*.xls), click the Select File(2) icon to open the Open Cell/Excel File dialog box, and then select the data file.

  • DBF File

  • If you need to use a database file (*dbf) containing addresses and/or telephone numbers, click the Select File(3) icon to open the Open DBF File dialog box, and then select the .dbf file.

    • Code Types

    • After you designate a .dbf file as the data file, you must select the code type from Korean (KS), Korean (KSSM), Chinese (GB), Chinese (BIG5), and Japanese (SJIS). The Select Records dialog box opens after you select a code and merge the mail. This dialog box displays the .dbf file which has been designated as the data file. From this dialog box, select the data you wish to use.

Select Sheet Dialog Box

This dialog box appears when you select Cell/Excel File from the Start Mail Merge dialog box.

To use data for mail merge, you must select the sheet containing the data from Sheet List.

Sheet List

Displays a list of sheets contained in the Cell/Excel file that was designated as the data file.

Select Records Dialog Box

This dialog box appears when you select Address Book, Outlook Address Book, Cell/Excel File, or DBF File from the Start Mail Merge dialog box, and then designate the data file.

You must select the records to mail merge from Address Book Records List. Press CTRL or SHIFT+CTRL to select some or all the records.

Address Book Records List

Displays a list of records contained in the data file.

Link Field Name Dialog Box

This dialog box appears if you designate a file and select Cell/Excel File from the Start Mail Merge dialog box. It is also displayed if field name of the template file and the field name of the data file do not match.

You must link the field name of the template file and the field name of the data file correctly, or modify the field name in the template file to match the field name of the data file.

Template Field Name

Displays the field name that exists in the template file but not in the data file.

Data File Field Name

Displays the list of registered field names of the data file. You can substitute the field names of the template file by selecting field names of your choice.
