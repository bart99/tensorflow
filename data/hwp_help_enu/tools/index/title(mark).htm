Mark as TOC
 

References > Table of Contents > Mark as TOC

Mark as TOC marks sections in the body so that they are used later to create a table of contents automatically.

To use this feature, find the sections you wish to mark as a table of contents element and click References > Table of Contents > Mark as TOC.

Steps

  1. Place the cursor on a body section you wish to mark as a table of contents element and click References > Table of Contents > Mark as TOC.
  2. Mark as TOC attribute is assigned at the cursor location.
  3. Repeat this process for all applicable paragraphs.
  4. Once you have completed marking all table of contents elements, move the mouse pointer to the top of the page where you wish to create the table of contents and click References > Table of Contents > Table of Contents.
  5. From the Table of Contents dialog box, select Table of Contents and Group into TOC Code and unselect Table of Tables, Table of Pictures, and Table of Equations under Table to Create.
  6. Select Create at > Cursor Position in Current Document and click Insert.
  7. A table of contents is created at the cursor position of the current document.

Note

TOC of Objects without Caption

When a TOC is made for pictures, tables, and equations without a caption, their frame numbers are used as contents.
