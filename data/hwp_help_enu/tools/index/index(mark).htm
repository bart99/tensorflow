Mark Entry
 

References > Index > Mark Entry

Index displays a list of keywords from the body at the very back of the book, along with the respective page numbers.

In order to use this feature, you must first mark the words and then click References > Index > Insert Index.

Steps

  1. Place the cursor after a word you wish to include in the index and click References > Index > Mark Entry.
  2. In the Mark Entry dialog box, the word preceding the cursor is entered in Main Entry automatically.
  3. Repeat this process for all applicable words.
  4. Once you have marked all the index entries, click References > Index > Insert Index.
  5. An index file is created in a new tab within the current window.

Note

Mark Index Word as Main Entry or Subentry

  • Main Entry Index

  • Arranges the index words in alphabetical order, along with their respective page numbers. To create a main entry index, enter the word in the Main Entry field of the Mark Entry dialog box and leave the Subentry field empty.

    • Pros: Indexing the words in alphabetical order provides an easy way to see if certain words have not been included in the index.
    • Cons: Because of the alphabetical arrangement, associated words are scattered throughout the index.
  • Subentry Index

  • Displays a list of headings in alphabetical order and organizes the index entries under the headings, along with their respective page numbers. To create a subentry index, enter the heading of the word in the Main Entry field and then the word itself in the Subentry field.

    • Pros: Groups associated index words under respective headings.
    • Cons: The reader is required to figure out which heading the word they seek is under and then look it up in the alphabetical list of index words belonging to that heading.

Index Font Size and Columns

Indices generally use the 9 pt font size and are formatted in the two-column format.

Caution

Spaces

Story and Story containing a space at the end are considered as two separate index entries. When inserting an index mark, be careful of unintended spaces preceding and proceeding the index work.
