Refresh TOC
 

References > Table of Contents > Refresh TOC

If you added or removed table of contents elements in your document, you can quickly update the table of contents.

Steps

  1. If table of contnets entries are modifed, click Reference > Table of Contents > Refresh TOC > Refresh TOC.
  2. Table of contents are updated.

More Information

Refresh TOC

Update the selected TOC.

Refresh All TOC

Update all TOC in the current document.

Note

The Refresh TOC menu can be disabled

When a table of contents was created using string instead of field, the Refresh TOC menu is disabled. To update a table of contents from time to time, you need to select Insert as Field in Type of the Create a Table of Contents dialog box when creating it.
