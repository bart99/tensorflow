Table of Contents
 

References > Table of Contents > Table of Contents

Table of Contents creates a list of lines containing a heading, table, picture, or equation, along with the page number of each line. Table of Contents is essential for creating a book or an essay.

If the page numbers of the document are changed after completing Table of Contents, you are required to run Table of Contents again.

Steps

  1. Move the mouse pointer to the page you wish to use for the table of contents using Style and TOC codes and click References > Table of Contents > Table of Contents.
  2. From the Create a Table of Contents dialog box, select Table of Contents and Group into Outline Paragraph and select Level 3 for Outline Level.
  3. Unselect Table of Tables, Table of Pictures, and Table of Equations, select Create at > Cursor Position in Current Document, and click Create.
  4. A table of contents is created at the cursor position of the current document.
  5. Specify the paragraph format and font format and save.

More Information

Create a Table of Contents Dialog Box

Table to Create

  • Table of Contents

  • You can create a table of contents by choosing one from the list or making multiple selections.

    • Group into Outline Paragraph

    • Displays outline paragraph lines in a specific part of the document, along with their page numbers. To create a table of contents using outline paragraphs, the headings must first be set as outline paragraphs.

    • Group into Style

    • Displays paragraph lines with a specific style in a specific part of the document, along with their page numbers.

    • Group into TOC Code

    • Displays paragraph lines inserted with TOC codes in a specific part of the document, along with their page numbers. To create a table of contents using TOC code paragraphs, the headings must first be inserted with TOC codes.

  • Table of Tables

  • Displays captions of all tables inserted in the document in the order of their placement in the document, along with their page numbers.

  • Table of Pictures

  • Displays captions of all pictures inserted in the document in the order of their placement in the document, along with their page numbers.

  • Table of Equations

  • Displays captions of all equations inserted in the document in the order of their placement in the document, along with their page numbers.

  • Create at

  • Specify the location where you want to create a table of contents.

    • Cursor Position in Current Document

    • Creates a table of contents at the cursor position in the current document.

    • New Tab

    • Creates a table of contents in a new tab in the current document window.

Format

  • Automatic Right Tab Stop

  • Selecting Format > Automatic Right Tab Stop inserts a right tab at the right end of the paragraph and then inserts the page number. With this option selected, the tab position changes along with any change to the right margin.

    However, you cannot select Pattern to Fill if you insert Automatic Right Tab Stop.

  • Right Tab Stop

  • Aligns the right ends of the page numbers.

  • Pattern to Fill

  • Select which type of line to use for the empty space between the end of a word and the beginning of a word in the current tab.

Note

TOC of Objects without Caption

When a TOC is made for pictures, tables, and equations without a caption, their frame numbers are used as contents.
