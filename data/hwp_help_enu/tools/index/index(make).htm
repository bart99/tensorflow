Insert Index
 

References > Index > Insert Index

An Index is a list that shows key items, terms, names, and places referenced in a book, an academic paper, or a research material, along with respective page numbers.

If the page numbers of the document are changed after completing Insert Index, you will need to perform Insert Index again.

Steps

  1. To create an index, place the cursor after a word you wish to include in the index and click References > Index > Mark Entry.
  2. Once you have marked all the index entries, click References > Index > Insert Index.
  3. An index file is created in a new tab within the current window.
  4. An index file lists Main Entry items (or Subentry items) in alphabetical order, along with the respective page numbers.

Note

Open as a New Tab

When you create an index file, it is created in a new tab in the current document window. Change the style of the contents of the index and then save or paste the index at the end of the current document.

Sort Order

Marked index words are sorted in the order of Latin, Chinese, and Korean characters.

Adding Recurring Index Entries

If you create an index after marking the same entry on numerous pages, the page numbers of the marked entry instances are inserted after the entry in the index.
