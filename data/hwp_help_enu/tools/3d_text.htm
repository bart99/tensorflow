Print 3D Text
 

Home > Print 3D Text

You can convert simple text to 3D object and create a STL file for 3D printing.

Steps

  1. Click Print 3D Text in the Home tab.
  2. Enter some text in the Contents area and click Apply.
  3. Set the options for Font, Panel Properties, and Veiw Properties as you want.
  4. Click the Save as STL button at the bottom of the window.
  5. Enter the file name and click Save.

Details

Properties for 3D Text

Font

Specify the font style for the entered 3D text.

  • Font

Specify the font type for the entered 3D text.

  • Size

Specify the font size for the entered 3D text.

  • Properties

Apply properties such as bold, italic, underline to the entered 3D text.

  • Depth

Specify the depth from the bottom to the top of the entered 3D text.

  • Effect

Apply various effects to the entered 3D text.

Panel Properties

Specify the properties of panel where 3D text is placed.

  • Horizontal Margins

Adjust the left and right margins of the panel.

  • Vertical Margins

Adjust the top and bottom margins of the panel.

  • Depth

Specify the depth from the bottom to the top of the panel.

View Properties

  • Color

Specify the color of the entered 3D text.

Note

Currently Only Supporting Save as STL

You can only save contents to a STL file for 3D printing and cannot open external STL files created out of Hancom Office.
