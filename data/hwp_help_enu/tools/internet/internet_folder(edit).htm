Edit Internet Folder
 

File > Manage Internet Folder Connection

You can edit the Server Info and Account Info of an existing internet folder account. Use this if you want to change the account address or the user ID.

Steps

To edit internet folder

  1. Click File > Manage Internet Folder Connection.
  2. In the Manage Internet Folder Connection dialog box, click Edit Internet Folder.
  3. In the Edit Internet Folder dialog box, modify items you want.
  4. Click Set to save modified information.

  You cannot edit the server information of ThinkFree and other default servers or delete these accounts.

More Information

Edit Internet Folder Dialog Box

Server Info

  • Server Name

    Type a name for the new server to register.

  • Server Address

    Type the address of the new server to register.

  • Port Info

    Select this option when you use a port number that is not the default port number.

Account Info

  • User ID

    Type a User ID for the internet folder you want to use.

Check Server Button

Tests the server.

Note

WebDAV

WebDAV (Web-based Distributed Authoring and Versioning) is an extended HTTP protocol that manages web server files through the web.
