Manage Internet Folder Connection
 

File > Manage Internet Folder Connection

This function manages internet folder connection. Internet folders that were once connected are automatically saved on the connection list.

For you to use internet folders, the server that provides them must support the WebDAV protocol.

Steps

To manage internet folder connection

  1. Click File > Manage Internet Folder Connection.
  2. In the Account List of the Manage Internet Folder Connection dialog box, select an account to log into and click Log in.
  3. Type User ID and Password and click Set to log into the selected account.
  4. To delete items from the connection list, select items to delete and click Delete Internet Folder.

More Information

Manage Internet Folder Connection Dialog Box

Account List

The currently registered account list appears. If you select an account you want and click Log in, you get logged into the account and Connection shows as Connected.

Log in/Log out Button

You can log in/log out of the selected account list.

Details on Icons

Icon Name Description
Register Internet Folder In the Register Internet Folder dialog box, you can register frequently used accounts or accounts you need.
Edit Internet Folder In the Edit Internet Folder dialog box, modify account names and User IDs.
Delete Internet Folder Delete account lists you do not use or need.

Note

WebDAV

WebDAV (Web-based Distributed Authoring and Versioning) is an extended HTTP protocol that manages web server files through the web.
