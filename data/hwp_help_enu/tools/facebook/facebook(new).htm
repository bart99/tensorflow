Register Account
 

File > SNS Service > Manage Facebook Account

You can register frequently used Facebook accounts.

After registering an account, you can upload text and pictures to that Facebook account directly from Thinkfree Office NEO.

Steps

To register a Facebook account

  1. Click File > SNS Service > Manage Facebook Account.
  2. In the Manage Facebook Account dialog box, click Register Account to add an account.
  3. In the Register Facebook Account dialog box, click Login Facebook in Facebook Account.
  4. In Login Facebook, type e-mail and a password in the E-mail and Password field.
  5. In Register Facebook Account, type the password in Password and type it again in Verify to record it correctly.
  6. Click OK. A Facebook account is created with the typed password.
  7. Check the registered account in Manage Facebook Account > Account List.

More Information

Register Facebook Account Dialog Box

Facebook Account

  • Login Facebook

  • Connect to the Facebook website and log in Facebook with the user ID/PW.

  • Password

  • Type in a new password for the account. The password must be at least 5 characters in length.

  • Verify

  • Enter the password again.

  • Sign in automatically

  • Select if you wish to always log in to this account automatically. By selecting this option, you can upload messages and pictures to this account without having to enter the username and password each time.

ThinkFree Account Information

  • User ID

    Enter the ThinkFree user ID (e-mail address).

  • User Password

    Enter the password you set when registering your ThinkFree user ID. If you can't remember the password, retrieve it from the ThinkFree website (http://www.thinkfree.com/).

  • Do not use ThinkFree service

    Select if you do not wish to use the ThinkFree service.
