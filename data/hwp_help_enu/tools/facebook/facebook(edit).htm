Edit Account
 

File > SNS Service > Manage Facebook Account

You can change the Account Name, Facebook Account Information, and ThinkFree Account Information settings of a registered Facebook account. Use this option if you wish to change the name of the account or the account password.

Steps

To edit a Facebook account

  1. Click File > SNS Service > Manage Facebook Account.
  2. Select an account to edit in the Account List.
  3. Click Edit Account.
  4. In Edit Facebook Account dialog box, type Hancom in Account Name, type the current password, and then click Change.
    The changed account name appears in Account List.

More Information

Edit Facebook Account Dialog Box

Edit What

You can select items to edit.

  • Account Name

  • Select if you wish to edit the account name.

  • Password

  • Select if you wish to change the password.

  • Auto Login

  • Select if you wish to always log in to this account automatically. By selecting this option, you can upload messages and pictures to this account without having to enter the username and password each time.

Facebook Account Information

You can check or modify the name of the current account. To modify the name, type a new name in the Account Name field. You can register a maximum of 20 account lists.

  • Account Name

  • Display the name of the current account. If you wish to use a different name for the account, enter it here.

  • Current Password

  • Type in a password for the account. The password must be at least 5 characters in length.

  • New Password

  • Type in a new password for the account. The password must be at least 5 characters in length.

  • Verify

  • Type in the new password again.

ThinkFree Account Information

  • User ID

    Enter the ThinkFree user ID (e-mail address).

  • User Password

    Enter the password you set when registering your ThinkFree user ID. If you can't remember the password, retrieve it from the ThinkFree website (http://www.thinkfree.com/).

  • Do not use ThinkFree service

    Select if you do not wish to use the ThinkFree service.
