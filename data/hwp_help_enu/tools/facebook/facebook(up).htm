Upload to Facebook
 

File > SNS Service > Upload to Facebook

Upload to Facebook uploads text and pictures from Thinkfree Office NEO directly to Facebook. You can conveniently post to Facebook from the comfort of a Thinkfree Office NEO application and without having to use a dedicated Facebook application.

Steps

To post to Facebook

  1. Compose a Facebook and set it as a block using the pointer.
  2. Click File > SNS Service > Upload to Facebook.
    If a registered Facebook account is not available, you will be prompted with the "You have not signed up for a Facebook account. Sign up now?" message.
    Click OK, and the Register SNS Account dialog box opens up so that you can register your Facebook account.
  3. To register your Facebook account, log in Facebook, enter your password, and then click Set in the Register SNS Account dialog box.
    If you selected Save Password, the text and picture are uploaded to Facebook without prompting you to log in.
  4. The content that was set as a block is automatically inserted into the Update Facebook Status dialog box up to 420 characters.
  5. If you click Update in the Update Facebook Status dialog box, you are prompted to upload the contents to Facebook.
  6. Click Connect to verify your new contents.

More Information

Update Facebook Status Dialog Box

Status

Type content to upload to Facebook. You can enter up to 420 characters.

Number of Typed Characters

Displays the number of characters inside the Facebook window.

Insert Picture

In the Insert Picture dialog box, select a picture file to upload to Facebook. Once you click Insert, the URL is inserted and the Insert Picture button is changed to the Cancel button.

Upload Document

  • File Name

    Upload the selected file to Facebook using ThinkFree Server.

  You need to sign up ThinkFree Online(http://member.thinkfree.com/) prior to uploading a document.
