Add New Label Page
 

Review > Labels > Add New Label Page

Labels are name tags attached to various tapes or diskettes to show their own features for the purpose of distinguishing them from one another.

By clicking Review > Labels > Create Label Template in Hwp, you can easily create labels for tapes and diskettes, barcodes, postcards, and business cards while looking at the label paper on the editing screen.

If you need to create multiple label pages, you can do so by clicking Labels > Add New Label Page.

Steps

  1. Click Review > Labels > Create Label Template.
  2. In the Create Label Template dialog box, select the Label Template Package tab.
  3. Select the manufacturer of the label paper you wish to use from the Label Manufacturer List. A list of label papers from the selected manufacturer is then shown in the Label Paper List.

    For this example, select Formtec Standard: 3107-Address Label (16).

  4. Check the label paper in Preview and click Open.
  5. A new document window opens and a label is created with red dots that exactly follow the label paper.
  6. Type the information into each label and then specify suitable Font and Paragraph settings to finish the labels.
  7. Click Labels > Add New Label Page if you require one additional page in the label document.
  8. A new page is added with the label document that was used inserted into the page.
