Create Label Template: Recent Label Templates
 

Review > Labels > Create Label Template

The Recent Label Templates tab of the Create Label Template dialog box shows a list of recently opened label documents. You can select and open a recently used label document directly from the list.

If you need to use the same type of label paper over and over again, you can use the Recent Label Templates list and open the required label template quickly and easily.

Steps

  1. Click Review > Labels > Create Label Template.
  2. The Create Label Template dialog box with the Recent Label Templates tab selected is shown.
  3. A list of recently used label template package names and label papers is shown.
  4. Select a label paper from the list box and click Open.
  5. The selected label document is opened in the edit screen.

More Information

Recent Label Templates Tab of Create Label Template Dialog Box

Details on Icons

Icon Name Shortcut Description
Delete Label Paper Delete Select and remove unnecessary label templates from the list of recently used label templates.

Note

Managing the Package

You cannot manage the label template package while the Recent Label Templates tab is selected. The Label Template Package tab must be selected in order to create, modify, or delete a label template.
