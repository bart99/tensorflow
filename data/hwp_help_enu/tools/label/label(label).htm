Create Label Template
 

Review > Labels > Create Label Template

Labels are name tags attached to various tapes or diskettes to show their own features for the purpose of distinguishing them from one another.

By clicking Review > Labels > Create Label Template in Hwp, you can easily create labels for tapes and diskettes, barcodes, postcards, and business cards while looking at the label paper on the editing screen.

Steps

  1. Click Review > Labels > Create Label Template.
  2. In the Create Label Template dialog box, select the Label Template Package tab.
  3. Select the manufacturer of the label paper you wish to use from the label manufacturer list. A list of label papers from the selected manufacturer is then shown in the label paper list.

    For this example, select Formtec Standard: 3107-Address Label (16).

  4. Check the label paper in Preview and click Open.
  5. A new document window opens and a label is created with red dots that exactly follow the label paper.

  6. A new document window opens and a label is created with red dots that exactly follow the label paper.
  7. Type the information into each label and then specify suitable Font and Paragraph settings to finish the labels.

Note

Label Paper

Hwp supports label papers manufactured by AnyLabel, Formtec, Avery standard, GreenLabel, NeoLabel, and LatechLabel. Select a label manufacturer. Specific product names and numbers for the selection are then shown in the list box. Select the appropriate label paper for your needs.

Label

Each of the square shapes on a label paper is called a Label. In Hwp, the labels are formatted as tables.

These tables are set to Fix Size, meaning the tables will not increase in size even if the text entered touches the border. Furthermore, the excessive portion of the text will not be visible on the screen.

The tables are also set to Lock Object and therefore cannot be selected with the pointer.

Label Borders

The tables that are used as labels are set to Dashes > None and therefore have transparent lines. When you Create a label template, however, the tables are automatically set to Show Gridlines to show the transparent lines with red dotted lines and thereby make the individual labels easier to distinguish. These gridlines are only visible on the screen and do not show in printouts.

Label Properties

Labels are formatted as Hwp Tables. Therefore, you can click Table > Cell Border/Background and change the border and fill styles of the labels as you would with a regular table that was created by clicking Insert > Table > Insert Table.

Vertical Position of Labels

The available vertical alignment setting for the information entered into the labels are Top, Middle, and Bottom. To specify the alignment, click Table > Properties, go to the Cell tab, and click Properties > Vertical Alignment.

Recent Label Templates

Recently opened label templates are shown in the Recent Label Templates tab of the Create Label Template dialog box. You can select and open a recently used label paper directly from the list.

If you need to use the same type of label paper over and over again, you can use the Recent Label Templates list and open the required label template quickly and easily.

To print current document as a label

You cannot compose a document containing label information only and then create a label template using that document. You must first open an empty label template by clicking Review > Labels > Create Label Template and then fill the labels with information.

Advanced Usage

Printing Multiple Sheets

If you need multiple pages of labels containing the same information, click File > Print and specify the number of sheets you need in Copies.

Table AutoFill

If you are creating a label template that will be used to print the same information over and over again, use the Table AutoFill option.

Click Review > Labels > Create Label Template, select the label paper you wish to print, and then enter information in the first field only. Select the labels you want to insert the same information to and set them as a cell block. Click Insert > Fill > Table AutoFill to compose the rest of the label template with ease.

Printing the Results of the Mail Merge

By creating Mail Merge content inside labels, you can have the mail merge results print on label paper instead of normal paper and create labels for letter envelopes with ease.
