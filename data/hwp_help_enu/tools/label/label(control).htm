Create Label Template: Label Template Package
 

Review > Labels > Create Label Template

In the Label Template Package tab of the Create Label Template dialog box, you can create new label papers or edit already registered label papers. You can also delete unnecessary label papers.

More Information

Label Template Package Tab of Create Label Template Dialog Box

Details on Icons

Icon Name Shortcut Description
Create Label Paper Insert Creates a document templates package to store template files in.
Modify Label Paper + Changes the name or path of registered packages.
Delete Label Paper Delete Deletes unused document package lists.

Note

Run from Context Menu

Place the mouse pointer on the label paper list box in the Label Template Package tab and right click to open the context menu. The context menu contains Create Label Paper, Modify Label Paper, and Delete Label Paper commands.
