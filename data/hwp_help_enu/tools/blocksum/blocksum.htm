Block Calculation
 

Review > Block Calculation

Block Calculation gains the sums and averages of numbers contained in a block area and inserts the result at the current location or pastes a calculation result that was stored in memory to a location designated by the user.

Steps

To gain block sum

  1. Type 10 4.0 5.5 -10 1,000 (-1,000) in the body.
  2. Set the entry as a block and click Review > Block Calculation > Block Sum.
  3. The Block Calculation Result dialog box appears and automatically displays Sum: 1009.5 and Average: 201.9.
  4. Click on the Copy button to the right of the sum result to store the calculation result in buffer memory.
  5. Move the cursor to the location where you want the block sum result to be inserted and then press CTRL+V to perform Home > Paste.
  6. 1009.5 is entered at the cursor position.

To gain block average

  1. Type 10 4.0 5.5 -10 1,000 (-1,000) in the body.
  2. Set the entry as a block and click Review > Block Calculation > Block Average.
  3. The Block Calculation Result dialog box appears and automatically displays Sum: 1009.5 and Average: 201.9.
  4. Click on the Copy button to the right of the average result to store the calculation result in buffer memory.
  5. Move the cursor to the location where you want the block average result to be inserted and then press CTRL+V to perform Home > Paste.
  6. 201.9 is entered at the cursor position.

Note

Use Block Formula in Tables

If you want to perform a calculation using numbers in a table, Block Formula is by far the more convenient option.

If you want to perform a calculation using numbers in the body, use Block Calculation.

Insert a calculation result

By clicking on the Insert button of the Block Calculation Result dialog box inserts the block average result immediately after the block.

Save calculation result

If you want to insert a block average result at a different location, click on the Copy button of the Block Calculation Result dialog box to store the average result in buffer memory. After that, place the cursor at the location where you want the average result to be inserted and click Home > Paste.

Caution

Restriction

Block Calculation is only available when a block is set. The content inside the block may include number and the +, -, ,, and . symbols. Make sure the block does not contain letters or numbers closed off using parentheses ([], {}, or ()).
