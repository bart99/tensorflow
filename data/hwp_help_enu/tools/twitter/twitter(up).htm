Upload to Twitter
 

File > SNS Service > Upload to Twitter

Upload to Twitter uploads text and pictures from Thinkfree Office NEO directly to Twitter. You can conveniently post to Twitter from the comfort of a Thinkfree Office NEO application and without having to use a dedicated Twitter application.

Steps

To post to Twitter

  1. Compose a Tweet and set it as a block using the pointer.
  2. Click File > SNS Service > Upload to Twitter.
    If a registered Twitter account is not available, you will be prompted with the "You have not signed up for a Twitter account. Sign up now?" message.
    Click OK, and the Register SNS Account dialog box opens up so that you can register your Twitter account.
  3. To register your Twitter account, get PIN, enter your password, and then click Submit > Set in the Register SNS Account dialog box.
    If you selected Save Password, the text and picture are uploaded to Twitter without prompting you to enter your password.
  4. The content that was set as a block is automatically inserted into the Tweet dialog box up to 140 characters.
  5. If you click Upload in the Tweet dialog box, you are prompted to upload the contents to Twitter.
  6. Click OK to verify your new Tweet.

More Information

Tweet Dialog Box

Tweet

Type content to upload to Twitter. You can enter up to 140 characters.

Number of Typed Characters

Displays the number of characters inside the Tweet window.

URL Shortener

If you selected a document from Upload Document or your Tweet contains a URL, pressing ENTER automatically shortens the URL and inserts it into the Tweet window. To disable URL shortening, select URL Shortener again.

Insert Picture

In the Insert Picture dialog box, select a picture file to upload to Twitter. Clicking Insert automatically inserts the converted URL into the Tweet window.

Upload Document

  • File Name

    Upload the selected file to Twitter using ThinkFree Server.
