Register Account
 

File > SNS Service > Manage Twitter Account

You can register frequently used Twitter accounts.

After registering an account, you can upload text and pictures to that Twitter account directly from Thinkfree Office NEO.

Steps

To register a Twitter account

  1. Click File > SNS Service > Manage Twitter Account.
  2. In the Manage Twitter Account dialog box, click Register Account to add an account.
  3. In the Register Twitter Account dialog box, click Import PIN in Twitter Account Info to type PIN information.
  4. Type a password in Password.
  5. Type the same password in Password again to record it correctly.
  6. Click Register. A Twitter account is created with the typed password.
  7. Check the registered account in Manage Twitter Account > Account List.

More Information

Twitter Account Registration Dialog Box

Twitter Account Information

You can check or modify the name of the current account. To modify the name, type a new name in the Account Name field. You can register a maximum of 20 account lists.

  • PIN

  • Displays the PIN information of the user's Twitter account.

  • Retrieve PIN

  • Retrieves the pin of the registered Twitter account.

  • Password

  • Type in a new password for the account. The password must be at least 5 characters in length.

  • Confirm Password

  • Enter the password again.

  • Automatic Login

  • Select if you wish to always log in to this account automatically. By selecting this option, you can upload messages and pictures to this account without having to enter the username and password each time.

ThinkFree Account Information

  • User ID

    Enter the ThinkFree user ID (e-mail address).

  • Password

    Enter the password you set when registering your ThinkFree user ID. If you can't remember the password, retrieve it from the ThinkFree website (http://www.thinkfree.com/).

  • Disable ThinkFree

    Select if you do not wish to use the ThinkFree service.

Note

PIN

You can verify yourself online with an ID and a password instead of your social security number. If you use a PIN ID and a password, you can protect your personal information because you sign up on websites without entering your social security number.
