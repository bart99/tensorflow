Manage Twitter Account
 

File > SNS Service > Manage Twitter Account

You can manage Twitter accounts by registering, editing, or deleting them.

Steps

To manage Twitter accounts

  1. Click File > SNS Service > Manage Twitter Account.
  2. In the Manage Twitter Account dialog box, click Register Account to add an account.
  3. In the Register Twitter Account dialog box, click Import PIN in Twitter Account Info to type PIN information.
  4. Type a password in Password.
  5. Type the same password in Password again to record it correctly.
  6. Click Register. A Twitter account is created with the typed password.
  7. Check the registered account in Manage Twitter Account > Account List.

More Information

Manage Twitter Account Dialog Box

Account List

The currently registered account list appears. The Display Selection icon is added to the beginning of the default connection account among registered accounts. You can register a maximum of 20 account lists.

Details on Icons
Icon Name Description
Register Account In the Register Account dialog box, you can register frequently used accounts or accounts you need.
Edit Account In the Edit Account dialog box, modify account name or user ID, or activate/deactivate Auto Connect.
Delete Account Delete accounts you do not use or need to reorganize the account list.
Set as Default Connection Account Set the selected account as the default connection account. If you set an account as the default connection account, the Display Selection icon gets added to the beginning of the account.
Up If you select a Twitter account from the Account List and click the icon, you can move the account one line up on the list.
Down If you select a Twitter account from the Account List and click the icon, you can move the account one line down on the list.

Note

PIN

You can verify yourself online with an ID and a password instead of your social security number. If you use a PIN ID and a password, you can protect your personal information because you sign up on websites without entering your social security number.
