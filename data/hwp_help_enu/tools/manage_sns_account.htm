Manage SNS Accounts
 

File > SNS Service > Manage SNS Accounts

You can manage Facebook accounts by registering, editing, or deleting them.

Steps

To change the default Facebook account

  1. Click File > SNS Service > Manage SNS Accounts.
  2. In the Manage SNS Accounts dialog box, check the previously registered account.
    If only one account is registered, it has already been set as the default account.
  3. Select an account to set as default one in the Account List, and then click Set as Default Connection Account.
  4. The Display Selection icon is added to the beginning of the default account.

More Information

Manage Facebook Account Dialog Box

Account List

The currently registered account list appears. The Display Selection icon is added to the beginning of the default connection account among registered accounts. You can register a maximum of 20 account lists.

Details on Icons
Icon Name Description
Register Account In the Register Account dialog box, you can register frequently used accounts or accounts you need.
Edit Account In the Edit Account dialog box, modify account name or user ID, or activate/deactivate Auto Connect.
Delete Account Delete accounts you do not use or need to reorganize the account list.
Set as Default Connection Account Set the selected account as the default connection account. If you set an account as the default connection account, the Display Selection icon gets added to the beginning of the account.
Up If you select a Facebook account from the Account List and click the icon, you can move the account one line up on the list.
Down If you select a Facebook account from the Account List and click the icon, you can move the account one line down on the list.
