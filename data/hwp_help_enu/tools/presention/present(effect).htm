Presentation Settings: Transition
 

View > Presentation > Presentation Settings

You can add different transition effects between sections of a presentation for greater visual simuli.

Steps

To apply a transition effect

  1. Click View > Presentation > Presentation Settings.
  2. From the Presentation Settings dialog box, select the Transition tab.
  3. Select Open to Left from the Effects list box in the Transition tab.
  4. A preview of the selected transition effect is shown in the Preview area.
  5. Click OK.

More Information

Transition Tab of Presentation Settings Dialog Box

Transition

  • Effects

  • Available transition effects are Open to Left, Open to Right, Open to Top, Open to Bottom, Open in Box , Shut in Box, Left Blind, Right Blind, Top Blind, Bottom Blind, Open Horizontal Curtain, Open Vertical Curtain, Close Horizontal Curtain, Close Vertical Curtain, Cover to Left, Cover to Right, Cover to Top, Cover to Bottom, and Random Choice, which applies a random transition effect for each page.

  • Preview

  • Each time you select a effect from the Effects list, a preview of the selected effect is shown in the Preview area.

  • Sound

  • You can make your presentation even more interesting by adding sound effects (*.wav) along with transition effects.

    Selecting the Sound check box opens the Open Sound File dialog box.

    • Sound File

    • The path and name of the selected sound file (*.wav) are shown in the Open Sound File dialog box.

      You can press the Select Sound icon and select a different sound file.

      • Embedded

      • Embeds the sound effect file (*.wav) in the document. Embedding the sound file in the document means you do not have to store that sound file separately.

Options

  • Black to White

  • Selecting this option when using a dark background changes black letters to white letters to make them more visible.

  • AutoDemo

  • To specify the time between page transitions in a presentation, select the AutoDemo option and then specify the After setting.

    • After

    • The time before transitioning over to the next page can be set as 1 sec through 600 secs.

  • Apply to

  • Transition effects can only be set on entire section basis. In other words, all the pages within a section can only have one type of transition effect. If you want to apply different transition effects to different pages, you must set each of the concerned pages as a section.

    Suggest settings for the section number, pointer position, and block setting of the active document. In most cases, you can select either Whole Document or New Section.

    • New Section: Create a new section by breaking pages at the current pointer position.
    • Whole Document: Applies the same presentation properties to the whole document even if the document is divided into two or more sections.
    • Current Section: You can select the Current Section in Apply to if the document is divided into two or more sections.

Note

Transition Sound Application

Transition sounds can only be set on entire section basis. In other words, all the pages within a section can only have one type of transition sound effect. If you want to apply different transition sound effects to different pages, you must set each of the concerned pages as a section.
