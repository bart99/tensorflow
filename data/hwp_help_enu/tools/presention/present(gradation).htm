Presentation Settings: Background
 

View > Presentation > Presentation Settings

You can decorate the background of a presentation by inserting a picture or applying a gradation effect.

Steps

  1. Click View > Presentation > Presentation Settings.
  2. In the Presentation Settings dialog box, select the Background tab.
  3. From the Type list under Gradation, select Autumn Sun.
  4. To insert a picture in the background, click Picture. Select a picture to use as the background and then specify the fill type and picture effects settings.
  5. Click OK.

More Information

Background Tab of Presentation Settings Dialog Box

Gradation

  • Beginning Color

  • Select the beginning color, which will spread outwards from the center of the object. Click the color table to expand the color palette.

  • Ending Color

  • Select the ending color, which will spread inwards from the outer edges of the object. Click the color table to expand the color palette.

  • Type

  • Select a gradation type from the numerous options available. Each gradation type is applied with specific style, center, angle, and blur settings.

    If you select Custom under Type, you can create and use your own gradation style instead of using one of the gradation templates. Specify the beginning color, ending color, style, center, angle, and blur settings and create a custom gradation style.

  • H/V Center

  • Designate the horizontal and vertical center of the object as a percentage (%) of the overall horizontal/vertical position.

  • Angle

  • Specify by what angle to tilt the gradation style using a value from 0¡Æ through 359¡Æ.

  • Blur Level

  • Specify spread level for the beginning color and the ending color.

    Level 0 means no face color. Level 2 only displays the beginning color and the ending color. This can be set from 0 through 255.

  • Blur Center

  • Specify the center of Blur Level. Specify a value from 0 through 100. The default value is 50.

    Assuming the distance between the beginning color and the ending color is 100, Blur Center is calculated as a ratio of that distance. Smaller the Blur Center value, closer the center is to the beginning color and vice versa.

Picture

Fills the background of the presentation with a picture. Selecting the Picture check box opens the Insert Picture dialog box so that you can select a picture file to insert.

  • Picture File

  • Click on the Select Picture icon and select a picture for the background.

    • Embedded

    • Include the picture file in the document as an inserted picture without connecting it to the document. If you select this option, the modification or position change of the original picture file will not affect the picture included in the document.

  • Fill Type

  • When filling the background with a picture, select Tile, Fit to Size, or Center as the fill type.

    • Fit to Size

    • Ignores the original dimensions of the picture and enlarges or reduces it to fit the current page.

    • Up/Center/Down

    • Maintains the original dimensions of the picture and fills the background in the designated direction without repeating the picture.

    • Tiled

    • Fills the background with the picture in its original dimensions. If the page is smaller than the picture, the picture is cropped. If the page is larger than the picture, the picture is tiled on.

  • Effects

  • Convert a picture file inserted into the background to a grayscale image, a black and white image, or a watermark.

  • Brightness

  • Adjust the brightness of the selected picture file. Brightness can be set from -100 through 100.

    -100 turns the image black, and 100 turns the image white. The default value is 0.

  • Contrast

  • You can adjust the contrast of the selected picture file. Contrast can be set from -100 through 100.

Apply to

You can specify the application range based on the number of sections in the current document, the cursor position, block setting status, etc. In general, available options are Whole Document and New Section.

  • New Section: Divide the document into new sections by breaking the page at the pointer and apply a new multilevel list style.
  • Whole Document: Apply the same multilevel list style to the whole document even if the document is divided into two or more sections.
  • Current Section: If the document is divided into two or more sections, you can select the Current Section in Apply to.

Note

Filling with a Picture and a Gradation Effect

You can use all image file formats supported in Hwp as a presentation background. Click on the Open Picture icon. From the Insert Picture dialog box, specify the path and name of the presentation background picture file.

Filling together with a Color or Gradation Effect

Presentation background is superimposed on top of a gradation background color layer. Therefore, setting Picture > Fill Type as Tile-Horizontal or Tile-Vertical fills areas not covered by the picture with the designated gradation effect.
