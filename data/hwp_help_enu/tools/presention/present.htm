Run Presentation
 

View > Presentation > Run Presentation

The presentation feature displays Hwp documents on top of a picture or gradation effect background for use as a presentation.

Steps

  1. Click View > Presentation > Run Presentation.
  2. If running the current document as a presentation for the first time, you will be asked if you wish to create a presentation section in the current document. Click Yes.
  3. In the Presentation Settings dialog box, specify the presentation background, transition effects, transition sounds, and application range and click Run.
  4. The Hwp document window is switched into full screen mode, and the current document is shown on top of the designated background.
  5. Right click on the presentation screen to open the context menu. You can use this menu to control your presentation with ease.
  6. To end the presentation, press ESC or right click and select Exit from the context menu.

Note

Presentation Section

In order to run a presentation, a separate section must be available in the current document to hold presentation background, transition effect, and transition sound files.

Therefore, clicking View > Presentation > Run Presentation from a document that has never been used in a presentation before prompts a dialog box that asks the user to create a presentation section in the document. Clicking Yes opens the Presentation Settings dialog box so that you can specify the presentation background type, transition effects, transition sounds, application range, and other presentation section settings.

If you click View > Presentation > Run Presentation from a document you have used in a presentation at least once before, the presentation starts from the current section if the current section is a presentation section or from the next presentation section if the current section is not a presentation section, using the previous settings.

Saving Presentation Settings to Current Document

After a presentation, the presentation settings designated by the user are saved along with the document. Therefore, you can avoid having to specify the settings each time you run a presentation on a particular document by specifying the most suitable background for the document in advance.

Hiding Presentation Screen

Press B while a presentation is in progress to hide the current presentation. Press B again to show the presentation again.

Press ENTER or PAGE UP/PAGE DOWN while the presentation is hidden to load the next or previous page and resume the presentation.

Press ESC to end the presentation and exit to the edit window.
