Add Add-ons
 

File > Manage Add-ons

With this feature, you can add the add-on you want.

Steps

To add add-ons

  1. Click File > Manage Add-ons.
  2. In the Manage Add-ons dialog box, click the Add icon .
  3. In the Add dialog box, select the type of the add-on to add in Type.
  4. Type a name for the add-on in Name and click the Open icon in Select File.
  5. In Find in of the Load dialog box, select a drive or folder that has the file (*.dll) you want.
  6. Select the file to add and click Open to close the Load dialog box.
  7. Click Add to close the Add dialog box.
  8. Check List of Add-ons in the Manage Add-ons dialog box and click Set.
  9. Exit the program and restart it.
  10. Select and activate add-ons you want from the add-on list.

More Information

Add Dialog Box

Name

Type a name you want for the add-on so that you can easily find and use it.

Select File

Select a dynamic link library file (.dll) that provides the add-on you want. You should select a file that corresponds to the type of the add-on.

Add-on Type

From Additional Task Pane, Automation, Automation Security Feature, and ActiveX, select the type that corresponds to the feature you wish to add.

Status

Decide if you will activate the added add-on now or just have it added.

Comment

If you select the type of the add-on, you can check descriptions about the load timing of the add-on.
