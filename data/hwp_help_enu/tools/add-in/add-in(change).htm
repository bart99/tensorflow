Edit Add-ons
 

File > Manage Add-ons

You can change the added add-ons to other add-ons or change the name of the added add-ons.

Steps

To modify add-ons

  1. Click File > Manage Add-ons.
  2. Select an add-on to change from the List of Add-ons in the Manage Add-ons dialog box and click the Edit icon .
  3. In the Modify dialog box, change the name of the add-on in Name and click the Open icon in Select File.
  4. In Find in of the Load dialog box, select a drive or folder that has the file (*.dll) you want.
  5. Select the file to change and then click Open to close the Load dialog box.
  6. Click Set to close the Modify dialog box.
  7. From the Add dialog box, check the added add-on and click Set.
  8. Exit the program and restart it.

More Information

Modify Dialog Box

Name

You can change the name of an add-on.

Select File

You can select other dynamic link library files (.dll). You should select a file that corresponds to the type of the add-on.

Add-on Type

You cannot change add-on type.

Status

You can only activate or deactivate add-ons.

Comment

As you cannot change add-on type, you can check descriptions about when to load the current add-on.
