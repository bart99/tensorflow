Manage Add-ons
 

File > Manage Add-ons

This feature allows you to add an add-on you want. You can expand the program feature by adding dynamic link library files (.dll) to the program.

More Information

Manage Add-ons Dialog Box

List of Add-ons

Display the list of the add-ons you added. If you have not added any add-on, the list will be empty.

Details on Icons

Manage Add-ons Dialog Box

Icon Name Description
Add In the Add dialog box, add an add-on you want.
Edit In the Modify dialog box, change the add-on to another one.
Delete Delete unnecessary add-ons.
