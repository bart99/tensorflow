Register SNS Account
 

File > SNS Service > Register SNS Account

You can register frequently used SNS accounts.

After registering an account, you can upload text and pictures to that Twitter account directly from Thinkfree Office NEO.

Steps

To register a Twitter account

  1. Click File > SNS Service > Manage SNS Accounts.
  2. In the Manage SNS Accounts dialog box, click Register Account to add an account.
  3. In the Register SNS Account dialog box, click Get PIN in Twitter Account to type PIN information.
  4. Type a password in Password.
  5. Type the same password in Password again to record it correctly.
  6. Click Set. A Twitter account is created with the typed password.
  7. Check the registered account in Manage SNS Accounts > Account List.

To register a Facebook account

  1. Click File > SNS Service > Manage SNS Accounts.
  2. In the Manage SNS Accounts dialog box, click Register Account to add an account.
  3. In the Register SNS Account dialog box, click Login Facebook in Facebook Account.
  4. In Login Facebook, type e-mail and a password in the E-mail and Password field.
  5. In Register SNS Account, type the password in Password and type it again in Verify to record it correctly.
  6. Click Set. A Facebook account is created with the typed password.
  7. Check the registered account in Manage SNS Accounts > Account List.

More Information

Register SNS Account Dialog Box

SNS Services

You can select the type of SNS between Twitter and Facebook.

Twitter Account Information

  • PIN

  • Displays the PIN information of the user's Twitter account.

  • Get PIN

  • Get the pin of the registered Twitter account.

  • Password

  • Type in a new password for the account. The password must be at least 5 characters in length.

  • Verify

  • Enter the password again.

  • Save Password

  • Select if you wish to always log in to this account automatically. By selecting this option, you can upload messages and pictures to this account without having to enter the username and password each time.

Facebook Account

  • Login Facebook

  • Connect to the Facebook website and log in Facebook with the user ID/PW.

  • Password

  • Type in a new password for the account. The password must be at least 5 characters in length.

  • Verify

  • Enter the password again.

  • Save Password

  • Select if you wish to always log in to this account automatically. By selecting this option, you can upload messages and pictures to this account without having to enter the username and password each time.

Netffice 24 Account Info

  • User ID

    Enter the Netffice 24 user ID (e-mail address).

  • User Password

    Enter the password you set when registering your Netffice 24 user ID. If you can't remember the password, retrieve it from the Netffice 24 website (http://www.netffice24.com).

  • Do not use Netffice 24 service

    Select if you do not wish to use the Netffice 24 service.

Note

PIN

You can verify yourself online with an ID and a password instead of your social security number. If you use a PIN ID and a password, you can protect your personal information because you sign up on websites without entering your social security number.
