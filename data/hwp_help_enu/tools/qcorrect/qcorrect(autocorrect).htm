AutoCorrect List: AutoFormat
 

Tools > AutoCorrect > AutoCorrect List

Automatically converts characters entered in the edit window into specific formats.

More Information

AutoFormat Tab of AutoCorrect List Dialog Box

Replace as you type

  • "Straight quotes'' with "smart quotes'

  • Changes a straight quote into an opening smart quote if there is a space preceding the quote. If there is a character preceding the quote (including opening quote), the quote is changed into a closing smart quote.

  • Hyphens (-) with dash (–)

  • Changes single hyphens (-) between letters to n dashes (–) and double hyphens (--) into m dashes (—).

  • Ordinals (1st) with superscript

  • Expresses English ordinals entered after a number without any spacing in between in superscript format.

  • Match opening and closing parentheses

  • If a closing parenthesis that does not match the opening parenthesis is entered, the closing parenthesis is automatically changed to match the opening parenthesis.

  • Internet and network paths with hyperlinks

  • Displays URLs, file locations, network paths, and email addresses as hyperlinks.

Apply as you type

  • Automatic bulleted lists

  • Automatically applies bulleted list formatting if *, -, or > is entered, the SPACE or TAB key is pressed once, and then characters are entered.

  • Automatic numbered lists

  • Automatically applies numbered list formatting if 1 is entered, a period is entered or the TAB key is pressed, and then characters are entered.

  • Insert horizontal line

  • Entering >, ~, #, *, -, _, or = three consecutive times and pressing ENTER changes the three symbols into a horizontal line.

Note

Ending Automatic Numbered List

To end an automatic numbered list while editing a document, press ENTER twice.
