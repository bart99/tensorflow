AutoCorrect List: AutoCorrect List
 

Review > AutoCorrect > AutoCorrect List

AutoCorrect is a feature that automatically corrects spelling and expression mistakes as you make them while working on a document. You can also add custom words to the feature.

More Information

AutoCorrect List Tab of AutoCorrect List Dialog Box

AutoCorrect Custom Word

Displays a list of custom AutoCorrect words. You can add words to the list and edit or remove registered words from the list.

Details on Icons

Icon Name Description
Add AutoCorrect Add a frequently misspelled word (command) to the list.
Edit AutoCorrect Edit the selected word (command).
Delete AutoCorrect Remove the selected word (command) from the list.
Open AutoCorrect Open the AutoCorrect list file.
Save AutoCorrect Save the current AutoCorrect list.
