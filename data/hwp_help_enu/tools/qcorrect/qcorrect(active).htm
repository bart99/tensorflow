Enable AutoCorrect
 

Review > AutoCorrect > Enable AutoCorrect

You can specify whether to enable AutoCorrect during document editing.

Steps

  1. Select Review > AutoCorrect > Enable AutoCorrect.
  2. A selection mark is shown in front of the Enable AutoCorrect option to indicate that it has been selected.
  3. To disable the feature, select Review > AutoCorrect > Enable AutoCorrect once more and remove the selection mark.

Note

AutoCorrect Activation Point

AutoCorrect is triggered when an incorrect text string is entered and then SPACE, TAB, or ENTER is pressed to complete the word entry.
