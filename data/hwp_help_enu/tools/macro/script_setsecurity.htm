Script Macro Security Settings
 

View > Macros > Security Settings

Script macros record and assign user-defined keyboard and pointer input sequences to specific shortcuts so that those actions can be repeated with ease.

Running a script macro that contains object-creation syntax (e.g., loading an external program in Hwp) can cause your system to become infected with a virus. To protect your system, use the script macro security feature in conjunction with antivirus software.

From the Script Macro Security Settings dialog box, you can change the security level, view a list of registered objects, and remove registered objects.

More Information

Script Macro Security Settings Dialog Box

Security Level Tab

You can set the security level for script macros.

Select the desired macro security level from Very High, High, Normal, and Low (Not Recommended).

Registered Objects Tab

You can view a list of registered objects and remove objects from the list.
