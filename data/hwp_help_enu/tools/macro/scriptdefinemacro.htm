Record Script Macros/Stop Recording
 

View > Macros > Record Script Macros/Stop Recording

Script macros record and assign user-defined keyboard and pointer input sequences to specific shortcuts.

Effectively using script macros allows you to perform repetitive document editing tasks quickly and with ease. as defined script macros repeat the remembered keyboard and pointer input sequence.

You can save defined macro scripts on the system before exiting from Hwp and can be use in later sessions as long as they have not been overwritten.

Steps

To define a script macro

  1. Type Record Script Macro in the edit window and press ENTER. Repeat two more times to produce three identical lines.
  2. Set Record Script Macro from the first line as a block and click View > Macros > Record Script Macro.
  3. From the list in the Record Script Macro dialog box, select ALT+SHIFT+3.
  4. Type Font Color in the Name field and click Define.
  5. Script macro recording commences, and Stop Recording and Pause Recording buttons are displayed on the status bar.

    (The sequence of keyboard and pointer input actions taken from this point on are recorded to ALT+SHIFT+3.)

  6. Click Home > Font and go to the General tab of the Font dialog box. Set Font Color to red and click Set.
  7. Point to the Status Bar and stop the macro recording by clicking the Stop Recording icon .
  8. A script macro sequence of changing the font color to red is now assigned to ALT+SHIFT+3.

To run a script macro

  1. Set Record Script Macro from the second line as a block.
  2. Press ALT+SHIFT+3 to run the script macro and change the font color of the block to red.
  3. You can change the font color of other text elements in the same manner.

More Information

Record Script Macro Dialog Box

Select a Macro to Define

Select a shortcut (ALT+NUMBER) to assign the new script macro to. Once defined, the script macro can be run by pressing the assigned shortcut (ALT+NUMBER).

The existing definition is erased and replaced with a new script if you select an already defined script macro.

Macro Name

Type in a name for the script macro you wish to assign to the selected shortcut.

Macros Status Bar Button

Open the Record Script Macro dialog box by clicking View > Macros > Record Script Macro or pressing ALT+SHIFT+H. Configure the settings and click Define. The Status Bar now shows Pause Recording and Stop Recording buttons.

Details on Icons

Icon Name Description
Pause Recording Pauses script macro recording. Cannot perform other tasks.
Click Pause Recording once more to resume script macro recording.
Stop Recording Finish script macro recording completely.

Note

Script Macro Shortcuts

Macros that have been created using Record Script Macro can be run while in body edit mode by holding down ALT+SHIFT and pressing the corresponding number key (1 through 0) or the - key at the top of the keyboard.

To run a script macro whose shortcut setting has been set to None, click View > Macros > Run Script Macros and select the macro.

Number of Script Macros

You can use script macros by assigning them to the 11 available shortcuts or without assigning them to a shortcut at all in Hwp. You are only limited by the number of macros allowed by your system.in terms of unassigned macros.

Advanced Usage

Main Menu Script Macros

To define a macro with a command that does not have a shortcut and is executed from the main menu, hold down ALT and click on the shortcut next to the command.

Pressing ALT+H opens the Home menu. To select Options, click O.

Dialog Box Tab Macros

General and Advanced in Font and General, Advanced, Tab Settings, and Border/Background in Paragraph are called dialog box tabs.

Depending on the situation, the same dialog box tab can open every time or the most recently used tab can open.

Therefore, you must use absolute values to define macros with commands from specific tabs. You can select the dialog box tab by holding down CTRL and pressing the number key that corresponds to the tab. Here, the tabs are numbered from 1 and on, starting from the leftmost tab.

Color Palette Macros

To define a macro for selecting the color red from Font > Font Color, press ALT+L, CTRL+1, and ALT+C to shift the focus to Font Color and then press the SPACEBAR to expand the color palette.

If you wish to apply the same color whenever you run the macro, regardless of whether the text has already been defined with a color, it is important that you press HOME and shift the focus to the first position, where black is. Press бщ twice to select red and then press ENTER.

When using the arrow keys such as in this example, you must first press HOME or END to define an absolute position.

Block Macros

To define a macro with Arrange, Font, or another command requiring block designation, set the appropriate section as a block and then start Record Script Macro.

To run such macro, set the corresponding section of the body as a block and then select Run Script Macros.

Caution

Script Macro Name Restrictions

Script macro names must start with a Latin or Korean character.

A message will prompt if you enter a space or a special character as the first character in a script macro name, and you will not be able to create the script macro.
