Script Macro Security Settings: Registered Objects
 

View > Macros > Security Settings

From the Registered Objects tab of the Script Macro Security Settings dialog box, you can view a list of registered objects and remove objects from the list.

Steps

To check for registered objects

  1. Click View > Macros > Security Settings.
  2. In the Script Macro Security Settings dialog box, select the Registered Objects tab.
  3. Check Registered Objects List for registered macro objects and then click Set.

More Information

Registered Objects Tab of Script Macro Security Settings Dialog Box

Registered Objects List

You can view a list of registered macro objects.

Details on Icons

Icon Name Description
Delete Object Removes selected registered objects from the list.

Note

Difference from Security Level

The macro may or may not be run, depending on the security level when a script macro containing syntax to create an object that has been registered by the user.

  • Security Level is set to Normal

  • Object-creation macros are allowed to run without displaying a Script Macro Security Warning message.

  • Security Level is set to High

  • A Script Macro Security Warning message prompts for your confirmation even if the object is a registered one when a script macro that contains object-creation syntax is run.
