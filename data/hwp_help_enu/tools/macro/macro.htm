Macros
 

View > Macros

Macros is a feature that remembers keyboard or pointer actions and repeats them.

You can assign macro command keys (ALT+1 through 0, - or ALT+SHIFT+ one through zero, -) to the simple and repetitive editing tasks so that you can perform the tasks conveniently using those keys.

Hwp supports two types of macros: Key Macro, which only remembers keyboard entries, and Script Macros, which remembers both keyboard and pointer actions.

You can name your macros so that they are easier to remember, save them as separate files, and edit defined macros.

Note

Macros saved on System

You can use defined macros from any document window and is not confined to the current document only. You can save all macros on the system when you exit from Hwp by clicking File > Ext. You can use macros again the next time you start up Hwp.

You can also save macros as individual files in desired folders.

Shortcut Use is Essential

Macros help you edit documents faster and are recommended for those who are somewhat familiar with the shortcuts supported in Hwp.

You need to know how to perform commands using the keyboard and not just using the basic keyboard controls and the pointer in order to define and use macros functions effectively.

Basic Macros

The basic script macros package is located in the User Data folder (\user\hwp\hwpwmkor80.hms). You can check the location of User Data folder by clicking > About Thinkfree Office NEO Hwp > Folder Info.
