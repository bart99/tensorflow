Run Script Macros
 

View > Macros > Run Script Macros

You can run a defined script macro by holding down ALT+SHIFT and pressing 1, 2, 3, 4, 5, 6, 7, 8, 9, 0, or -. You can define up to 11 different script macros.

From the Run Script Macros dialog box, you can also define a macro to repeat itself.

Steps

To run a script macro

  1. Click View > Macros > Run Script Macros.
  2. From the Run Script Macros dialog box, select a macro and click Run.

   If you know the shortcut of the defined script macro, hold down ALT+SHIFT and press the shortcut to run it directly without having to open the Run Script Macros dialog box.

To repeat a script macro

  1. Click View > Macros > Run Script Macros.
  2. Select a macro from the list in the Run Script Macros dialog box.
  3. Type 5 in the Repeat Macro field and click Run.
  4. The selected macro is run 5 times.

More Information

Run Script Macros Dialog Box

Select a macro to run.

Select a script macro to run.

You can run a macro by holding down ALT+SHIFT and pressing the shortcut, without opening the Run Script Macros dialog box if you know the name and shortcut or the macro.

Repeat Macro

Repeats the selected macro by the specified number of times. The maximum number of times a macro can be repeated per run is 500.

Description

Displays a description about the selected macro.

Edit Code Button

Displays the entire script code of the selected script macro in the Script task pane.

Details on Icons

Icon Name Description
Edit Script Macro You can edit the name and description of the selected script macro.
Delete Script Macro Deletes the name and recorded contents of the selected macro.
Open Script Macro Opens a script macro file (*.msr) and assigns a shortcut.
Save Script Macro Saves the content of the selected macro script as a script macro file (*.msr).
Save Script Macro Package Saves all designated script macros as a script macro package file (*.hms).
