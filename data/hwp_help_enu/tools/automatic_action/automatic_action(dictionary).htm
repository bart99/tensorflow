AutoCorrect List: Typing Commands List
 

Review > AutoCorrect > AutoCorrect List

If you are composing a document and type in a text string containing a recognized command (applying styles, running macros, and performing actions), Typing Commands automatically performs that command. You can directly add typing commands to Typing Commands and remove existing commands.

More Information

Typing Commands List Tab of AutoCorrect List Dialog Box

Typing Commands Custom Words

Displays a list of custom typing commands. You can add new typing commands and edit or remove existing typing commands.

Typing command types are Styles, Command, and Macro.

Details on Icons

Icon Name Description
Add Typing Commands Add a custom typing command.
Edit Typing Commands Edit a typing command from the list.
Delete Typing Commands Delete a typing command from the list.
Move One Line Up Change the order of registered typing commands. Select a command and click this button to move it one place up.
Move One Line Down Change the order of registered typing commands. Select a command and click this button to move it one place down.
Open Typing Commands Open the file (*.QCM) containing the custom typing commands.
Save Typing Commands Save typing commands from Custom Commands List as a separate file.
