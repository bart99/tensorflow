AutoRun Typing Command
 

Review > AutoCorrect > AutoRun Typing Command

You can decide whether to use typing commands while editing documents.

Steps

  1. Click Review > AutoCorrect > AutoRun Typing Command.
  2. The selection indicator is shown in front of the AutoRun Typing Command option and typing command is enabled.
  3. Click Review > AutoCorrect and unselect AutoRun Typing Command if typing command is not being performed.

Note

Disable Automatic Web/E-Mail Address Link

If you type a web address or an e-mail address in the body with AutoRun Typing Command selected, the address is turned into a hyperlink automatically.

If you do not wish to format web and e-mail addresses as hyperlinks automatically, cancel the selection AutoRun Typing Command.
