Add Typing Commands
 

Review > AutoCorrect > AutoCorrect List

If you are composing a document and type in a text string containing a recognized command (applying styles, running macros, and performing actions), Typing Commands automatically performs that command. You can directly add typing commands to Typing Commands and remove existing commands.

Steps

  1. Click Review > AutoCorrect > AutoCorrect List.
  2. From the AutoCorrect List dialog box, select the Typing Commands List tab and click Add Typing Commands.
  3. From the Content of Typing Commands dialog box, select the Command tab.
  4. From the Position list, select Anywhere and then select Shapes from the Item list.
  5. Once the Shapes option is shown in the Functions List, select Shadow Style, type Shadow in the Input field, and click Add.
  6. Once the Shadow command is added to the AutoCorrect List dialog box, click Close.
  7. Verify that Review > AutoCorrect > AutoRun Typing Command is selected.
  8. Type Shadow in the body and press the SPACEBAR once. Type in Hancom. The word Shadow is removed, and the word Hancom is shown with a shadow.

More Information

Content of Typing Commands Dialog Box

Styles Tab

You can add specific strings you want to apply styles to. When you type in a registered text string while composing a document, Hwp automatically applies the corresponding style.

Command Tab

You can register specific text strings. When you type in a registered text string while composing a document, Hwp automatically runs the corresponding command.

Macro Tab

You can register specific text strings that will execute macros. When you type in a registered text string while composing a document, Hwp automatically runs the corresponding macro.

Note

AutoRun Typing Command Activation

AutoRun Typing Command is only triggered if you type in a registered typing command while composing a document and press SPACE, TAB, or ENTER to mark the end of that text string.

Checking Custom Commands

You can check custom typing commands from the Typing Commands List tab in the AutoCorrect List dialog box. Of the custom typing commands registered to Type, those for applying a style are indicated as Styles, those for performing a feature as Command, and those for running a macro as Macro.

Priority for AutoCorrect List

If the same word has been registered to Typing Commands List and AutoCorrect Words, AutoCorrect overrides the typing command.

When adding a AutoRun Typing Command, you can register Input words that have the same name. For words with the same name, the command that is higher on the Typing Commands List is performed.
