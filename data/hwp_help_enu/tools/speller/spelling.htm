Spelling <F7>
 

Review > Spelling

It compares the current document against the dictionary and suggests the correct spelling for misspelled words.

Steps

  1. Open a document you wish to run a spell check on and click Review > Spelling.
  2. In the Spelling Check/Proofing dialog box, click Start to start spelling check.
  3. When a spelling or grammar error is found, correct the error as below.
    • Type a new word in the Replace with edit box and click Replace to change the word with an error to the correct word.
    • Select a word in Suggestion and click Replace to change the word with an error to the correct word.
    • You can directly correct words with a spelling or grammar error in the body text with the Spelling Check/Proofing dialog box open. Correct words in the document and click Restart in the Spelling Check/Proofing dialog box to continue spelling check.
  4. To finish spelling check, click Close.

   The words pointed out as errors in the Spelling Check/Proofing dialog box are displayed in red.

More Information

Spelling Check/Proofing Dialog Box

Replace with

If you click Replace, the words pointed out as errors in the body text are replaced with the word in Replace with.

You can either type a word in Replace with or select a word from the Suggestion list.

Suggestion

The spelling checker suggests correct words in a list for the words pointed out as errors.

Select a word from the Suggestion list and click Replace to change the word with an error to this word.

Button Description

Button Name Description
Start* Start spelling check.
When spelling check starts, the Start button changes to the Ignore button.
Ignore* Skip words pointed out as errors without correcting them.
Restart* Restart spelling check.
If you edit the body text in the edit window with the Spelling Check/Proofing dialog box open, and return to the Spelling Check/Proofing dialog box, the Ignore button changes to the Restart button.
Ignore All Skip words pointed out as errors without correcting them.
Even if the same words appear later, they will be skipped without being pointed out as Incorrect Word.
Replace Replace the words pointed out as errors with the word typed in Replace with or the one selected from the Suggestion list.
Replace All Replace the words pointed out as errors with the word typed in Replace with or the one selected from the Suggestion list.
Replace all instances of the word in the document including those located before the current word.
Close Close the dialog box.

 The Start, Ignore, and Restart buttons appear in the same position one at a time depending on the situation.

Spelling Light

You can tell spelling error level of the current word by the spelling light on the left of the spelling description box.

Light Description
Indicate that there is no word with an error.
Indicate Level 1 error such as punctuation mark error, honorific error, ambiguous word error, and previous word reference error.
Indicate Level 2 error such as word not in the dictionary, redundant words, and space after punctuation.
Indicate Level 3 error such as spelling error, incorrect term, and suffix (-i/-hi) error. Level 3 errors are errors that must be corrected.

Spelling Description Box

Describe errors of words in detail.

Click on the Spelling Assistant icon on the right of the spelling description box to see a more detailed description and examples.

Details on Icons

Icon Name Description
Add Spelling Dictionary You can add words that have been flagged as incorrect to the custom dictionary from the Spelling Check/Proofing dialog box.
If you add a word to the spelling dictionary, the word will be skipped from the next proofing.
Set Show the Spelling Check/Proofing Settings dialog box to set spelling check method and language for a more detailed check.

Note

Spell Check Range

Clicking Spelling performs a spell check from the current cursor position to the end of the document. When the end of the document is reached, you are asked if you wish to continue the spell check from the beginning of the document.

If you wish to perform a spell check on a part of the document only, set the part as a block and then click Spelling. When spell checking on the block is complete, you are asked if you wish to continue the spell check after the block.

Parentheses Check

For sentences containing parentheses, the word in front of the opening parenthesis and the words inside the parentheses are checked separately and, if the word after the closing parenthesis is a particle, the association between the word before the opening parenthesis and the word after the closing parenthesis is checked.

Control Codes between Words

If there are Control Codes mixed in between words, the words before and after the codes are handled as separate words. However, the words are recognized as a joined word if the word after the control codes is a particle.

If there are header or footer control codes between two words, spell check is performed on the linked header or footer area.

Spelling Check/Proofing Dialog Box

Once you open the Spelling Check/Proofing dialog box and start running a spell check, it continues to perform the check without disturbing the other tasks in operation. You can switch back and forth between other tasks and the dialog box by using cursor.

Check Spelling as you type

If you click File > Options and enable Check Spelling as you type from the Options > Others tab, misspelled words are underlined in red as you type them.
