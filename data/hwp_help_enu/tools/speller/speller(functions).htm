Various Functions of Spelling Checker
 

Review > Spelling

Hwp Spelling Checker is a powerful spelling check tool that references terms from the dictionary to check spelling.

Note

Check Punctuation

Check a series of punctuation marks, the end of sentences, conjunctives, responsive words, and dates to see if punctuation marks are used correctly.

EX) It snows.. ¡æ It snows.

     I can'not believe it. ¡æ I cannot believe it.

Check Abbreviation

Check English abbreviations in sentences.

Check Internet Address and E-mail Address

Check if the typed Internet address and E-mail address are in the correct format.

Add Custom Supplementary Dictionary

You can create a supplementary dictionary by combining words you added to the custom dictionary. Once you create a supplementary dictionary, it is added to the list of supplementary dictionaries and you can select whether to use it or not. Custom supplementary dictionary entries can be added and deleted.
