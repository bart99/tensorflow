Spelling Check/Proofing Settings
 

Review > Spelling

In the Spelling Check/Proofing dialog box, you can specify various spell check settings, check ambiguous words, manage supplementary dictionaries, and check registered symbols.

Steps

  1. Click Review > Spelling.
  2. In the Spelling Check/Proofing Settings dialog box, click the Set icon .
  3. The Spelling Check/Proofing Settings dialog box appears.
  4. Select the Options, Search Dictionary, Supplementary Dictionary, or Symbol tab and perform the necessary task.
