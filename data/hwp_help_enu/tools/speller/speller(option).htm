Spelling Check/Proofing Settings: Options
 

Review > Spelling

In the Options tab of the Spelling Check/Proofing Settings dialog box, you can set spelling check method and language for a more detailed check.

Steps

  1. Click Review > Spelling.
  2. In the Spelling Check/Proofing Settings dialog box, click the Set icon .
  3. In the Spelling Check/Proofing Settings dialog box, click the Options tab.
  4. Select About Spelling Check > Separate Auxiliary Verbs/Adjectives and Grammar > Respectful Language. Click Close.
  5. Return to the Spelling Check/Proofing dialog box and restart spelling check.
  6. Spelling check starts with Spelling Check/Proofing Settings applied.

More Information

Options Tab of Spelling Check/Proofing Settings Dialog Box

About Spelling Check
  • Spacing

    Check for words that should be separated by a space and for words that should be combined together.

  • Redundant word

    Consider compound nouns including redundant nouns as incorrect when there are redundant words.

  • English Abbreviation

    In English, consider words with the first letter in uppercase and the rest in both upper case and lowercase, as abbreviations and treat them accordingly. If you do not select this option, such words will be considered misspelled.

  • Punctuation

    Check if any punctuation mark is omitted or misused.

  • Sound Effect

    Sound goes off ever time a misspelled word appears.

Language

Select Korean, Chinese, or English as the language for spelling check.
