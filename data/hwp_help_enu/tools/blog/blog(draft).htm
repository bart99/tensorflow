Publish to Blog as a Draft
 

File > SNS Service > Publish to Blog as a Draft

Publish as a Draft uploads a document created in Hwp to a blog as a draft post directly from Hwp. Editing a post and publishing it as a draft replaces the previous version of that post. When publishing a post as a draft, you can specify whether to publish it as a Public or Private post and enter a Title for the post. It is a convenient feature to use if you wish to publish posts as you work on them.

Steps

  1. Create a blog post using Hwp.
  2. Click File > SNS Service > Publish to Blog as a Draft.
    If there is no registered blog account, you will be prompted with the "Blog account is missing. Register a blog account?" message.
    Click Yes. The Register Account dialog box opens so that you can register a blog account.
  3. If there is a registered account, a dialog box prompting you to enter the User ID and API Connection will open.
    If you have enabled Auto Connect in Register Account, the post is published as a draft under the designated blog category without prompting you to enter the User ID and API Connection.
  4. Enter the User ID and API Connection and then click OK.
  5. From the Publish to Blog as a Draft dialog box, specify the Post Title and the Post Location.
    If Post Location is empty, that means there is only one category in that blog.
  6. Set Publish Options to Public or Private and then click Add.
  7. When you are prompted with the "Post has been successfully registered to the blog. View registered post?" message, click OK.
  8. The registered blog is shown so that you can verify the post.

More Information

User Account Dialog Box

User ID

Enter the user ID of the blog you wish to register.

API Connection

Enter the password for the API Connection password of the blog you wish to register.

Auto Connect

Select Auto Connect if you wish to publish posts to the registered blog without having to enter the User ID and API Connection.

Publish to Blog Dialog Box

Post Title

Type in the title of the post you wish to publish as a draft.

Post Location

Select the blog category you wish to publish the post as a draft under. Blog categories can be added and modified from the settings or management screen of the blog.

Publish Options

Specify whether to publish the post as a Public or Private draft. You can change the Public/Private setting from the blog at a later point in time.
