Publish to Blog
 

File > SNS Service > Publish to Blog

Publish to Blog uploads a document created in Hwp to a blog directly from Hwp. This feature allows you to publish a post on your blog directly from a familiar program, instead of having to use an HTML editor or a blog editor.

Steps

  1. Create a blog post using Hwp.
  2. Click File > SNS Service > Publish to Blog.
    If there is no registered blog account, you will be prompted with the "Blog account is missing. Register a blog account?" message.
    Click Yes. The Register Account dialog box opens so that you can register a blog account.
  3. If there is a registered account, the Publish to Blog dialog box opens. If you have selected Auto Connect in the Register Account dialog box, the post is automatically registered to the designated blog category without undergoing account information verification.
  4. Specify the Post Title, select the Post Location, and then click Add.
    If Post Location is empty, that means there is only one category in that blog.
  5. When you are prompted with the "Post has been successfully registered to the blog. View registered post?" message, click OK.
  6. The registered blog is shown so that you can verify the post.

More Information

User Account Dialog Box

User ID

Enter the user ID of the blog you wish to register.

API Connection

Enter the password for the API Connection password of the blog you wish to register.

Auto Connect

Select Auto Connect if you wish to publish posts to the registered blog without having to enter the User ID and API Connection.

Publish to Blog Dialog Box

Post Title

Type in the title of the post you wish to publish.

Post Location

Select the blog category you wish to publish the post under. Blog categories can be added and modified from the settings or management screen of the blog.

Publish Options

Specify whether to publish the post as Public or Private. You can change the Public/Private setting from the blog at a later point in time.
