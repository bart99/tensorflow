Manage Blog Accounts
 

File > SNS Service > Manage Blog Accounts

This feature is used to manage blog accounts. Register your frequently used blog accounts in advance to publish Hwp documents on a registered account directly from Hwp.

Steps

  1. Click File > SNS Service > Manage Blog Accounts.
  2. In the Manage Blog Accounts dialog box, click Register Account to add an account.
  3. From the Register Account dialog box, type in the address of the blog you wish to register in Blog > API Address.
    Hwp is included with MetaWeblog API.
  4. Enter the User ID and API in Account Info and click Set.
  5. The account you registered to Manage Blog Accounts > Account List is created.

More Information

Manage Blog Accounts Dialog Box

Account List

The currently registered account list appears. The Display Selection icon is added to the beginning of the default connection account among registered accounts. You can register a maximum of 20 account lists.

Details on Icons
Icon Name Description
Register Account In the Register Account dialog box, register frequently used accounts or accounts you need.
Edit Account In the Edit Account dialog box, modify account name or user ID, or activate/deactivate Auto Connect.
Delete Account Delete accounts you do not use or need to reorganize the account list.
Set as Default Connection Account Set the selected account as the default connection account. If you set an account as the default connection account, the Display Selection icon gets added to the beginning of the account.

Note

MetaWeblog API

To use the Publish feature of Hwp, you must use a blog that supports MetaWeblog API.

MetaWeblog API is a program interface that enables blog post publishing and editing from external editors, without having to connect to the blog directly. It is also known as RFC: MetaWeblog API (MWA).
