Sort
 

Home > Sort/Table > Sort

It rearranges a section of a document in the alphabetical order or the reverse alphabetical order. Using this feature, you can sort a list of addresses or phone numbers by contact name, place of residence, or age.

Steps

  1. Set the range you wish to arrange as a block.
  2. Click Home > Sort or Table > Sort.
  3. From the Sort dialog box, set the first Location field to Paragraph and Format to Characters (A to Z).
  4. Click Run to arrange the information in the block in the alphabetical order, based on the first letter of each paragraph.

More Information

Sort Dialog Box

Sort by

Specify the location of the character you wish to use as the basis for the arrangement and the method of the arrangement.

Separate Fields at

Specify the separator to use to separate the contents into fields.

Sort Option

Specify the sort options.

Note

Set as a Block and then Sort

Sort can only be performed if a block has been set.

Sort by Paragraphs

The basic unit of Sort is a paragraph. In other words, the orders of the paragraphs are rearranged in accordance with the specified conditions. Paragraphs with only a portion of its text included in the block are also subjects of rearrangement, and the contents of paragraphs cannot be arranged.

Separate Fields at

Sort compares the first letters of paragraphs or specific fields, depending on the setting specified under Location.

The fields are divided where the character or symbol specified under Separate Fields at is found, and the number of fields from the paragraph with the most fields is shown in Location.

Setting Separate Field at as Symbol and entering multiple characters or symbols in the input field results in separation of fields at wherever that set of characters or numbers is found.

From Last Character to Previous Character

Compares the fields from the last character to first and arranges the paragraphs in the specified order.

For example, arranging AA, AB, BA, CA, and AC with From Last Character to Previous Character selected produces AA, BA, CA, AB, and AC.

Spaces in Fields

If Separate Field at is set to Text and not Space, spaces in the fields are ignored and arrangement is done based on the letters only. If set to Code, however, fields containing spaces are always arranged to the very beginning since their code values precede all other characters.

Table Sort

You can place the cursor on a table row you wish to arrange and select two or more rows as a cell block. However, you cannot specify the Location setting when arranging a table or arrange two or more columns that have been set as a cell block.
