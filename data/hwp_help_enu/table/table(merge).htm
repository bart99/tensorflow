Merge Cells <M>
 

Table > Merge Cells

Merge Cells merges a cell block consisting of two or more cells into a single cell.

Steps

  1. Set the cells you wish to merge as a cell block.
  2. Click Table > Merge Cells.
  3. The cells in the cell block are merged into one.

Note

Rectangular Cell Block

In order to use Merge Cells, the cell block must be rectangular. Merge Cells will not work on a discontinuous or non-rectangular cell blocks.

Merged Cell Style

Background, line type, diagonal line type, and other properties of the upper leftmost cell of the cell block are applied to the cell merged through Merge Cells.
