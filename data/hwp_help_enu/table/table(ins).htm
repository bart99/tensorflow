Add Rows/Columns <ALT+INSERT>
 

Table > Insert > Cell

When you work on a table, sometimes you need to add more rows and/or columns. In this case, use Table > Add Rows/Columns to insert additional rows and/or columns to the table.

Steps

  1. Place the cursor inside the cell to which you want to add rows and click Table > Insert > Cell.
  2. From the Add Rows/Columns dialog box, click Top.
  3. Enter 3 in Number of Rows/Columns and click Add.
  4. 3 additional rows are added to the top of the current cell.

More Information

Add Rows/Columns Dialog Box

Add

When adding rows, you must specify whether you wish to add them to the top or bottom of the cell at the current cursor position. When adding columns, you must specify whether you wish to add them to the left or right of the cell at the current cursor position.

Icon Name Description
Top Adds rows to the top of the cell at the current cursor position.
Bottom Adds rows to the bottom of the cell at the current cursor position.
Left Adds columns to the left of the cell at the current cursor position.
Right Adds columns to the right of the cell at the current cursor position.

Number of Rows/Columns

Specify the number of rows or columns to add. Up to 63 additional rows or columns may be added.

Note

Cell Block

To use Add Rows/Columns on a cell block, you must set only one cell as a cell block. Add Rows/Columns cannot be used if two or more cells are set as a cell block.

TAB and CTRL+ENTER

Press TAB to move between cells. Pressing TAB from the last cell, however, creates a new row.

In addition, pressing CTRL+ENTER from inside the table creates a new row below the current row.
