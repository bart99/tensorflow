Text to Table
 

Insert > Table > Text to Table/Table > Convert > Text to Table

It converts the text in a block into a table.

Steps

  1. Type in words, pressing TAB after each word. Once you have typed in all the words for a given line, press ENTER to break the line and type in the words for the next line.

    Korean > TAB > English > TAB > Math > ENTER

    80 > TAB > 90 > TAB > 85 > ENTER

    70 > TAB > 80 > TAB > 70 > ENTER

  2. Set all the entries as a block and click Insert > Table > Convert Text to Table.

  3. From the Text to Table dialog box, select Automatic Insert and click Set.
  4. A table containing the entries from the block is created.

More Information

Text to Table Dialog Box

Automatic Insert

Automatically generates a table containing information from the selected block. Line breaks made using the ENTER key are used to determine the number of rows, whereas TAB entries or three or more spaces between words are used to determine the number of columns.

Separate by

Converts information from the selected block into a table and determines the number of columns based on a division method specified by the user. The number of rows are determined by the number of line breaks (ENTER).

  • Tab

  • Uses the TAB entry as the column separator.

  • Comma

  • Uses the comma as the column separator.

  • Three or more spaces

  • Uses three or more spaces as the column separator.

  • Other Chars

  • Uses user-defined Korean or Latin characters or symbols as the column separator.

    For example, specifying 한글 under Other Chars designates 한 and 글 as the separators.

  • Text

  • Uses a user-defined word as the column separator.

    For example, specifying 한글 under Text designates 한글 as the separator.

Options

  • Keep Separators

  • This option leaves the separators in the cells of the table.

Note

Applying to All Cases

You can convert a text block into a table even if there is a table included in the block. Also, you can convert non-body text, such as the header or footer, into a table as well.

Maintaining Spaces to the Right

Converts text into a table and shows the spaces between an entry and its separator inside the cells.

Caution

Single Column Block

Text to Table does not work if you have selected a single column block by pressing F4.
