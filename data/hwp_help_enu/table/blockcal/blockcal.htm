Block Formula
 

Table > Block Formula

You can obtain Block Sum, Block Average, Block Product, and other calculation results after specifying a cell block within a table. Block Formula is useful for calculating the sum or average in a document containing a complete set of numerical data, such as a report card or an account book.

Steps

To apply block sum

  1. In the table, select cells for block calculation.
  2. Click Table > Block Formula > Block Sum.
  3. The sum is obtained and inserted into an empty cell on the right or at the bottom automatically.

To apply block average

  1. In the table, select cells for block calculation.
  2. Click Table > Block Formula > Block Average.
  3. The average is obtained and inserted into an empty cell on the right or at the bottom automatically. The average is calculated to two decimal places.

To apply block product

  1. In the table, select cells for block calculation.
  2. Click Table > Block Formula > Block Product.
  3. The product is obtained and inserted into an empty cell on the right or at the bottom automatically.

Note

One Number from a Single Cell

If there are multiple numbers in a single cell, only the first number is reflected in the calculation.

Block Formula from the Body

If you need to perform a calculation by using numbers in a table, Block Formula is the more convenient option.

If you need to perform a calculation by using numbers in the body, you can use Review > Block Calculation.

Recalculating Formula

If you modify the cell data that is used in a block calculation, the calculation result gets updated automatically. If you modify the result, then recalculation does not occur until you modify the cell content you originally used for block calculation or modify the formula.

Ignoring Previous Calculation Result

If a Formula result is contained in the cell directly preceding the block formula result cell, which is the last cell of the selected range of cells, this value is included in calculation of the block average. This happens because in many cases the total score is obtained using Formula, and then the average is also obtained in the next cell by using Formula.

Caution

Restriction

Block Formula cannot be used on cell blocks consisting of a single cell and on cell blocks with discontinuous cell selection. Block calculation cannot be performed if the cell required to enter the calculation result (the cell to the right or at the bottom of the block) is not available.
