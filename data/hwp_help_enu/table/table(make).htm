Insert Table
 

Insert > Table > Insert Table

You can use a table for organizing complicated content or numbers in a document.

Insert Table in Hwp enables you to create a table inside a table and automatically divides tables that go over a page.

Steps

To create a table using the menu

  1. Click Insert > Table > Insert Table.
  2. From the Insert Table dialog box, specify the number of rows and columns and click Insert.
  3. The mouse pointer changes to indicate readiness to draw a table.
  4. Place the mouse pointer where you want to insert a table and click and drag until a table of the desired size is created.
  5. When a table is inserted, the pointer is in the first column of the first row.
  6. Type text into the table as you would in the body.
  7. To exit from the table, press SHIFT+ESC or mouse click anywhere outside the table.

More Information

Insert Table Dialog Box

Rows/Columns

Specify the number of rows and columns of a table. The maximum number of rows and columns of a table is 128 each.

However, it does not mean that you cannot make a table of 129 or more rows and columns. When making a table, you can adjust the number of rows and columns within 16,384 cells. For example, if you select 1,600 for the number of rows, you can only select up to 10 for the number of columns. This means that you cannot make a table with 16,384 or more cells.

Specify Size

In order to specify the size of the table, the Create by Dragging option under Others must not be selected.

  • Width
    • Fit to Column

    • Creates a table that fits the width of the column at the current cursor position.

    • Align to Paragraph

    • Creates a table that fits the width of the paragraph at the current cursor position.

    • Random Value

    • Create a table by specifying the width.

  • Height
    • Automatic

    • Creates a table equaling the size of the standard cell height multiplied by the number of rows.

    • Random Value

    • Creates a table with specified height.

Others

Specify whether to set the new table as In Line with Text, which handles placement of the table like it were text, and to enable Create by Dragging, which lets you set the location and size of the table using the mouse.

  • In Line with Text

  • Treats the table like normal text. Therefore, the table position changes as you type or delete text.

  • Create by Dragging

  • Selecting this option allows you to create tables at the designated locations by dragging, instead of always inserting the table at the current cursor location.

    If left unselected, a table that fits the width of the column at the current cursor position is created.

Templates Button

Opens the Quick Tables dialog box from which you can select table formats to create new tables or switch the style of an existing table.

Properties Button

Opens the Table/Cell Properties dialog box from which you can use predefined table border and background settings to create new tables.

Note

Table Tab

The Table tab automatically appears if the cursor is placed inside an existing table. From this tab, you can perform basic table editing tasks including create a table, change table/cell properties, split cells, merge cells, and distribute rows/columns.

Moving Table Position

If you click inside a table while pressing ALT, the pointer changes to a four-headed arrow . You can hold a spot inside of the table and drag it to a position you want. You can move a table past page boundaries.

Creating Table Inside Table

Click Insert > Table > Insert Table from within a table to create and edit a new table.

If the table inside the active table is bigger than cells of the active table, the rest is hidden. You can adjust cell size according to the size of the table inside the active table.

Minimum Table Size

If you create a table using Random Value settings for width and height and those values are too small for the number of rows and columns specified, a table whose width equals the minimum cell width multiplied by the number of columns and whose height equals the minimum cell height multiplied by the number of rows is created.

Specifying Table Origin Using Mouse

To enable table location and placement selection using the mouse, click File > Options > Object.

Overlapping Tables

Two tables whose Wrapping Style is set to Top and Bottom cannot be overlapped. To enable table overlapping, select the tables, click Table/Cell Properties > Basic, and select Position > Allow Overlap.
