Merge Table
 

Table > Merge Table

You can merge two or more tables into one by placing the cursor in the first table and click Table > Merge Table. Merge Table can only merge two tables if there is a space and a line break (ENTER) between the two tables.

Steps

  1. Of the two tables to merge, place the cursor inside the first table and click Table > Merge Table.

  2. The second table is then merged with the first table.

Note

Table Properties and Size

When two or more tables are merged, Table Properties of the first table are applied to the rest of the tables.

The width of the merged table follows the width of the widest table.

Copy and Paste

Merge Table is usually used to merge the current table with the table below it. To merge a table to a column or to paste a table within a table, use Paste Cell instead.

Caution

Restriction

Merge Table will not work if there is any non-table element between the tables, such as text, a picture, or a text box. Merge Table will only work if there is a space or a line break between two tables.

In addition, a warning message will show if there is no table after the current one.
