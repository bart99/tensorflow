Table Eraser
 

Table > Delete > Table Eraser

Eraser erases rows and columns from an existing table. You can erase rows simply by dragging on them with the mouse.

Steps

  1. With a table inserted into the body, click Table > Delete > Table Eraser or click Table Eraser from the Table tab.
  2. Move the mouse pointer over to the edit window, and it changes to the eraser icon .
  3. Place the mouse pointer on the row you wish to erase and drag in the direction of the row while holding down the left mouse button. The row is displayed as a thick line.
  4. Release the left mouse button to erase the row.

Note

Erase Method

If the surface selected using Table Eraser is being used in conjunction with an adjacent cell, the current cell is merged into one with the adjacent cell. Otherwise, the line is turned into a transparent one.
