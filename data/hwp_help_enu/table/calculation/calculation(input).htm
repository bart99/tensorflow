Inserting Formula
 

Table > Formula

You can insert the sheet functions of a formula and the calculation range. If you select a sheet function from the Function list, it automatically enters the function in the Formula field.

Note

Enter Manually

You can also enter functions, such as a basic arithmetic function (+, -, *, /), the sum function, and the average function directly in the Formula field. To type in a formula directly into the Formula field, use the prefix = or @ before entering the sheet function.

Insert Formula

For example, the following formula may be used:

a1+(b3-3)*2+sum(a1:b5,avg(c3,e5-3))

(?1:?3) and (A?:C?)

If you are calculating column values only, you can create a formula such as SUM(?1:?3) and copy it repeatedly. You can use the formula SUM(A?:C?) if you are calculating row values only.
