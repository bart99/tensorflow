Formula
 

Table > Formula

You can use basic arithmetic functions such as add, subtract, multiply, and divide, as well as sheet functions, such as sum and average and formulas containing left, right, below, and above range designators such as sum(left) in tables.

Steps

  1. Click Insert > Table > Insert Table, and then enter values in the cells.
  2. Place the pointer on the cell in which the result of the calculation must be displayed, and then click Table > Formula.

    For example, place the cursor in the cell located on row 5, column 1, and then click Table > Formula.

  3. In the Formula dialog box, click Function, and then select the PRODUCT(..) function.
  4. The PRODUCT() is entered in the Formula field automatically, and the cursor is displayed inside the parentheses.
  5. Type B1, a comma (,), and then C3, inside the parentheses.

    PRODUCT(B1,C3) must be entered in the Formula field.

  6. Click OK, and then enter the calculation result in the cell.

    2 ¡¿10 is calculated, and 20 is displayed as the result.

More Information

Formula Dialog Box

Formula

A function selected from the Function list is automatically entered. You can also enter a basic arithmetic function (+ - * /), the sum function, or the average function directly into formula field.

To type a formula directly into the formula field, use the prefix = or @, before entering the sheet function.

Function

Click on list, and then select a sheet function. The function is automatically entered into the Formula field.

Quick Selection

You can select LEFT for farthest left to the current cell, RIGHT for farthest right to the current cell, ABOVE for topmost to the current cell, and BELOW for bottommost to the current cell.

Format

You can specify the result display format. Select from Default Format, Integral Numbers, 1 Decimal, 2 Decimal, 3 Decimal, and 4 Decimal.

Thousands Separator

If you select this option, a comma (,) is placed after each group of three digits in the calculation result.

Note

Cell Number

When the pointer is moved from one cell to another, the name of the cell is displayed on the Status Bar in the form of A1, A2, A3, and so on. The columns are designated in the order of A, B, C, D, E, and so on, and the rows in the order of 1, 2, 3, 4, 5, and so on.

When creating a formula in a table, use the cell names instead of directly typing in the contents of the corresponding cells.

Cell Separator

After entering a sheet function in the Formula field, you must enter the names of the cells inside the parentheses.

You can separate cell names inside the parentheses by using commas (,) or (colons). A comma is used to obtain the sum of the specified cells, whereas a colon is used to specify the start and end cells, and obtain the sum of all cells within the range.

For example, type in =SUM(A1,C1) if you wish to obtain the sum of A1 and C1. If you wish to obtain the sum of all cells from A1 to C1, type =SUM(A1:C1).

Recalculating Formula

If you modify the cell content you used in Formula, the result gets applied automatically. If you modify the result, recalculation does not occur until you modify the cell content you used in Formula or modify the formula.

Using Field Name

If you use Field Name in the formula, you can only use Quick Selection, and the ALL function. After Quick Selection of ALL function, type a colon (:), and then enter the field name.
