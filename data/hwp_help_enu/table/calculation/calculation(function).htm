Sheet Functions
 

Table > Formula

If you select sheet functions to use in a formula, it automatically enters them in the Formula field. The sheet functions available with Hwp are as follows:

Note

SUM

Calculates the sum of cells in a defined range.

AVERAGE (or AVG)

Calculates the average of cells in a defined range.

PRODUCT

Calculates the product of cells in a defined range.

MIN

Calculates the minimum of cells in a defined range.

MAX

Calculates the maximum of cells in a defined range.

COUNT

Counts the number of non-empty cells in a defined range.

COS

Calculates the cosine of a single cell.

SIN

Calculates the sine of a single cell.

TAN

Calculates the tangent of a single cell.

ACOS

Calculates the arccosine of a single cell.

ASIN

Calculates the arcsine of a single cell.

ATAN

Calculates the arctangent of a single cell.

ABS

Calculates the absolute value of a single cell.

EXP

Calculates the power of e of a single cell.

LOG

Calculates the power of a natural logarithm with e as the base of a single cell.

LOG10

Calculates the base-10 common logarithm of a single cell.

SQRT

Calculates the root of a number of a single cell.

RADIAN

Calculates the radian of an angle (general).

SIGN

Calculates the number in a single cell as 1 if positive, 0 if 0, and -1 if negative.

CEILING

Calculates the larger or equal smallest integer for a single cell.

EX) CEILING(A2) ¢¡ -3, CEILING(B2) ¢¡ 5

FLOOR

Calculates the smaller or equal largest integer for a single cell.

EX) FLOOR(A2) ¢¡ -4, FLOOR(B2) ¢¡ 4

INT

Calculates the integer of a single cell.

EX) INT(A2) ¢¡ -3, INT(B2) ¢¡ 4

ROUND

Rounds the number up to the specified digit for a single cell.

MOD

Calculates the remainder of a quotient of two cells.
