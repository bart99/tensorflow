Table
 

Insert > Table / Table > Table

You can use a table for organizing complicated content or numbers in a document.

Insert Table in Hwp enables you to create a table inside a table and automatically divides tables that go over a page.

More Information

Insert Table

Specify the number of rows and columns of a table. The maximum number of rows and columns of a table is 128 each.

However, it does not mean that you cannot make a table of 129 or more rows and columns. When making a table, you can adjust the number of rows and columns within 16,384 cells. For example, if you select 1,600 for the number of rows, you can only select up to 10 for the number of columns. This means that you cannot make a table with 16,384 or more cells.

Draw Table

By using Draw Table, you can create a table, add rows and columns to a table, draw diagonal lines, change the line type, and more with just the mouse. This allows you to create even complex tables with ease.

Eraser

Eraser erases rows and columns from an existing table. You can erase rows simply by dragging on them with the mouse.

Text to Table

Convert the text in a block into a table.

Convert to Text

Remove table borders and convert the content into text.

Note

Moving Table Position

If you click inside a table while pressing ALT, the pointer changes to a four-headed arrow . You can hold a spot inside of the table and drag it to a position you want. You can move a table past page boundaries.

Creating Table inside Table

Click Insert > Table > Insert Table or Table > Insert > Insert Table from within a table to create and edit a new table.

If the table inside the active table is bigger than cells of the active table, the rest is hidden. You can adjust cell size according to the size of the table inside the active table.
