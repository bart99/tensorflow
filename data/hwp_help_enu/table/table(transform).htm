Transpose Table <T>
 

Table > Transpose Table

You can use the Transpose Table feature to transpose a table based on rows or columns or to rotate a table by 90¡Æ.

For example, if you need to transpose a timetable or a statement of accounts based on the rows or the columns, you can do so in a single step by using Transpose Table.

Steps

  1. Click Insert > Table > Insert Table.
  2. Fill the table with data.
  3. Select the table or place the cursor inside of it.
  4. Click Table > Transpose Table.
  5. From the Transpose Table dialog box, select Rotation > Counterclockwise 90 Degree and click Flip.
  6. The selected table is rotated by 90¡Æ counterclockwise.

More Information

Transpose Table Dialog Box

Flip

  • Invert Rows

  • Transpose the selected table based on the rows.

  • Invert Columns

  • Transpose the selected table based on the columns.

  • Invert Rows/Columns

  • Transpose the selected table diagonally.

Rotation

  • Counterclockwise 90 Degree

  • Rotate the selected table by 90¡Æ counterclockwise.

  • 180 Degrees

  • Rotate the selected table by 180¡Æ.

  • Clockwise 90 Degree

  • Rotate the selected table by 90¡Æ clockwise.

Options

  • Flip Margin

  • Specify whether to flip or rotate the margins along with the selected table.

Note

Table Position

Transpose Table can be used if the position of the table is set to Text, Paragraph, Page, or Paper.

Fix Size and Lock Object

Transpose Table cannot be performed if the selected table is in Fix Size or Lock Object state. Select Table/Cell Properties and disable Fix Size and Lock Object settings before using Transpose Table.
