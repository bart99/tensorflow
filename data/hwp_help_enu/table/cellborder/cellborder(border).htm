Cell Border/Background: Borders <L>
 

Table > Cell Border/Background > Apply to Each Cell/Set as One Cell

In the Borders tab of the Cell Border/Background dialog box, you can specify border styles for individual cells of a table, and designate sides to display or not display a border.

The Apply to setting under the Borders tab is available if you click Table > Cell Border/Background > Apply to Each Cell. It also provides All Cells and Selected Cells as options depending on the cell block status of the current table.

Steps

  1. Position the pointer inside a table, and click Table > Cell Border/Background > Apply to Each Cell.
  2. In the Cell Border/Background dialog box, select the Borders tab.
  3. Click the Border Type list, and select the triple line, which is at the end of the list.
  4. Weight is automatically set to 2 mm.
  5. In the border preview area, click Outside Borders, and apply the selected border style to all outside cell borders.
  6. Click Set to change the outside borders to triple lines.

More Information

Borders Tab of Cell Border/Background Dialog Box

Borders

There are 12 Types of available cell borders. You can specify the Weight and Color for each border type selected.

  • Type

  • You can select the desired type of line from the 12 available options. You can also specify the weight and color of your selection.

  • Weight

  • You can select the weight from the list. Custom weight value is not supported.

  • Color

  • You can click the color table to display the color palette. From the color palette, select a color of your choice.

  • Apply Line Type Now

  • You can apply the border style in the preview area by selecting this option.

    You can also click Borders to select a style and apply it.

Apply to

This option is displayed if you click Table> Cell/Border Background > Apply to Each Cell. It is not displayed if you click Table > Cell/Border Background > Set as One Cell.

  • All Cells

  • You can apply Cell Border/Background properties to all cells in the active table.

  • Selected Cells

  • If you select cells as a cell block, the Selected Cells show their default values and the Cell Border/Background properties are applied only to the selected cells.

Table Border/Backgrounds Button

To use a picture as the background for all cells, in the Cell Border/Background dialog box, click Table Border/Backgrounds.

Select line type, line color, and line weight to change the border of the required side.If you click the required border direction icons in the preview area, the borders for the selected directions begin to toggle between current style and previous style.
