Set as One Cell
 

Table > Cell Border/Background > Set as One Cell

You can set multiple cells in the block as a single cell by clicking Cell Border/Background > Set as One Cell after defining cell block in a standardized pattern cell.

From the Cell Border/Background dialog box, you can decorate the table by changing the cell border style, drawing diagonal lines, and/or applying a background color or image to individual cells.

Note

Cell Block

Cell Border/Background > Set as One Cell is available only if the cell block is set in the standardized pattern.

Set as One Cell is not available if you set two or more cells as a cell block in a non-standard pattern or as a discontinuous cell block.

Borders

Select the border weight and color for the cells within the defined cell block range.

The cells are recognized as a single cell for which you can specify the cell border style by setting multiple cells as a cell block and then clicking Set as One Cell.

Background

You can specify the background (color, gradation, or picture) for the defined cell block range.

The cells are recognized as a single cell for which you can specify the cell background by setting multiple cells as a cell block and then clicking Set as One Cell.

Diagonal

You can draw and insert various types of Diagonal lines in the defined cell block range.

The cells are recognized as a single cell for which you can specify the diagonal line style by setting multiple cells as a cell block and then clicking Set as One Cell.

Applying by Cell

Cell Border/Background is applied on individual cells. If you set a cell block and specify the cell background, it applies the background to individual cells and not to the entire cell block.

Table Border/Background

To create a background for all the cells of the table, in the Cell Border/Background dialog box, click Table Border/Backgrounds.

Shortcut

You can use the following shortcuts to easily adjust the Cell Border/Background, when cells are selected as a block.

  • Cell Border: <L>
  • Cell Background: <C>

Format Painter in Table

In the table, click Home > Format Painter to copy the current font, paragraph style, cell property, line type, and cell background settings and apply them to other cells.

Copying the cell property, cell border style, and cell background is available only if you click Home > Format Painter from inside the table.
