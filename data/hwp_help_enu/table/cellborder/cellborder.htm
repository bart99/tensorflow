Cell Border/Background
 

Table > Cell Border/Background

You can decorate the table by changing the cell border style, drawing diagonal lines, and/or applying a background color or image to individual cells.

More Information

Cell Border/Background

  • Apply to Each Cell (Apply to > Selected Cells)

  • You can apply the desired border style, background, and diagonal line style to selected cells in a table only.

    Select cells in a table, right click on the mouse to access the context menu, and click Cell Border/Background > Apply to Each Cell.

    The background settings you specify in the Background tab of the Cell Border/Background dialog box are applied to individual cells as shown below:

  • Apply to Each Cell (Apply to > All Cells)

  • You can apply the desired border style, background, and diagonal line style to all cells in a table.

    Select cells inside a table, right click to access the context menu, and click Cell Border/Background > Apply to Each Cell. Specify the background in the Background tab of the Cell Border/Background dialog box and set Apply to to All Cells to apply the background to all cells as shown below:

  • Set as One Cell

  • You can apply the desired border style, background, and diagonal line style to a range of multiple cells in a table.

    Select cells in a table, right click to access the context menu, and click Cell Border/Background > Set as One Cell. Background settings you specify in the Background tab of the Cell Border/Background dialog box are applied to a range of multiple cells as shown below:

Table Border/Backgrounds

To make a background that covers all the cells of the table, click Table Border/Backgrounds in the Cell Border/Background dialog box. Specifying the background in the Cell Border/Background dialog box applies the background to all cells in a table as displayed below:

Note

Shortcut

If cells are selected as a block, you can use the following shortcuts to easily adjust the Cell Border/Background.

  • Cell Border: <L>
  • Cell Background: <C>

Format Painter in Table

In the table, click Home > Format Painter to copy the current font, paragraph style, cell property, line type, and cell background settings and apply them to other cells.

Cell Properties, Cell Border, and Cell Background function are available if you click Home > Format Painter inside the table.
