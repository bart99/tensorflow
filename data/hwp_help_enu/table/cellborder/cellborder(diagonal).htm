Cell Border/Background: Diagonal
 

Table > Cell Border/Background > Apply to Each Cell/Set as One Cell

In the Cell Border/Background dialog box from the Diagonal tab, you can draw different types of diagonal lines or non-dividing virtual center lines inside individual cells of a table.

The Apply to setting under the Diagonal tab is available only if you click Table > Cell Border/Background > Apply to Each Cell. It also provides All Cells and Selected Cells as options depending on the cell block status of the current table.

More Information

Diagonal Tab of Cell Border/Background Dialog Box

Diagonal

There are 12 diagonal line Types available. You can also specify the Weight and Color for each diagonal line type selected.

  • Type

  • You can select the desired line type from the 12 available options. You can also specify the weight and color of your selection.

  • Weight

  • You can select the weight of the diagonal line from the list only. Custom weight value is not supported.

  • Color

  • You can click the color table to display the color palette, and then select a diagonal line color of your choice.

¡¬ Diagonal

Draws a diagonal from the top- left corner to the bottom- right corner of the cell. When you select a diagonal line type from the 8 available options, the selected line type is displayed in the preview area. To remove the diagonal line, click Remove Diagonal.

£¯ Diagonal

Draws a diagonal from the top- left corner to the bottom- right corner of the cell. When you select a diagonal line type from the 8 available options, the selected line type is displayed in the preview area. To remove the diagonal line, click Remove Diagonal.

+ Center Line

Draws a horizontal or vertical center line or horizontal and vertical center lines that pass through the center of the cell. When you select a center line type from the 3 available options, the selected line type is shown in the preview area. To remove the center line, click Remove Center Line.

Apply to

This option is displayed if you click Table > Cell/Border Background > Apply to Each Cell. It is not displayed if you click Table > Cell/Border Background > Set as One Cell.

  • All Cells

  • You can apply Cell Border/Background properties to all cells in the active table.

  • Selected Cells

  • If you select cells as a cell block, the Selected Cells show their default values and the Cell Border/Background properties are applied only to the selected cells.

Table Border/Backgrounds Button

If you want to use a picture as the background for all cells, click Table Border/Backgrounds in the Cell Border/Background dialog box.

Note

Drawing a Diagonal Line

With 5 line type options, diagonal lines can be drawn from top-left to bottom-right, top-right to bottom-left, or in both directions.

You can specify the weight and color of diagonal lines, as the cell borders.
