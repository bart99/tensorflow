Cell Border/Background: Background <C>
 

Table > Cell Border/Background > Apply to Each Cell/Set as One Cell

In the Background tab of the Cell Border/Background dialog box, you can add fill the cell background with a color, pattern, or picture or give it a gradation effect.

The Apply to setting under the Background tab is available when you click Table > Cell Border/Background > Apply to Each Cell, and provides All Cells and Selected Cells, as options depending on the cell block status of the current table.

Steps

  1. Place the pointer on a cell to change its background color, press F5 to set it as a block, and then click Table > Cell Border/Background > Apply to Each Cell.
  2. In the Cell Border/Background dialog box, select the Background tab.
  3. Select Color, and then click on the color table next to the Face Color option.
  4. Select the desired background color from the expandable color palette, and then click Set.
  5. The background of the cell is filled with the selected color.

More Information

Background Tab of Cell Border/Background Dialog Box

No Fill

Select No Fill, if a color or a gradation effect is not required or to remove an existing background effect.

Color
  • Face Color

  • You can click the color table to display the color palette, and then select a background color from the color palette.

  • Pattern Color

  • If Pattern is selected, click the color table in Pattern Color to display the color palette. In the color palette, select a pattern color to fill the background.

  • Pattern

  • From the Pattern list, select a pattern to fill the background.

Gradation

Fill the background with gradation for gradual color spread.

Picture

Open a picture file, and fill the background by using the picture.

Apply to

This option is displayed if you click Table > Cell/Border Background > Apply to Each Cell. It is not displayed if you click Table > Cell/Border Background > Set as One Cell.

  • All Cells

  • Apply Cell Border/Background properties to all cells in the active table.

  • Selected Cells

  • If you select cells as a cell block, the Selected Cells show their default values and the Cell Border/Background properties are applied only to the selected cells.

Table Border/Backgrounds Button

To use a picture as the background, in the Cell Border/Background dialog box, click Table Border/Backgrounds.

Note

Applying by Cell

Cell background is applied on the basis of individual cells. If you insert a background picture to a group of cells, the picture is applied as the background for individual cells, and not to the entire range of cells.

To use a picture as the background, in the Cell Border/Background dialog box, click Table Border/Backgrounds.

Setting Simultaneously

You cannot apply a color and a gradation effect to a cell background simultaneously. However, you can set two effects like Fill with Both Color and Picture and Fill with Both Gradation and Picture simultaneously.

Applying in Cell Block State

To apply the border/background setting to the current cell, click Cell Border/Background in cell block state.

If you position the pointer in the cell you need to apply border/background without setting it as a part of a cell block, and click Cell Border/Background to apply the changes to all cells.
