Table/Cell Properties: Borders
 

Table > Table/Cell Properties

Show All

From the Borders tab of the Table > Table/Cell Properties dialog box, you can specify the type, weight, and color of table borders.

More Information

Borders Tab of Table/Cell Properties Dialog Box

Borders
  • Type

  • Select the desired table border type from the 12 available options.

  • Weight

  • Specify the table border weight.

  • Color

  • Click the color table to display the color palette and then select a color for the table border.

  • Preview Line Style

  • Selecting this option immediately applies the border style in the preview area, on the selected sides.

    If you do not select this option, you have to click Borders for the direction to apply a border style after selecting the style so that you can change border styles.

  • Preview Area

  • Specify the border type, weight, and color and specify the which sides to apply the style to using Left, Right, Top, Bottom, or All icons from the preview area. This applies the style on the selected sides.

  • Cell Spacing

  • This setting specifies the spacing between individual cells within the table. Type a value or use the increase and decrease buttons to adjust.

Boundary Settings of Auto Split Table

You can specify the type, weight, and color of the wireframe if the table has been set to split automatically on the border.

  • Type

  • Select the desired table border type from the 12 available options.

  • Weight

  • Specify the table border weight.

  • Color

  • Click the color table to display the color palette and then select a color for the table border.

Note

Cell Spacing

The table border style specified in Table/Cell Properties > Borders is applied only when a value is entered for Cell Spacing in Table/Cell Properties > Table.
