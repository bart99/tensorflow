Table/Cell Properties: Table
 

Table > Table/Cell Properties

Show All

From the Table tab of the Table > Table/Cell Properties dialog box, you can specify the table to span over multiple pages and specify the margin between the table borders and its content and the spacing between cells.

More Information

Table Tab of Table/Cell Properties Dialog Box

Same as Whole Table
  • At Page Break

  • Specifies where to split a table that spans over multiple pages.

    Selection Description
    Separate When a table is split by the page boundary, continue to the next page like normal body text.
    Break by Cell When a cell is split by the page boundary, place the whole cell on the next page.
    Do not split Always place the whole table on a single page. Sections outside the page area are not shown on the screen or printouts.
  • Repeat Header Row Automatically

  • Selecting Repeat Header Row Automatically for a table that spans over multiple pages automatically displays the title row of the table at the beginning of each page.

    However, the first row must be set as Title Cell from the Cell tab in order for Repeat Header Row Automatically to function.

Margins Inside All Cells
  • Left

  • The margin between the left border and the content.

  • Right

  • The margin between the right border and the content.

  • Top

  • The margin between the top border and the content.

  • Bottom

  • The margin between the bottom border and the content.

  • All

  • Increase or decrease left, right, top, and bottom margins of the cell at the same time.

Note

In Line with Text

Setting the grid origin of a table as In Line with Text negates the Same as Whole Table setting. In addition, the table cannot be larger than a single page. If a table exceeding the page in size was created, the section that exceeds the page size is not shown on the screen or printouts.

Split Table

If a table with Do not split set for the At Page Break setting or a table set to In Line with Text runs over to the next page, click Table > Split Table at the appropriate location and split the table into two. Split Table splits the table beneath the row at the cursor position.
