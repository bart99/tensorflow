Field Name
 

Table > Table/Cell Properties

You can assign names to cells so that external programs can read and write the Hwp document. By assigning field names, you can use the Formula feature more effectively.

Steps

To assign field names

  1. In the following table, place the cursor in the Expenses for 2006 cell and click Table > Table/Cell Properties.

  2. From the Table/Cell Properties dialog box, select the Cell tab and type a name for the cell in the Field Name field.

    For this example, type EX as the field name. Click Set.

  3. Place the cursor in the Expenses for 2007 cell, click Table > Properties, and assign EX as the field name.
  4. Place the cursor in the Total Expenses cell, click Table > Properties, and assign SUM as the field name.
  5. Place the cursor in the Costs for 2006 cell, click Table > Properties, and assign CO as the field name.
  6. In the same manner, assign CO as the field name for the Costs for 2007 cell.
  7. Place the cursor in the Total Costs cell, click Table > Properties, and assign SUM as the field name.

To insert a formula

  1. Place the cursor in the Total Expenses cell and click Table > Formula.
  2. From the Formula dialog box, enter =SUM(ABOVE:EX) in the Formula field to add all the values existing above the current cell.
  3. Click OK. 280 is entered automatically.

  4. Place the cursor in the Total Costs cell, click Table > Formula, type =SUM(ABOVE:CO) in the Formula field, and click OK. 190 is entered automatically.

  5. Place the cursor in the Grand Total cell, click Table > Formula, and type =SUM(ALL:SUM) in the Formula field to gain the sum of all fields with the name SUM from the current table.
  6. Click OK. 470 is entered automatically.

Note

Same Field Names

A single table may contain multiple instances of the same field name, which can be designated in English or Korean.

To Use with Formula

To use field names in the Formula field, only the Quick Selection and ALL functions are supported. Type the Quick Selection or ALL function, a colon (:), and then the field name.

ALL Function

Calculates all fields in the current table bearing the designated field name. For example, the SUM(ALL:INCOME) formula gains the sum of all cells with the name INCOME from the current table.

Quick Selection for Continuous Cell Range Only

Formulae that combine a Quick Selection function such as LEFT, RIGHT, ABOVE, and BELOW with a field name only calculate field names in continuation from the current cell.

For example, typing SUM(ABOVE:EX) in the no.5 cell of the following table only calculates cells 3 and 4, which are continued from the current cell.

  1)EX  
     
  3)EX  
  4)EX  
  5)Current Cell
