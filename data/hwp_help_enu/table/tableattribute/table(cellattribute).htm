Table/Cell Properties: Cell
 

Table > Table/Cell Properties

From the Cell tab of the Table/Cell Properties dialog box, you can adjust the height and width, inside margins, vertical alignment position, cell protection setting, title cell setting, and other properties of the current cell or cells within a cell block.

Steps

  1. Define as a cell block the cells whose properties you want to specify, and click Table > Table/Cell Properties.
  2. From the Table/Cell Properties dialog box, select the Cell tab.
  3. Select Preferred Cell Margins and specify left/right and top/bottom margin settings.
  4. Click Set to apply the margin settings.

More Information

Cell Tab of Table/Cell Properties Dialog Box

Preferred Cell Size

Specify the Width and Height settings for the selected cell. Cell size settings apply to the entire row and column. In other words, clicking Preferred Cell Size and specifying the Width and Height settings does not mean the changes apply to the selected cell only. The specified width and height settings are applied to both the row and column on which the selected cell is located.

You cannot specify cell size settings that are smaller than the current settings.

Preferred Cell Margins

Select Preferred Cell Margins and specify the inside cell margins. The margin value specified in Cell Margins is ignored, and the Preferred Cell Margins setting is applied.

Properties
  • Vertical Alignment

  • Specify whether to align the cell content to the top, bottom, or middle.

    Icon Name Description
    Top Aligns to the top of the cell.
    Center Aligns to the middle of the cell.
    Bottom Aligns to the bottom of the cell.
  • Vertical Writing

  • Arranges the content inside the cell so that it is arranged from top to bottom.

    Icon Name Description
    English Down Set English and numbers to be laid down when text is entered vertically.
    English Up Set English and numbers to be put up when text is entered vertically.
  • Single Row

  • The height of a cell is adjusted automatically to contain text typed into the cell without going over to another cell or into the body. With Single Row selected, the cursor stays on the same row even if it touches the cell border and keeps the entire text on a single row.

    While editing a document, you sometimes need to change the information inside cells without changing the shape of the table. If the replacement text is too long, however, the cells are resized automatically and the shape of the table is changed. To prevent the table from resizing, go to Table > Table/Cell Properties and select the Fix Size option. The downside, however, is that text that is longer than the size of the cell is not shown in its entirety. As an effective solution, use the Single Row option.

  • Cell Protection

  • To protect selected cells in a document form from being edited, use the Cell Protection option. The cursor cannot be placed inside cells that have been protected in this manner.

    To deactivate the protection, set an unprotected cell as a cell block and move it to the protected cell. From here, go to Table > Table/Cell Properties and unselect Cell Protection.

  • Title Cell

  • If a table spans over multiple pages, select cells and set the Title Cell option to display the selected cells on each of the pages. Title Cell must be selected in order for the Repeat Header Row Automatically option under the Table tab to function.

    However, the first row of the table must be included when using the Title Cell option. If you apply the Title Cell option to the second row without including the first row, the Repeat Header Row Automatically option will not function.

Field
  • Field Name

  • Assigns names to cells so that external programs can read and write the Hwp document (DDE). By assigning field names, you can use the Formula feature more effectively.

  • Enable Editing in Form Mode

  • The body of a document that has been opened in form mode cannot be edited. However, Field and Cell entries made with this option selected can be edited in form mode. This feature is used to control Hwp from an external program.

Note

Copying Cell Properties

By clicking Home > Format Painter from inside a table, you can copy the properties of the cell at the cursor position including font and paragraph styles, as well as Preferred Cell Margins, Vertical Alignment, Title Cell, Single Row, and Vertical Writing settings.
