Table/Cell Properties: Margin/Caption
 

Table > Table/Cell Properties

Show All

From the Margin/Caption tab of the Table > Table/Cell Properties dialog box, you can specify the margin between the table and the body and insert a caption in to the table.

More Information

Margin/Caption Tab of Table/Cell Properties Dialog Box

Distance from Text
  • Left

  • Adjust the distance between the left edge of the object and the body.

  • Right

  • Adjust the distance between the right edge of the object and the body.

  • Top

  • Adjust the distance between the top edge of the object and the body.

  • Bottom

  • Adjust the distance between the bottom edge of the object and the body.

  • All

  • Click All to specify the margin between the left, right, top, and bottom edges and the body, all at once.

Caption
  • Insert Caption

  • Caption content will be inserted outside the borders of the object.
    Choose from Top, Bottom, Upper Left, Center Left, Lower Left, Upper Right, Center Right, and Lower Right as the caption position.

  • Size

  • With the caption position set to Top or Bottom, caption size remains the same as the width of the object. With the caption position set to Left or Right, however, you can specify the size of the caption.

  • Distance from Object

  • Specify how much distance to keep between the object and caption content. The default value is 3mm. Hwp supports negative distance from object values. If necessary, you can specify a negative value for Distance from Object.

  • Include Margin

  • Even if outside margin is set for the object, the caption starts from the object area excluding the outside margin part. Selecting this option places the caption starting from the left or right outside margin.

  • Single Row

  • Selecting this option keeps the entire caption on a single line, even if the cursor reaches the right border while typing the caption.

Note

Relationship with Table

Outside margin is the distance between the table and the in-body text. Therefore, it does not affect distance between the table and another table that has not been set as In Line with Text.

Wrapping Style

Outside margin only applies to tables that have been set as In Line with Text or whose wrapping style has been set as Square or Top and Bottom. Outside margin does not apply to tables whose wrapping style has been set as In Front of Text or Behind Text.

Outside Margin and Caption

Even if outside margin is set for the table, the caption starts from the object area excluding the outside margin part. Selecting Caption > Include Margin starts the caption from the left or right outside margin.
