Table/Cell Properties <P>
 

Table > Table/Cell Properties

You can specify the size, location, margins, and borders of a table inserted in the body.

Steps

To set table properties

  1. Click Insert > Table > Insert Table, specify the number of rows and columns, and click Insert to insert a table.
  2. Select the table and press the P key or click Table > Table/Cell Properties.
  3. From the Table/Cell Properties dialog box, select the Background tab.
  4. Under Gradation, set Beginning Color to white and Ending Color to black and then click Set.
  5. The gradation background is applied to the table.

To set cell properties

  1. Click Insert > Table > Insert Table, specify the number of rows and columns, and click Insert to insert a table.
  2. Type Hancom in the first cell.
  3. Click Table > Table/Cell Properties.
  4. From the Table/Cell Properties dialog box, select the Cell tab.
  5. Select Preferred Cell Margins, set the top and bottom margins to 5.00mm, and click Set.
  6. The margins for the cell containing Hancom are increased.
