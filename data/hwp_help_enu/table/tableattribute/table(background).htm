Table/Cell Properties: Background
 

Table > Table/Cell Properties

Show All

From the Background tab of the Table > Table/Cell Properties dialog box, you can fill the table background with a color, pattern, gradation effect, or picture.

More Information

Background Tab of Table/Cell Properties Dialog Box

No Fill

Select No Fill when not applying Color, Gradation, or Picture Fill.

Color
  • Face Color

  • You can click the color table to display the color palette, and then select a background color from the color palette.

  • Pattern Color

  • If Pattern is selected, click the color table in Pattern Color to display the color palette. In the color palette, select a pattern color to fill the background with.

  • Pattern

  • From the Pattern list, select a pattern to fill the background with.

Gradation

Fill the background with gradation for gradual color spread.

  • Beginning Color
    Click the color table to display the color palette. In the color palette, select a color that spreads from the center of the object to the outside.

  • Ending Color

  • Click the color table to display the color palette. In the color palette, select a color that comes into the center from the outside of the object.

  • Type

  • Select a style you want from various gradation templates. By selecting Custom, you can create a type or style you want.

  • H Center

  • Display the position of the horizontal center of the object in percent.

  • V Center

  • Display the position of the vertical center of the object in percent.

  • Angle

  • Specify how many degrees to rotate gradation.

  • Blur Level

  • Specify blur level for the beginning color and the ending color. Level 0 means no blur effect. Blur level can be set from 0 through 255.

  • Blur Center

  • Specify spread level for the beginning color and the ending color. Level 0 means no face color. Level 2 only displays the beginning color and the ending color. This can be set from 0 through 100.

Picture

Open a picture file and fill the background with the picture.

  • Picture File

  • By clicking Select Picture, open a picture file to fill the background with.

  • Embedded

  • Include the picture file in the document as an inserted picture without connecting it to the document. If you select this option, the modification or position change of the original picture file will not affect the picture included in the document.

  • Fill Type

  • When filling the background with a picture, select Tile, Fit to Size, or Center from the list for fill type.

  • Effects

  • Apply grayscale or black and white to the opened picture.

  • Watermark Effect

  • Apply 70% brightness and -50% contrast to the opened picture file to change it to a bright and low-contrast picture.

  • Brightness

  • Adjust the brightness of the selected picture file. Brightness can be set from -100 through 100.

  • Contrast

  • You can adjust the contrast of the selected picture file. Contrast can be set from -100 through 100.
