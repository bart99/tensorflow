Quick Formula
 

Table > Quick Formula

You can calculate the horizontal sum, vertical sum, horizontal average, vertical average, horizontal product, and vertical product of cells adjacent to the current cell by using Table > Quick Formula. Place the cursor on the cell into which you want the calculation result to be inserted and select the desired formula.

Note

One Formula per Cell

Quick Formula inserts one formula per cell. For example, clicking Table > Quick Formula > Horizontal Sum on a cell already containing a horizontal sum result recalculates the horizontal sum. Conversely, clicking Table > Quick Formula > Horizontal Average erases the horizontal sum value and replaces it with the horizontal average formula.
