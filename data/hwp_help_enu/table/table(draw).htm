Draw Table
 

Insert > Table > Draw Table / Table > Draw Table

You can create a table, add rows and columns to a table, draw diagonal lines, change the line type, and more with just the mouse by using Draw Table. This allows you to create even complex tables with ease.

Steps

  1. Click Insert > Table > Draw Table or Table > Draw Table.
  2. Move the mouse pointer over to the edit window, and it changes to the pencil icon .
  3. Point the mouse to where you want to insert a table and drag to draw a rectangular table.

  4. Place the mouse pointer on the left border of the table, then click and drag to the right border to create rows.

  5. Place the mouse pointer on the top border of the table, then click and drag to the bottom border to create columns.

  6. After drawing a table, press ESC or left click once on the outside of the table to exit from Draw Table.

Note

Line Style

When using Draw Table to create a table, the line type selected in Line Style is applied.

Data Entry in Draw Table State

After drawing a table using Draw Table, you can enter information into the table, move the cells, and edit the table while the mouse pointer remains as the pencil icon. This means you do not have to press ESC and exit from Draw Table.
