Distribute Rows <W>
 

Table > AutoFit > Distribute Rows

Distribute Rows applies the same width value to cells of disparate widths in a table.

Steps

  1. Set the cells you want to make the same width as a cell block.
  2. Click Table > AutoFit > Distribute Rows.
  3. The cells in the cell block are turned into the same width.

Caution

Restriction

In order to use Distribute Rows, the cell block must be rectangular. Distribute Rows will not work on a discontinuous or non-rectangular cell block.
