Split Table
 

Table > Split Table

If a table you created extends over to the next page, you can click Table > Split Table at the appropriate location to split the table into two or more tables. Split Table splits the table beneath the row at the cursor position.

Steps

  1. Click Insert > Table > Insert Table and create a table made up of 5 rows and 5 columns.
  2. Place the cursor in any cell on the 3rd row and click Table > Split Table.

  3. The table is split after the 3rd row.

Note

Table Properties

If you split a table into two or more tables, Properties of the original table are applied to the new tables.

Split Table Automatically

Hwp automatically splits tables that do not fit on a single page.

If a table whose At Page Break setting is set to Remove Split or a table set to In Line with Text extends over to the next page, click Table > Split Table at the appropriate location and split the table into two or more tables.

Caution

Restriction

If you perform Split Table on the first cell, Cannot run Split Table in the first row. is shown. Split Table is not performed.
