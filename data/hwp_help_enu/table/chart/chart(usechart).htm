Using Charts
 

Insert > Illustrations > Chart

Chart is a feature that organizes variations in data using visual graphs. With Hwp, you can create charts with ease by clicking Insert > Illustrations > Chart.

When you are in chart edit mode, you can design the chart by specifying the chart style, data entry, chart title, footnotes, legends, and axis display options.

Steps

To set chart legend

  1. Right click on the chart and select Legend Styles from the context menu.
  2. In the Legend Styles dialog box, select the Position tab.
  3. Select Show and then click Position > Center Left.
  4. Select the Background tab and then Color.
  5. Click on the color table next to Face Color and select the desired color.
  6. Click Apply, check the preview of the settings, and then click Set to exit from the dialog box.

More Information

Chart Screen Layout

When you are in chart edit mode, right click to access the context menu or double-click on the desired part of the chart to modify the chart area, chart location, background, series, axes, legends, title, and footnote properties.

Name Description
General Change general chart settings including background, background lines, and chart composition.
Chart Change the format, series, position, and background of the chart.
Series Change series, series label, data point, and data point label.
Axis Change axis, axis label, and title.
Legend Set the font, background, and position of the legend.
Footnote Set the style, background, font, and position of the footnote.

Note

Chart Type Selection

Hwp provides 2-D and 3-D chart types. In addition, you can specify the style of individual charts.

Set Chart Format

Double-clicking on a chart component, such as chart format, title, footnote, legend, or any other part you wish to change, opens up the corresponding dialog box. Alternatively, right clicking while in chart edit mode opens the context menu. You can use the context menu to change the chart format.

Adjusting Size

Select a chart that has been inserted into the body and place the mouse pointer over a round control point of the chart. The mouse pointer then changes to the size adjustment icon . In this state, you can simply drag the mouse to adjust the size of the chart.

Click on a point in the middle of a border and drag to change the size in the selected direction only. Clicking on a corner point and dragging adjusts the size in vertical and horizontal directions.

Rotation

If you select a 3-D chart, place the mouse pointer near a square control point, and drag on the point while holding down the CTRL key, X-axis, Y-axis, and Z-axis arrows are shown. In this state, you can rotate the orientation of the chart in 3-D.
