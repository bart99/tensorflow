Chart
 

Insert > Illustrations > Chart

Chart is a feature that organizes variations in data using visual graphs. With Hwp, you can select an table or set a section of a table as a cell block and produce charts with ease.

Steps

To create a chart using the menu

  1. Click Insert > Illustrations > Chart or click the Chart icon in the Insert tab.
  2. A 3-D column chart is inserted into the edit window.
  3. Select the inserted chart and right click to access the context menu. Click Title Styles.
  4. From the Title Styles dialog box, select the Text tab and type Report Card in the Contents field.
  5. From the Position tab, select Show, click Position > Bottom, and then click Set.
  6. Select the inserted chart and right click to access the context menu. Click Legend Styles.
  7. From the Legend Styles dialog box, select the Position tab, select Show, click Position > Bottom, and click Set.
  8. To close Create Chart, click outside the chart area.

To create a chart using table data

  1. Create a table to create a chart from.
  2. Press F5 three times from inside the table to set a cell block from the very first cell of the table to the very last.
  3. Click Insert > Illustrations > Chart or navigate to the Table group under the Insert tab and click on the Chart icon.
  4. A 3-D column chart based on the data of the selected table is inserted into the edit window.
  5. To close Create Chart, click outside of the chart area.

Note

Linking to Current Table

Selecting a table or placing the cursor inside one and then clicking Create Chart produces a chart using the default settings of Hwp.

Setting an entire table or a portion of a table as a cell block and then clicking Create Chart produces a chart that is linked to the data set from the selected cell block.

Leaving First Cell in Table Empty

Data that exists in the first cell on the far left of the table used to create a chart is not reflected in the chart. Since the first cell on the far left is the corner where the two axes meet, it should be left empty.
