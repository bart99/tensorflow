Insert Table
 

Insert > Table > Insert Table/Table > Insert > Insert Table

You can use a table for organizing complicated content or numbers in a document.

Insert Table in Hwp enables you to create a table inside a table and automatically divides tables that go over a page.

Steps

To create a table using the mouse

You can create tables from the toolbar, without having to specify the number of rows and columns separately.

  1. From the Insert tab, click on the icon underneath Insert Table to expand the table grid.
  2. Drag on the table grid until the desired number of rows and columns is selected.

    The number of selected rows and columns is displayed at the top of the grid in the rows ¡¿ columns format.

  3. Left click on the mouse.
  4. When a table is inserted, the pointer is on the first column of the first row.

More Information

Insert Table Dialog Box

Rows/Columns

Specify the number of rows and columns of a table. The maximum number of rows and columns of a table is 128 each.

However, it does not mean that you cannot make a table of 129 or more rows and columns. When making a table, you can adjust the number of rows and columns within 16,384 cells. For example, if you select 1,600 for the number of rows, you can only select up to 10 for the number of columns making the total of 16,000 cells. This means that you cannot make a table with 16,384 or more cells.

Specify Size

To specify the size of the table, the Create by Dragging option under Others must not be selected.

  • Width
    • Fit to Column

    • Create a table that fits the width of the column at the current cursor position.

    • Align to Paragraph

    • Create a table that fits the width of the paragraph at the current cursor position.

    • Random Value

    • Create a table by specifying the width.

  • Height
    • Automatic

    • Create a table equaling the size of the standard cell height multiplied by the number of rows.

    • Random Value

    • Create a table with specified height.

Others

Specify whether to set the new table as In Line with Text, which handles placement of the table like it were text, and to enable Create by Dragging, which enables you to set the location and size of the table using the mouse.

  • In Line with Text

  • Treats the table like normal text. Therefore, the table position changes as you type or delete text.

  • Create by Dragging

  • Select this option to create tables at the designated locations by dragging, instead of always inserting the table at the current cursor location.

    If it is left unselected, a table that fits the width of the column at the current cursor position is created.

Templates Button

Open the Quick Tables dialog box from which you can select table formats to create new tables or switch the style of an existing table.

Properties Button

Open the Table/Cell Properties dialog box from which you can use predefined table border and background settings to create new tables.

Note

Tables Toolbar

The Table tab is displayed automatically if the cursor is placed inside an existing table. From the toolbar, you can perform basic table editing tasks including create a table, change table/cell properties, split cells, merge cells, and distribute rows/columns.

If you close the table toolbar after it is displayed, it will not viewable even if you place the cursor inside the table.

Moving Table Position

If you click inside a table while pressing ALT, the pointer changes to a four-headed arrow . You can hold a spot inside of the table and drag it to a position you want. You can move a table past page boundaries.

Creating Table inside Table

Click Insert > Table > Insert Table from within a table to create and edit a new table.

If the table inside the active table is bigger than cells of the active table, the rest is hidden. You can adjust cell size according to the size of the table inside the active table.

Minimum Table Size

If you create a table using Random Value settings for width and height and those values are too small for the number of rows and columns specified, a table whose width equals the minimum cell width multiplied by the number of columns and whose height equals the minimum cell height multiplied by the number of rows is created.

Specifying Table Origin Using the Mouse

To enable table location and placement selection using the mouse, click File > Options > Object.

Overlapping Tables

Two tables for which the Wrapping Style is set to Top and Bottom cannot be overlapped. To enable table overlapping, select the tables, click Table/Cell Properties > Basic, and select Position > Allow Overlap.
