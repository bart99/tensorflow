Quick Tables
 

Insert > Table > Quick Tables / Table > Quick Tables

Quick Tables contains table formats featuring a wide array of border and background styles. By using Quick Tables, you can create new table using appropriate styles or load existing table styles.

Steps

  1. Place the cursor inside a table and click Insert > Table > Quick Tables.
  2. In the Quick Tables dialog box, select the desired table from Table Templates List and click Set.
  3. The line type and cell settings of the selected table style are instantly applied to the selected table.

More Information

Quick Tables Dialog Box

Table Templates List

Quick Tables contains more than 70 table styles including Normal Style, Color Fill, Vertical Pattern, Horizontal Pattern, Accentuated Rows, Accentuated Columns, Embossed, and Engraved.

Select a format from Table Templates List, and the selected format is shown in the Preview window.

Format to Apply

Specify which of the Border, Font/Paragraph, Cell Background, and Grayscale settings of the format selected from Table Templates List to apply to the current table. The style of the selected table is maintained for settings that are not selected.

Selecting Grayscale changes the line color and cell background color of the selected Quick Table to equivalent grayscale.

Apply to

From Header Row, Last Line, First Column, and Last Column, select which of the options to apply the selected table format to. The style of the selected table is maintained for settings that are not selected.

Note

Creating a Table using Quick Tables

You can create a table and change the style using Quick Tables. Alternatively, you can specify the border and background styles first and then create a table. Click Insert > Table > Insert Table. From the Insert Table dialog box, click Templates and select the desired table format.

Preview

The preview area of Quick Tables shows a preview of the selected preview format. It provides a preview of the selected format with all styles applied, all styles except line type applied, or with only the grayscale setting applied.

Undo

To undo application of a Quick Tables format, click Home > Undo.

Caution

Restrictions

Quick Tables can only be used if the cursor is placed inside a table or the entire table is selected. Quick Tables cannot be used if a cell block has been set.

New items cannot be added to the Quick Tables list

You cannot register new table styles to Quick Tables. What's more, you cannot edit or remove registered Quick Tables formats. If you have formats that you frequently use, we recommend you register them in AutoText.
