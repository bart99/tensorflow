Thousands Separator
 

Table > Thousands Separator

It is used to place a comma after each group of three digits for numbers contained inside a table.

Selecting Thousands Separator > Insert Separator inserts a separator (,) after each group of three digits. To remove all number separators, click Thousands Separator > Remove Separator.

Steps

To insert separator

  1. Click Insert > Table > Insert Table.
  2. From the Insert Table dialog box, specify the number of rows and columns and click Insert.
  3. Type numbers that are in the thousands into the newly created table.
  4. Place the cursor inside the table and set the entire table as a Cell Block by pressing F5 three times.
  5. Click Table > Thousands Separator > Insert Separator.
  6. A separator (,) is inserted after each group of three digits in all cells containing a number within the cell block.

To remove separator

  1. Click Insert > Table > Insert Table.
  2. From the Insert Table dialog box, specify the number of rows and columns and click Insert.
  3. Type numbers that are in the thousands into the newly created table.
  4. Place the cursor inside the table and set the entire table as a Cell Block by pressing F5 three times.
  5. Click Table > Thousands Separator > Insert Separator.
  6. A separator (,) is inserted after each group of three digits in all cells containing a number within the cell block.
  7. Set the entire table as a Cell Block by pressing F5 three times again.
  8. Click Table > Thousands Separator > Remove Separator.
  9. All the separators (,) that have been inserted are removed.

Note

Thousands Separator

Selecting Thousands Separator under Table > Formula inserts a separator (,) after each group of three digits for numbers containing the calculation result.

Set Area

Area setting when inserting separators after each group of three digits works only when a Cell Block has been set by pressing F5. Insert Separator does not work properly if the cursor is placed outside the table and F3 or F4 is pressed to set a cell block or an area.

No Separator

  1. A continuing series of numbers that is 3 digits or less in length
    EX) 200
  2. Numbers proceeding the decimal point
    EX) 200.55555
  3. The 4-digit year portion of Date/Time Codes
    EX) 9/10/2010
  4. A number made up of 4 or more digits and inserted between Date/Time Code fields
