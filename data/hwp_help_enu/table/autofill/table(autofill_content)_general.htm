AutoFill Content: Default
 

Insert > Fill > AutoFill Content

To insert data at the pointer position, select data from the default AutoFill list.

Steps

  1. Click Insert > Fill > AutoFill Content.
  2. From the AutoFill Contents dialog box, select the Default tab.
  3. From the AutoFill List, select jan,feb,mar,apr,may,jun,jul,aug,sep,oct,nov,dec ,and then click Fill.
  4. jan,feb,mar,apr,may,jun,jul,aug,sep,oct,nov,dec is filled in the cells below the current cursor location.
