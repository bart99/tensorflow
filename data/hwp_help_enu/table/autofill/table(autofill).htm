Table AutoFill
 

Insert > Fill > Table AutoFill

This feature is used to analyze the rule of some cells of a table and automatically fill all selected cells in accordance to the rules.

Steps

  1. Click Insert > Table > Insert Table and create a table containing 5 rows and 5 columns.
  2. In the first cell of the table, type 1, press the TAB key to move the cursor to the second cell, and then type 2.
  3. Set the entire table as a cell block and then click Insert > Fill > Table AutoFill.
  4. The empty cells beginning from the third cell, are automatically filled with a sequence of numbers ranging from 3 to 25.

Note

Fill Rule

The AutoFill rules are as follows:

  • If multiple rows and columns are selected as a cell block, AutoFill is performed on the basis of the higher number of rows or columns. If there is an equal number of rows and columns in the block, AutoFill is performed on the rows first.

  •   EX) A block with 3 rows and 4 columns ¡æ Fill columns

    A block with 4 rows and 3 columns ¡æ Fill rows

    A block with 3 rows and 3 columns ¡æ Fill columns

  • You can fill all the selected cells in accordance with the rules. AutoFill first fills the empty cells in the first row or column, and if the first cell on the next row or column is empty, continues to fill the cells in the next row or column according to the rules specified in the previous row or column.

  • If there is an empty section in the middle of the cell block, AutoFill finds rules from cells preceding the first empty space and overwrites the other cells.

  • If only the first cell contains a number, AutoFill fills all other cells in the block with the same value.
  • If there are numbers and characters, AutoFill fills the selected cells with an available match from the AutoFill list. If there is no match available in the list, and a numerical rule is identified, AutoFill fills the remaining cells according to the rule, but retains the characters before and after the number.

Repeat Fill

If there are no rules, AutoFill fills the entire cell block with the entered text strings. The properties (font, paragraph style, cell format, and line type) of the entered cell are applied to all the remaining cells.

No AutoFill

You cannot perform AutoFill if the first cell of the block is empty, if all the numbers of cells in the block do not match, and the block is not continuous.

Adding Custom Data

You can add custom data to the AutoFill Content available by default.

To add custom data, click Insert > Fill > AutoFill Content, and select the Custom tab from the AutoFill Contents dialog box.

Advanced Usage

Create Label Template

To create a label template for printing multiple copies of identical content, use the AutoFill feature as follows:

Click Review > Labels > Create Label Template, select a label paper, and type data into the first cell only. Select the labels to fill with the data, and click Insert > Fill > Table AutoFill.
