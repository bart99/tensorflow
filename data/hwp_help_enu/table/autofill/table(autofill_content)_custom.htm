AutoFill Content: Custom
 

Insert > Fill > AutoFill Content

Check the AutoFill custom list that you registered, and add a new list or modify an existing AutoFill list.

You can also save entered data as an AutoFill saved file (*.dat) for other computers or load existing AutoFill saved files.

Steps

To add Custom Data

  1. Click Insert > Fill > AutoFill Content.
  2. In the AutoFill Contents dialog box, select the Custom tab.
  3. In the Custom List, select (New), and enter the title in the Title field.

    For example, type Seasons in the Title field.

  4. In the Content field, enter the following data. You can press ENTER after each word to separate the data.

    Spring

    Summer

    Autumn

    Winter

  5. Click Add after the data entry is complete.
  6. The new AutoFill data is registered under Custom List.

To apply AutoFill by using Custom Data

  1. Click Insert > Fill > AutoFill Content.
  2. In the AutoFill Contents dialog box, select the Custom tab.
  3. Under Options, check Show AutoFill List in the Menu, and then click Close.
  4. In the edit window, position the cursor where you need to insert data, and click Insert > Fill.
  5. From the AutoFill list under Fill, select 1 Spring, Summer, Autumn, Winter.
  6. The custom data is automatically entered downwards starting from the cursor location.

More Information

Custom Tab of AutoFill Contents Dialog Box

Custom List

You can display a list of custom AutoFill data titles and contents.

You can also add, edit or delete custom data, and import or export AutoFill saved files (*.dat).

Details on Icons

Icon Name Description
Delete AutoFill List You can delete selected custom data from Custom List.
Move Up You can select a data from Custom List, and then click the icon to move the selected data to the previous row.
Move Down You can select a data from Custom List, and then click the icon to move the selected data to the next row.
Import AutoFill List You can import a saved AutoFill saved file (*.dat).
Save AutoFill List You can back up or save the AutoFill data that is defined in the current document as an AutoFill saved file (*.dat) for use on other computers.

Title

To enter new custom data, select (New) from the Custom List, and then enter the title in the Title field.

Contents

After entering the title of the new custom data in Title, enter the content of the data in the Contents field. You can press ENTER after each entry to separate them.

Add Button

You can add the new AutoFill data with the defined Title and Contents information to the Custom List.

Edit Button

In the Custom List, select the item you need to edit, and click Edit to load the information in Title and Content fields. The Edit button changes into the Apply button.

Edit the title and content, and click Apply to update the Custom List. The Apply button changes back to the Edit button.

Options

  • Show AutoFill List in the Menu

  • If you check this option, it displays Custom List data when you click Insert > Fill for quick selection of custom AutoFill data. You can specify the number of AutoFill data to display from 1 through 9.

Note

Default AutoFill File Path

If you have not modified AutoFill data, data from the autofill.dat file which is located on the default path, is displayed under AutoFill List in the Default tab.

\Program Files\HNC\hwp80\autofill.dat

If you add or delete custom data from the Custom tab, Autofill.dat is copied to the following path.

C:\Documents and Settings\Custom ID\Application Data\HNC\User\Shared80\Dics\Autofill80.dat

To check the locations of Hwp program folder and user data folder, click on , and then click About Thinkfree Office NEO Hwp > Folder Info.
