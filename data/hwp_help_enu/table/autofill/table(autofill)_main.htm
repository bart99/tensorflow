Insert > Fill

Table AutoFill searches for the rules specified in some cells of a table and automatically fills all selected cells in accordance to the rules.

You can insert large volumes of predefined data into the cursor position and easily complete tasks that require repeated entry of identical data by using AutoFill Content. You can also save entered data as an AutoFill saved file (*.dat) for use on other computers or load existing AutoFill saved files.
