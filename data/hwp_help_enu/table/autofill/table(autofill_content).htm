AutoFill Content
 

Insert > Fill > AutoFill Content

To insert data at the pointer position, select data from the default AutoFill list.

Check the AutoFill custom list that you registered, and add a new list or modify an existing AutoFill list.

More Information

AutoFill Contents Dialog Box

Default Tab

Default tab displays a list of basic AutoFill data available with Hwp.

Custom Tab

You can create and add custom AutoFill data from the Custom tab.

You can also edit or delete custom AutoFill data, and import or export AutoFill saved files (*.dat).
