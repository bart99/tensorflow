Table Border/Backgrounds
 

Table > Cell Border/Background

You can change the outline type for a table that has been set with a Cell Spacing value and specify a background color or insert a background picture for the entire table.

Note

Difference between Cell Border and Background

Table Border/Background settings are applied on the entire table. The option should therefore be used if you wish to create a background for the entire table.

In contrast, Cell Border/Background settings apply to individual cells only. Setting a cell block and specifying Cell Border/Background settings applies the settings to individual cells and not over the entire cell block.
