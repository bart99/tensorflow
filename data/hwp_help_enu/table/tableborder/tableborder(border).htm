Table Border/Background: Borders
 

Table > Cell Border/Background

Show All

You can change the type of the outline of a table with a set cell spacing value and specify which borders to show or hide.

Steps

  1. Place the cursor inside a table and click Table > Cell Border/Background > Apply to Each Cell.
  2. From the Cell Border/Background dialog box, select the Borders tab.
  3. Click Table Border/Backgrounds.
  4. From the Table Border/Backgrounds dialog box, select the Borders tab.
  5. Click on the Borders Type list button and select the triple line, which is located at the very bottom. Weight is automatically set to 2 mm.
  6. In the Preview Line Style area, click All to apply the selected border style to all outside cell borders, and click Set.
  7. Click Set once more to open the Cell Border/Background dialog box.

More Information

Borders Tab of Table Border/Backgrounds Dialog Box

Borders
  • Type

  • Select the desired cell border type from the 12 available options.

  • Weight

  • Specify the cell border weight.

  • Color

  • Click the color table to display the color palette and then select a color for the cell border.

  • Preview Line Style

  • Selecting this option immediately applies the border style in the preview area, on the selected sides.

    If you do not select this option, you have to click Borders for the direction to apply a border style after selecting the style so that you can change border styles.

  • Preview Area

  • Specify the border type, weight, and color and specify which sides to apply the style to using Left, Right, Top, Bottom, or All icons from the preview area. This applies the style on the selected sides.

Apply to

Specify the range of border style application.

  • All Cells

  • Selecting All Cells applies cell border and background properties to all cells in the active table.

  • Selected Cells

  • Applies the border and background properties to the selected cells only. If a cell block has been set, Selected Cells is provided as the default value.

Note

Line Style

There are 11 line types available for the table border. In addition, you can specify the weight and color for each border type selected. Line weight can be specified from the list only. Custom weight value is not supported.

Change Borders

Select line type, line color, and line weight to change the border of the side you want. Select the desired border sides in the preview area, and the borders on the selected sides begin to toggle between current style and previous style.

Adjust the Border of Each Side

You can adjust the border of each side in the preview area for borders. Click on the left/right, top/bottom, and outside border buttons, and the borders on the selected sides begin to toggle between current style and previous style.

Preview Line Style

Selecting this option immediately applies the border style in the preview area, on the selected sides.

If you do not select this option, you have to click Borders for the direction to apply a border style after selecting the style so that you can change border styles.

Cell Spacing

Table Border is only shown on the outline of a table that has been set with a Cell Spacing value. Therefore, in order to show the table border, click Table > Properties, open the Table tab, and set an appropriate value for Cell Spacing under Others.

Caution

Applying Table Borders

The table border style specified in Properties > Borders is applied only when a value is entered for Properties > Cell Spacing.
