Distribute Columns <H>
 

Table > AutoFit > Distribute Columns

The height of the cells can change as you edit a table using options such as Split Cells and Merge Cells. In this case, use Distribute Columns to set all the cells to the same height.

Steps

  1. Set the cells you want to make the same height as a cell block.
  2. Click Table > AutoFit > Distribute Columns.
  3. The cells in the cell block are turned into the same height.

Caution

Restriction

In order to use Distribute Columns, the cell block must be rectangular. Distribute Columns will not work on a discontinuous or non-rectangular cell block.
