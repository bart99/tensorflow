Paste Cell <CTRL+V>
 

Home > Paste

You can paste a table that has been copied or removed by using the Copy or Cut command at the current cursor position in the specified direction.

Steps

  1. In a table, set the cells you wish to copy as a cell block.
  2. Click Home > Copy.
  3. Move the cursor to the cell into which you wish to paste the copied content and click Home > Paste > Paste.
  4. From the Paste Cell dialog box, select Overwrite Content and click Paste.
  5. Contents of the copied cells are pasted in.

More Information

Paste Cell Dialog Box

Paste

  • Icon Name Description
    Top Inserts the table that has been copied or removed using the Copy or Cut command to the top of the current cursor position.

  • Overwrite
  • Icon Name Description
    Overwrite Pastes the cell style (properties, borders, and background) and content of the table that was copied or removed using the Copy or Cut command to the cell at the current cursor position.
    Overwrite Content Pastes only the content of the table that was copied or removed using the Copy or Cut command to the cell at the current cursor position.
    Insert Table into Cell Pastes the table that was copied or removed using the Copy or Cut command to the cell at the current cursor position as a table.

Note

Paste to Body

You can paste a table that was copied or removed using the Copy or Cut command into the body of a normal document at the current cursor position using the Paste command.

Cut Cells from Table

If a defined cell block is rectangular and includes entire rows or columns, selecting the Cut command cuts the contents of the selected cells and the selected cells as well.

When prompted with the "Cut the selected cells. Leave the cell format?" message, select Leave to cut the contents of the cells only. Selecting Delete cuts out the cells of the cell block and changes the layout of the table.

If the cell block does not include entire rows or columns, performing the Cut command removes the cell contents only and does not prompt the user for confirmation.
