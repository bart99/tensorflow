Split Cells <S>
 

Table > Split Cells

This feature is used to split the cell at the current cursor location or a group of cells set as a cell block into multiple cells.

Steps

  1. Place the cursor inside the cell you wish to split and click Table > Split Cells.
  2. From the Split Cells dialog box, specify how many rows and columns to split the cell into. The selectable range is 2 through 63.
  3. Click Split.
  4. The current cell is split into the specified number of cells. Properties of the initial cell are applied to the new cells.

More Information

Split Cells Dialog Box

Split Rows/Columns

  • Rows

  • Specify the number of rows to split the cell into by typing in a number or using the increase and decrease buttons.

  • Columns

  • Specify the number of columns to split the cell into by typing in a number or using the increase and decrease buttons.

Options

  • Split Rows Evenly

  • To have the new rows the same height, select Split Rows Evenly and then split.

  • Merge and Split Cells

  • To set a group of cells into a cell block and perform Split Cells, you can specify whether to split individual cells or to merge the cells into one and then split it. Selecting this option merges the cells into one and then splits it into the specified number of rows and columns.

Note

Split into 2 Rows or 2 Columns

By using the Split Cell into 2 Rows/Columns icon under the Table tab, you can split the current cell or cells in a cell block into two rows or columns without having to open up the Split Cells dialog box.

Split Rows/Columns Simultaneously

By selecting both Rows and Columns before splitting a cell, you can split it into both rows and columns at the same time. However, you must at least select either Rows or Columns. In other words, you must select either Rows or Columns or both.

Split using Draw Table

Splitting a cell that contains data into two rows using Split Cells enters the data in the first row and leaves the second row empty.

However, you can use Draw Table to split the cell by rows.

Caution

Restriction

In order to use Split Cell, the cell block must be rectangular. Split Cells will not work on a discontinuous or non-rectangular cell block.
