Delete Rows/Columns <ALT+DELETE>
 

Table > Delete > Delete Rows/Columns

You sometimes end up with unnecessary rows and/or columns after editing a table. Therefore, it becomes necessary to remove these extra rows and columns.

Steps

  1. Place the cursor in any cell on the row or column you wish to delete.
  2. Click Table > Delete > Delete Rows/Columns.
  3. From the Delete Rows/Columns dialog box, select Row and then click Delete.
  4. The entire row of the cell in which the cursor is located is deleted.

More Information

Delete Rows/Columns Dialog Box

Delete

Deletes the entire row or column at the current cursor location.

Icon Name Description
Col Deletes the entire column at the current cursor location.
Line Deletes the entire row at the current cursor location.

Note

Cell Block

To use Delete Rows/Columns on a cell block, you must set only one cell as a cell block. Delete Rows/Columns cannot be used if two or more cells are set as a cell block.

Unable to Delete Misaligned Rows/Columns

In order to delete a row or column, the entire row or column on which the cell at the current cursor position is located must be straight with the same width or height throughout.

  • Unable to Delete Columns

  • Unable to Delete Rows

Delete and Undo

If you have used Delete Rows/Columns by accident, click Home > Undo. This reverses the previous Delete Rows/Columns action and restores the deleted row/column.
