Cell Block <F5>
 

Insert > Table > Insert Table

In a table consisting of multiple fields, each field is called a cell. Because a table is made up of numerous cells, cells are the most basic unit when it comes to editing a table.

If you wish to change the font or paragraph style of specific cells in a table, merge or split specific cells, or adjust cell size or margin, you must set the corresponding cells as a cell block.

Steps

  1. Place the cursor inside a cell and press the F5 key. The color of the cell is inversed to indicate its selection as a cell block.
  2. In this cell block state, you can change the size or style of the cell or expand the cell block to include additional cells.
  3. To undo the cell block selection, press the ESC key.

Note

Setting Cell Block with Mouse

Placing the mouse pointer on the cell you wish to set as a cell block and left clicking the mouse while holding down the CTRL key sets that single cell as a cell block.

Place the mouse pointer on the starting cell of the cell block you wish to set and drag while holding down the left mouse button. Release the button, and all the cells selected in the process are set as a cell block.

Placing the cursor inside the starting cell of the cell block you wish to set and left click on the last cell of the desired cell block while holding down the SHIFT key sets all the cells between the two cells as a rectangular cell block.

Setting Cell Block with Keyboard

If you place the pointer inside a cell and press F5, the cell gets selected as a block. Press the F5 key once more or the F3 in this state and then use the arrow keys to select the cells you wish to set as a cell block.

Alternatively, place the cursor inside a cell, press SHIFT+F5, and then use the arrow keys to select the cells you wish to set as a cell block.

Entire Table as Cell Block

Place the cursor inside a cell and press the F5 key three times to set the entire table as a cell block.

Row as Cell Block

If you place the pointer inside a cell and press F5, the cell gets selected as a block. Press the F8 key in this state to set the entire row as a cell block.

Column as Cell Block

If you place the pointer inside a cell and press F5, the cell gets selected as a block. Press the F7 key in this state to set the entire column as a cell block.

Discontinuous Cell Block

Placing the mouse pointer on a cell and left clicking the mouse while holding down the CTRL key sets that cell as a cell block. In this state, move the mouse pointer over to a different cell and left click while holding down the CTRL key to add that cell to the cell block. In other words, you can hold down the CTRL key and select cells to add them to the cell block.

To remove a cell from the cell block, left click on the cell while holding down the CTRL button.

Cell Size Adjustment

With a cell block set, hold down the CTRL, ALT, or SHIFT key and use the arrow keys to adjust the cell size.

Alternatively, place the cursor inside a cell and use the arrow keys while holding down CTRL+ALT to adjust the cell size.

Shortcut

With a cell block set, use the following shortcuts to edit the cell block with ease:

Feature Shortcut
Split Cells S
Merge Cells M
Cell Border L
Cell Background C
Add Rows/Columns ALT+INSERT
Delete Rows/Columns ALT+DELETE
Distribute Columns H
Distribute Rows W
AutoFill A

Advanced Usage

Pressing F5 Once

Pressing the F5 key once and then pressing the arrow keys moves the cell block over instead of expanding it. In this state, you can use PAGE UP, PAGE DOWN, HOME, and END keys to move the cells quickly.

Cell Block and ENTER

Pressing ENTER while a cell block is set removes the cell block selection and places the cursor in the current cell.
