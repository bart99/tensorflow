Resizing Table
 

Insert > Table > Create Table

You can resize tables inserted into the body by using the mouse or the keyboard.

Steps

To resize the table using the mouse

  1. Place the mouse pointer on the bottom cell border. The mouse pointer icon changes.
  2. In this state, left click and drag to resize the table.

To resize the table using the keyboard

  1. Place the pointer inside a cell and press F5 to set the cell as a block.
  2. Press CTRL+ก้ to increase the size of the table.
  3. Press CTRL+ก่ to decrease the size of the table.

Note

Adjust Using the Mouse

Place the mouse pointer on a cell border. The mouse pointer icon , changes. In this state, left click and drag to increase or decrease the size of the table.

Adjust with the Keyboard and the Mouse
  • CTRL+Drag

  • Place the mouse pointer on a table border and drag while holding down CTRL to increase or decrease the size of the cells and the overall size of the table.

  • SHIFT+Drag

  • Place the mouse pointer on a table border and drag while holding down SHIFT to increase or decrease the size of the cell. However, you cannot make the cell smaller than its content.

Adjust Using the Keyboard

After setting a cell block, press CTRL, ALT, or SHIFT along with the arrow keys to adjust the size of the cells.

If a discontinuous cell block has been set, however, using the ALT or SHIFT key only changes the size of the current cell.

  • CTRL+Arrow Keys

  • To change the size of entire rows or columns without affecting the size of other cells, use the arrow keys while holding down CTRL. The size of the table also changes by the amount of change to row or column size.

    Shortcut Description
    CTRL+ก็ Decreases the overall width of the cell columns that have been set as a cell block.
    CTRL+กๆ Increases the overall width of the cell columns that have been set as a cell block.
    CTRL+ก่ Decreases the overall height of the cell rows that have been set as a cell block.
    CTRL+ก้ Increases the overall height of the cell rows that have been set as a cell block.
  • ALT+Arrow Keys

  • To adjust the size of rows or columns containing cells that have been set as a cell block without changing the overall size of the table, use the arrow keys while holding down ALT. The size of the applicable rows or columns changes, and the size of adjacent rows or columns increases or decreases to maintain the original table size.

    Shortcut Description
    ALT+ก็ Decreases the width of all cells in the column and increases the width of the cells immediately to the right.
    ALT+กๆ Increases the width of all cells in the column and decreases the width of the cells immediately to the right.
    ALT+ก่ Decreases the height of all cells in the row and increases the height of the cells immediately below.
    ALT+ก้ Increases the height of all cells in the row and decreases the height of the cells immediately below.
  • SHIFT+Arrow Keys

  • To change the size of cells that have been set as a cell block without changing the overall size of the table, use the arrow keys while holding down SHIFT. A single adjacent cell is resized by the amount of change to the selected cell. The size of all other cells is maintained.

    Shortcut Description
    SHIFT+ก็ Decreases the width of a single cell and increases the width of the cell immediately to the right.
    SHIFT+กๆ Increases the width of a single cell and decreases the width of the cell immediately to the right.
    SHIFT+ก่ Decreases the height of a single cell and increases the height of the cell immediately below.
    SHIFT+ก้ Increases the height of a single cell and decreases the height of the cell immediately below.

Caution

Overall Table Size

Unlike with other objects, you cannot adjust the size of a table using the 8 control points that appear when the table is selected.

Furthermore, you cannot specify the height or width of a table in the Basic tab of Table/Cell Properties. To resize a table, you must adjust the size of the cells.

Fix Size

Tables with a fixed sizes are fixed and cannot be resized.

To fix the size of a table, go to the Basic tab of the Table/Cell Properties dialog box and click Size > Fix Size.
