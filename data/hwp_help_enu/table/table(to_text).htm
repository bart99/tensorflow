Convert to Text
 

Insert > Table > Convert to Text / Table > Convert > Convert to Text

It removes the table borders and converts the content into text.

Steps

  1. Create a table and type in entries.
  2. Place the cursor inside the table and click Table > Convert > Convert to Text.

  3. From the Convert to Text dialog box, select Separate by > Tab and click Set.
  4. The table borders are removed, leaving the text only. The columns are separated by TAB entries and rows by line breaks (ENTER).

More Information

Convert to Text Dialog Box

Separate by

When converting a table into text, the rows are separated by line breaks (ENTER). However, you need to specify the method of separating the table columns (TAB, comma, space, or other text).

  • Tab

  • Separates text from the same row using TAB entries.

  • Comma

  • Separates text from the same row using commas.

  • Spaces

  • Separates text from the same row using spaces.

  • Other Text

  • You can specify which method to use to separate the text from the same row. Open the context menu or press CTRL+F10 and select the separator from Symbol.

Note

Setting as a Block from Outside the Table

To convert a table to text, you must place the cursor inside the table or set the entire table as a block. Here, the block must be set from outside the table. This feature cannot be performed if the entire table has been set as a cell block from inside the table.
