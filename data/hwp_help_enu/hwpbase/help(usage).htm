Introduction to Help
 

For the convenience of users, Hwp provides help options such as basic help, quick help, balloon tips, and status bar tips.

Steps

To run Basic Help

  1. Press F1 or click the Help icon to the left of the Close button .
  2. In Thinkfree Office NEO Hwp Help, click a title in the Contents tab.

    For example, click File > New.

  3. Help content about New feature appears on the right pane.

   To view the subfolders under a title with subfolders in the Contents tab, double-click the title or click the plus icon .

More Information

Types of Help

Basic Help

To view Thinkfree Office NEO Hwp Help, press F1 or click the Help icon in Hwp.

  • Contents Tab

  • Display help topics in the form of a tree. Information is provided in the menu order. You can select a topic you want and look up Help.

  • Index Tab

  • Help contents are sorted in the ascending order of topic titles. In the keyword box, you can type a part of the title to view a list below that automatically changes to what you have typed. You can see if there is information about a title you are looking for.

  • Search Tab

  • If you type a keyword in the Type in the word(s) to search for box and then press ENTER, you get search results in a list below. When you click a title in the list, you can see content that matches the keywords in the right pane.

  • Favorites Tab

  • If you register information you frequently refer to in Favorites, you can access the information faster.

Quick Help

If you right-click an item in a dialog box, quick help about the item appears.

Balloon Tips

When you rest the pointer on top of an icon in a toolbar, task pane, status bar, or image collection, a balloon tip appears and provides details about the icon.

You can activate or deactivate the balloon tips for icons of each toolbar. You can also specify whether the balloon tips for functions should include details or not. To change balloon tip settings, go to Options > Balloon Tips Settings in the Toolbar tab of the Customize dialog box.

  • To set Balloon Tips of toolbars
  • Name Description Reference
    Show with function details Display function details in balloon tips along with icon names and hotkeys.
    Show without function details Display icon names and hotkeys in balloon tips.
    If the icon is labeled with a name or has no hotkey, information will not be displayed.
    Hide Hide balloon tips.

Status Bar Tips

If you rest the pointer on a menu name or an icon on a toolbar, a status bar tip appears at the end of the status bar.
