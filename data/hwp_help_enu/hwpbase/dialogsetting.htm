Dialog Box Settings Configuration
 

You can use Dialog Box Settings Configuration to save settings in a file and select values from Dialog Box Settings List as needed, if settings in certain dialog boxes have standardized patterns.

You can create a value but if the value has already been created as a default value, you can select it on Dialog Box Settings List to change dialog box settings.

Steps

  1. Click Home > Font.
  2. In the Font dialog box, click Properties > Bold and Underline on the Basic tab.
  3. Click Configuration next to Dialog Box Settings at the bottom of the dialog box.
  4. In the Dialog Box Settings Configuration dialog box, click the Add Dialog Box Settings icon.
  5. In the Add Dialog Box Settings dialog box, type Bold/Underline in Dialog Box Settings Name and click Set.
  6. Confirm that Bold/Underline is added to Dialog Box Settings List of the Dialog Box Settings Configuration dialog box, and then click Close.
  7. Confirm that Bold/Underline is added to the Dialog Box Settings list of the Font dialog box, and click Set.
  8. Select the content to apply Bold/Underline to in the edit window and click Home > Font.
  9. In the Font dialog box, select Bold/Underline from the Dialog Box Settings list at the bottom and click Set.
  10. Bold/Underline is applied to the selection.

Details on Icons

Icon Name Description
Add Dialog Box Settings Type a name for dialog box settings to create new Dialog Box Settings.
Edit Dialog Box Settings Edit the name of the existing Dialog Box Settings.
Delete Dialog Box Settings Delete the selected settings from the Dialog Box Settings List.
Move One Line Up Move the selected settings one line up on Dialog Box Settings List.
Move One Line Down Move the selected settings one line down on Dialog Box Settings List.

Note

Dialog Boxes with Dialog Box Settings Configuration

Hwp supports Dialog Box Settings Configuration feature in Print, Page Setup, Columns, Font, Paragraph, Note Style, Page Border/Background, Drop Cap, and Table/Cell Properties dialog boxes.

Specify required settings in each tab of a dialog box and save the settings by clicking Configuration in Dialog Box Settings. You can select them from the Dialog Box Settings list later.
