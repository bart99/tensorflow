Hancom Configuration Utility
 

You can set the Hwp environment in Hancom Configuration Utility.

Steps

Hwp Configuration Utility

  1. On the Windows taskbar, click Start > All Programs > Hancom > Thinkfree Office NEO > Hancom Configuration Utility.
  2. In the Hancom Configuration Utility dialog box, click Customize.
  3. Select the Hwp tab in the Hancom Configuration Utility dialog box.
  4. Select the Revert User-Defined Data to Initial State check box in Others and click OK.
  5. When you start Hwp, the user-defined data reverts to the initial state.

More Information

Hancom Configuration Utility Dialog Box

When you run Hancom Configuration Utility for the first time, the Restore Defaults button and the Customize button appear in the Hancom Configuration Utility dialog box.

If you click Restore Defaults > OK, the dialog box closes and the program is automatically reset to its initial environment.

If you click Customize, the following tabs appear in the Hancom Configuration Utility dialog box. They can help you to specify details about the environment.

HncSetup Tab

  • Restore Defaults Button

  • Automatically reset the program to its initial environment.

  • Customize Button

  • Select required items in the HncSetup, Hwp, Theme, File Association, and Languages tabs.

Hwp Tab

Select the following items in Others.

  • Revert Hwp to Initial State

  • Delete the customized values of toolbox, menu, document view, registered favorites, and options, and revert the program to its initial state.

  • Revert User-Defined Data to Initial State

  • Revert the user-defined data to the initial values that existed when Hwp was first installed.

Security Tab

Set options for malware block.

  • Malware Block when Opening Document

  • Detect malware when opening a document and prevent a malware infection. If malware is detected from a document, you will be immediately notified of warning message.

  • Real-time Malware Block

  • Monitor running Hancom Office in real time and prevent a malware infection. If malware is detected in real time, you will be immediately notified of warning message.

Theme Tab

You can decorate the program with program themes, which include various screen components like skin, toolbar, hotkey, and task pane.

When you select a program theme type from a list, the theme is applied to the program immediately.

File Association Tab

Select a file type from the file type tree on the left side and select a program to link the file to on the Associated Program list.

Languages Tab

Select the language you want from the Language Settings list.

If you select a language from the list and start the program, the program skin, toolbar, and menu appear in the selected language.

Caution

Restriction

To run Hancom Configuration Utility, exit all programs related to Thinkfree Office NEO.
