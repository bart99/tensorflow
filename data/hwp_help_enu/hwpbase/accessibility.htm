Increasing Accessibility
 

Hwp has enhanced accessibility with various features for ease of use. You can make Hwp more convenient to use by changing the mouse or keyboard or setting the user environment in Options.

More Information

To enter content through screen reader or handwriting recognizer

You can use the system keyboard to enter body text that can be recognized by the screen reader or to enter text by handwriting.

  1. To use the system keyboard, click Review > Keyboard > Change Keyboard.
  2. In the Keyboard Settings dialog box, select Current Keyboard > Windows Keyboard in the Change Keyboard tab and click Set.
  3. The text that is recognized by the screen reader or the handwriting recognizer is entered in the body text.
To run functions by using hotkeys

When you cannot use the mouse or if you have difficulty reading menus, you can run functions by using hotkeys.

  • The functions that are commonly used in Hwp can be run by combining function keys such as F1 and F2 and keys like CTRL, ALT, and SHIFT.
    For example, press ALT+TAB to switch to the next document when multiple documents are open.
  • If you press ALT and the letter in parentheses next to each tab name, the pull-down menu for the tab opens.
  • If you press ALT and the letter in parentheses next to each function name in menus or dialog boxes, you can set or run the function.
To apply color scheme of system to screen and dialog boxes

If you use high contrast due to the difficulty in reading the screen, you can apply the color scheme of the system to the Hwp screen.

  1. To apply the color scheme of the system, click File > Customize.
  2. In the Customize dialog box, select the Skin tab.
  3. Select System Style from the Skin list and then click OK.
  4. The color scheme of the system is applied.
To zoom edit window or icons

The Zoom function is for visually challenged users so that the edit window content and the toolbox icons look bigger.

  • You can zoom in or zoom out edit window by using the View tab of the toolbox or by the Zoom icon at the bottom of the screen.
  • To zoom in or zoom out an edit screen, press CTRL and move the mouse wheel when the pointer is on the edit screen.
  • To zoom in or zoom out the icons and function names of the toolbox, press CTRL and move the mouse wheel when the pointer is on the toolbox.
To use task panes by using keyboard

When you cannot use the mouse, you can use the keyboard to open a task pane or return to the editing area from the task pane.

  1. To open the Task Pane menu, select View > Task Pane or press ALT+U, W.
  2. To open a task pane, select a task pane by using the arrow keys of the keyboard.
  3. To return to the editing area, press ESC when using the task pane.
  4. To hide all task panes, press ALT+U, W, X.
To return toolbars or task panes to their original location

If you have difficulty viewing the screen and you use toolbars and menus by remembering their location, you can return toolbars and task panes back to their original location.

  1. To return all toolbars and task panes back to their original location, exit all running Thinkfree Office NEO programs.
  2. On the Windows taskbar, click Start > All Programs > Hancom > Thinkfree Office NEO > Hancom Configuration Utility.
  3. In the Hancom Configuration Utility dialog box, click Customize.
  4. The Hwp tab of the Hancom Configuration Utility dialog box is selected.
  5. Select the Revert Hwp to Initial State check box in Others and click OK.
  6. The customized settings of toolbox, menu, document view, registered favorites, and options get deleted and reverted to their initial settings.
To access multiple document windows by using keyboard

If you cannot use the mouse, you can use the keyboard to move, select, and close document windows.

  • To check currently open documents, go to View > Window List by using hotkeys. You can select a document to open it.
  • You can view up to 9 recent documents by using View > Window List. You can select a document to open it.
  • To switch to a document opened in a new window, press ALT+TAB. You can close the document by pressing ALT+F4.
  • To switch to a sheet opened in a new window, press CTRL+TAB.
To fold or unfold toolbars by using hotkeys

If you cannot use the mouse and thus work with pull-down menu, you can make the document editing area spacious by folding toolbars not in use.

You can fold or unfold toolbars into three layers by clicking the Fold/Unfold icon or pressing CTRL+F1.

To enter text by using mouse

Users who cannot use the physical keyboard can enter text with the mouse by using a keyboard on the screen.

  1. To view a keyboard on the screen and the key layout of the current keyboard, click Review > Keyboard > View Keyboard Layout.
  2. You can enter text with the keyboard or mouse by viewing the key layout.
To insert tables by using keyboard without mouse

You can insert tables with the keyboard when you cannot use the mouse.

  1. Click Insert > Table > Insert Table.
  2. In the Insert Table dialog box, clear the Create by Dragging check box in Others.
  3. Click Insert to insert a table at the cursor.
To insert pictures by using keyboard without mouse

You can insert pictures with the keyboard when you cannot use the mouse.

  1. Click Insert > Illustrations > Picture. In the Insert Picture dialog box, clear the Set Size with Mouse check box.
  2. Select the picture file you want and click Insert to insert it.
To search fonts quicker by using keyboard

When you are editing a document, you can search fonts by using the keyboard and change the font to the one you want.

  1. Click Home > Font. In the Font dialog box, select Font in the Basic tab.
  2. If you type a part of a font name, the list of fonts that include the spelling will be shown.
  3. You can select a font by using the up and down arrow keys. The selected font gets applied to the body text.

   The Font Search function is also available in the font list on the Formatting toolbar.
