Screen Layout
 

Hwp is easy to use for users who have not used Hwp before, because it follows the Windows standard interface structure.

More Information

Hwp Screen Layout

Name Description
Title The program title, the Maximize button, the Minimize button, and the Close button are displayed.
Menu Various menus are grouped according to their functions.
Standard Toolbar Frequently used functions of each menu are grouped in the form of menu tabs.
Toolbox for each menu is displayed by default. Depending on the function, toolbars for certain objects or conditions appear.
Formatting Toolbar Functions that are frequently used during document editing are grouped under icons.
Task Pane You can work on documents efficiently by using task panes. You can reduce the document editing time and increase work speed.
Status Bar Information of the current status of the edit window and the pointer location is displayed.
Tab Navigation Icons When multiple tabs are open, you can go to the previous tab or the next tab.
Document Tab Display active documents and their file names.
The names are displayed in red for unsaved documents, blue for automatically saved documents, and black for saved documents.
New Tab Add a new tab to the document.
Edit Window A workspace for inserting contents such as text and pictures.
Horizontal Scroll Bar When the document is bigger than the edit screen, you can use the horizontal scroll bar to move the screen left or right.
Move Page Icon Move by page when the document has multiple pages.
View Preferences Icon Select functions related to view preferences including Print Layout, Show/Hide Hard Return, Show/Hide Control Codes, Transparent Line, ViewGridOption, Find, Go to Page, Go to Section, Go to Line, Go to Style Settings, Go to Control Codes Settings, Go to Bookmark Settings, and Go to Object Settings.
Vertical Scroll Bar When the document is bigger than the edit screen, you can use the vertical scroll bar to move the screen up or down.
Vertical Ruler Assist you in checking the vertical position and height of an object.
Horizontal Ruler Assist you in checking the horizontal position and height of an object.

Note

To copy file names and file paths

If you right-click the title bar, a context menu appears. You can use functions like Copy File Path, Copy File Path and Name, and Open File Save Location.

By using these functions, you can easily copy the file name and path of the current document and run Windows Explorer in the current document location.

However, this function might not be supported in Windows Vista if Customize > Skin is set to System Style.
