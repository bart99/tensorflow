Keyboard Toolbar
 

When you start Hwp, the Keyboard toolbar appears on the desktop.

On the Keyboard toolbar, you can change the language and type of the current keyboard and also display the keyboard on the screen.

Steps

To change the language of the current keyboard

  1. When you start Hwp, a document window and the Keyboard toolbar appear on the desktop.
  2. In the edit window, type Hancom.
  3. On the Keyboard toolbar, click the Current Language icon which is set as English.
  4. From the list of languages, select Korean.
  5. The Current Language icon and the keyboard change to Korean .

  To specify the location of the Keyboard toolbar, in the Keyboard Settings dialog box, click Toolbox/Icon > Show Icon in System Tray or Show Toolbox on Desktop. You can use the Keyboard toolbar only when both of the items or at least one of them is selected.

More Information

Details on Icons

The basic icons of the Keyboard toolbar are as follows.

To add the Fullwidth/Halfwidth icon or the Language Options icon on the Keyboard toolbar, in the Keyboard Settings dialog box, go to Toolbox/Icon > Toolbox Items list.

The Language Options icon is available only for the Chinese and Japanese keyboards.

Icon Name Description
Current Language Specify the language of the current keyboard.
You can select a language from various languages that are displayed.
Fullwidth/Halfwidth Set the width of text.
Every time you click this icon, the icon switches between the Fullwidth icon and the Halfwidth icon .
Language Options Display the Conversion tab of the Keyboard Settings dialog box. You can specify how to convert characters to Japanese or Chinese.
This icon is available only when the Current Language is set to Japanese, Simplified Chinese, or Traditional Chinese.
Current Keyboard Select a keyboard type for the language that is selected in Current Language.
You can select a keyboard from various keyboards that are displayed.
Change Keyboard The Change Keyboard tab of the Keyboard Settings dialog box is displayed to change the settings of the current keyboard.
View Keyboard Layout A keyboard appears on the screen and shows the layout of the currently selected keyboard.
Options The Toolbox/Icon tab of the Keyboard Settings dialog box is displayed to change the settings of the Keyboard toolbar.
Help The Help dialog box is displayed.

Note

Show Toolbox on Desktop

If you clear the Show Toolbox on Desktop check box in the Toolbox/Icon tab of the Keyboard Settings dialog box, the Keyboard toolbar does not show on the desktop.

In that case, to change the keyboard, right-click Hancom Multi-language Input System on the Windows taskbar or change the settings on the Keyboard toolbar to appear again.
