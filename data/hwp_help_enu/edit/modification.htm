Object Properties
 

Home > Object Properties

Object Properties enables you to modify content in the document such as Picture, Table, Text, Drawing Object, Formula, Wordshop, Clip Art, Cross-reference, Header/Footer, Enclose Characters, Footnote/Endnote, Phonetic Guide, Page Number Position, Hidden Content, Field Entry, Hidden Note, Mark Entry, Hyperlink, Bookmark, Mark, and Formula.

Steps

  1. Click Home > Object Properties in the body text.
  2. Picture, Table, Text, Drawing Object, Formula, Wordshop, Clip Art, Cross-reference, Header/Footer, Enclose Characters, Footnote/Endnote, Phonetic Guide, Page Number Position, Hidden Content, Field Entry, Hidden Note, Mark Entry, Hyperlink, Bookmark, Mark, and Formula in the body text will be searched forward from the current pointer position and the Object Properties dialog box appears for the first item that is found.
  3. Make changes you want and click Set to close Object Properties.

Note

Order of Application

Modify content at the current position through Object Properties if any of the following is located at the current pointer position: Picture, Table, Text, Drawing Object, Formula, Wordshop, Clip Art, Cross-reference, Header/Footer, Enclose Characters, Footnote/Endnote, Phonetic Guide, Page Number Position, Hidden Content, Field Entry, Hidden Note, Mark Entry, Hyperlink, Bookmark, Mark, and Formula.

If none of them is located at the current pointer position, search upward from the pointer position and modify the first item that is found.

If there is nothing to modify in the area before the pointer position, there will be no activity.
