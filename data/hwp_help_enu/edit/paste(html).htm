Paste HTML Document
 

Home > Paste

When copying content of an HTML document and pasting it in a Hwp document, you can keep the original data type or just paste the text of the HTML document.

Steps

  1. Select content to copy as a block in the HTML document and press CTRL+C to copy.
  2. Point to the edit window of the Hwp document and click Home > Paste > Paste.
  3. In the Paste HTML Document dialog box, select Select Data Type > Paste as Text and click OK.
  4. The content selected as a block gets pasted at the pointer in the Hwp document.

More Information

Paste HTML Document Dialog Box

Select Data Type

Select data type when copying content of an HTML document and pasting it in a Hwp document.

  • Keep Original Format

  • Paste the content in a Hwp document while keeping the data type of the copied HTML document.

  • Paste as Text

  • Paste only the text of the copied HTML document in a Hwp document.

Options

Select whether to set the selected format of pasting HTML documents as default or not.

  • Set as Default

  • Select the value selected in Select Data Type as the default value.

    If you use Paste From Clipboard > Paste Options for HTML Document in the Code Format tab of the Options dialog box, you can change data type for pasting HTML documents and apply the change.
