Text Direction
 

Page Layout > Text Direction

You can select the text direction in a document when you are in normal edit mode. You can select either horizontal or vertical as text direction.

Steps

To write vertically

  1. Open a document to edit.
  2. While in normal edit mode, click Page Layout > Text Direction > Vertical.
  3. The text direction in the opened document changes to vertical.

To write horizontally

  1. Open a document to edit.
  2. While in normal edit mode, click Page Layout > Text Direction > Horizontal.
  3. The text direction in the opened document changes to horizontal.
