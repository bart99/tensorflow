Copy <CTRL+C> / <CTRL+INSERT>
 

Home > Copy

This feature allows you to copy selected content on the Clipboard. You can import the copied content to the required location by clicking Home > Paste. This function is mostly used when you require the same content at many locations.

Steps

  1. Select the content from the document.
  2. Click Home > Copy to save your selection in the Clipboard.
  3. Place the pointer on the specific location, and click Home > Paste, the selected content will be pasted at the required location.

Note

Copy and Cut

  • Copy: Copy the selection on the Clipboard.
  • Cut: Copy the selected content on the Clipboard and then delete it from the current location.

Drag with the Pointer

You can copy and paste selected content by placing the pointer within the selected content, holding-down CTRL, and then dragging it to the required location.

Clipboard

Clipboard is a temporary memory storage used in Windows for data transfer between applications.

In an application, you can send data to the Clipboard by clicking Copy or Cut, and insert data from the Clipboard by clicking Paste.

If you type clipbrd in Start > Run on Windows and click OK, ClipBook Viewer is displayed and you can check the content stored in the Clipboard.

To Copy Inside a Table

If you select cells and click Copy, not only content in the table but also the selected table pane will be copied.

If you Paste inside the table, the Paste Cell dialog box appears. Select the option to insert the copied table, overwrite the active table, or insert the copied table as a table inside a cell.

Advanced Usage

Using also in the Edit Box of Dialog Boxes

You can also use Copy, Cut, and Paste in most edit boxes, like File Name in the Open dialog box where you can type.

If you memorize the shortcuts CTRL+C, CTRL+X, and CTRL+V, they will be very useful.

Copying Page Setup State

If you click Select All to select the entire document and click Copy or Cut, document content, page setup, and page border/background will also be copied.

If you click New or New Tab to create a new document, and click Paste, the document content and page setup will be pasted on the new document.

To copy content and page setup at the same time, you have to select the entire document and then copy.
