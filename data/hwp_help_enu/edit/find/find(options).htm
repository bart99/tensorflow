Options
 

Home > Find > Find > Options

You can experience more powerful and advanced search and replace features by using Options from the Find/Replace dialog box.

More Information

Options

  • Match Case

  • Search for exactly matching English letters that even match case.

    For example, when you search for computer, Computer and COMPUTER are skipped.

      Match Case is applied only to Find What in the Find/Replace dialog box.

  • Whole Words Only

  • Search for only whole words without other characters (punctuation marks such as period and comma included) before or after them.

    For example, when you search for total, subtotal and total are skipped.

      Whole Words Only is applied only to Find What in the Find/Replace dialog box.

  • Find Multiple Words

  • Search for multiple words by using a comma (,) or semicolon (;) as the separator.

    For example, if you type add;sum, add and sum will be found.

      Find Multiple Words is applied only to Find What in the Find/Replace dialog box.

  • Ignore Spacing

  • Search for all the characters typed in Find What by ignoring the spaces.

    For example, when you find login, both log in and login will be found.

      Ignore Spacing is applied only to Find What in the Find/Replace dialog box.

  • Wildcards

  • If you are not sure about part of the text you want to find, use ? or *. ? represents one letter and * represents more than one letters.

    If you search with ?, spaces are not recognized as wildcards.

    For example, if you want to search for a four-letter word that starts with re, type re?? in Find What and specify Wildcards in Find > Options. Words like real and rest will be found in the body text.

    You can find all words that start or end with a specific letter by using *.

    For example, if you type re* in Find What and specify Wildcards in Find > Options, words like real, rest, re, and respect will be all found in the body text.

      Wildcards is only applied to Find What in the Find/Replace dialog box.

      If you select Wildcards and Find Chinese Using Korean at the same time, the latter might not work properly.

  • Use Conditional Expression

  • Type reserved symbols (wildcards) used in conditional (pattern or regular) expressions for advanced search.

    Options that can be used with conditional expressions are Match Case and Whole Words Only.

      Use Conditional Expression is only available in the Find dialog box.

Conditional Expression

Conditional (pattern or regular) expressions are concise and flexible expressions that use simple parsing to find text patterns.

Select Use Conditional Expression in Options of the Find dialog box and use the following conditional expressions in Find What to find text or numbers in a small or large search range.

Note

Multiple Selections in Options

You can select multiple items in Options but you cannot select Ignore Spacing and Wildcards at the same time. You can use Find/Replace in a variety of ways by using it differently in different situations.

Caution

Options Applied Only to Find What or Find/Replace

Most items in Options can be applied only to Find What of the Find/Replace dialog box.

  • Items That Are Applied Only to Find What of Find/Replace

  • Match Case, Whole Words Only, Find Multiple Words, Ignore Spacing, Wildcards, Use Conditional Expression
