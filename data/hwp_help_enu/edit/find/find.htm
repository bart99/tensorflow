Find<CTRL+F>
 

Home > Find > Find

This feature is used to search for specific words (text) in the active document. You can search with advanced options by using Options.

Steps

  1. Click Home > Find > Find.
  2. In Find What of the Find dialog box, type the required text to search for in the active document.
  3. Click Find or press ENTER on the keyboard.
  4. A word that matches Find What is displayed as a block. If the text you want to find does not exist, you will be notified.
  5. Click Find or press ENTER again to continue search.

More Information

Find Dialog Box

Find What

Type content to search for. If you click Find Format next to the box and select a format, you can search text with a specific format, font, paragraph, or style. You can also search for text inserted as a phonetic guide.

Options

You can search with various options such as case matching and spacing matching.

Find Button

If you click Find, a word that matches Find What is found and displayed as a block.

Find All Button

If you click Find All, all words that match Find What are found and displayed as blocks.

Replace Button

If you click Replace, the Find dialog box changes to the Replace dialog box and you can run Replace.

Close Button

Stop searching and close the Find dialog box.

As Find is used repeatedly in many cases, it remains on the screen until you close the dialog box by clicking Close.

Note

Select as a Block and Find

To search for only a part of a document, select the part as a block and click Find.
After the search in the block is complete, the user gets prompted to continue/stop the search in the rest of the document.

Range

Search will be performed from the current pointer position to the end/top of the document if Search Direction is Down/Up. After the search, the user will get prompted to continue/stop the same search in the rest of the document.
If Search Direction is set to Whole Document, search will be performed from the current pointer position to the end of the document and then from the top of the document to the current pointer position.

Find Format

Find does not just find specific text in the body text. If you click Find Format on the right of the Find What box, you can find special characters that display tabs, paragraph ends, or line breaks as well as formats such as specific font, paragraph, and style in the body text.
