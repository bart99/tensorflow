Find Format
 

Home > Find > Find > Find Format

You can use Find not only for searching for specific text in the body text but also for replacing special characters that display tab, paragraph end, or line break as well as formats such as specific font, paragraph, and style in the body text.

Steps

  1. Click Home > Find > Find.
  2. In Find What of the Find dialog box, type the text to be found in the active document.
  3. If you click the Find Format icon on the right of the Find What text box, sub-menus of Find Format are displayed.
  4. Select Tab and click Find or press ENTER.
  5. A word that matches Find What is displayed as a block.
  6. Click Find or press ENTER again to continue search.

  If the text to find does not exist, a notification message appears.

Note

Submenus of Find Format

  • Special Characters - Tab, Paragraph End, Line Break, Fixed-Width Space, Nonbreaking Space

  • Search only for special characters such as Tab (^t), Paragraph End (^n), Line Break (^l), Fixed-Width Space (^s), and Nonbreaking Space (^r), or text with special characters combined.

    For example, if you type done in Find What and specify Paragraph End (^n), instances of done in the middle of paragraphs will not be found. Only the instances at the end of paragraphs will be found.

  • Font to Find

  • You can find specific fonts in the body text.

    For example, you can find instances of Haansoft in Gothic+Bold among all instances and change them to Hancom in Outline+Italic.

  • Paragraph to Find

  • You can find specific paragraphs in the body text.

    For example, you can find instances of Hancom only in paragraphs with Indent: 10pt.

  • Style to Find

  • Find specific styles in the body text.

    For example, you can find instances of Hancom only in the paragraphs with the style Multilevel List 1 applied.

  • Ignore Content to Find

  • When you run Find for specific Font to Find or Paragraph to Find, the search is performed for all the text with the specific font or paragraph regardless of Find What.

Clearing the Selection in Find Format

If you select a format and run Replace, the selection mark stays in front of the selected format menu. Therefore, even if you type something different in Find What and run Replace, only the words with the specific format will be found.

After selecting a format and running Replace, you must select Find Format again to remove the selection mark in front of the menu.

Typing Special Characters

If you type ^t, ^n, ^l, ^s, or ^r in the Find What box, the search result will not include these characters typed in the body text. Rather, special characters represented by ^t (tab), ^n (paragraph end), ^l (line break), ^s (fixed-width space), or ^r (nonbreaking space) will be found.
