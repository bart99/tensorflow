Find All
 

Home > Find > Find > Find All

From the Find dialog box you can use Find All to search for all instances of specific words (text) in the active document and check them altogether.

Steps

  1. Click Home > Find > Find.
  2. In Find What of the Find dialog box, type Hancom as the text to be searched for in the active document.
  3. Click Find All.
  4. Words that match Find What are highlighted.
  5. Click Find. The first word found will be displayed as a block.
  6. The block moves to the next highlighted whenever you click Find.
  7. Click Close. The Find dialog box closes and the highlighting disappears.

Note

Highlight

From the Find dialog box, if you click Find All, the words that match Find What in the entire document are highlighted and the highlighting will stay as long as the Find dialog box is open.

If you close the Find dialog box by clicking Close, highlighting in the body text disappears.

Phonetic Guide Is Included in Search

Terms inserted as a phonetic guide are included in the search of Find All.
