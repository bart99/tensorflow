Find Again <CTRL+L>
 

Home > Find > Find > Find Again

You can run Find or Go to again without opening the Replace dialog box. The content to find was previously specified in Find or Go to. Find the content from the current pointer position to the Up or the Down, depending on Find > Search Direction.

Steps

  1. Click Home > Find > Find.
  2. In Find What of the Find dialog box, type the text to find in the active document.
  3. Click Find or press ENTER on the keyboard.
  4. Words that match Find What are highlighted. If the matching text does not exist, then you will be notified.
  5. Click Close or close the Find dialog box by pressing ESC.
  6. Click Home > Find > Find Again or press CTRL+L to run Find to find the content. 

Note

Find Backward <CTRL+Q,L>

If you want to run Find Again backward after running it forward, press CTRL+Q,L once. This command changes the direction of the search.
If you press CTRL+L again, Find Again continues in the changed direction.

Use Shortcuts for Find Again and Find Backward

Find Again and Find Backward are functions that enable you to repeat search under the existing conditions by using shortcuts without opening a dialog box.
Use the Find Again shortcut, CTRL+L, and the Find Backward shortcut, CTRL+Q,L, for your convenience.

Remember Only the Last

Find Again remembers the last content searched through Find or Go to and finds it again. It does not remember the content of Find and Go to separately.

Find Again after Replace

Find Again is a function that finds the content you typed in Find What in Find or Replace.
If you run Find Again after running Replace, only the Find function runs to find the content in Find What. The Replace function does not run.
