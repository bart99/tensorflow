Paste <CTRL+V>
 

Home > Paste

You can paste the content on the Clipboard saved by clicking Home > Copy or Home > Cut back at the pointer.
You can repeatedly Paste the content that has once been stored in the Clipboard.

Steps

  1. Select content inside the document you would like to move, as a block.
  2. Click Home > Copy.
  3. Point to a position you want and click Home > Paste > Paste. The block will be pasted at the pointer.

Note

Paste in Another Document

You can paste content copied or cut from a document into another document.
When you need content of a certain document in another document, click Copy in the document with the content and point to the other document and click Paste > Paste.

Paste in Another Application

You can use the content saved on the Clipboard in other applications as well as the Hwp program.
However, details such as pictures, fonts, and paragraphs might change depending on the support provided by each application.

Paste Inside a Table

If you select cells as a block and click Copy, not only content in the table but also the table pane selected as a block will be copied.
If you Paste inside the table, the Paste Cell dialog box appears. Select whether to insert the copied table, overwrite the active table, or insert the copied table as a table inside a cell.

Paste Special

When you copy the object you worked on in a different Windows program and insert the content saved on the Clipboard into a Hwp document, Paste Special is very useful.

Advanced Usage

Using also in the Edit Box of Dialog Boxes

You can also use Copy, Cut, and Paste in most edit boxes like File Name in the Open dialog box where you can type.
If you memorize the shortcuts CTRL+C, CTRL+X, and CTRL+V, they will be very useful.
