Select All <CTRL+A>
 

Home > Select All

You can set the entire content of a document as a block.
This function is useful for changing fonts or paragraphs in the entire document or Copy, Cut, or Delete the entire document.

Steps

  1. Click Home > Select All.
  2. The entire document gets selected as a block.
  3. Run a command.
  4. Press ESC to remove the block.

Note

Notes Regarding Select All

  • Select All With the Mouse

  • If you point to the empty space on the left of the edit screen (left margin of paper), the pointer reverses and points to the right instead of left.

    If you click three times, the entire document gets selected as a block.

  • Select All With the Keyboard

  • If you place the pointer on some text in and press F3 twice, the word gets selected as a block. If you press the key three times, the paragraph where the word is gets selected as a block. If you press the key four times, the entire document gets selected as a block just like when you use Select All.

  • Select All Inside a Table or Text

  • If you place the pointer on a table or on text and run Select All, the content of only the active cells or text is selected as a block.
