Delete Control Codes
 

Home > Delete Control Codes

You can select and delete specific objects in the entire document, a block, or a section.

Steps

  1. Select the required content in the document as a block (F3) or a section (F4).
  2. Click Home > Delete Control Codes.
  3. Select the objects to delete from the Control Code List box of the Delete Control Codes dialog box.

    For example, select the Footnote and Picture check boxes.

  4. Click Delete when all the objects to delete are selected.
  5. All footnotes and pictures inserted in the selected area will be deleted.

More Information

Delete Control Codes Dialog Box

Select Object

A list of approximately 60 control codes is displayed. Select multiple objects from the list and click Delete to delete them.

You can delete the following the objects by using Delete Control Codes.

Footnote, Footnote Number, Hide, Formula

Start Section, Draw, Picture, Picture Number, Wordshop, Text Box, Enclose Characters, Footer, Click Here, Phonetic Guide,

Creation Date, Header, Comments, Mail Merge, Summary, Endnote, Endnote Number, User Number, Inserted Picture, Cross-reference,

New Footnote Number, New Picture Number, New Column Definition, New Endnote Number, New Equation Number, New Page Number, New Table Number,

Equation, Equation Number, Hidden Note, Linked Picture, Page Number, Page Number Position, Index, File Path,

Table, Table Number, Hyperlink, Current Date, Start from Odd Page, OLE Object

Select All

If you click Select All, the button switches between Select All and Release All.

  • Select All: To select the 60 control code objects from the Control Code List.
  • Release All: To release all the control code objects selected from the Control Code List.
