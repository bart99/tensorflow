Go to <CTRL+G>
 

Home > Find > Go to

You can point and move to a specific position such as line or page, or a position with a picture, table, or bookmark inserted.

Hwp displays and records font or paragraph styles as properties of the font or paragraph, and inserts invisible Control Codes at the pointer for pictures, tables, headers, footers, and bookmarks. You can find such control codes by clicking Home > Go to instead of Home > Find.

In Go to, you can find Page, Section, Row, Control Codes, Style, and Bookmark.

Steps

  1. In a multi-page document, click Home > Find > Go to.
  2. In the Go to dialog box, select Page.
  3. Calculate the number of pages in the active document and display the total page number.
  4. In the box, type 3 and click Go.
  5. The pointer moves to the beginning of Page 3.

More Information

Go to Dialog Box

Page

Display the total page number of the active document in the list box.

If you type a page number in the text box or select one from the list and click Go, the pointer moves to the page.

Section

Display the total section number of the active document in the list box.

If you type a section number in the text box or select one from the list and click Go, the pointer moves to the section.

Line

Display the total line number of the active document in the list box.

If you type a line number in the text box or select one from the list and click Go, the pointer moves to the line.

Style

Display a list of all styles saved in the active document.

If you select a style from the list and click Go, the pointer moves to the position where the style is applied.

Control Codes

Display all control codes of Hwp in a list box.

If you select a control code from the list and click Go, the pointer moves to the position where the control code is inserted.

Bookmark

Display a list of all bookmarks in the active document.

If you select a bookmark from the list and click Go, the pointer moves to the position where the bookmark is inserted.
