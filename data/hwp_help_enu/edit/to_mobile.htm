Save as Mobile Optimized Document
 

File > Save as Mobile Optimized Document

From the edit screen, you can save the document as a mobile optimized document.

Steps

To save as mobile optimized document

  1. Open a document to save as a mobile optimized document.
  2. Click File > Save as Mobile Optimized Document, the Save as Mobile Optimized Document dialog box appears.
  3. Specify the location in which to save the mobile optimized document and its file name, and click Save.
  4. A message appears with a request to save the document as a mobile optimized document.
  5. Click Save. A mobile optimized document will be created in the specified location.

Caution

Save a Document with Objects as a Mobile Optimizes Document

Click the Save as Mobile Optimized Document function to save a document optimized for the mobile environment. The layout of the document stays the same but the properties of some objects might change.

  • Save charts, wordshops, OLEs, form controls, and grouped objects as pictures.
  • Save picture files in low resolution and change them to an image format that supports mobile.
