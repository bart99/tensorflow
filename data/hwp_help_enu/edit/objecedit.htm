OLE Properties
 

Home > OLE Properties

You can modify objects by clicking Insert > Object > Insert Object.

Steps

  1. Double-click an object to modify whether it is embedded or linked, or click Home > OLE Properties.
  2. In the OLE Properties dialog box, modify the object and click OK.
  3. The properties of the object will change.
