Insert/Modify <INSERT>
 

The Insert status and the Modify status are useful when you edit an existing document.

In the more commonly used Insert status, content that is typed or copied and pasted gets inserted at the pointer. Therefore, the existing content only gets pushed back according to how many characters are inserted. While in the Modify status, however, the existing content gets deleted according to how many characters are inserted. Therefore, Insert is also called Insert Writing and Modify is also called Overwrite.

Steps

To change to insert status

  1. Type Hwp 2014 on the edit screen and move the pointer in front of 2 of 2014.
  2. Type Word Processor while Insert is displayed on the status bar.
  3. 2014 gets pushed back and the result is Hwp Word Processor 2014.

To change to modify status

  1. Type Hwp 2014 on the edit screen and move the pointer in front of 2 of 2014.
  2. Press INSERT on the keyboard while Insert is displayed on the status bar. The status changes to Modify.
  3. While in the Modify status, type Word Processor.
  4. 2014 located behind the pointer gets deleted and the result is Hwp Word Processor.
  5. Press INSERT again. Modify changes to Insert.

    INSERT is a key that changes the status between Insert and Modify.

Note

Run from the Status Bar

You can easily switch the status between Insert and Modify by clicking the Insert/Modify part on the status bar.

Lock Insert

If you select Lock Insert in Edit of the File > Options > Others tab, the status locks as Insert. Even if you press INSERT, the status does not switch between Modify and Insert.
