Change Case
 

Home > Change Text > Change Case

You can select English letters typed on the screen as a block and change them to uppercase or lowercase.

You can select various options. For example, you can change all the English letters in the block to uppercase or lowercase or in only the first letter of all sentences to uppercase.

Steps

  1. Select part of the body text that includes English letters, as a block.
  2. Click Home > Change Case > Change Case.
  3. In the Change Case dialog box, select Replace Method > Uppercase and click Replace.
  4. All the English letters in the block change to uppercase.

More Information

Change Case Dialog Box

Replace Method

  • Uppercase

  • Change all the English letters in the block to uppercase.

  • Lowercase

  • Change all the English letters in the block to lowercase.

  • Toggle Case

  • Change uppercase English letters in the block to lowercase and lowercase English letters to uppercase.

  • Capitalize Each Word

  • Change the first letter of all English words in the block to uppercase and the rest of the letters to lowercase.

  • Sentence Case

  • Change the first letter of all sentences in the block to uppercase and leave the rest of the letters unchanged.

  • Sentence Case Only

  • Change the first letter of all sentences in the block to uppercase and change the rest of the letters lowercase.

Note

Not Related to Non-English Characters

Even if there are non-alphabet characters in the block selected for case changing, case changing is performed only on alphabets.

Not Related to Non-English Foreign Characters

The Change Case function only works for English. It does not work for characters used in other languages like German or French.

Advanced Usage

Undo

Click Home > Undo to undo case changes.
