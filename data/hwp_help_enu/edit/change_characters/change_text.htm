Change Text
 

Home > Change Text

You can select English letters typed on the screen as a block and change them to uppercase/lowercase or half-width/full-width.

Note

Not Related to Non-English Characters

Even if there are non-alphabet characters in the block selected for Change Case, case changing is performed only on alphabets.

Not Related to Non-English Foreign Characters

The Change Case function works only for English. It does not work for characters used in other languages like German or French.

Advanced Usage

Undo

Click Home > Undo to undo case changes.
