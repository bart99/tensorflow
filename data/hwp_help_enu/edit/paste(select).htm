Paste Special <CTRL+ALT+V>
 

Home > Paste > Paste Special

You can copy the object you worked on in a different Windows program and insert the content saved on the Clipboard into a Hwp document.

Steps

  1. Click Home > Paste > Paste Special.
    Only when any content or object is copied on the Clipboard, you can run this function.
  2. In the Paste Special dialog box, select a paste type in the Data Type box and click OK.
  3. Content of the Clipboard gets pasted at the pointer in the body text.

More Information

Paste Special Dialog Box

Paste

Paste the content copied on the Clipboard at the pointer.

Paste Link

Link the content copied on the Clipboard to the pointer and paste it at the pointer.

If the original document changes, the content pasted into the edit window changes too.

Data Type

From the list, select a data type to paste.

Show as Icon

Specify whether to display content or an icon that shows the content in the file.

If you run Paste Special by selecting Show as Icon and double-click the icon in the edit window, you can open the content.

Note

Clipboard

The clipboard is a temporary memory storage used in Windows for data transfer between applications.
In an application, you can send data to the Clipboard by clicking Copy or Cut and insert data from the Clipboard by clicking Paste.
In Clipboard Display, a Windows system tool, you can check content stored in the Clipboard.

Device Independent Bitmap

If you paste the object copied on the Clipboard in Device Independent Bitmap format, the original content changes but the content pasted into a Hwp document remains unchanged.
