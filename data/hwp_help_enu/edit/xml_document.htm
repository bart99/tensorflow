XML Document
 

File > XML Document

XML (eXtensible Markup Language) is a script language that is similar to HTML (Hyper Text Markup Language).

XML is a standardized markup language in text format, which is designed for transferring structured documents online. It is simpler than SGML (Standard Generalized Markup Language) and is capable of expressing documents that can be used on the Internet.

Hwp supports the function of creating XML documents so that users can write and use XML documents easily.

Note

XML Document Formats Supported by Hwp

When you are running Open/Save XML Document in Hwp, the XML documents that are in HTML schema format can be open and they are saved as Hwp XML templates.

If you open or save an XML document that is not in HTML schema format, the document will be processed as binary.
