Redo <CTRL+Y>
 

Home > Redo

You can redo commands or actions canceled by clicking Home > Undo.

Steps

  1. Type text in the document on the edit screen or edit the document, for example, by changing fonts.
  2. Click Home > Undo.
  3. Check on the edit screen that the last action has been cancelled.
  4. Click Home > Redo.
  5. Check that the action cancelled by Undo has been redone.

Note

Nothing to Redo

Redo is the function that performs the action cancelled by Undo. Until you run Undo, there is nothing to redo.
Redo is displayed blurred on menus and toolbars and cannot be used.

Only Available after Running Undo

You can run Redo only after running Undo. You cannot run Redo after you run Undo and then perform another action.
The action to Redo is the last action cancelled by Undo. If you run Undo and then type or edit something without running Redo right away, the saved actions will be deleted.
If you run Undo, the action to Redo will be created again.

Run as Many Times as Undo was run

Just like Undo remembers the actions of the user step by step, Redo remembers the actions cancelled by Undo step by step.
Therefore, you can run Redo as many times as you previously ran Undo.
