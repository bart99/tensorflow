Cut <CTRL+X>
 

Home > Cut

This feature allows you to delete the selection and copy its content on the Clipboard. You can import the copied content to the required location by clicking Home > Paste.

It is usually used when you move the content to a new location.

Steps

  1. Select content from the document.
  2. To delete the selected content will be deleted, click Home > Cut.
  3. Place the pointer on the specific location, and click Home > Paste, the selected content will be pasted at the required location.

Note

Copy and Cut

  • Copy: Copy the selection on the Clipboard.
  • Cut: Copy the selected content on the Clipboard and delete it from the current position.

Drag with the Pointer

You can cut and paste selected content by placing the pointer within the selected content, and then dragging it to the required location.

Clipboard

Clipboard is a temporary memory storage used in Windows for data transfer between applications.

In an application, you can send data to the Clipboard by clicking Copy or Cut and insert data from the Clipboard by clicking Paste.

In Clipboard Indicator, a Windows system tool, you can check content stored in the Clipboard.

Cut Cells from a Table

If you select a range of cells and click Cut, not only the cell content but also the selected cells will be cut.

If you click Keep in Body, when "Cut the selected cells. Leave the cell format?" message appears, then the cell content will be cut and the cell format will remain intact. If you click Delete, the selected cells selected will be cut and the table will look different.

If you select a section of entire rows or columns and click Cut, only the cell content will be cut.

Advanced Usage

Using also in the Edit Box of Dialog Boxes

You can also use Copy, Cut, and Paste in most edit boxes, like File Name in the Open dialog box.

If you memorize the shortcuts CTRL+C, CTRL+X, and CTRL+V, they will be very useful.
