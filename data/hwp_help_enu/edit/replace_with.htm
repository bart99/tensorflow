Replace <CTRL+H>
 

Home > Find > Replace

You can find a specific word (text) in the active document and replace it with another word. You can also replace words with advanced features by using Options.

Steps

  1. Click Home > Find > Replace.
  2. In the Replace dialog box, enter text in Find What that you may want to find in the active document.
  3. In Replace With, type text that will replace the text in Find What.
  4. Click Replace or press ENTER.
  5. Words that match Find What will be displayed as blocks in the body text.
  6. Click Replace to replace the found words. Click Find Next to ignore them without replacing them.
  7. To finish replacing, click Close to close the Replace dialog box.

More Information

Replace Dialog Box

Find What

Type content that is required to be found. If you click Find Format next to the box and select a format, you can search text with a specific format, font, paragraph, or style.

Replace With

Type text to replace with the content that has been found. If you click Find Format next to the box and select a format, you can replace the search results with text with a specific format, font, paragraph, or style.

Options

You can search with various options such as case matching and spacing matching.

Replace Button

If you click Replace when one instance of the word in Find What is found and selected as a block, the word changes to the one in Replace With and the next instance is shown as a block.

Find Next Button

If you click Find Next when one instance of the word in Find What is found and is selected as a block, the word stays unchanged and the next instance is displayed as a block.

Replace All Button

When clicked, Replace All finds all instances of the word in Find What and replaces them with Replace With within the designated range without prompting for user confirmation. It then indicates how many replacements were made.

Close Button

Stop searching and close the Find dialog box.

As Find and Replace are used repeatedly in many cases, they stay on the screen until you close the dialog box by clicking Close.

Note

Select as a Block and Replace

To replace certain words of a document only in part, select part of a document as a block and run Replace.

After replacement is complete in the block, the user gets prompted to continue/stop the same replacement.

Range

Replacement will be performed from the current pointer position to the end/top of the document if Search Direction is set to Down/Up. After the replacement, the user will get prompted to continue/stop the same replacement in the rest of the document. If Search Direction is set to Whole Document, search will be performed from the current pointer position to the end of the document and then from the top of the document to the current pointer position.

Replace Format

Replace not only finds specific text in the body text but also replaces special characters that display tabs, paragraph ends, or line breaks as well as format such as specific font, paragraph, and style in the body text.

The Replace dialog box does not close automatically

As Replace is used repeatedly in many cases, it stays on the screen until you close the dialog box by clicking Close.

Advanced Usage

Cancel Replace

Click Home > Undo to cancel the action performed by Replace.
