Font List Setup
 

File > Options > Fonts

You can customize the font list by using the Font List Setup.

From the Fonts tab of the Options dialog box, you can select Display in Tabbed List to add tabbed lists of fonts that are grouped by use to the Fonts list of the formatting toolbar.

Steps

To add font list

  1. Click File > Options.
  2. In the Options dialog box, select the Fonts tab.
  3. In Tabbed List in Font List, select Display and click Font List Setup.
  4. From the Font List Setup dialog box, click the Add Font List icon .
  5. In the Add Font List dialog box, enter Education Material Font for the List Name, and then click Add.
  6. In the Font List of Font List Setup, check that Education Material Font is added.
  7. Select All Fonts from the Source font list. Select multiple fonts of your choice from the font list registered in a subbox, and then click the Copy icon .
  8. Check if Education Material Font is copied to the Current list, and then click Set.
  9. When Font List Setup dialog box is closed, click Set in the Options dialog box.
  10. In the Format dialog box of the Fonts list box, click the drop-down arrow .
  11. Check if the Education Material Font tab and the font are added in the drop-down list of each tab.

More Information

Font List Setup Dialog Box

Font List

You can display default font lists such as Title Font, Body Font, and Adorn Font.

You can add a font list of your choice, and then register fonts you need.

Current

Currently registered fonts are displayed on the font list selected from the Font List.

Source

You can display the name of the font list added to the Font List, and fonts registered on the font list, so that you can select fonts to copy from All Fonts.

Details on Icons

Icon Name Description
Add Font List Display the Add Font List dialog box to allow addition of custom font lists to Font List.
Edit Font List Display the Edit Font List dialog box to change the name of the custom font list selected from the Font List.
Delete Font List Delete the selected custom font list from Font List.
Delete font from current Font List Delete the selected font from the Current font list.
Copy Copy the font that was selected from the Source list to the Current font list.
Copy All Copy all fonts from the Source list to the Current font list.
