Translate
 

Review > Translate

With this feature, you can translate part of contents or whole document by using machine translation service.

When you have a document written in other languages that you haven't known, this feature helps you read and understand the document instantly.

Steps

Translate contents as a block

  1. Type anything in Korean on the document.
  2. Select the sentence as a block by dragging the mouse.
  3. Click Review > Translation > Translate Selected Text.
  4. In the Translation task pane, select the languages to translate the selected sentence from and to, for example, English and Korean, and then click Translate.
  5. The translation result appears in the Translation List area on the task pane in English.
  6. Click the translated sentence, click the drop-down arrow on the right, and then click Overwrite. The translation result is applied on the current document.

   If you have applied the translation results on your document, they disappear on the Translation List area of the task pane.

   If you select part of contents as a block and translate, the selected contents are translated only. If you want to translate a whole paragraph, place the cursor in the middle of paragraph and then click Review > Translation > Translate Paragraph.

Translate a whole document

  1. Open a document to translate.
  2. Click Review > Translation > Translate Document.
  3. The translation results appear on the Translation List area.
  4. Click the translated sentence, click the drop-down arrow on the right, and then click Overwrite. Or, click Overwrite All at the bottom of the Translation task pane.

More Information

Translation Task Pane

Select Languages

  • Translate from

Select a language to translate from. If you want to translate the contents written in English into Korean, select English.

  • Translate to

Select a language to translate to. If you want to translate the contents written in English into Korean, select Korean.

Translate/Stop Button

Start or stop translation.

Reset Button

Delete the existing translation results on the Body area of the Translation task pane.

Translation List Area

List up the translation results.

Overwrite All Button

Apply all of the translation results on the Body area into the document by overwriting the source text.

Insert Button

Insert the translated sentence before or after text.

  • Insert Above/Below Paragraph

Insert the translated sentence above or below a paragraph.

Note

Internet should be connected

To use the translation service, Internet should be connected. Contents to translate in your document will be sent over the Internet in a secured format to Hancom or a third-party translation service provider.

Caution

Some objects can be lost when translating

When you try to apply the translation results to your document, some objects such as memo, hyperlink can be lost.
