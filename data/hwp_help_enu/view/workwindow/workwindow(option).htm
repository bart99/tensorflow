Task Pane View Settings
 

View > Task Pane > Task Pane View Settings

You can select a task pane to display on the screen. The task pane consists of Search Dictionary, View Outline, Quick Launch, Script, Styles, Form Control Properties, View Page Format, Clipboard, XML Document Structure, XML Document Tree, and Hwp Assistant.

Steps

  1. Click View > Task Pane > Task Pane View Settings.
  2. From the Task Pane View Settings dialog box, select the Form Control Properties check box under Task Pane Types and click Close.
  3. Form Control Properties tab is added to the task pane, and Form Control Properties task pane is opened.
  4. To remove the Form Control Properties tab from the task pane, click Task Pane > Task Pane View Settings again.
  5. From the Task Pane View Settings dialog box, unselect the Form Control Properties check box under Task Pane Types and click Close.
  6. The Form Control Properties task pane is removed.

More Information

Task Pane View Settings Dialog Box

Task Pane Types

  • Search Dictionary

  • You can look up for words in the dictionaries included in Hwp while editing a document.

  • View Outline

  • View Outline tab summarizes all outlines used in the document that was opened. Clicking on an outline that is to be moved to sends the cursor to it.

  • Quick Launch

  • Quick Launch tab automatically records all features used by the user in Hwp and allows you to select frequently used features quickly and with ease.

  • View Page Format

  • View Page Format tab provides previews of the individual pages of the document that was opened in Task Pane.

  • Clipboard

  • Clipboard tab remembers up to 16 copied or cut contents, which can be pasted in at the cursor position directly from the clipboard.

  • Form Control Properties

  • Form Control Properties tab is used to change the font, size, background color, and other settings for form controls inserted into the Hwp document.

  • Script

  • Script tab allows you to create and control new Hwp features using scripts. The tab is also used to edit existing scripts.

  • XML Document Tree

  • This task pane appears only when an XML file is opened.

    Fields created from the XML document structure tree cannot be deleted or copied from the body. Instead, they can be deleted, copied, and edited from Task Pane > XML Document Tree.

  • XML Document Structure

  • This task pane appears only when an XML file is opened.

    When an XML template file (*.hwt) is opened, a tree showing the entire structure of the document and its individual nodes is shown on the left side of the document window.

  • Styles

  • Styles tab provides previews of Style List items. Double-clicking on a style from Style List applies the style at the current cursor position in the edit window. You can also select Edit Style or Delete Style, change the order of the styles in the list, or open Style Templates in this tab.

Reset Button

Resets Task Pane to its default setting, which has View Outline, Quick Launch, Search Dictionary, Script, Styles, View Page Format, Clipboard, and Hwp Assistant task panes selected.

Task Pane Direction
  • All To Left Button

  • Set all task panes to display on the left.

  • All To Right Button

  • Set all task panes to display on the right.

Note

Changing Task Pane Position

The task pane gets separated from the screen by dragging the title bar of a task pane. You can change the position of the task pane by dragging the title bar to the required position, Double-click the tabs to separate the task panes if multiple task panes are divided into tabs of a window.
