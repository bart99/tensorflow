Task Pane
 

View > Task Pane

With this feature, you can choose to show or hide the task pane and position it anywhere on the screen.

You can work on documents efficiently by using task panes. You can reduce the document editing time and increase work speed.

Steps

To fold the task pane

  1. Click View > Task Pane > Styles.
  2. From the Styles task pane, click Fold/Unfold.
  3. The task pane folds and only the Task Pane tab is displayed.
  4. Click Fold/Unfold one more time to minimize the task pane.

To unfold the task pane

  1. Click Fold/Unfold when the task pane is minimized.
  2. Select the tab you want when the Task Pane tab displays. Select the Styles tab again.
  3. The Styles task pane is unfolded.

Note

Separating Task Pane from Edit Window

Opening the task pane for the first time fixes it to the right side of the Hwp edit window.

To separate the task pane from the edit window and keep it on the screen at the desired location at all times, click on the title area of the task pane and drag it to the desired location.

The task pane gets separated from the edit window by double-clicking the title area of a task pane. It gets fixed again by double-clicking the title area of the task pane again.
