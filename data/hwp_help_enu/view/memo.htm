Comments
 

View > Comments

Inserting a comment on a specific word or text block within the current document creates an editable comment box on the right side of the page.

Comment options available under View are Show/Hide Comments, Show Comments Guidelines, and View Comments. If working on a document containing comments, use these options according to your needs.

With Show/Hide Comments enabled, right click on a comment and select Edit Comment to edit it or Delete Comment to delete it.

Note

Show/Hide Comments

Show or hide comments inserted into the document. Disable on a document containing comments if you do not want to see the comments.

View > Comments > Show/Hide Comments toggles between show comments and hide comments settings. A selection mark in front of the Show/Hide Comments indicates the comments are shown. Without the selection mark, Show/Hide Comments is disabled.

With Show/Hide Comments enabled, right click on a comment and select Edit Comment to edit it or Delete Comment to delete it.

Show Comments Guidelines

Show or hide the guidelines to text boxes containing comments.

View > Comments > Show Comments Guidelines toggles between show guidelines and hide guidelines settings. The guidelines are shown if a selection mark is placed in front of Show Comments Guidelines. If there is no selection mark in front of Show Comments Guidelines, the guidelines are hidden.

View Comments

Use this option to view the comments in the comments window.

Clicking View > Comments > View Comments loads the comments window at the bottom of the document and displays the comments in that window.

The comments window is shown if a selection mark is placed in front of View Comments. If there is no selection mark in front of View Comments, the comments window is not shown.

With Control Codes Enabled

If control codes are enabled, Comments Start and Comments End fields are shown in the edit screen and comment text boxes are not shown on the screen.
