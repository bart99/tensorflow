Set Document Window Style
 

View > Document Window > Set Document Window Style

Two types of document window styles have been provided to ensure a more convenient user environment.

You can choose the document window style that is right for your needs and enjoy greater work efficiency.

Steps

  1. Click View > Document Window > Set Document Window Style.
  2. From the Document Window Style dialog box, select Windows Style and click OK.
  3. The selected document window style is applied on the edit screen immediately.

More Information

Document Window Style Dialog Box

Normal Style

This is the default document window style. The style contains the document tab, the horizontal scroll bar, and the status bar placed below the scroll bar.

Windows Style

This is the default Windows style and contains the horizontal scroll bar on the very top line, followed by the document tab, and the status bar.
