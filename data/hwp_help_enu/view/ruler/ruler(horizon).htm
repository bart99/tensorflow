Horizontal Ruler
 

View > Document Window > Horizontal Ruler

The horizontal ruler on the top of the edit screen displays horizontal scales for the width of the paper. It is optional to have this ruler display on the edit screen.

Steps

To view/hide horizontal ruler

  1. Select View > Document Window > Horizontal Ruler.
  2. The selection icon in front of the Horizontal Ruler option is removed, along with the horizontal ruler from the edit screen.
  3. Select View > Document Window > Horizontal Ruler again.
  4. The selection mark is displayed again in front of the Horizontal Ruler option, and the ruler is restored on the top of the edit screen.

Note

Unit

The ruler consists of 1mm scales with a number marking at every 10mm.

Viewing Symbols

The horizontal ruler contains left/right indentation marks, the first row begins mark, and the remaining rows begin mark and s tab locations and types.

Margin Adjustment

You can adjust left/right indentations without clicking Page Layout > Page Setup. This is done by dragging the left/right indentation marks on the horizontal ruler.

Located on the left end of the dark gray margin indication area of the horizontal ruler is the gutter margin adjustment icon, which allows you to adjust the Gutter Margin using the pointer.

Holding down ALT and dragging the pointer hides the scales and displays the margins in cm for precision adjustment.

Changing Tab Type

Displayed on the left end of the horizontal ruler is the change tab type icon . Each click on this icon switches the tab type in the order of Left Tab > Right Tab > Center Tab > Decimal Tab.

However, the change tab type icon is only visible when both vertical and horizontal rulers are set to view.

Ruler Movement

The horizontal ruler moves along with horizontal movement of the edit screen. You can tell how many mm the current pointer position is horizontally from the print page.

Context Menu

Place the pointer on the horizontal ruler in the area between the left and right indentation marks. Right click on the pointer to open the context menu, which contains options such as Insert Left/Right/Center/Decimal Tab, Delete All Tab Stops, Tab Settings, and Paragraph.

Caution

Viewing Print Layout

You cannot adjust the indentations from the horizontal ruler while View > Print Layout is disabled.
