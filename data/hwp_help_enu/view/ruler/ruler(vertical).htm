Vertical Ruler
 

View > Document Window > Vertical Ruler

The vertical ruler on the left side of the edit screen displays vertical scales for the length of the paper. It is optional to have this ruler display on the edit screen.

Steps

To view/hide vertical ruler

  1. Select View > Document Window > Vertical Ruler.
  2. The selection icon in front of the Vertical Ruler option is removed, along with the vertical ruler from the edit screen.
  3. Select View > Document Window > Vertical Ruler again.
  4. The selection mark is displayed again in front of the Vertical Ruler option, and the ruler is restored on the left side of the edit screen.

Note

Unit

The ruler consists of 1mm scales with a number marking at every 10mm.

Viewing Symbols

The vertical ruler contains indications for the top and bottom margins and header/footer areas.

Margin Adjustment

You can adjust top and bottom margins without clicking Page Layout > Page Setup. This is done by dragging the top and bottom margin level indications on the vertical ruler.

Holding down ALT and dragging the pointer on the indicators hides the scales and displays the margins in cm for precision adjustment.

Ruler Movement

The vertical ruler moves along with vertical movement of the edit screen. Therefore, you can tell how many mm the current pointer position is vertically from the print page.

Caution

Viewing Print Layout

You cannot adjust the margins from the vertical ruler while View > Print Layout is disabled.
