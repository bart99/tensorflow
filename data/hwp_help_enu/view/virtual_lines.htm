Transparent Line
 

View > Show/Hide > Transparent Line

You can select the line types for tables and text boxes. If you select No Lines, the lines will not show on the edit screen or printouts. If you select View > Show/Hide > Transparent Line, red dotted lines are shown where lines should be so that it becomes much easier for you to edit the document.

Steps

To show/hide transparent line

  1. Click View > Show/Hide > Transparent Line.
  2. A selection mark appears in front of the Transparent Line option to indicate that transparent lines have been applied.

    In the document window, the table or text box lines that have been applied with the No Lines setting are displayed as red dotted lines.

  3. Click View > Show/Hide > Transparent Line again.
  4. The selection mark in front of the Transparent Line option is removed, and lines applied with the No Lines setting are no longer visible.

Note

Print Transparent Lines

The lines that appear with Transparent Line enabled are shown on the screen only and do not appear in printouts.

Transparent Line Color

The line that appears with Transparent Line enabled is always red-dotted. You cannot change the color or style of these lines.
