Status Bar
 

View > Document Window > Status Bar

Status Bar, which is located on the lowermost line of the Hwp document window, is an area where key information about the current cursor position, such as the page number, column number, row number, section number, insert info, and modification info, is shown. You can choose to show or hide the status bar from the edit screen.

Steps

To show or hide the status bar

  1. Click View > Document Window > Status Bar.
  2. The selection mark in front of the Status Bar option is removed to indicate that the status bar is now hidden from the edit screen.
  3. Click View > Document Window > Status Bar again.
  4. The selection mark is restored in front of the Status Bar option to indicate that the status bar is again shown on the lowermost line of the document window.

More Information

Information on Status Bar

In the order from left to right, the status bar shows Pages, Columns, Rows/Columns, Editing Status, Sections, and Insert/Modify Status information about the current cursor position.

Check the status bar often, and you will become familiar with a wider range of features.

  • Pages

  • Displays the Current Page/Total Pages info about the current cursor position. For example, 2/4 indicates that the cursor is on the second page out of four total pages.

    Clicking on the page info opens the Go to dialog box so that you can perform Find Page with ease.

    • Renumber Pages

    • If the status bar displays incorrect page information, clicking on the page info opens the Go to dialog box and the correct page information is displayed after renumbering the pages.

      Clicking Insert Code > Renumber Pages from the Header/Footer tab or clicking Insert > AutoText > Renumber Pages also renumbers the pages and updates the information.

  • Columns

  • Displays which column the cursor is currently located at.

  • Number of Rows/Columns

  • Displays which row and column of the page the cursor is currently located at.

    Clicking on the row/column info opens the Go to dialog box so that you can perform Find Row with ease.

  • Editing Status

  • Displays the current edit status. Shows Paragraph Break, Cell Address, and Object Type. Clicking on the control code info opens the Go to dialog box so that you can perform Find Control Code with ease.

  • Sections

  • Displays which section the cursor is currently located at. Clicking on the section info opens the Go to dialog box so that you can perform Find Section with ease.

  • Insert/Modify Status

  • In insert mode, new characters entered are inserted in place at the cursor location without writing over the existing characters. In modify mode, however, new characters entered overwrite the existing characters at the cursor position.

    The insert/modify status can be switched by pressing INSERT or by clicking on the insert/modify status section of the status bar.

  • Status Bar Tips

  • Resting the pointer on a command from the menu or toolbar displays tips about that command.

Status Bar in Preview Mode

In Preview mode, the status bar indicates Current Page/Total Pages, Paper Type, and Zoom Ratio.

The zoom ratio is shown as a percentage. Fit to Window indicates that the document window is in single-page view mode. If the document page is in multiple-page view mode, the viewing layout is indicated as a grid value, such as 2x2.

Note

Show/Hide Status Bar

You can choose to show or hide the status bar for each document window that has been opened. However, the show/hide status bar setting of the current edit screen applies to a new document window that is opened by clicking File > New.

Editing Status Bar

Unlike with the menu or toolbar, the components of the status bar cannot be edited or rearranged. In addition, you cannot separate the status bar from the document window.

Lock Insert

With Lock Insert selected under File > Options > Others, pressing INSERT does not change the Insert to Modify or vice versa.
