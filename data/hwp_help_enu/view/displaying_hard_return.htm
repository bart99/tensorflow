Hard Return
 

View > Show/Hide > Hard Return

This option is to insert a line break symbol in all locations where ENTER was pressed to break the line. ENTER is usually pressed at the end of a paragraph, and this paragraph-breaking ENTER is called a hard return.

Steps

To show/hide hard return

  1. Click View > Show/Hide > Hard Return.
  2. A selection mark is shown in front of the Hard Return option to indicate that hard returns are now shown.

    In addition, a blue line break symbol is inserted each time you press ENTER while in the edit window.

  3. Click View > Show/Hide > Hard Return again.
  4. The selection mark in front of the Hard Return option is removed to indicate that hard returns are now hidden.

Note

Printing Hard Returns

Hard returns are shown on the edit screen only and do not appear in printouts.

Hard Return Color

Hard returns are blue in color, and you cannot change the color or style of hard returns.

Line Break

Pressing SHIFT+ENTER at the end of a line breaks the line without breaking the paragraph.

A blue line break symbol is inserted each time you press SHIFT+ENTER.

Showing or Hiding Control Codes and Hard Returns

Hard return display setting is not affected by the Show/Hide Control Codes setting.

In other words, you can set hard returns to show, set control codes to show, and then set control codes to not show without affecting the hard return display setting. The same applies if you set hard returns to not show.
