Customize: Menu
 

File > Customize

From the Menu tab of the Customize dialog box, you can customize the context menu by adding new options and removing unused ones.

Steps

To add new menu to context menu

  1. Click File > Customize.
  2. In the Customize dialog box, select the Menu tab.
  3. Selecting Body Block under Context Menu to Edit opens the Body Block context menu next to the Customize dialog box.
  4. Click on the Command tab from the Customize dialog box and select Item > Input then Function > Hyperlink in that order.
  5. Drag Hyperlink to the bottom end of the Body Block context menu.
  6. The Hyperlink option is now added to the Body Block context menu.
  7. In the document window, set the body text as a block and right click to open the context menu, which now includes the Hyperlink option.

To remove an option from context menu

  1. Click File > Customize.
  2. In the Customize dialog box, select the Menu tab.
  3. Selecting Body Block under Context Menu to Edit opens the Body Block context menu next to the Customize dialog box.
  4. Drag the Hyperlink option from the Body Block context menu away from the menu.
  5. The Hyperlink option is now removed from the Body Block context menu.

More Information

Menu Tab of Customize Dialog Box

Context Menu

  • Context Menu to Edit

  • Select the context menu to edit from the list. A complete list of context menus that can be opened by right clicking from the edit window of Hwp is shown.

Reset Button

Click Reset to return the context menu selected in Context Menu to Edit to its original state.

Reset All Button

To return all context menus to their original state, click Reset All. Cancel all the edited menus and return them to their original state.

Note

Using Context Menu in Customize Mode

In customize mode, right clicking on a toolbar icon or command opens the context menu. You can change the name of a tool icon or menu, change the icon of a toolbar, set icon display, insert/delete a separator, copy/cut, delete, and paste by using the context menu.
