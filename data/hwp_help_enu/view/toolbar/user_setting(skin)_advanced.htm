Skin Settings
 

File > Customize > Skin

If you select Default Style, you can change the color of the skin by using Advanced setting.

Steps

To change skin settings

  1. Click File > Customize.

    Click Customize in the Settings group of the Tools tab.

  2. In the Customize dialog box, select the Skin tab.
  3. Select Default Style from the Skin list and click Advanced.
  4. From the Skin Settings dialog box, select the Apply Now check box under Adjust Color and then click on the Dark Violet icon.
  5. After confirming that the dark violate skin has been applied, click Close.
  6. Click OK in the Customize dialog box.

More Information

Skin Settings Dialog Box

Effects

  • Use Animation Effect

  • Set various animation effects applied to the program to start.

Adjust Color

Seven types of basic skins are provided including default, gold, grass, blue violet, silver, dark gray, and dark violet.

If you click each icon, you can preview the applied skin in Preview of the Skin tab of the Customize dialog box.

Apply a skin after checking its style.

  • Apply Now

  • When you select this option and select a skin in the Adjust Color box, the skin gets applied to the program immediately.

RGB Gamma

Adjust the skin color by changing the RGB gamma value. You can type a value, click the increase or decrease icon, or adjust the slider.

The value for Red, Green, and Blue can be set from 10% through 190%.

  • Keep same color ratio for RGB Gamma

  • Adjusting the Red, Green, or Blue level with this option selected proportionately adjusts the RGB gamma ratio.

HSL

Adjust the skin color by adjusting the HSL value. You can type a value, click the increase or decrease icon, or adjust the slider.

The value for Color and Brightness can be set from -50% through 50%. The value for Saturation can be set from -100% through 100%.
