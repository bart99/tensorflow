View Tab
 

View Tab

The View tab provides icons for commands related to viewing options including view, document window, show, edit screen, and zoom.

More Information

View Tab

  • View
  • Print Layout CTRL+G,L Preview the paper margin or the header/footer position.
    Full Screen Reading CTRL+G,L View the document in full screen reading view to maximize readability.
  • Document Window
  • Set Document Window Style Set the document window style to Normal style or Windows style.
    Horizontal Scroll Bar Display or hide the horizontal scroll bar in the Hwp edit window.
    Vertical Scroll Bar Display or hide the vertical scroll bar in the Hwp edit window.
    Status Bar Display or hide the status bar in the Hwp edit window.
    Horizontal Ruler Display or hide the horizontal ruler in the Hwp edit window.
    Vertical Ruler Display or hide the vertical ruler in the Hwp edit window.
    Document Tab Display or hide the document tab over the status bar.
  • Show
  • Hard Return CTRL+G,T Display or hide hard return in the Hwp edit window.
    Control Codes CTRL+G,C Display or hide control codes in the Hwp edit window.
    Transparent Line If you select No Lines as the line type for a table or text box, lines will not be visible in the edit screen or on printouts. If you select View > Show > Transparent Line, however, a red dotted line for all areas applied with the No Lines setting is displayed.
    Image Display or hide images in the Hwp edit window. If you hide the images, the images are removed and are replaced with insertion marks.
    Comments Display or hide comments in the Hwp edit window.
    Hide Task Pane Display or hide the task pane in the Hwp edit window.
    Gridlines Display or hide grid on the Hwp edit window.
    • Show Gridlines: Displays gridlines in the body as per settings under Gridline Settings.
    • Gridline Settings: You can change details of grid by setting gird operation, grid spacing, grid style, and grid origin.
    Object Guidelines Display or hide guidelines automatically based on the object position when you are moving or resizing an object.
  • Color/Grayscale
  • Color Display the editing document with original colors.
    Grayscale Apply the grayscale effect to the editing document.
  • Zoom
  • Zoom Zoom in on or out from the content being edited.
    100% Scale the window to actual size without zooming in or out.
    Fit to Width Zoom in/out the document so that its width fits that of the document window.
    Fit to Page Zoom in/out the document so that one whole page is displayed on the screen.
  • Window
  • Remove Split Restore the vertically or horizontally split edit window to the normal edit window without split.
    Split Horizontally Split the edit window into two horizontal screens.
    Split Vertically Split the edit window into two vertical screens.
    Split to 4 Windows Split the edit window into four screens.
    Arrange Windows Set the arrangement style for the currently open Hwp document windows. You can arrange the document windows in cascade or tile format or minimize them all as icons on Windows taskbar.
    • Arrange Horizontally: Fit all open windows to screen and display them in a tiled horizontal arrangement.
    • Arrange Vertically: Fit all open windows to screen and display them in a tiled vertical arrangement.
    • Cascade: Arrange all open windows in a cascading configuration.
    • Minimize All: Minimize all open document windows into document icons on the taskbar. The foremost opened window will go to the leftmost position.
    Switch Windows Display the list of at most nine Hwp document windows that are currently open.
    • Window List: View a list of windows that are currently open in the Window List dialog box.
  • Macros
  • Define Macros Records mouse pointer input as a new script macro.
    Run Macros Runs or edits a defined script macro.
    Pause Recording Pauses the script macro recording.
    Stop Recording Stops the script macro recording.
    Security Settings Sets the script macro security level.
