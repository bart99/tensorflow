Hwp supports most of the shortcuts used in previous versions of Hwp to facilitate users who are familiar with using shortcuts and the keyboard. Therefore, if you have been using shortcuts in previous versions of Hwp, you should not face any difficulties using Hwp in spite of the revised document editing environment.

Pressing a Shortcut while holding down CTRL, ALT, or SHIFT run the command the shortcut is bound to. The shortcut for each command is displayed to the right of the command option in the menu. Sequential Shortcuts are shortcuts like CTRL+N,T, where you have to press N and then T while holding down CTRL.

If F3 (or F4)°Ê is shown in front of a shortcut, it means you need to set the applicable area as a block or section before executing the shortcut command.

If you are familiar with using Hwp, using shortcuts to execute commands instead of opening up the menu will drastically reduce the time it takes to edit documents.

Steps

To use shortcuts

  1. Hold down CTRL and press F.
  2. Home > Find is selected and the Find dialog box opens up.
