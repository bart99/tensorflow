Customize: Toolbar
 

File > Customize

From the Toolbar tab of the Customize dialog box, you can create a new custom toolbar that contains icons you specify or modify an existing custom toolbar.

Steps

To create customized toolbar

  1. Click File > Customize.
  2. In the Customize dialog box, select the Toolbar tab and click the New icon .
  3. In the Create New Toolbar dialog box, type My Toolbar in the Toolbar Name edit box and click Create.
  4. My Toolbar gets added to the end of the Toolbar Type list of the Toolbar tab and also appears in the edit window.

  5. Select the Command tab of the Customize dialog box. Then from the Item list, select an item that includes a command to add to My Toolbar.

    Here, select Picture from the Item list.

  6. Once the commands included under the Picture item show up in the Function list, drag the desired commands to My Toolbar.

    For this example, select and drag Change Picture from the Function list to My Toolbar.

  7. When you finish dragging all icons you need, a customized toolbar is completed as below.

More Information

Toolbar Tab of Customize Dialog Box

Toolbar Style

In Hwp, you can change the toolbar type. Available toolbar types are as follows:

Name Description
Hwp 2010 This is the default Hwp 2010 toolbar style and displays tab-specific toolboxes.
Hwp 2010 Vertical Group Name Arranges the group names in each of the toolboxes in Hwp 2010 toolbar in the vertical manner. You can enlarge space in the edit window.
Microsoft Word 2003 Applies the Microsoft Word 2003 style to the toolbar.
  • Save As Button

  • Save the toolbar style selected from the list with a different name. You can save toolbars per toolbar style when editing or adding them.

  • Delete Button

  • Delete the customized toolbar style selected from the list.

Toolbar Type

Display a list of all editable toolbars.

When Hwp is launched, the check box in front of the name of the toolbar that shows up by default is checked. You can check or uncheck this box to change the view/hide setting.

Details on Icons

Button Defined Names Description
New Type a name for the customized toolbar that has only the functions you need.
Rename Select the customized toolbar from the Toolbar Type list and click Rename to change the name.
Delete Select the customized toolbar from the Toolbar Type list and click on the Delete icon to delete the toolbar.

Reset Button

Resets the selected toolbar to default Hwp settings. You cannot reset a new toolbar you added.

Reset All Button

Resets all edited toolbars to default Hwp settings. You cannot reset a new toolbar you added.

Options

  • Balloon Tips Settings

  • Select whether to display Balloon Tips or not when you point to an icon on a toolbar. You can also select whether to display function descriptions in balloon tips or not.

    Name Description
    Show with function details Displays function details in balloon tips along with icon names and shortcuts.
    Show without function details Displays icon names and shortcuts in balloon tips.
    Hide Hides balloon tips.

Note

Customizing Toolbar in Document Window

You can edit toolbars in normal edit mode without clicking File > Customize.

If you click an icon on a toolbar while pressing ALT on the keyboard and drag it to another position in the toolbar or to another toolbar, the position of the icon changes accordingly. If you move an icon while pressing ALT and CTRL at the same time, the icon gets copied.
