Customize: Skin
 

File > Customize

The Skin tab of the Customize dialog box offers a wide array of skins.

Steps

  1. Click File > Customize.
  2. In the Customize dialog box, select the Skin tab.
  3. Select a desired style from Skin List. A preview of the selected style is displayed in the preview area.
  4. Click Apply to apply the selected skin on the document window.
  5. To accept the selected skin, click OK.

More Information

Skin Tab of Customize Dialog Box

Skin List

Name Description
Default Style Skin unique to Thinkfree Office NEO.
System Style Follow default screen color schemes and button styles set by the computer.
Windows Aero effects are supported on Windows Vista.
Traditional Style Follow default screen color schemes set by the computer.
Microsoft Office 2003 Style Microsoft Office style skin.

Select a desired skin type from Skin List. A preview of the selected style is displayed in the preview area. Check the preview of the selected skin and then click Apply to apply the skin.

Advanced Button

This button is displayed only when you select Default Style from the Skin list.

If you click this button, the Skin Settings dialog box appears and you can customize a skin by adjusting RGB gamma and HSL values.

Apply Button

Applies the skin selected from Skin List.
