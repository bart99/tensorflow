Customize
 

File > Customize

You can use the Customize feature to change the theme, toolbar, command icon, shortcut, and context menu settings of Hwp to your preference and customize the program environment to meet your needs.

More Information

Customize Dialog Box

Toolbar Tab

From the Toolbar tab, you can specify whether to have frequently used toolbars to be displayed by default when Hwp is launched and remove unnecessary toolbars. You can also create or modify a customized toolbar by selecting icons of choice.

Shortcut Tab

In the Shortcut tab, you can check shortcuts currently assigned to all commands or create a new one. You can also create a shortcut style and select only the shortcuts of choice.

Menu Tab

From the Menu tab, you can customize the context menu by adding new options and removing unused ones.

Command Tab

From the Command tab, you can add all command icons of Hwp to existing toolbars or to a new toolbar created by using the Toolbar tab. You can reorganize toolbars to your convenience.

Toolbox Tab

From the Toolbox tab, you can specify only the required Hwp command icons and create a custom tab.
