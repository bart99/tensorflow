Customize: Command
 

File > Customize

From the Command tab of the Customize dialog box, you can add all command icons used in Hwp to Formatting Toolbar.

You can reorganize Formatting Toolbar for your convenience.

Steps

To add icon to toolbar

  1. Click File > Customize.
  2. In the Customize dialog box, select the Command tab.
  3. Selecting Item > Draw and clicking Function > Shadow Color changes the pointer to the selection shape.
  4. Drag a function name to the desired location on the formatting toolbar in the document window.

    When you drag the pointer, it changes from selected arrow to a four-headed arrow .

  5. The Shadow Color icon is added to the Shapes tab.

To remove an icon from toolbar

To remove an icon from the toolbar, click File > Customize and take the following steps while keeping the Customize dialog box on the screen.

  1. Clicking on the Preview icon on the toolbar in the document window changes the pointer to the selection shape.
  2. Drag the icon to the edit window.

    When you drag the pointer, it changes from selected arrow to a four-headed arrow .

  3. The Preview icon is now removed from the formatting toolbar.

More Information

Command Tab of Customize Dialog Box

Item

If you select items from the list, the functions of the items is displayed on the Function list.

Feature

Displays a categorized list of Hwp menu-related functions. If you select Item, functions in the items is displayed on the Function list.

Separator

Separator is a line that groups similar icons or menus from toolbars or menu lists, and divides the groups.

You can insert a separator in front of an icon on the toolbar of the document window or delete an existing separator by using Separator. You can also insert a separator in front of a menu or delete an existing separator.

  • Insert Button

  • When you select an icon on the toolbar of the document window or a menu from the menu list of the document window, the Insert button gets activated. Once you click Insert in this mode, a separator gets inserted in front of the icon or menu.

    However, if you select the icon or menu right after a separator, the Insert button does not get activated.

  • Delete Button

  • When you select the icon right after the separator on the toolbar of the document window or the menu right under the separator on the menu list of the document window, the Delete button gets activated. Once you click Delete in this mode, you can delete the separator in front of the icon or menu.
