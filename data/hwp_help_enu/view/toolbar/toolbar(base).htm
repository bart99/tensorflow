Standard Toolbar
 

View > Toolbar > Default

The Standard toolbar area displays the menu tab by default and shows object and condition tabs that are specific to the given situation.

Note

Default Toolbox Items

Standard Toolbar Items displays the Menu tab by default and the Object tab and the Condition tab in certain situations.

  • Menu Tab

  • If you click each menu, the Menu tab unfolds with frequently used functions categorized into grouped icons.

    Menu tab types include Home, Insert, Page Layout, References, Review, and View. The File menu does not provide a menu tab.

  • Object Tab

  • If you insert or select an object, the Object tab is automatically displayed with functions of the object displayed as icons.

    Object tab types include Table, Drawing Tools, Picture Tools, Wordshop Tools, Chart Tools, and Comments.

  • Condition Tab

  • If you open File > Preview, Page Layout > Master Page, References > Insert Footnote or open an XML document, the appropriate condition tab is loaded automatically. Icons for commands specific to the selection are provided for greater usability.

    Condition tab types include Preview, Footnotes, XML, Header/Footer, and Compare Versions.
