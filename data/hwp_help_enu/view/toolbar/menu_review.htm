Review Tab

The Review tab provides icons for document review-related commands, such as spelling check, comments, and change markup.

More Information

Review Tab

  • Proofing
  • Spelling Check You can use the Spelling and Check Spelling as you type options to make sure the document does not contain spelling mistakes. When you select Spelling Check, the Spelling Check/Proofing dialog box appears.
    • Spelling: Compare the active document against dictionary (Korean, Chinese, English spelling) and display spelling suggestions for misspelled words.
    • Check Spelling as you type: Insert a red underline beneath words that have been entered incorrectly while creating a document in Hwp.
  • Translate
  • Translate Open the Translation task pane to machine-translate the whole document or selected text.
  • Language
  • Keyboard You can register and use custom keyboards made up of specified letters and symbols. The program speeds up your typing by creating a custom keyboard with frequently used symbols. In addition, you can change the keyboard, view keyboard layout, and change language options.
    • Change Keyboard: Open the Change Keyboard tab of the Keyboard Settings dialog box so that you can change the settings for the current keyboard, Keyboard1, Keyboard2, Keyboard 3, and Keyboard 4.
    • View Keyboard Layout: Display the layout of the currently selected keyboard. You can enter text while viewing the layout of the currently selected keyboard.
    • Create Keyboard: Allow you to create custom keyboards with the characters and symbols of choice.
    • Language Options: Allow you to specify character conversions and character input formats. This option is available only when the current language of the keyboard is Japanese or Chinese.
  • Comments
  • New Comment Insert a comment at the mouse pointer.
    Delete Delete the selected comment.
    Previous Navigate to the previous comment.
    Next Navigate to the next comment.
  • Track
  • Track Changes You can switch on Track Changes, specify the user name, and set passwords for the changes.
    • Track Changes: Track all changes such as additions, deletions, and formatting changes.
    • Change User Name: Change the user name that will be used to sort reviewers to display changes.
    • Protect Changes: Set passwords to prevent unauthorized people from modifying the changes.
    Show In Comments Show changes in balloons. You can accept or reject changes by clicking On or Off in balloons.
    Show Markup You can specify the kind of markup to show in the document.
    • Insertions and Deletions: Display inserted and deleted segments.
    • Formatting: Mark formatting changes for convenient tracking of changes.
    • Reviewer: Selectively display changes made by specific reviewers.
    • Change Tracking Options: Open the Track Changes dialog box. You can change colors for the markup, balloon width, and track formatting.
  • Show Markup
  • Original Display the original version with all changes removed.
    Original with Markup Display the document content with emphasis on the original version. Insertions, deletions, and formatting changes are displayed in the manner designated by the user. Choose this setting to view changes and comments made by the author of the document.
    Final with Markup Display the document content with emphasis on the revised version. Insertions, deletions, and formatting changes are displayed in the manner designated by the user. Choose this setting to view changes and comments made by the editor and other reviewers.
    Final Display the final document with all changes applied.
  • Changes
  • Accept Change Accept the selected changes.
    • Accept and Move to Next: Accept the current change and move to the next change.
    • Accept Change: Accept the change that is currently selected.
    • Accept All Changes Shown: Accept all the displayed changes. For example, if you have specified the reviewers to display changes, the changes made by other reviewers are not applied.
    • Accept All Changes in Document: Accept all the changes made in the document.
    Reject Change Delete the selected changes.
    • Reject and Move to Next: Reject the current change and move to the next change.
    • Reject Change: Reject the change that is currently selected.
    • Reject All Changes Shown: Reject all the displayed changes. For example, if you have specified the reviewers to display changes, the changes made by other reviewers are not rejected and you can view the changes by other reviewers before you reject all of them.
    • Reject All Changes in Document: Reject all the changes made in the document.
    Previous Navigate to the previous change.
    Next Navigate to the next change.
  • Label
  • Label Labels are name tags attached to various tapes or diskettes to display their own features for distinguishing them from one another. You can easily create labels for tapes and diskettes, barcodes, postcards, and business cards while looking at the label paper on the editing screen by clicking Review > Labels > Create Label Template in Hwp.
    • Create Label Template: Open the Create Label Template dialog box so that you can create label templates.
    • Add New Label Page: Use this feature if you want to make multiple pages of a label template.
  • Start Mail Merge
  • Mail Merge Merges a data file containing the names, addresses, and other information about numerous contacts with a template file.
    • Start Mail Merge: Combine a data file with a template file to produce tens or even hundreds of identical e-mails addressed to multiple recipients with different positions and addresses.
    • Mark: Mark areas of a template file where names, positions, addresses, and other information from the data file will be substituted. You can use Windows Address Book, Outlook Address Book, Hwp files, .dbf files, Cell files, and Excel files as data files.
  • Replace Private Info
  • Replace in Place Change characters within an area that has been set as a block with other characters selected by the user.
    Search/Replace Search and replace characters within an area that has been set as a block with characters selected by the user. Use this feature on an important section to hide important information by substituting it with different symbols. From the Hide Private Info dialog box, you can configure Private Info Options and Select Masking Text settings.
