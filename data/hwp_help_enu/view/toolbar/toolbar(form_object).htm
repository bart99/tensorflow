Form Controls
 

Insert > Form Controls

You can insert form controls to the edit screen by clicking Insert > Form Controls.

In Hwp 2010, you can insert five types of form controls including the Command Button, Check Box, Combo Box, Radio Button, and Edit Box.

Steps

To insert form controls

  1. Click Insert > Form Controls > Insert Radio Button.
  2. Select the Insert Radio Button icon and insert three radio buttons.
  3. After inserting the radio buttons into the body, click on the View Properties icon .
  4. Once the Form Control Properties task pane is displayed on the left side of the edit window, type Hwp, Cell, and Show in the Caption fields and set CharShape to Gulim, 9pt.
  5. From Form Controls, click on the Form Edit Mode/Disable icon to disable form control edit mode.
  6. From the body, clicking on the Hwp radio button selects Hwp.

More Information

Form Controls Icon Description

Icon Name Description
Form Edit Mode/Disable Form Edit Mode is selected by default when Form Controls is selected so that you can insert form controls.
In form edit mode, all Form Controls icons are active.
If the icons are pressed, Form Edit Mode is on. If the icons are not pressed, Form Edit Mode is off. With Form Edit Mode turned on, you can insert form controls like the Command Button, Check Box, Combo Box, Radio Button, and Edit Box.
When you are running form controls inserted in the document, turn off Form Edit Mode and select a form control.
View Properties Opens the Form Control Properties task pane.
From the Form Control Properties task pane, you can set or edit the internal properties of the selected form control including the font, size, and background color settings.
View Code Opens the Script task pane.
From the Script task pane, enter scripts for all form controls inserted into the document and run those scripts.
By typing a script directly, you can control new functions in Hwp. Type a script by using only JavaScript.
Insert Command Button Inserts a command button.
Command button is a form control that runs the assigned script when clicked on.
Insert Check Box Inserts a check box.
Check boxes are form controls that are used when selection of one or more options within a group or category is needed and supports multiple selection.
Insert Combo Box Inserts a combo box.
Combo boxes are form controls that are used to have the user make a selection from a list of options with assigned values.
Insert Radio Button Inserts a radio button.
Radio buttons are form controls that are used when selection of a single option within a group or category is needed.
Insert Edit Box Inserts an edit box.
Edit boxes are form controls that are used to process input and output of simple text.
