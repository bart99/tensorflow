References Tab
 

References Tab

The References tab provides icons for reference-related features like table of contents, footnotes, caption, and index.

More Information

References Tab

  • Table of Contents
  • Table of Contents Table of Contents creates a list of lines containing a heading, table, picture, or equation, along with the page number of each line. Table of Contents is essential for creating a book or an essay. If the page numbers of the document are changed after completing Table of Contents, you are required to run Table of Contents again.
    Mark as TOC Mark the location in the body as a Table of Contents topic and create a title file.
    Hide from TOC Insert Hide from TOC to hide certain content of the paragraphs with a TOC mark.
  • Footnotes
  • Insert Footnote Insert a footnote at the mouse pointer.
    Insert Endnote Insert an endnote at the mouse pointer.
  • Footnotes
  • Previous Note Navigate to the previous footnote.
    Next Note Navigate to the next footnote.
    Save Notes Saves the contents of all footnotes from the body as a separate Hwp file.
    Delete Notes Deletes the current footnote.
  • Position
  • Footnote Insert a footnote at the mouse pointer.
    Endnote Insert an endnote at the mouse pointer.
    Column Footnote Specify the footnote position for when there are multiple columns.
  • Note Number
  • Numbering Styles Changes the numbering style of the current footnote.
    Normal Display footnote/endnote numbers in normal size.
    Smaller Display footnote/endnote numbers in smaller size.
  • Separator
  • Set Separator Length Set the length of the footnote/endnote separator manually.
    More Outline Colors Changes the color of the separator line between footnotes and endnotes.
    Line Styles Changes the line style of the separator between footnotes and endnotes.
    Weight Changes the weight of the separator between footnotes and endnotes.
  • Convert Footnotes/Endnotes
  • All Footnotes to Endnotes Change all footnotes in the current file to endnotes.
    You cannot select this option in a document only with endnotes.
    All Endnotes to Footnotes Change all endnotes in the current file to footnotes.
    You cannot select this option in a document only with footnotes.
    Swap Footnotes with Endnotes Change all footnotes to endnotes and all endnotes to footnotes in the current file.
    You can select this option in a document only when both footnotes and endnotes are present.
  • Caption
  • Caption Insert a caption at bottom of the object.
  • Index
  • Mark Entry Attaches a mark after a word you wish to index.
    Insert Index Collects instances of Mark Entry inserted into the document and creates an index file. If the page numbers of the body change after performing Insert Index, Insert Index is required to be performed again.
