Customize: General
 

File > Customize

From the General tab of the Customize dialog box, you can specify the theme of Hwp or design custom themes with preferred skin, toolbar, and shortcut settings.

Steps

To create customized theme

  1. Click File > Customize.
  2. In the Customize dialog box, select the General tab.
  3. Select the desired settings, such as Skin > Traditional Style and Toolbar > Hwp, and click Save As.
  4. From the Save As dialog box, type a name for the custom theme and click Save.
  5. The name of the customized theme is added to the Program Theme list of the Customize dialog box.

More Information

General Tab of Customize Dialog Box

Program Theme

Select one of the basic themes included in Hwp, such as Hwp, Microsoft Word 2010, and Microsoft Word 2003, or a custom theme. The selected theme is applied immediately.

  • Save As Button

  • Saves the theme that was selected from the list under a different name. You can edit themes and then save them, hence they are easier to manage.

  • Delete Button

  • Delete the customized theme selected from the list.

Skin

Changes the skin of Hwp. If you click a skin, the skin applies to the program immediately and you can preview it.

Name Description
Default Style Skin unique to Thinkfree Office NEO.
System Style Follow default screen color schemes and button styles set by the computer.
Traditional Style Follow default screen color schemes set by the computer.
Microsoft Office 2003 Style Microsoft Office style skin.

Toolbar

You can choose Hwp or a custom toolbar style.

Shortcut

Select Hwp Shortcut or a custom shortcut style.

Task Pane

Changes the composition of task panes in Hwp.

Selecting a task pane type from the list applies the selection immediately. Available task pane styles are as follows:

Name Description
Horizontal Tab Layout Arranges the task panes as horizontal tabs.
Vertical Tab Layout Arranges the task panes as vertical tabs. Use the Fold/Unfold button to fold or unfold the task pane in three levels and gain more edit window space.

Note

Deleting a Custom Theme

To delete a custom theme, select the theme from the Program Theme list and click Delete.

Default themes included in Hwp cannot be deleted.

Changing Master Document

When you create a new document, Hwp opens a master document which is different depending on the Program Theme. When you create a document in Microsoft Word 2010 theme, the master document have some disabled features and it will be enabled by changing the program theme.

When you change program theme while documents are opened, the master document applied to the documents will not be changed. To change the master document, exit and run Hwp again.
