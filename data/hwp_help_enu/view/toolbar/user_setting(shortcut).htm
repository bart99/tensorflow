Customize: Shortcuts
 

File > Customize

In the Shortcut tab of the Customize dialog box, you can check shortcuts currently assigned to all commands or create a new shortcut. Or, you can create a shortcut style and select only the shortcuts you want.

Steps

To configure shortcuts

  1. Click File > Customize.
  2. From the Customize dialog box, select the Shortcut tab. The mode changes to customize mode for editing menu easily.
  3. Select Classification > File and then Command > New Tab.
  4. Place the pointer in the Key to specify box and, at the same time, press CTRL+F11 from the keyboard.
  5. Once CTRL+F11 in entered in the Key to specify field, check for Not Defined under Set Command and then click Add.
  6. Once the "Cannot modify the default shortcut style. Create copy?" message appears, click Yes.
  7. From the Save As dialog box, type My Shortcuts in the shortcut style name field and then click Save.
  8. The name of the current shortcut style in the Shortcut Style list under Customize > Shortcut is changed to My Shortcuts, and CTRL+F11 is added to the Current shortcut list of the selected command.
  9. Click Close to exit Customize Shortcut.
  10. Pressing CTRL+F11 in the edit window executes File > New > New Tab.

To delete an assigned shortcut

  1. Click File > Customize.
  2. In the Customize dialog box, select the Shortcut tab.
  3. Click Classification > File and then Command > New Tab.
  4. Once CTRL+F11 appears in the Current shortcut list, select CTRL+F11 and then click Delete.
  5. Click Close to exit Customize Shortcut.

More Information

Shortcuts Tab of Customize Dialog Box

Shortcut Style

Select Cell Shortcuts, Microsoft Excel Shortcuts, or a custom shortcut style.

  • Save As Button

  • Save the shortcut style selected from the list with a different name. You can save shortcuts per shortcut style when newly assigning them or modifying them.

  • Delete Button

  • Delete the customized shortcut style selected from the list.

Classification

Displays a categorized list of Hwp menu-related functions. If you select items from the list, functions in the items appear on the Command list.

Command

Display a list of commands related to the items selected in Classification.

Current

If you click a function from the Command list and the function is assigned with a shortcut, the shortcut is displayed on the Current list.

Description

When you select a function from the Command list, a brief description of the function appears.

Set Command

Once you select Classification and Command in this order and enter a shortcut already assigned to a function in the Key to Specify edit box, the name of the function appears in Set Command.

Key to Specify

You can assign up to eight shortcuts to the function by selecting a function from the Command list and pressing a key on the keyboard with the pointer in the Key to Specify edit box.

However, you cannot assign shortcuts used for other functions. If you try to assign shortcuts used for other functions, the name of the functions appears in Set Command.

Only when the shortcut is a combination of multiple keys including ALT and CTRL, it is shown in the format of ALT(CTRL)+currently pressed key. If you press another key or ALT(CTRL)+another key, the shortcut is shown in the format of ALT(CTRL)+currently pressed key, another key and you cannot enter more key values.

Pressing BACKSPACE in the Key to specify field deletes the entered keys one group at a time.

Note

Creating Customized Shortcut Style

Assign a shortcut to a function by selecting the function from the Classification and Command lists. Then click Save As and give the shortcut style a different name to add the name of the customized shortcut style to the Shortcut Style list.

Pressing CTRL, ALT, or SHIFT and a Key at the same time

Shortcuts enable certain functions to run when pressed with CTRL, ALT or SHIFT. When assigning a shortcut, you have to press a key while holding down CTRL, ALT, or SHIFT.
