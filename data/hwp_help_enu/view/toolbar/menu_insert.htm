Insert Tab

The Insert tab provides icons for frequently used insert-related commands like table, object, reference, insert assistant, and find.

More Information

Insert Tab

  • Page
  • Page Break   Insert a page break at the mouse pointer.
  • Table
  • Insert Table CTRL+N,T Inserts a new table into the document.
  • Illustrations
  • Picture Select a prepared picture file to insert at the mouse pointer position.
    Clip Art Draw easily and quickly by importing objects from Clip Art.
    Shapes Select and insert a shape of choice. If you click the drop-down arrow , an image collection is displayed.
    Chart Create and insert a chart object.
    Screenshot Add a screenshot to your document without leaving the program that you are working on.
    Video Insert a video with two ways; embed a video file and link to a video file.
    Form Controls You can insert controls that are used to fill out and edit form objects.
    • Insert Command Button: Inserts a command button (PushButton).
    • Insert Check Box: Inserts a check box.
    • Insert Combo Box: Insert a combo box.
    • Insert Radio Button: Inserts a radio button (RadioButton).
    • Insert Edit Box: Inserts an edit box (Edit).
    • Properties: Opens the Form Control Properties task pane. From the Form Control Properties task pane, you can specify or edit internal properties of the selected form object including the font, size, and background color settings.
    • View Code: Opens the Script task pane. From the Script task pane, you can insert scripts for all form objects that have been inserted into the body and run scripts. You can control new functions in Hwp. Make sure to type a script by using JavaScript by typing a script directly.
    • Form Edit Mode: Enables selecting the Form Object commands loads Object Edit Mode with all settings set to default. While you are in Form Edit Mode, you can insert Command Buttons, Check Boxes, List Boxes, Radio Buttons, Edit Boxes, and other form objects. When you are running form controls inserted in the document, turn off Form Edit Mode and select a form control.
  • Link
  • Hyperlink CTRL+K Insert a hyperlink at the cursor, the selected object, or a block area.
    Bookmark CTRL+SHIFT+F5 Insert a bookmark at the cursor.
  • Header/Footer
  • Header   Create or edit a header.
    Footer   Create or edit a footer.
  • Text
  • Text Box Insert a horizontal or vertical text box.
    Quick Parts Register frequently used text and document properties for future use.
    Wordshop Select a Wordshop from the image collection.
    Drop Cap Format the first letter of the paragraph as a dropped capital.
    Date/Time String Inserts the current system date as a text.
    Object Inserts an OLE object into the document.
  • Symbol
  • Equation Enter an equation using the Equation Editor.
    Symbol The Enter from Symbol dialog box appears if this feature is run for the first time. If a recently entered character is available, it is entered. If you click the drop-down arrow , the following menu is displayed.
    • Recent Symbols: Display up to 16 recently used symbols.
    • Symbol: Display the Enter from Symbol dialog box.
