Home Tab
 

Home Tab

The Home tab provides icons for frequently used document editing commands like clipboard, font, paragraph, style, change text, and find.

More Information

Home Tab

  • Clipboard
  • Paste CTRL+V Paste the content on the Clipboard saved by Edit-Copy or Edit-Cut back at the mouse pointer position.
    • Paste: Pastes the clipboard content at the cursor position.
    • Paste Special: Display the Paste Special dialog box to paste the data type you need besides the original program type.
    Cut CTRL+X Delete the selection and save its content on the Clipboard.
    Copy CTRL+C Save the selection on the Clipboard.
    Format Painter ALT+C Copy the text format, paragraph format, and style at the mouse pointer position to another location.
  • Font
  • Rep.   Select a language among Rep, Korean, English, Chinese, Japanese, Other Languages, Symbol, and User.
    Rep is a collection of complementary fonts for different languages. When you change the Rep font, complementary fonts are loaded for other languages as per predefined settings.
    Font   Specify the font to use.
    Text Size   You can increase or decrease font size.
    Character Border   Draw borders for the selected text. You can specify the weight and color of the character border.
    Bold ALT+SHIFT+B Make the selected text bold.
    Italic ALT+SHIFT+I Italicize the selected text.
    Underline ALT+SHIFT+U Underline the selected text.
    Strikethrough   Inserts a strikethrough line across the selected characters.
    Subscript CTRL+Equal Sign Make letters subscript.
    Superscript CTRL+SHIFT+Equal Sign Make letters superscript.
    Text Highlight Color   Highlights the selected characters in the selected color.
    Font Color   Changes the color of the selected characters.
    Enclose Characters   Overlaps two or three characters on top of one another to create a new character.
    Decrease Letter Spacing Decreases the spacing between characters by 1%.
    Increase Letter Spacing Increases the spacing between characters by 1%.
    Change Case SHIFT+F3 Changes the English characters in the block to uppercase or lowercase.
    Phonetic Guide   Inserts a phonetic guide above or underneath the original word.
  • Paragraph
  • Bullets   When you are listing multiple items, you can add bullets to the head of the paragraph.
    • Bullet Styles: Opens the Bullets tab of the Numbering/Bullets dialog box so that you can change the Bullets style.
    Numbering   When you are listing multiple items, you add numbers or bullets to the heads of the paragraphs. Numbering provides seven levels of numbers. When there is a change in the order of numbered sentences, the numbering changes automatically.
    • Numbering Styles: Opens the Numbering tab of the Numbering/Bullets dialog box so that you can change the Numbering style.
    Outline   You can number items by level with ease to summarize the content or to create a table of contents.
    • Multilevel List: You can change multilevel list styles after inserting multilevel lists.
    Restart Numbering   Restart numbering from the current paragraph.
    Promote Paragraph ALT+SHIFT+Right Arrow Each click of Promote Paragraph promotes the outline paragraph by one level.
    Demote Paragraph ALT+SHIFT+Left Arrow Each click of Promote Paragraph demotes the outline paragraph by one level.
    Increase Left Margin CTRL+ALT+F6 Click to increase the left margin of the current paragraph by 1 pt.
    Decrease Left Margin CTRL+ALT+F5 Click to decrease the left margin of the current paragraph by 1 pt.
    First Line Outdent CTRL+T Set the left margin of an entire paragraph excluding the first line to be pushed as much to the right as the negative indent value specifies. You can adjust indentation while checking it in the Preview window below.
    First Line Indent CTRL+SHIFT+T Set the first line of a paragraph to be pushed to the right from the left margin of the paragraph. You can adjust indentation while checking it in the Preview window below.
    Align Left CTRL+SHIFT+L Align paragraphs to the left margin.
    Align Center CTRL+SHIFT+C Align paragraphs to the center.
    Align Right CTRL+SHIFT+R Align paragraphs to the right margin.
    Justify CTRL+SHIFT+M Align paragraphs to both left and right margins. This is the most commonly used alignment setting.
    Distributed CTRL+SHIFT+T Justifies regardless of the number of characters but applies even spacing between characters.
    Spread   Justifies regardless of the number of characters but applies even spacing between words.
    Line Spacing   Change the line spacing of the active paragraph. The spacing between lines in the same paragraph is the same.
    Sort   Arranges the block in the vertical direction.
    Delete Control Codes   Select and delete multiple control codes from the entire document, a block, or a section.
  • Styles
  • Styles CTRL+SHIFT+S Changes the Font and Paragraph settings of the paragraph at the cursor position to the selected style.
    Add Style   Display the Add Style dialog box to add a new style.
    Style Templates   Apply the style selected in Style Templates to the active document.
  • Find/Replace
  • Find CTRL+F Provides find-related functions like Find and Go to.
    • Find: Display the Find dialog box.
    • Go to: You can point and move to a specific position such as line or page, or a position with a picture, table, or bookmark inserted.
    Replace CTRL+H Replace: Opens the Replace dialog box.
    Select Object   Select the desired object.
    Quick Search   Type a keyword in the field and press ENTER to perform a quick search.
    Previous   Find the text entered in the edit box in the content before the current pointer position.
    Next   Find the text entered in the edit box in the content after the current pointer position.
