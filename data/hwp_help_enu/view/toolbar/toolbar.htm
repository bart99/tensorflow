Toolbar
 

View > Toolbar

The toolbar is a collection of icons for commands that are frequently used while editing documents.

Hwp provides toolboxes that change dynamically, and a toolbar which contains static icons for the most frequently used commands. Toolboxes consist of large icons and menu option text to ensure more efficient menu option placement and function identification than with the previous toolbar design.

The basic toolbar area contains the menu tab by default and dynamically displays the object tabs and status tabs that are specific to the given situation.

Steps

To fold or unfold toolbars
  1. Click the Fold/Unfold Toolbar icon in the menu area.
  2. The toolbox folds up, and only the menu tab and the formatting toolbar are.
  3. Click Fold/Unfold one more time. The Formatting toolbar unfolds and only the Menu tab remains.
  4. With all the toolbars folded up, click the Fold/Unfold Toolbar icon.
  5. The toolbox and the formatting toolbar are displayed.

Note

Toolbox Items

Standard Toolbar Items displays the Menu tab by default and the Object tab and the Condition tab in certain situations.

  • Menu Tab: Click on a menu option to open the toolbox of the concerned menu tab. Hwp provides file, home, insert, page layout, reference, review, view, and table menu options. The File menu does not provide the Menu tab.
  • Object Tab: When you insert or select a table, shape, picture, Wordshop, chart, or comment object, the object tab and toolbox of the concerned object open up automatically.
  • Condition Tab: If you select preview, master page, footnote, XML, header/footer, or compare versions, the condition tab and toolbox open up.

The formatting toolbar is a collection of commonly used document editing commands and, unlike the dynamic nature of the default toolbar, consists of static components.

By clicking Fold/Unfold, you can gradually hide or unhide toolbars on the screen.

Custom Toolbar

From the Toolbox tab of the Customize dialog box, you can configure the customize toolbar to include selected icons only.

Balloon Tips

If you are unfamiliar with the function of a toolbar icon, resting the pointer on the icon opens a balloon tip containing the name of the icon and describing the shortcut and its function.

In Options > Balloon Tips Settings in the Toolbar tab of the Customize dialog box, you can activate or deactivate balloon tips for the icons of each toolbar and specify whether balloon tips for functions should include details or not.
