Customize: Toolbox
 

File > Customize

Show All

From the Toolbox tab of the Customize dialog box, you can create a User tab in the toolbox and add Hwp commands to the tab. You can gather frequently used functions in one tab for easy use.

Steps

To create a user tab and add icons to the tab

  1. Click File > Customize.
  2. In the Customize dialog box, select the Toolbox tab.
  3. Select Edit from the Item list and Cut from the Function list and then click Add.

  4. A User tab with Cut added to Group 1 is created.

    If you added a function for the first time, tab, group, and place items get automatically added to the User Tab list.

  5. Select One Vertical Line from the User Tab list and click Add Placement Settings.
  6. Select the newly added One Vertical Line. Then select Three Horizontal Lines from the Placement list and three from the Rows list.
  7. Select Edit from the Item list and select Copy, Paste, and Format Painter from the Function list to add those functions to the Edit Function group of the actual User tab.

More Information

Toolbox Tab of Customize Dialog Box

Item

If you select items from the list, the functions of the items appear on the Function list.

Feature

Displays a list of Hwp features. If you select Item, functions in the items appear on the Function list.

User Tab

Displays the components of the User tab in tree structure. Select a function from the Function list and then click Add to add the function to the User tab.

Because the tab components are in tree structure, it is easy to check them. You can fold submenus by clicking the minus icon .

Details on Icons

Icon Name Description
Add The function selected from the Function list is added to User Tab.
To Sub The function selected from the Function list is added as a submenu of the function selected in User Tab.
Add Group Adds a new group.
Add Placement Settings Adds placement settings. Placement settings you can choose from are One Vertical Line, Two Horizontal Lines, and Three Horizontal Lines.
Delete Deletes the selected function from the tree list.
Move one row up Moves one line up on the tree list.
Move one row down Moves one line down on the tree list.

Options

  • Name

  • Changes the group or function name. Select a group or function from the User Tab tree list to change its name.

  • Rows

  • If you selected Horizontal in Placement, specify how many lines you need for placing the function.

    The default value is one line. If you select two or three from the list, placement will be done in two or three lines from top to bottom.

  • Placement

  • Select One Vertical Line, Two Horizontal Lines, or Three Horizontal Lines as the method of placing icons and text.

    The vertical setting arranges the icons and text in a vertical manner, and the horizontal settings arranges the icons and text in a horizontal manner.

  • Show as Icon

  • Specify whether to show the icon of the selected function.

    This option is activated only if you selected the sub-functions of Two Horizontal Lines or Three Horizontal Lines from the User Tab list.

  • Show Text

  • Specify whether to show the text of the selected function.

    This option is activated only if you selected the sub-functions of Two Horizontal Lines or Three Horizontal Lines from the User Tab list.

  • Align Left

  • Align to the left if function placement is set as horizontal.

    This option is activated only when you select Two Horizontal Lines or Three Horizontal Lines from the User Tab list.

Note

Adding Settings for Group Adding and Placement

After adding groups, the names of the added groups, such as Group 1 and Group 2, are shown in the toolbox, along the top.

Added placement settings do not appear in the toolbox. Placement settings are only used to specify the arrangement of the icons, and you can add multiple settings to an individual group.

Options

Options can be set based on the items selected from User Tab.

Show Icons and Show Text can be configured if a horizontally placed function is selected from User Tab.

Placement can be configured after selecting a placement setting from User Tab.

Rows and Align Left can be configured if Two Horizontal Lines or Three Horizontal Lines is selected from User Tab.
