Page Layout Tab
 

Page Layout Tab

The Page Layout tab provides icons for page-related commands like page setup, text direction, page format, page break, section, and paper.

More Information

Page Layout Tab

  • Text Direction
  • Horizontal Writing Select this option to switch from vertical writing to horizontal writing.
    Vertical Writing While you are in normal editing mode, click Vertical Writing. Alternatively, you can specify the target area (block, specific sections, or the entire document) and apply the Vertical Writing setting only to that area. The available vertical writing options are Vertical and Stacked.
  • Page Setup
  • Page Margins Provides paper margin sets. You can select from Default, Narrow (including header/footer margins), Wide (including header/footer margins), Narrow, or Wide. In addition, you can select Modify Page Margins to change the top/bottom and left/right margins.
    • Custom Margins: The Page Setup dialog box appears so that you can change the paper size, orientation, and margin settings.
    Orientation Specify the paper orientation as portrait or landscape.
    Columns Divides the page into multiple columns. The available options are 1, 2, 3, left page multiple, and right page multiple.
    Breaks Inserts a page, column, or section break.
    Line Number Settings Display line numbers in all or part of the document.
  • Section
  • Section Settings Set various section properties such as starting page number and starting object number. From the Section Settings dialog box, you can specify whether to include the header/footer, master page, page borders/background, and page number in the starting pages of sections only. You can also specify whether to remove empty lines that appear in the beginning of a page, what the default tab size should be, and other settings specific to individual sections.
  • Page Background
  • Watermark Provides a picture or text watermark.
    Page Background Specify the page background color or apply fill effects.
    Page Borders You can apply borders or a picture background to all pages of the current document.
