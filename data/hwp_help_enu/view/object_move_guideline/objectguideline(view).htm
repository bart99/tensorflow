View Object Guidelines
 

View > Object Guidelines > Display Object Guidelines

Selecting Display Object Guidelines displays a 3X3 guideline grid in the edit screen when an object is selected so that you can move it with precision. The guidelines are very useful when lining up two objects that are placed apart from one another.

Steps

  1. Click View > Object Guidelines > Display Object Guidelines.
  2. Select Insert > Illustrations > Shapes > Ellipse and insert the object into the document.
  3. Left click the inserted ellipse object to select it and then move it to the left.
  4. A 3X3 object guideline grid is displayed on the screen.
     

More Information

Object Movement Information Display

If you move an object after enabling object guidelines, information about the movement of the object is displayed as a balloon tip.
