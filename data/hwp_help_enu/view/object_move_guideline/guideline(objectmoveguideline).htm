Object Guidelines
 

View > Object Guidelines

This feature displays object guidelines so that you can move the selected object with precision. Object guidelines are useful if you want to line up two objects that are located apart from one another.

Enabling Display Object Guidelines opens a 3X3 guideline edit screen to help you move the selected object with precision. Object Guidelines can be enabled or disabled from View > Object Guidelines > Display Object Guidelines.
