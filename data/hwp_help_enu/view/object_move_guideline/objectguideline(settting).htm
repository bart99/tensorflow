Object Guidelines Settings
 

View > Object Guidelines > Object Guidelines Settings

You can change the operation, type, and origin settings of object guidelines and related options.

Steps

  1. Click View > Object Guidelines > Object Guidelines Settings.
  2. From the Object Guidelines Settings dialog box, select Snap Selected Object to Other Objects under Guideline Type and click Set.
  3. When moving an object on the edit screen, object guidelines remain visible even when going past the selection point of not only the selected object, but other objects on the edit screen as well. This makes it possible to move the object and accurately maintain the distance even if the object is placed far away.

More Information

Object Guidelines Settings Dialog Box

Display Object Guidelines

View or hide object guidelines.

Use the Solid Line and Dashed options at the bottom to set object guidelines as solid or dashed lines, respectively.

  • Solid Line

  • Displays the horizontal and vertical lines of object guidelines as solid lines.

  • Dashed

  • Displays the horizontal and vertical lines of object guidelines as dashed lines.

Guideline Type

Specify whether to display object guidelines on the selected object only or on other objects inserted in the document as well.

  • Snap Objects to Selected Object

  • When selecting and moving an object, object guidelines are created for the selected object only.

  • Snap Selected Object to Other Objects

  • Object guidelines are created even when moving an object past the selection points of other object inserted into the document.

Guideline Origin

Specify the origin for object movement coordinates that are displayed as a balloon tip.

  • Page

  • Calculates horizontal and vertical positions based on the page where the object is located.

    Displays object movement information with the upper left corner of the body editing area, which is Page Setup minus top, bottom, left, right, header, and footer margins, as the origin for the guidelines.

  • Paper

  • Calculates horizontal and vertical positions based on the paper where the object is located.

    Displays object movement information with the upper left corner of the paper as the origin for the guidelines.

Options

Specify whether to give snap property to object guidelines and whether to display position data as a balloon tip.

  • Snap to Gridlines

  • Specify whether to snap the object when moving it.

    With this option selected, the highlighted object is snapped if placed close to another object.

    Specify how close the object needs to be to another object before being snapped.

  • Show Position Info with Balloon Tips

  • Specify whether to display coordinates as balloon tips when moving an object.

  • Show Size Info with Balloon Tips

  • Specify whether to display sizes as balloon tips when moving an object.

View Coordinates on Status Bar

When moving an object, displays its coordinates as X and Y values on the status bar.
