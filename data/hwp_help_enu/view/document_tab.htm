Document Tab
 

View > Document Window > Document Tab

You can enlarge the edit screen by the width of the document tab by checking or unchecking the document tab located above the status bar.

Steps

To show or hide the document tab

  1. Click View > Document Window > Document Tab.
  2. The selection mark in front of the Document Tab option is removed to indicate that the document tab is now hidden.

    The document window is now enlarged by an amount equaling the size of the document tab.

  3. Click View > Document Window > Document Tab again.
  4. The selection mark is restored in front of the Document Tab option to indicate that the document tab has been restored above the status bar.

Note

Moving Document Tab

If you have multiple document tabs on the screen, you can switch between the tabs by pressing CTRL+TAB instead of selecting Document Tab from the View menu.
