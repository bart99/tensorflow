Formatting Toolbar
 

View > Toolbar > Formatting Toolbar

Formatting Toolbar groups icons for frequently used template-related commands into once place so that you can access those commands without having to open up the dialog box.

You can choose to show or hide the formatting toolbar. In addition, you can click Customize and rearrange the toolbar components to your preferences.

More Information

Toolbox Icon Description

Icon Name Shortcut Description
New ALT+N Create a new document.
  • Hwp Document Opens a Hwp document. Compatible Document is used to open a Hwp document while a non-Hwp program is selected.
  • Microsoft Word Compatible Document: Hwp starts and a Microsoft Word-compatible document opens in a new window.
  • Document Templates: Opens the Document Template dialog box and then the Recent Document Templates package.
  • Cell Document: Hwp starts and a Hwp workbook opens in a new window.
  • Show Document: Slide starts and a Show document opens in a new window.
Open CTRL+O Opens an existing file in the edit window to view, edit, or print its contents.
Save CTRL+S Saves the document currently being edited on the screen in the file format exclusive to Hwp.
  • Save As: Saves the document being edited under a different name or as a different format.
  • Save as PDF: Saves the document being edited as a .pdf file without sending it to the printer.
Print CTRL+P Print the active document.
Preview   Preview the document to print on the screen before sending it to the printer.
Undo CTRL+Z Undo the action previously taken while editing the document.
Redo CTRL+Y Redo the command or action canceled by Home > Undo.
Styles F6 Specify the style to use.
Language   Select a language among Rep, Korean, English, Chinese, Japanese, Other Languages, Symbol, and User. The Rep option combines complementary fonts from different languages.
Font   Specify the font to use.
Text Size   You can increase or decrease font size. Adjusting the font size changes both horizontal and vertical values at the same time.
Bold ALT+SHIFT+B Displays the selected characters in bold.
Italic ALT+SHIFT+I Displays the selected characters in italics.
Underline ALT+SHIFT+I Underline the selected text.
Strikethrough   Draw a strikethrough on the selected text.
Font Color   Change the color of the selected font.
Justify CTRL+SHIFT+M Align paragraphs to both left and right margins.
Align Left CTRL+SHIFT+L Align paragraphs to the left margin.
Align Center CTRL+SHIFT+C Align paragraphs to the center.
Align Right CTRL+SHIFT+R Align paragraphs to the right margin.
Distributed CTRL+SHIFT+J Make the space between each word in a paragraph equal.
Spread   Align to both left and right and margins maintain even spacing between words.
Line Spacing   Change the line spacing of the active paragraph.

Note

Color Palette

Click Font Color to show the color palette. If there is no desired color in the color palette, click Others.

In the Colors dialog box, you can specify the exact color that you wish to use.

Toolbar Separation

You can separate the color palette from the formatting toolbar.

On the formatting toolbar, click the Font Color icon. Click on the separator at the top of the color palette that opens up and drag it to the edit screen. This separates the color palette as Font Color toolbar.
