Control Codes
 

View > Show/Hide > Control Codes

Commands issued by the user while editing in Hwp are normally recorded as control codes and hidden from the document window. Control codes are not shown on normal edit screens. View > Show > Control Codes is used to show these codes on the edit screen. Even if set to show on the edit screen, control codes are not shown in printouts.

Steps

To show/hide control codes

  1. Click View > Show/Hide > Control Codes.
  2. Selection marks are shown in front of Control Codes and Hard Return to indicate that control codes are now shown.

    In addition, red control codes are shown at document window areas where a table, text box, picture, or header/footer is inserted; blue hard return marks are shown where ever ENTER was pressed; and blue spacing marks are shown where ever the SPACEBAR was pressed.

  3. Click View > Show/Hide > Control Codes again.
  4. The selection marks in front of Control Codes and Hard Return options are removed to indicate that control codes are now hidden.

Note

Relationship with Hard Returns

Setting control codes to show also sets hard returns to show. In other words, selecting Show/Hide Control Codes results in hard returns being shown as well. If you selected Show/Hide Hard Return first, selecting or unselecting Show/Hide Control Codes does not affect the display of hard returns.

Font

With Control Codes selected, the control codes in the document are shown in red letters, enclosed in brackets (e.g., Rectangle, Picture). The font style of the body applies to the control codes.

Go to

To go to a specific control code in the body, click Home > Find > Go to. Select Control Codes to display a list of all control codes that can be inserted in Hwp.

Control Codes in Front

Control codes for the header or footer are inserted at the very front of the paragraph. Under the Hide Control Codes setting, you cannot place the cursor before these control codes. You can only place the cursor in front of control codes under the Show Control Codes setting.

Delete Control Codes

Pressing DELETE or BACKSPACE from where there is a control code prompts the "Delete (Control Code Name)?" message.

To delete two or more control codes at the same time, use CTRL+Y or ALT+Y. In this case, you are asked for confirmation to delete the codes, starting with the one in the back.

Checking Current Control Code

The type of the control code at the current cursor position can be checked on the status bar at the bottom of the edit screen.
