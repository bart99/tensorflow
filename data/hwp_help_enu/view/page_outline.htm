Print Layout <CTRL+ALT+P>
 

View > Print Layout

Documents created with Hwp need to be printed on paper in order to see what the layout of each page is. If you wish to preview the paper margins and the header and footer before printing, however, click View > Print Layout and set Hwp to show the print layout.

Showing the print layout displays the margins, header/footer, page borders, and other elements that appear on printouts only so that you can adjust or edit them from the screen.

Steps

To show/hide print layout

  1. Click View > Print Layout.
  2. The selection mark in front of the Print Layout option is removed to indicate that print layout is no longer shown.
  3. Click View > Print Layout again.
  4. The selection mark in front of the Print Layout option is restored to indicate that print layout is now shown.

Note

Print Layout Status

The current print layout viewing status is maintained even after opening a new document window by clicking File > New.

The print layout viewing status is also remembered when exiting from Hwp. Launching Hwp again opens a document window using the remembered setting.

Auto Activation of Print Layout

If you create a Header/Footer or set Zoom to Many Pages without turning on the printout layout, View > Print Layout is automatically selected.

In this situation, the print layout viewing status is maintained when you finish editing the header/footer or change the zoom ratio.

Differences

If you reach the end of the current page as you type text, the text is automatically continued over on the next page. In this situation, the top and bottom margins are not shown and instead a dotted boundary line is shown to indicate the page split if print layout has not been set to show.

If print layout has been set to show, on the other hand, the top/bottom margin, left/right indentation, footnote/endnote, and paper size settings as specified in Page Setup and all other print elements contained in the current page are shown on the edit screen.

Margin Adjustment using Rulers

With print layout set to show, you cannot adjust the margins from the horizontal and vertical rulers.
