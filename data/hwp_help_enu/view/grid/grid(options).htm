Gridline Settings
 

View > Gridlines > Gridline Settings

You can change details of grid by setting grid operation, grid spacing, grid style, and grid origin.

Steps

  1. Click View > Gridlines > Gridline Settings.
  2. From the Gridline Settings dialog box, select Snap > Snap only to Gridline and click Set.
  3. On the edit screen, objects cannot be drawn in between grids. They are drawn snapped to the gridlines only.

More Information

Gridline Settings Dialog Box

Show Gridline

Displays gridlines on the edit screen.

  • Dot

  • Displays gridlines by inserting dots at the horizontal/vertical positions specified in Gridline Spacing.

  • Line Grid

  • Displays gridlines by connecting the horizontal/vertical positions specified in Gridline Spacing with lines.

    • Checker: Draws horizontal and vertical lines at specific intervals to produce gridlines.

    • Horizontal: Draws horizontal lines at specific intervals to produce gridlines.

    • Vertical: Draws vertical lines at specific intervals to produce gridlines.

Gridline Location

Specify whether to display gridlines behind the text or in front of it.

Snap

Specify whether to allow objects to be drawn anywhere in relation to the gridlines or drawn snapped up against a gridline only.

  • Ignore Gridline

  • Gridlines are used as ruler references in drawing or relocating pictures.

  • Snap to Gridline

  • Snap only to Gridline is triggered when a picture is drawn or when a drawn object is moved inside the area specified in Snap Range. Ignore Gridline is applied for areas outside the gridline range.

  • Snap only to Gridline

  • Only allows pictures to be snapped to the gridline when being drawn or moved. Accordingly, pictures cannot be drawn or moved to in between gridlines.

Gridline Spacing

Specify the horizontal and vertical spacing between grid points.

  • Horizontal

  • The default value is 5mm. Can be specified from 1mm through 25mm.

  • Vertical

  • The default value is 5mm. Can be specified from 1mm through 25mm.

Gridline Origin

Specify whether to display gridlines with the paper as its origin or the body editing area as its origin. For detailed information, refer to the section on Gridline Origin.

  • Page

  • Displays gridlines with the upper left corner of the body editing area, which excludes the top, bottom, left, right, header, and footer margins from Page Setup, as the gridline origin.

  • Snap to Gridline

  • Displays gridlines with the upper left corner of Page Setup as the origin.

Snap Range

Snap Range can only be set if Snap to Gridline is selected under Snap. Specify how close an object needs to be to a grid point before being snapped to the gridline.

  • Horizontal

  • The default value is 2mm. Can be specified from 0mm through 25mm.

  • Vertical

  • The default value is 2mm. Can be specified from 0mm through 25mm.

Caution

Snap Range Limit

If a value greater than half of Gridline Spacing is set for Snap Range, Snap only to Gridline is applied.
